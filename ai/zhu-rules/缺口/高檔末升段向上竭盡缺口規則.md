# 高檔末升段向上竭盡缺口規則

- **Rule ID**: R-GAP-11
- **名稱**: 高檔末升段向上竭盡缺口規則
- **分類**: 缺口
- **原文與頁碼**: P09-C2-向上突破缺口的研究.md（下冊 p.218-219，圖表9-2-17、9-2-18）
- **解讀**: 發生在上漲趨勢接近高檔或末升段，此時仍出現強力向上跳空，但實際上代表資金後續動能漸漸耗盡，稱為「向上竭盡缺口」。三項判斷條件（缺一即證據不足，三者同時成立才確認竭盡缺口）：①**位置**：缺口出現在連續上漲的高檔，或波段的末升段（需先確認股價已走了一大段漲幅）。②**爆量**：缺口附近1、2天出現爆量上漲的長紅K或跳空上漲紅K，但後面幾天量能明顯萎縮，價格呈現小幅上漲、走平或下跌，顯示後繼無力。③**回補**：缺口出現後2、3天內，股價不漲或回跌，將缺口回補——回補是竭盡缺口成立與趨勢即將反轉的確認訊號。與[[打底完成向上突破缺口規則]]不同，竭盡缺口的缺口下緣**不是可靠支撐**——一旦被回補即代表支撐失守，需轉為戒備頭部反轉（可接續觀察是否出現[[高檔島型反轉規則]]或[[做頭完成向下跌破缺口規則]]）。
- **可程式化**: 是
- **所需資料**: 波段漲幅/相對高檔位置判斷、缺口事件、跳空當日與前後數日成交量、近N日均量、缺口出現後2-3日收盤價
- **計算公式**:
```
function detect_exhaustion_gap_up(gap, bars, i, is_late_stage_rally, avg_volume_N):
    if gap.type != "up_gap":
        return None
    if not is_late_stage_rally:   # 需先確認位於連續上漲高檔或末升段
        return None

    had_huge_volume = bars[i].Volume > K_exhaust * avg_volume_N
    volume_shrinking_after = all(
        bars[j].Volume < bars[i].Volume for j in range(i+1, i+1+M_days)
    )
    price_stalling = all(
        abs(bars[j].Close - bars[j-1].Close) / bars[j-1].Close < small_move_threshold or bars[j].Close <= bars[j-1].Close
        for j in range(i+1, i+1+M_days)
    )

    if not (had_huge_volume and volume_shrinking_after and price_stalling):
        return {"category": "疑似竭盡缺口（條件不完整，需持續觀察）"}

    is_filled = is_gap_filled(gap, bars, i, within_days=3)
    result = {
        "category": "向上竭盡缺口",
        "support_reliable": False,   # 竭盡缺口下緣不是可靠支撐
        "confirmed": is_filled
    }
    if is_filled:
        result["signal"] = "缺口2-3天內回補，趨勢反轉確認，戒備頭部反轉"
    else:
        result["signal"] = "尚未回補，竭盡缺口暫未確認，持續觀察"
    return result
```
- **參數**: K_exhaust = 爆量門檻倍數（書中未給精確值）；M_days = 量能萎縮觀察天數（書中稱「1、2天爆量、後面幾天萎縮」，建議預設2天爆量+後續2-3天觀察）；small_move_threshold = 「小幅上漲」判斷門檻（書中未給精確值）
- **可回測**: 是
- **信心**: 88/100 高（三項條件〔位置／爆量／回補〕書中以條列方式明確給出，且與9-3章末跌段竭盡缺口逐字對稱可交叉驗證；僅各項的精確量化門檻〔爆量倍數、小幅上漲定義〕未給，需自行校準）
