# 低檔末跌段向下竭盡缺口規則

- **Rule ID**: R-GAP-16
- **名稱**: 低檔末跌段向下竭盡缺口規則
- **分類**: 缺口
- **原文與頁碼**: P09-C3-向下跌破缺口的研究.md（下冊 p.228-229，圖表9-3-10）
- **解讀**: 與[[高檔末升段向上竭盡缺口規則]]鏡射對稱（書中原文與該節文字幾乎逐字對稱，僅多空方向相反，可交叉驗證）。下跌趨勢走到低檔或波段的「末跌段」時，經常出現強力向下的跳空缺口，此時資金的後續下跌動能反而逐漸「增加到極限後衰竭」，是下跌行情即將結束的訊號，稱為「向下竭盡缺口」。三項判斷條件（三者同時成立才確認）：①**位置**：缺口出現在連續下跌的低檔，或波段的末跌段。②**爆量**：缺口附近1、2天出現爆量下跌的長黑K，或跳空下跌的黑K；緊接著後面幾天成交量明顯縮減，股價呈小幅價漲、價平或價跌，顯示後續下跌動能無力。③**回補**：出現缺口後的2、3天內，股價不再破底或回漲，將缺口回補。三個條件同時成立時，應靜待行情轉折，視為另一波段（可能翻多或止跌）的好機會，而非追空時機。
- **可程式化**: 是
- **所需資料**: 波段跌幅/相對低檔位置判斷、缺口事件、跳空當日與前後數日成交量、近N日均量、缺口出現後2-3日收盤價
- **計算公式**:
```
function detect_exhaustion_gap_down(gap, bars, i, is_late_stage_decline, avg_volume_N):
    if gap.type != "down_gap":
        return None
    if not is_late_stage_decline:   # 需先確認位於連續下跌低檔或末跌段
        return None

    had_huge_volume = bars[i].Volume > K_exhaust * avg_volume_N
    volume_shrinking_after = all(
        bars[j].Volume < bars[i].Volume for j in range(i+1, i+1+M_days)
    )
    price_stalling = all(
        abs(bars[j].Close - bars[j-1].Close) / bars[j-1].Close < small_move_threshold or bars[j].Close >= bars[j-1].Close
        for j in range(i+1, i+1+M_days)
    )

    if not (had_huge_volume and volume_shrinking_after and price_stalling):
        return {"category": "疑似竭盡缺口（條件不完整，需持續觀察）"}

    is_filled = is_gap_filled(gap, bars, i, within_days=3)
    result = {
        "category": "向下竭盡缺口",
        "resistance_reliable": False,   # 缺口上緣不是可靠壓力，一旦回補即失守
        "confirmed": is_filled
    }
    if is_filled:
        result["signal"] = "缺口2-3天內回補，趨勢反轉確認，視為止跌/翻多的潛在機會，非追空時機"
    else:
        result["signal"] = "尚未回補，竭盡缺口暫未確認，持續觀察"
    return result
```
- **參數**: K_exhaust = 爆量門檻倍數（書中未給精確值）；M_days = 量能萎縮觀察天數（建議預設2天爆量+後續2-3天觀察）；small_move_threshold = 「小幅」判斷門檻（書中未給精確值）
- **可回測**: 是
- **信心**: 88/100 高（三項條件書中明確條列，且與[[高檔末升段向上竭盡缺口規則]]逐字對稱可交叉驗證；精確量化門檻需自行校準）
