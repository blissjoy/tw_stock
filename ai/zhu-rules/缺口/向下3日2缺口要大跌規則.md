# 向下3日2缺口要大跌規則

- **Rule ID**: R-GAP-23
- **名稱**: 向下3日2缺口要大跌規則
- **分類**: 缺口
- **原文與頁碼**: P09-C5-缺口的進階應用.md（下冊 p.252-253，圖表9-5-20～9-5-22）
- **解讀**: 與[[向上3日2缺口要大漲規則]]完全鏡射對稱。連續3根交易日的K棒之間，出現**2次同方向（向下）**的跳空缺口，為空方對稱版本，定義方式相同（僅方向相反）。四種情境：①**做頭位置**：空頭做頭時，無論型態的第1個頭或第2個頭，出現連續下跌3根黑K、且3天內出現2個缺口，代表頭部完成，是弱勢空頭股，可鎖股做空。②**起跌位置**：空頭下跌的關鍵起跌點，連續3日向下出現2個跳空缺口，是主力強力做空的宣示；只要缺口沒有被回補（見[[缺口真封口判定規則]]），股價容易大跌。③**行進位置**：下跌行進過程中再度出現3日2缺口，若是低檔爆量後股價不再上漲，容易再跌一波。④**低檔位置（警訊）**：空頭下跌到低檔才出現3日2缺口且同時爆大量，要特別注意股價**容易反彈**——此時可能轉變為類似[[低檔末跌段向下竭盡缺口規則]]的最後殺盤訊號，而非續跌訊號。
- **可程式化**: 是
- **所需資料**: 連續3根K棒OHLC、2次缺口偵測、當時股價所在位置（做頭/起跌/行進/低檔）、成交量、近N日均量
- **計算公式**:
```
function detect_3day_2gap_down(bars, i):
    gap1 = detect_gap(bars, i-1)   # bars[i-2] -> bars[i-1]
    gap2 = detect_gap(bars, i)     # bars[i-1] -> bars[i]
    if gap1 is None or gap1.type != "down_gap":
        return None
    if gap2 is None or gap2.type != "down_gap":
        return None
    if not (is_black_candle(bars[i-2]) and is_black_candle(bars[i-1]) and is_black_candle(bars[i])):
        return None   # 需連續3根黑K

    result = {"category": "向下3日2缺口", "gaps": [gap1, gap2]}
    position = classify_position(bars, i)   # "做頭" / "起跌" / "行進" / "低檔"

    if position in ("做頭", "起跌"):
        result["signal"] = "強力續跌訊號，可鎖股做空；只要缺口不被回補，股價容易大跌"
    elif position == "行進":
        result["signal"] = "若低檔爆量後不再上漲，容易再跌一波"
    elif position == "低檔":
        had_huge_volume = bars[i].Volume > K_huge * avg_volume_N(bars, i)
        if had_huge_volume:
            result["warning"] = "低檔爆量3日2缺口，股價容易反彈，可能轉為竭盡缺口式最後殺盤訊號，非續跌訊號"

    return result
```
- **參數**: K_huge = 爆大量門檻倍數（書中未給精確值）；`classify_position` 依賴上游趨勢/型態判定模組
- **可回測**: 是
- **信心**: 76/100 中高（與向上版完全鏡射對稱，四種位置的多空意涵書中皆有清楚文字與圖例對照；位置分類需依賴上游規則判定準確度）
