# 向上跳空缺口四層價位分級支撐規則

- **Rule ID**: R-GAP-03
- **名稱**: 向上跳空缺口四層價位分級支撐規則
- **分類**: 缺口
- **原文與頁碼**: P09-C4-缺口成交量支撐與壓力.md（下冊 p.233-236，圖表9-4-5～9-4-8）
- **解讀**: 這是缺口篇最核心的支撐精細化規則，補完[[向上跳空缺口支撐規則]]（支撐壓力分類，只處理「整個缺口區間」二分支撐/失效）未定義的部分。書中明確指出：向上跳空缺口形成支撐時，**不是把整個缺口區間當成同一等級支撐，而是以缺口上漲前、後兩根紅K線的高低點，切出由強到弱4個價位**：
  - **上高（最強）** = 缺口上漲後那根紅K線的高點
  - **上沿（次強）** = 缺口上漲後那根紅K線的低點（＝缺口上邊界，等同[[缺口基本定義與偵測規則]]中的 `upper_edge`）
  - **下沿（較弱）** = 缺口上漲前那根紅K線的高點（＝缺口下邊界，等同`lower_edge`）
  - **下底（最弱／最後防線）** = 缺口上漲前那根紅K線的低點

  適用前提：這套分級法只適用於**放量**的向上跳空缺口（放量代表強力上漲，日後回跌時這幾個價位才具支撐意義；無量缺口請見[[缺口成交量強弱與支撐壓力角色互換規則]]）。

  支撐效力是**階梯式衰減**，不是一次性有效/失效：
  1. 收盤跌破「上高」→ 多頭氣勢轉弱（第一道警訊，仍屬多頭，不足以出場）。
  2. 收盤跌破「上沿」→ 股價正式跌入缺口區間，容易多空拉鋸、上下震盪；若缺口被完全回補（見[[缺口真封口判定規則]]判斷是否為真回補），代表原本「強力上漲」缺口降級為「一般多頭」，**尚未翻空**（回補是「降級」不是「失效」）。
  3. 收盤跌破「下沿」→ 不僅跌破缺口，還跌入上漲前那根紅K線實體內，多頭氣勢**完全轉弱**，須提防下一道「下底」不可再破。
  4. 收盤跌破「下底」→ 跌破上漲前紅K線實體最低點，判定為**多空易位**，趨勢正式轉由空方主導。
- **可程式化**: 是
- **所需資料**: 缺口事件（[[缺口基本定義與偵測規則]]輸出）、缺口前一根K棒（K_before）與缺口當根K棒（K_after）的OHLC、跳空當日成交量、近N日均量、後續逐日收盤價
- **計算公式**:
```
function build_up_gap_tiers(gap, K_before, K_after, avg_volume_N):
    if K_after.Volume < K_vol * avg_volume_N:
        return None   # 未放量，不適用四層分級法（見角色互換規則的無量缺口處理）

    tiers = {
        "上高": K_after.High,   # 最強
        "上沿": K_after.Low,    # 次強，= gap.upper_edge
        "下沿": K_before.High,  # 較弱，= gap.lower_edge
        "下底": K_before.Low    # 最弱／最後防線
    }
    tiers["state"] = "強力多頭"   # 初始狀態：四層皆未破
    return tiers

function update_up_gap_state(tiers, close_t, is_true_fill):
    # is_true_fill: 由 缺口真封口判定規則 判斷缺口是否已被真封口
    if close_t < tiers["下底"]:
        tiers["state"] = "多空易位（正式翻空）"
    elif close_t < tiers["下沿"]:
        tiers["state"] = "氣勢完全轉弱（提防下底不可再破）"
    elif close_t < tiers["上沿"]:
        if is_true_fill:
            tiers["state"] = "降級為一般多頭（缺口已真封口，非強力多頭，但未翻空）"
        else:
            tiers["state"] = "缺口內拉鋸震盪"
    elif close_t < tiers["上高"]:
        tiers["state"] = "多頭轉弱警訊（仍屬多頭）"
    else:
        tiers["state"] = "強力多頭（四層皆未破）"
    return tiers["state"]
```
- **參數**: K_vol = 放量門檻倍數（書中未給精確值，建議預設1.5～2倍N日均量，可比照[[大量長紅K的二分之一價支撐規則]]的K參數校準）；N = 均量計算天數（預設5或20日）
- **可回測**: 是
- **信心**: 90/100 高（四層價位定義、支撐強度排序、階梯式衰減與「回補是降級不是失效」的判讀邏輯，書中以完整圖例（圖表9-4-6～9-4-8）逐階段示範，條理明確；僅放量門檻的精確倍數未給，需自行校準）
