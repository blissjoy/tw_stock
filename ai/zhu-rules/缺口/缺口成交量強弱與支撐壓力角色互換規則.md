# 缺口成交量強弱與支撐壓力角色互換規則

- **Rule ID**: R-GAP-05
- **名稱**: 缺口成交量強弱與支撐壓力角色互換規則
- **分類**: 缺口
- **原文與頁碼**: P09-C4-缺口成交量支撐與壓力.md（下冊 p.231-233，圖表9-4-1～9-4-4）
- **解讀**: 本規則是[[向上跳空缺口四層價位分級支撐規則]]與[[向下跳空缺口四層價位分級壓力規則]]的前置總論，處理三個跨缺口的通用原則：
  1. **量能決定支撐壓力強弱**：向上跳空缺口若伴隨大量，是強力向上訊號，日後回檔時該缺口支撐力道強，跌破後同一位置日後的壓力也大（強支撐＝強壓力，同一位置角色可互換，呼應第2點）；「無量」向上跳空缺口支撐力道弱，書中特別舉例：無量漲停跳空，日後一旦反轉下跌，該位置多半無支撐可言，容易出現跌停板式的急速下跌。此點是四層分級法「適用前提為放量缺口」的理論依據——無量缺口不建議套用四層分級法，應直接視為弱支撐/壓力處理。
  2. **支撐壓力角色互換原則**：向上跳空缺口原本是支撐，一旦被向下跌破（收盤有效跌破），同一價格區退化為日後反彈的壓力（書中稱「缺口過壓」）；向下跳空缺口原本是壓力，一旦被向上突破，會轉為日後回檔的支撐。此原則補充了[[向上跳空缺口支撐規則]]／[[向下跳空缺口壓力規則]]所述「缺口失效後不再具原方向支撐/壓力」——失效後並非單純消失，而是角色反轉為對側功能。
  3. **大量缺口區＝多空交戰區**：股價再次進入當初以大量形成的缺口區間時，容易出現忽漲忽跌、震盪加大，不宜預設單邊方向，應觀察區間內拉鋸結果（例如搭配四層分級法中「缺口內拉鋸」狀態）再判斷。
- **可程式化**: 是
- **所需資料**: 缺口事件（含跳空當日成交量）、近N日均量、缺口邊界價位、後續逐日收盤價
- **計算公式**:
```
function gap_volume_strength(gap_volume, avg_volume_N):
    ratio = gap_volume / avg_volume_N
    if ratio < 1.0:
        return "無量缺口：支撐/壓力力道弱，不建議套用四層分級法；反轉時風險較高（可能急速跌停/急速漲停）"
    elif ratio < K_vol:
        return "普通量能：支撐/壓力力道中等"
    else:
        return "放量缺口：支撐/壓力力道強，可套用四層分級法"

function role_flip_on_breach(gap, breach_direction):
    # breach_direction: "down" 表示原支撐被有效跌破；"up" 表示原壓力被有效突破
    if gap.type == "up_gap" and breach_direction == "down":
        gap.role = "轉為未來反彈的壓力（缺口過壓）"
    elif gap.type == "down_gap" and breach_direction == "up":
        gap.role = "轉為未來回檔的支撐"
    return gap.role

function is_battleground_zone(price_t, gap):
    if gap.was_formed_with_large_volume and gap.lower_edge <= price_t <= gap.upper_edge:
        return "大量缺口區多空交戰：預期震盪加大，不宜預設單邊方向"
    return None
```
- **參數**: K_vol = 放量門檻倍數（書中未給精確值，與R-GAP-03/04共用同一套校準）；N = 均量計算天數
- **可回測**: 是
- **信心**: 90/100 高（量能強弱、角色互換、多空交戰三項原則書中皆有明確文字敘述與對照圖例；僅量能門檻的精確倍數需自行校準）
