# 打底完成向上突破缺口規則

- **Rule ID**: R-GAP-09
- **名稱**: 打底完成向上突破缺口規則
- **分類**: 缺口
- **原文與頁碼**: P09-C2-向上突破缺口的研究.md（下冊 p.213-216，圖表9-2-11～9-2-14）
- **解讀**: 發生在底部打底已完成、股價向上突破盤整區間上緣（壓力線）當下的缺口，稱為「突破缺口」，是主力強勢做多的表態，意義重大，屬強力買進訊號，也是本章6型態中訊號等級最高者之一。三項判斷條件：①需有明顯大量配合，且成交量須持續增加；出大量的向上跳空通常容易繼續上漲、短時間不會回補。②缺口開口幅度越大，漲勢力道越強；跳空漲停力道最強，容易讓空手投資人措手不及追買，或空單來不及回補而軋空，造成股價快速上漲。③（假突破過濾）若缺口在3天內跌回起漲點、把缺口回補，則視為「假突破」，須提高警覺（此為9-2章的粗略版本，精確的真封口/假封口三要件判斷見[[缺口真封口判定規則]]與[[缺口假封口判定規則]]）。操作原則：打底完成出現突破缺口，可依進場紀律買進，但仍須設好停損、嚴守紀律。支撐位置：突破缺口下緣（原本盤整區壓力線，突破後轉為支撐，即「壓力轉支撐」）是關鍵支撐，回檔測試不破為型態延續的健康表現；跌破則視同假突破。
- **可程式化**: 是
- **所需資料**: 前置底部盤整區間（[[盤整區上下頸線支撐壓力規則]]或[[密集盤整區區間支撐壓力規則]]）、缺口事件、跳空當日與後續數日成交量、近N日均量、跳空後3日內收盤價
- **計算公式**:
```
function detect_breakaway_gap_up(gap, consolidation_range, bars, i, avg_volume_N):
    if gap.type != "up_gap":
        return None
    if gap.lower_edge < consolidation_range["upper"]:
        return None   # 未突破盤整區上緣，屬普通缺口，見 底部盤整區普通缺口規則

    is_large_volume = bars[i].Volume > K_break * avg_volume_N
    volume_increasing = bars[i].Volume > bars[i-1].Volume and bars[i+1].Volume >= bars[i].Volume  # 需持續增量

    result = {
        "category": "突破缺口（打底完成）",
        "signal": "強力買進訊號" if is_large_volume else "缺乏大量配合，訊號強度降低",
        "gap_size_signal": "缺口越大漲勢越強；跳空漲停力道最強" ,
        "support": gap.lower_edge   # 原壓力轉支撐
    }

    # 假突破過濾：3天內跌回起漲點並回補缺口
    if is_gap_filled(gap, bars, i, within_days=3):
        result["warning"] = "3天內回補，視為假突破，提高警覺"

    return result
```
- **參數**: K_break = 大量門檻倍數（書中未給精確值，需自行校準）；假突破觀察窗＝3天（書中明確）
- **可回測**: 是
- **信心**: 90/100 高（三項條件、支撐位置與操作原則書中皆有明確文字與多張圖例佐證，是6型態中描述最完整的一類；僅大量的精確倍數需自行校準）
