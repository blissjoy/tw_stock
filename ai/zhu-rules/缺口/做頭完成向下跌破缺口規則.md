# 做頭完成向下跌破缺口規則

- **Rule ID**: R-GAP-14
- **名稱**: 做頭完成向下跌破缺口規則
- **分類**: 缺口
- **原文與頁碼**: P09-C3-向下跌破缺口的研究.md（下冊 p.225-226，圖表9-3-6～9-3-8）
- **解讀**: 與[[打底完成向上突破缺口規則]]鏡射對稱，但有一項關鍵不對稱之處。「頭部做頭」型態已完成之後才出現向下跌破的跳空缺口，屬主力「強勢做空」表態，訊號強度高於單純跌破。判斷條件：①多頭上漲出現「頭頭低」，若第2個頭是以跳空下跌黑K構成，後續反轉為空頭的機率很大，持有多單應立刻出場。②**「向下跳空跌破」不需要明顯大量配合**——即使沒有大量的向下跳空，通常也已是下跌表態，可視為空頭確認（與突破缺口「必須大量」的條件明確不同，此為書中原文特別強調的不對稱之處，下跌訊號比上漲訊號更容易在無量下成立）。③向下跌破的缺口開口幅度越大，下跌訊號越明顯，代表多頭結束的力道越強。④風險提示：若缺口在出現後3天內就被漲回起跌點回補，應提高警覺，小心是「假跌破」（粗略版本，精確判斷見[[缺口真封口判定規則]]與[[缺口假封口判定規則]]）。操作原則：做頭完成＋向下跌破缺口雖可依紀律進場做空，但仍要有風險意識，設好停損、嚴守紀律。
- **可程式化**: 是
- **所需資料**: 前置頭部型態完成判定（頭頭低結構，見[[頭頭低底底低空頭趨勢判定]]）、缺口事件、跳空後3日內收盤價
- **計算公式**:
```
function detect_breakaway_gap_down(gap, topping_pattern_confirmed, bars, i):
    if gap.type != "down_gap":
        return None
    if not topping_pattern_confirmed:   # 需先確認頭部做頭已完成（頭頭低第2個頭）
        return None

    result = {
        "category": "做頭完成向下跌破缺口",
        "signal": "空頭確認，持有多單應立刻出場",
        "volume_requirement": "不需要大量配合（與突破缺口不同，無量亦成立）",
        "gap_size_signal": "缺口越大，下跌訊號越明確",
        "resistance": gap.upper_edge   # 原支撐轉壓力
    }

    if is_gap_filled(gap, bars, i, within_days=3):
        result["warning"] = "3天內回補，視為假跌破，提高警覺"

    return result
```
- **參數**: 假跌破觀察窗＝3天（書中明確）；無其他量能門檻參數（本規則明確不要求大量）
- **可回測**: 是
- **信心**: 91/100 高（判斷條件明確，且「不需大量配合」為書中明確強調的特殊條件，與突破缺口版本形成清楚對照，降低誤讀風險）
