# 底部盤整區普通缺口規則

- **Rule ID**: R-GAP-08
- **名稱**: 底部盤整區普通缺口規則
- **分類**: 缺口
- **原文與頁碼**: P09-C2-向上突破缺口的研究.md（下冊 p.213-214，圖表9-2-11）
- **解讀**: 出現在底部盤整區域**內部**（尚未突破整理區上緣）的缺口，稱為「區域缺口」或「普通缺口」。成因多為消息面影響或短期買賣力不平衡，非主力有計劃性攻擊（可搭配[[缺口成因三分類規則]]交叉判斷）。特徵：很容易在1、2天內就封閉（回補），無法產生突破力道或助漲力道。判斷原則：只要缺口出現在盤整區「內部」（尚未突破區間上緣，即缺口的上邊界仍低於盤整區壓力線），一律視為區域缺口/普通缺口，對趨勢判斷參考價值低，不應以此類缺口作為進出場依據，且無明顯支撐意義（因容易被快速回補，不構成有效支撐區）。此規則的存在意義主要是作為「排除條件」——用於與[[打底完成向上突破缺口規則]]做對照區分：同樣是底部區域出現的向上缺口，若未突破盤整區上緣即為本規則的普通缺口，若突破則升級為突破缺口。
- **可程式化**: 是
- **所需資料**: 缺口事件、盤整區間上下緣（可參考[[盤整區上下頸線支撐壓力規則]]或[[密集盤整區區間支撐壓力規則]]界定）、缺口出現後1-2日是否回補
- **計算公式**:
```
function classify_common_gap(gap, consolidation_range, bars, i):
    # consolidation_range = {"upper": 盤整區壓力線, "lower": 盤整區支撐線}
    if gap.type != "up_gap":
        return None
    if gap.upper_edge >= consolidation_range["upper"]:
        return None   # 已突破區間上緣，屬突破缺口，見 打底完成向上突破缺口規則，非本規則範圍

    # 缺口仍在盤整區內部
    filled_within_2_days = is_gap_filled(gap, bars, i, within_days=2)
    return {
        "category": "區域缺口／普通缺口",
        "significance": "低，不宜作為進出場依據",
        "support_significance": "無",
        "filled_within_2_days": filled_within_2_days   # 書中預期通常為 True
    }
```
- **參數**: 無額外校準參數；依賴上游盤整區間界定規則的參數
- **可回測**: 是
- **信心**: 62/100 中（「盤整區內部缺口＝普通缺口」的判斷原則與「1-2天內封閉」特徵書中明確描述；但屬於單一對照圖例，樣本較少，且依賴上游盤整區間界定的準確度）
