# 高檔島型反轉規則

- **Rule ID**: R-GAP-13
- **名稱**: 高檔島型反轉規則
- **分類**: 缺口
- **原文與頁碼**: P09-C3-向下跌破缺口的研究.md（下冊 p.223-224，圖表9-3-3～9-3-5）
- **解讀**: 與[[低檔島型反轉規則]]鏡射對稱。股價先向上跳空，到高檔後出現無力現象，接著立刻改以向下跳空方式下跌，圖形左右各一個缺口，中間墊高的股價像海中孤島。定義核心：高檔「向上跳空缺口」＋「無力盤頭」＋「向下跳空缺口」三段式結構，且兩缺口的價格區間**不重疊**，缺口將中間股價孤立出來。此型態不常出現，一旦在高檔出現，後續常是大跌走勢，是書中所稱「必殺做空」的好機會。結構天數不定：可能僅1根K線構成反轉，也可能2、3天，或經過較多天盤整才反轉；**盤整天數越多，日後跌幅越大**。壓力位置：向下跳空缺口上緣即為關鍵壓力（原缺口區轉為日後反彈的壓力）。
- **可程式化**: 是
- **所需資料**: 連續K棒OHLC與成交量、缺口偵測（兩次，方向相反）
- **計算公式**:
```
function detect_island_reversal_top(bars, i_up, i_topping_end, i_down):
    up_gap   = detect_gap(bars, i_up)      # 向上跳空
    down_gap = detect_gap(bars, i_down)    # 向下跳空
    if up_gap is None or up_gap.type != "up_gap":
        return None
    if down_gap is None or down_gap.type != "down_gap":
        return None

    # 兩缺口區間不重疊：向上缺口的上邊界 必須高於 向下缺口的下邊界（否則不構成孤島架空）
    if not (up_gap.upper_edge > down_gap.lower_edge):
        return None

    topping_days = i_down - i_up - 1   # 中間無力盤頭天數（可為0）

    return {
        "category": "高檔島型反轉",
        "topping_days": topping_days,
        "expected_decline": "盤整天數越多，日後跌幅越大（正相關，書中未給精確函數）",
        "resistance": down_gap.upper_edge,
        "action": "積極布空的好機會（必殺做空）"
    }
```
- **參數**: 無額外校準參數；「不重疊」為精確幾何條件（書中未逐字寫出但由圖例可明確推論為必要條件）
- **可回測**: 是
- **信心**: 76/100 中高（三段式結構、罕見性、盤整天數與跌幅正相關、壓力位置皆有明確文字與圖例佐證，與低檔版完全鏡射對稱可交叉驗證；精確跌幅函數未給）
