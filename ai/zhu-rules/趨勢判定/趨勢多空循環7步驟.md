# 趨勢多空循環7步驟

- **Rule ID**: R-TREND-18
- **名稱**: 趨勢多空循環7步驟
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.105-119（見 P02-C5-趨勢綜合應用.md）
- **解讀**: 整合第5章「一次弄懂多頭」與「一次弄懂空頭」的完整趨勢生命週期，是全書趨勢演變的總覽框架，多空兩版本互為鏡射：①趨勢反轉前，先在低檔/高檔出現爆量後，再次下跌/上漲時收盤無法跌破前低/突破前高，形成底部/頭部盤整。②大量K線突破/跌破該盤整，趨勢確認（多頭/空頭確認，對應R-TREND-03/04）。③反彈/回檔測試月線（MA20），若股價維持在月線之上/之下橫盤，待月線轉向後再度出現大量K線突破/跌破該橫盤，確認上漲/下跌。④股價觸及或突破/跌破季線（MA60），若季線同向逆勢（多頭時季線仍下彎、空頭時季線仍上揚），股價容易在此震盪或反彈/回檔。⑤月線向上穿越季線（或向下穿越）形成4線多頭/空頭排列，週線同時突破/跌破20週均線，趨勢正式轉為中長期格局。⑥上漲/下跌過程中出現大量或爆量K線創新高/新低後，趨勢延續。⑦上漲/下跌一波（約10%～15%以上）後，會出現修正，修正方式分4種情況：情況1弱勢回檔/反彈（幅度不到二分之一，未破月線/前低或前高，趨勢繼續，即回後買上漲/彈後空下跌）；情況2強勢回檔/反彈（幅度超過二分之一，跌破/突破月線或前低/前高，容易進入頭頭低/底底高盤整，需等待重新符合原趨勢架構才續做）；情況3盤整突破/跌破（盤整結束，延續原趨勢方向）；情況4 ABC修正結束（短空/短多回測失敗，突破/跌破原始切線，延續原趨勢方向）。
- **可程式化**: 部分（各階段觸發條件多數可程式化，但完整7步驟的「循環狀態機」需要整合R-TREND-01～13多條子規則作為輸入，工程實作複雜度較高；10%～15%修正幅度門檻書中有明確數字）
- **所需資料**: 收盤價、MA20（月線）、MA60（季線）、週線收盤價與週20均線、成交量、轉折波頭/底序列（R-TREND-01）、多頭/空頭趨勢判定（R-TREND-03/04）、盤整判定（R-TREND-05）
- **計算公式**:
```
# 以多頭方向為例（空頭版本方向對稱互換）
stage = "無趨勢"

# 步驟1-2：打底與趨勢確認
if 底部盤整且再次下跌不破前低:
    stage = "底部盤整"
if 大量紅K突破底部盤整上頸線:
    stage = "多頭確認"    # 對應 R-TREND-03/06

# 步驟3-4：月線季線測試
if stage == "多頭確認" and 股價在MA20之上橫盤 and slope(MA20) 由平轉揚 and 大量紅K突破橫盤:
    stage = "多頭確認上漲"
if 股價觸及或突破MA60 and slope(MA60) < 0:
    note = "季線下彎，股價容易在此震盪或回檔"

# 步驟5：中長多格局
if MA20 上穿 MA60 and 4線多排(MA5>MA10>MA20>MA60, 皆向上) and 週收盤 > 週MA20:
    stage = "中長多格局"

# 步驟6：續漲確認
if 大量或爆量K線 and Close[t] > 前一個轉折高點:
    stage = "趨勢延續"

# 步驟7：修正分類（漲幅達 threshold=10%~15% 以上後才適用）
if 累積漲幅 >= 0.10:  # 可設 0.10~0.15
    pullback_pct = (波段高點 - 回檔低點) / (波段高點 - 波段起漲點)
    if pullback_pct < 0.5 and 未破MA20 and 未破前低:
        correction_type = "情況1 弱勢回檔：回後買上漲，多頭繼續"
    elif pullback_pct >= 0.5 and (跌破MA20 or 跌破前低):
        correction_type = "情況2 強勢回檔：容易進入頭頭低盤整，需等重新符合多頭架構再做多"
    elif is_consolidation and Close[t] > upper_neckline:
        correction_type = "情況3 盤整突破：多頭繼續"
    elif ABC修正結束 and Close[t] > 原始下降切線:
        correction_type = "情況4 ABC修正結束：多頭繼續"
```
- **參數**: 波段漲幅/跌幅門檻＝10%～15%以上（書中明確數字，超過此幅度才適用修正分類）；回檔/反彈幅度分界＝原波段的二分之一（書中明確數字，用以區分弱勢/強勢修正）
- **可回測**: 部分（各子條件可回測，但完整狀態機需整合多條規則，實作與驗證成本較高）
- **信心**: 62/100 中（整體流程框架描述清楚，且10%～15%、二分之一等關鍵數字明確＝高；但作為連續性「循環」的完整狀態機判斷，需要組合大量子規則且部分轉換時機書中僅用示意圖呈現，工程化時仍需補充邊界判斷，綜合評為中）
