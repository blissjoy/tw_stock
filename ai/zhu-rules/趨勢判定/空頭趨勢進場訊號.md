# 空頭趨勢進場訊號（彈後空下跌與盤整跌破）

- **Rule ID**: R-TREND-07
- **名稱**: 空頭趨勢進場訊號（彈後空下跌與盤整跌破）
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.073-074、p.093-094（見 P02-C3-空頭趨勢總論.md）
- **解讀**: 與多頭進場訊號（R-TREND-06）完全鏡射對稱。書中空頭進場口訣為「①彈後空下跌、②盤整的跌破」。「彈後空下跌」：空頭下跌一段後反彈修正，只要不突破前一個轉折高點，出現帶量長黑K線下跌，收盤跌破前一日最低點，即為進場做空點。「盤整的跌破」：空頭下跌一段後走勢橫向盤整，出現帶量長黑K線收盤跌破盤整下頸線，即為進場做空點。成功率較高的空頭下跌當天做空條件為「股價下跌、有大量、實體黑K線、收盤跌破昨低」。
- **可程式化**: 是
- **所需資料**: 收盤價、最高價、最低價、成交量、5日均線、盤整下頸線（R-TREND-05）、空頭趨勢判定結果（R-TREND-04）、K線實體跌幅
- **計算公式**:
```
# 訊號1：彈後空下跌
if is_bear_trend and 反彈未突破前一個轉折高點 (High_rebound < prev_head):
    if Close[t] < MA5[t] and Close[t] < Low[t-1] and Volume[t] > Volume[t-1] \
       and (Open[t] - Close[t]) / Open[t] >= 長黑K門檻(預設2%):
        signal = "彈後空下跌"

# 訊號2：盤整的跌破
if 前一趨勢為空頭 and is_consolidation (見 R-TREND-05):
    if Close[t] < lower_neckline and Volume[t] > 前均量*攻擊量倍數(預設1.3) \
       and (Open[t] - Close[t]) / Open[t] >= 長黑K門檻(預設2%):
        signal = "盤整的跌破"
```
- **參數**: 長黑K實體跌幅門檻（借用同章短線做空SOP的「黑K實體棒>2%」，預設2%）；放量倍數（借用同章SOP的「較前一日增加1.3倍以上」，預設1.3）
- **可回測**: 是
- **信心**: 63/100 中（訊號方向與觸發條件明確＝高；「帶量」「長黑K」精確門檻本節未直接給出，需借用短線做空SOP的代理數字，綜合信心中）
