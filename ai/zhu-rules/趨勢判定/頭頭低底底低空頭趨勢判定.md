# 頭頭低底底低空頭趨勢判定

- **Rule ID**: R-TREND-04
- **名稱**: 頭頭低底底低空頭趨勢判定
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.071（見 P02-C3-空頭趨勢總論.md）
- **解讀**: 空頭趨勢判定與多頭判定（R-TREND-03）完全鏡射對稱。依道氏觀察理論：波浪下跌時，股價不斷創新低，反彈的頭部高點會一次比一次低，下跌的底部低點也一次比一次低，此現象稱為「頭頭低、底底低」。只要最近一個轉折頭部（頭2）比前一個轉折頭部（頭1）低，「同時」最近一個轉折底部（底2）比前一個轉折底部（底1）低，兩個條件缺一不可，即判定目前走勢為空頭趨勢。判定方法同樣是「由最後一天收盤K線往左」，找最近的頭與底比較。確認為空頭架構後，操作原則是順勢只做空。
- **可程式化**: 是
- **所需資料**: 轉折波頭/底序列（R-TREND-01輸出，至少需要最近2個頭與2個底）
- **計算公式**:
```
# Head[-1], Head[-2] 為最近與次近的轉折頭部；Bottom[-1], Bottom[-2] 為最近與次近的轉折底部
is_bear_trend = (Head[-1] < Head[-2]) and (Bottom[-1] < Bottom[-2])

# 若條件不同時成立，則不成立空頭架構，應視為盤整（見 R-TREND-05）
```
- **參數**: 無額外可調參數（純比較最近2組頭/底的高低）；可調整的是上游轉折波取點所用的均線天期（5/10/20日，見R-TREND-01）
- **可回測**: 是
- **信心**: 94/100 高（書中原文明確標註「2個條件要同時成立，缺一不可」，判定方式描述清楚，與多頭判定完全對稱）
