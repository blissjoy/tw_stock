# 空頭確認爆大量與做空時機判斷

- **Rule ID**: R-TREND-13
- **名稱**: 空頭確認爆大量與做空時機判斷
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.080-081（見 P02-C3-空頭趨勢總論.md）
- **解讀**: 當趨勢在高檔出現「頭頭低、底底低」空頭架構（R-TREND-04）確認，並伴隨大量長黑K時，代表下跌力道較強，持有多單者應立刻賣出。此時是否適合立即進場做空，取決於股價與月線（MA20）、季線（MA60）的相對位置：若該位置已跌破月線（且月線同時下彎），但尚未跌破季線，可視為適合做短空的時機；若該位置尚未跌破月線，則應等待下一次「彈後空下跌」訊號（R-TREND-07）出現後再進場，不宜此時追空。
- **可程式化**: 部分（月線/季線相對位置與方向的判斷邏輯明確可程式化；「大量」的精確倍數門檻本節未給出具體數字，需自訂）
- **所需資料**: 收盤價、成交量、MA20（月線）、MA60（季線）、空頭趨勢判定結果（R-TREND-04）
- **計算公式**:
```
if is_bear_trend (頭頭低底底低確認, 見 R-TREND-04) and Volume[t] >= 大量門檻(需自訂) and 長黑K:
    action_existing_long = "立刻賣出多單"

    if Close[t] < MA20[t] and slope(MA20) < 0 and Close[t] > MA60[t]:
        signal = "可做短空"
    elif Close[t] >= MA20[t]:
        signal = "尚未跌破月線，等待下一次彈後空下跌訊號再進場（見 R-TREND-07）"
```
- **參數**: 大量倍數門檻（書中本節未給精確數字，僅用「大量」字眼，需自訂；可參考 R-TREND-12 的3～5倍/2倍作為代理起點）；月線方向需向下、且股價尚未跌破季線為書中明確描述條件
- **參考來源查證（2026-07-22，工程補充）**：與R-CANDLE-23同一批查證，「爆大量」台股常見定義為2倍5日均量（參考：[豐雲學堂](https://www.sinotrade.com.tw/richclub/hotopic/%E9%AB%98%E6%AA%94%E7%88%86%E9%87%8F%E6%98%AF%E4%BB%80%E9%BA%BC%E6%84%8F%E6%80%9D-%E8%AD%A6%E8%A8%8A%E9%82%84%E6%98%AF%E8%B2%B7%E9%BB%9E-%E5%85%A9%E5%BC%B5%E5%9C%96%E8%A1%A8%E6%95%99%E4%BD%A0%E6%8E%8C%E6%8F%A1%E5%B8%82%E5%A0%B4%E9%97%9C%E9%8D%B5%E8%A8%8A%E8%99%9F-64d9e9c81c00522a7c5a0756)），此處採用2倍5日均量作為本規則「大量」的代理門檻，與R-TREND-12的爆量下限一致。
- **可回測**: 部分（月線/季線相對位置判斷可直接回測；「大量」門檻需自訂數字才能精確回測）
- **信心**: 65/100 中（月線/季線相對位置的操作決策邏輯描述清楚＝高；「大量」的量化定義本節未給出精確倍數＝拉低整體信心，綜合評為中）
