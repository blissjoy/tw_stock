# 多轉空做頭量價與均線排列鎖股訊號（含一日反轉）

- **Rule ID**: R-TREND-11
- **名稱**: 多轉空做頭量價與均線排列鎖股訊號（含一日反轉）
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.086-092（見 P02-C3-空頭趨勢總論.md）
- **解讀**: 與空轉多打底量價訊號（R-TREND-10）完全鏡射對稱，描述多頭上漲末端如何辨識即將反轉空頭的完整量價流程，是多頭轉空頭的「做頭兩個頭」框架：**第1個頭**——多頭上漲到高檔出現爆量紅K或下跌黑K止漲，視為第1次出貨量；若回檔到月線（MA20）遇支撐出現紅K上漲，但股價無法再突破第1個頭的高點（出現頭頭低），視為出現壓力，應立刻出場；若回檔進一步跌破月線且跌破前低（出現底底低），代表空方力道轉強，趨勢改變，不宜再做多。**第2個頭**——第1個頭止漲後再度出現大量回檔，若高點不過第1個頭高點（頭頭低），代表做頭盤整成形，回檔低點成為支撐、第2個頭高點成為壓力；股價跌破支撐低點即反轉確認空頭。做頭期間需同步觀察均線排列：當10日均與20日均轉為向下空頭排列，代表做頭接近完成，可鎖股準備做短空；股價跌破60日均線、均線4線空排後，可鎖股準備做短中長空。此外，書中另定義「一日反轉」訊號：高檔出現爆量長黑K或爆量長上影線，次日股價即跌破當天最低點，構成一日反轉，多單應立刻出場，常見於急漲個股高檔。
- **可程式化**: 部分（均線排列與MA60跌破條件可完全程式化；一日反轉的觸發邏輯明確可程式化；「爆量」的精確倍數門檻本節未給出具體數字，需自訂）
- **所需資料**: 收盤價、最高價、最低價、成交量、MA10、MA20、MA60
- **計算公式**:
```
# 第1個頭
if 多頭上漲中 and Volume[t] >= 爆量門檻(需自訂) and (長紅K止漲 or 下跌黑K):
    leg1_high = High[t]
    後續回檔至MA20附近遇支撐（紅K）:
        if 之後反彈高點 < leg1_high:               # 頭頭低，出現壓力
            action = "立刻出場，留意反轉"
        if 回檔跌破MA20 and 跌破前一個轉折低點:        # 底底低
            action = "趨勢改變，不宜再做多"

# 第2個頭
if 第1個頭止漲後再度爆量回檔 and 回檔後反彈高點 < leg1_high:   # 頭頭低
    leg2_high = 反彈高點
    support = min(第1個頭回檔低點, 第2個頭回檔前低點)
    if Close[t] < support:
        趨勢反轉為空頭（見 R-TREND-04 確認）

# 均線排列鎖股條件
if MA10[t] < MA20[t] and slope(MA10) < 0 and slope(MA20) < 0:
    signal = "做頭接近完成，鎖股準備做短空"
if Close[t] < MA60[t]:
    signal = "均線4線空排，鎖股準備做短中長空"

# 一日反轉
if 高檔出現(爆量長黑K or 爆量長上影線) at day t:
    day_low = Low[t]
    if Low[t+1] < day_low:
        signal = "一日反轉，多單立刻出場"
```
- **參數**: 爆量倍數門檻（書中未給精確數字，需自訂，可參考 R-TREND-13 的相關描述）；均線排列條件（MA10<MA20且皆向下、跌破MA60）為書中明確描述
- **可回測**: 部分（爆量門檻需自訂數字才能精確回測；均線排列、MA60跌破、一日反轉條件皆可直接回測）
- **信心**: 62/100 中（整體流程方向、均線排列條件、一日反轉觸發邏輯描述清楚＝高；「爆量」量化倍數本節未給具體數字＝拉低信心，綜合評為中）
