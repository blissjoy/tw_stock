# 轉折波定義與取點演算法（5日/10日/20日短中長線）

- **Rule ID**: R-TREND-01
- **名稱**: 轉折波定義與取點演算法（5日/10日/20日短中長線）
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.033-040（見 P02-C1-認識轉折波.md）
- **解讀**: 轉折波是全書趨勢判定的基礎輸入資料，是後續「頭頭高底底高」等多空趨勢判定規則的必要前置步驟。核心邏輯是用「收盤價相對均線的位置」把連續K線切成「正價區段」（收盤在均線上方）與「負價區段」（收盤在均線下方）；每次收盤價由正轉負（即「收盤跌破均線」）時，把前面連續的正價區段K線＋當天跌破的K線一起取區間內最高點（含上影線）作為一個轉折高點（頭）；每次收盤價由負轉正（即「收盤突破均線」）時，把前面連續的負價區段K線＋當天突破的K線一起取區間內最低點（含下影線）作為一個轉折低點（底）。依時間序把取出的頭、底依序連接，即為轉折波。5日、10日、20日均線只是套用同一套演算法時的參數不同，均線週期越短轉折點越多越敏感（適合短線操作），週期越長轉折點越少越平滑（適合判斷中長期趨勢及支撐壓力）。書中口訣：「突破5日均取低點，跌破5均取高點」（10日、20日同理）。
- **可程式化**: 是
- **所需資料**: 每日OHLC（開盤、最高、最低、收盤價）、N日均線（以收盤價計算的簡單移動平均）
- **計算公式**:
```
function compute_turning_points(bars, N):
    ma = SMA(bars.Close, N)
    state = None          # "positive"（收盤在均線上）或 "negative"（收盤在均線下）
    group = []             # 目前累積中的同狀態K線區段
    turning_points = []    # 輸出：[(type, price, date), ...]

    for i, bar in enumerate(bars):
        if bar.Close > ma[i]:
            cur = "positive"
        elif bar.Close < ma[i]:
            cur = "negative"
        else:
            cur = state    # 收盤剛好等於均線：書中未定義，預設沿用前一狀態

        if state is None:
            state = cur
            group = [bar]
            continue

        if cur == state:
            group.append(bar)
        else:
            group.append(bar)          # 當天K線併入取值區間（跌破/突破當天也算在內）
            if state == "positive" and cur == "negative":
                # 收盤跌破均線 -> 取這段區間「最高點」（含上影線）為轉折高點（頭）
                head_bar = max(group, key=lambda b: b.High)
                turning_points.append(("head", head_bar.High, head_bar.date))
            elif state == "negative" and cur == "positive":
                # 收盤突破均線 -> 取這段區間「最低點」（含下影線）為轉折低點（底）
                bottom_bar = min(group, key=lambda b: b.Low)
                turning_points.append(("bottom", bottom_bar.Low, bottom_bar.date))
            state = cur
            group = [bar]               # 重新累積下一段

    return turning_points   # 依時間順序交替排列：頭1, 底1, 頭2, 底2, ...
```
- **參數**: 均線天期 N（可選 5＝短線轉折波，10＝中線轉折波，20＝長線轉折波；預設 N=5）
- **可回測**: 是
