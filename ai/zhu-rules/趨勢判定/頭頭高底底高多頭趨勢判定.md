# 頭頭高底底高多頭趨勢判定

- **Rule ID**: R-TREND-03
- **名稱**: 頭頭高底底高多頭趨勢判定
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.033、p.048（見 P02-C1-認識轉折波.md、P02-C2-多頭趨勢總論.md）
- **解讀**: 這是全書多頭趨勢判定的核心定義。依道氏（道瓊）觀察理論：波浪上漲時，頭部高點會一次比一次高，回落修正時底部低點也一次比一次墊高，此現象稱為「頭頭高、底底高」。只要最近一個轉折頭部（頭2）比前一個轉折頭部（頭1）高，「同時」最近一個轉折底部（底2）比前一個轉折底部（底1）高，兩個條件缺一不可，即判定目前走勢為多頭趨勢。判定方法是「由最後一天收盤K線往左」，找出最近的頭與底進行比較。此規則完全依賴 R-TREND-01（或R-TREND-02簡化版）所輸出的轉折高低點序列作為輸入，是後續「多頭趨勢進場訊號」「多頭趨勢改變先知先覺訊號」等規則的判斷基礎。確認為多頭架構後，操作原則是順勢只做多。
- **可程式化**: 是
- **所需資料**: 轉折波頭/底序列（R-TREND-01輸出，至少需要最近2個頭與2個底）
- **計算公式**:
```
# Head[-1], Head[-2] 為最近與次近的轉折頭部；Bottom[-1], Bottom[-2] 為最近與次近的轉折底部
is_bull_trend = (Head[-1] > Head[-2]) and (Bottom[-1] > Bottom[-2])

# 若條件不同時成立（例如頭頭高但底底不高，或底底高但頭頭不高），
# 則不成立多頭架構，應視為盤整（見 R-TREND-05）
```
- **參數**: 無額外可調參數（純比較最近2組頭/底的高低）；可調整的是上游轉折波取點所用的均線天期（5/10/20日，見R-TREND-01）
- **可回測**: 是
- **信心**: 94/100 高（書中原文明確標註「2個條件要同時成立，缺一不可」，且清楚說明判定方式為「由最後一天收盤K線往左，和最近的頭及底比較」）
