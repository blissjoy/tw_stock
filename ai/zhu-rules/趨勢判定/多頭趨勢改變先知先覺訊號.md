# 多頭趨勢改變先知先覺訊號

- **Rule ID**: R-TREND-08
- **名稱**: 多頭趨勢改變先知先覺訊號
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.051、p.059-062、p.068（見 P02-C2-多頭趨勢總論.md）
- **解讀**: 在多頭趨勢已確認（頭頭高底底高，R-TREND-03）的前提下，持續監控最新一次回檔或上漲是否仍維持該架構。只要出現「回檔跌破前一個開始上漲的低點」（即底底低）或「上漲未能創新高就轉跌」（即頭頭低）任一種現象，即使尚未達到空頭完整確認（頭頭低與底底低同時成立），也代表多頭方向已出現改變的早期訊號，應先視為進入盤整、做短線的多單要出場。這是比「等空頭完整確認才出場」更早一步的預警機制（先知先覺），也是書中短線做多獲利方程式明確列出的出場條件之一（收盤出現頭頭低即出場）。
- **可程式化**: 是
- **所需資料**: 轉折波頭/底序列（R-TREND-01輸出）、多頭趨勢判定狀態（R-TREND-03）
- **計算公式**:
```
if is_bull_trend_previously_confirmed:
    if Bottom[-1] < Bottom[-2]:      # 回檔跌破前低，出現「底底低」
        warning = "多頭出現改變（底底低），趨勢轉入盤整，短線多單出場"
    elif Head[-1] < Head[-2]:        # 反彈未過前高就轉跌，出現「頭頭低」
        warning = "多頭出現改變（頭頭低），趨勢轉入盤整，短線多單出場"
    else:
        multi_trend_intact = True    # 多頭架構仍維持
```
- **參數**: 無額外可調參數（純比較最近轉折頭/底的高低）
- **可回測**: 是
- **信心**: 91/100 高（書中明確列出2種先知先覺情況，並在短線做多獲利方程式中直接寫明「收盤出現頭頭低，出場」為具體可執行規則）
