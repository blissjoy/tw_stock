# 做空進場10大戒律

- **Rule ID**: R-TREND-17
- **名稱**: 做空進場10大戒律
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.094-095（見 P02-C3-空頭趨勢總論.md）
- **解讀**: 與做多進場10大戒律（R-TREND-16）完全鏡射對稱，列出做空進場時應避免的10種情境：盤頭未跌破月線、連續下跌第3根以上位置殺低、量價背離搭配KD低檔＋乖離過大＋均線開口大、週線遇支撐前、反彈突破月線再下跌未跌破月線、反彈突破前高再下跌、盤整區內、一般多頭的回檔、連續急跌的大量長黑K低檔、進場位置是價跌的紅K。同樣是反向過濾清單，命中任一條即不應進場做空。
- **可程式化**: 部分（多數條件可轉換為可程式化的量化過濾條件，但「量價背離」「乖離過大」「均線開口大」等條件書中本節未逐一給出精確數字）
- **所需資料**: 收盤價、MA5/MA10/MA20/MA60、週線資料、K線實體與位置、成交量、KD指標、乖離率、多頭/空頭/盤整趨勢判定結果（R-TREND-03/04/05）
- **計算公式**:
```
avoid_short = False
reasons = []

if 盤頭做頭尚未跌破MA60(季線):
    avoid_short = True; reasons.append("盤頭未跌破月線")
if 目前是連續下跌第3根以上K線 (連續下跌天數 >= 3):
    avoid_short = True; reasons.append("殺低風險")
if 量價背離 and KD處於低檔 and 乖離率過大(需自訂門檻) and 均線開口過大(需自訂門檻):
    avoid_short = True; reasons.append("量價背離＋指標過熱")
if 週線遇支撐區間附近:
    avoid_short = True; reasons.append("週線支撐")
if 反彈曾突破MA20 and 再下跌未跌破MA20:
    avoid_short = True; reasons.append("反彈突破月線未跌破")
if 反彈突破前一個轉折高點 and 再下跌:
    avoid_short = True; reasons.append("突破前高再下跌，架構已破壞")
if is_consolidation:                                    # 見 R-TREND-05
    avoid_short = True; reasons.append("盤整區間不進場")
if is_bull_trend and 目前是多頭中的回檔:                  # 見 R-TREND-03
    avoid_short = True; reasons.append("一般多頭回檔不做空")
if 連續急跌(需自訂天數門檻) and 出現大量長黑K低檔:
    avoid_short = True; reasons.append("連續急跌大量低檔")
if 當前K線為價跌但收紅K (Close < Close[t-1] but Close > Open):
    avoid_short = True; reasons.append("進場位置是價跌的紅K")

can_enter_short = not avoid_short
```
- **參數**: 殺低天數門檻＝3根以上（書中明確）；乖離率過大、均線開口過大、連續急跌天數等門檻書中未給精確數字，需自訂
- **可回測**: 部分（多數條件可回測；「量價背離」「乖離過大」「均線開口大」需先自訂量化門檻才能精確回測）
- **信心**: 62/100 中（10條戒律的方向與邏輯描述清楚＝高；部分條件涉及模糊描述、書中未給明確數字門檻＝拉低信心，綜合評為中）
