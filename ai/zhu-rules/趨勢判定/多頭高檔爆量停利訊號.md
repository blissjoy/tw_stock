# 多頭高檔爆量停利訊號

- **Rule ID**: R-TREND-12
- **名稱**: 多頭高檔爆量停利訊號
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.062-065（見 P02-C2-多頭趨勢總論.md）
- **解讀**: 多頭上漲到高檔出現異常大量時，容易發生反轉，應立刻停利出場。書中給出兩種具體情境，皆附有明確數字門檻：**情境①**——單日出現前一日成交量3～5倍以上的異常天量，且當天股價收長黑K下跌，容易發生「1日反轉」，常見於飆股急漲後的高檔，應立刻停利出場。**情境②**——上漲過程中連續2～3天出現5日均量2倍以上的大量，但股價不漲或下跌，應立刻停利出場，此時股價容易快速回檔；若此現象出現在多頭的末升段，後續做頭反轉的機率更高，需提高警覺。
- **可程式化**: 是
- **所需資料**: 收盤價、開盤價、成交量、5日均量
- **計算公式**:
```
# 情境①：異常天量長黑
if Volume[t] >= 3 * Volume[t-1] and Close[t] < Open[t]:   # 長黑K，倍數可設3~5倍區間
    if 目前處於多頭高檔(依趨勢位置或漲幅/乖離判斷):
        signal = "異常天量長黑，立刻停利出場"

# 情境②：連續爆量不漲
window = last 3 trading days
count = count(d in window where Volume[d] >= 2 * MA_Volume5[d])
if count >= 2 and (Close[t] <= Close[t-1]):   # 股價不漲或下跌
    signal = "連續爆量不漲，立刻停利出場"
    if 位於多頭末升段:
        note = "後續做頭反轉機率高，加強觀察"
```
- **參數**: 天量倍數＝前一日量的3～5倍（書中明確數字，取下限3倍為保守觸發門檻）；連續爆量倍數＝5日均量的2倍（書中明確數字）；連續天數＝2～3天
- **可回測**: 是
- **信心**: 93/100 高（書中直接給出「前一日量3～5倍以上」「5日均線量2倍以上」等具體數字門檻，且分別對應長黑K與連續不漲兩種明確觸發條件）
