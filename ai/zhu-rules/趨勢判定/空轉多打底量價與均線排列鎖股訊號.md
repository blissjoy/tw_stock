# 空轉多打底量價與均線排列鎖股訊號

- **Rule ID**: R-TREND-10
- **名稱**: 空轉多打底量價與均線排列鎖股訊號
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.054-058（見 P02-C2-多頭趨勢總論.md）
- **解讀**: 描述空頭下跌末端如何辨識即將反轉多頭的完整量價流程，是空頭轉多頭的「打底兩支腳」框架：**第1支腳**——空頭下跌到低檔出現爆量下跌黑K或止跌紅K，視為搶反彈的進貨量；若反彈到月線（MA20）遇壓拉回，但股價不再跌破第1支腳低點（出現底底高），視為出現支撐，應立刻回補空單；若反彈進一步突破月線且突破前高（出現頭頭高），代表多方力道轉強，趨勢改變，不宜再做空。**第2支腳（黃金右腳）**——第1支腳止穩後再度出現大量反彈，若低點不破第1支腳低點（底底高），代表打底盤整成形，兩腳之間的反彈高點成為壓力、第2支腳低點成為支撐；股價突破兩腳中間最高點即反轉確認多頭。打底期間需同步觀察均線排列：當10日均與20日均轉為向上多頭排列，代表打底接近完成，可鎖股準備做短多；股價進一步突破60日均線後，可鎖股準備做短中長多。
- **可程式化**: 部分（均線排列與60日均突破條件可完全程式化；「爆量」「異常大量」的精確倍數門檻本節未給出具體數字，需自訂）
- **所需資料**: 收盤價、最高價、最低價、成交量、MA10、MA20、MA60
- **計算公式**:
```
# 第1支腳
if 空頭下跌中 and Volume[t] >= 爆量門檻(需自訂，例如前均量的X倍) and (長黑K or 止跌紅K):
    leg1_low = Low[t]
    後續反彈至MA20附近遇壓拉回（黑K）:
        if 之後回檔低點 > leg1_low:               # 底底高，出現支撐
            action = "立刻回補空單，留意反轉"
        if 反彈突破MA20 and 突破前一個轉折高點:      # 頭頭高
            action = "趨勢改變，不宜再做空"

# 第2支腳（黃金右腳）
if 第1支腳止穩後再度爆量反彈 and 反彈後回檔低點 > leg1_low:   # 底底高
    leg2_low = 回檔低點
    resistance = max(第1支腳反彈高點, 第2支腳反彈前高點)
    if Close[t] > resistance:
        趨勢反轉為多頭（見 R-TREND-03 確認）

# 均線排列鎖股條件
if MA10[t] > MA20[t] and slope(MA10) > 0 and slope(MA20) > 0:
    signal = "打底接近完成，鎖股準備做短多"
if Close[t] > MA60[t]:
    signal = "鎖股準備做短中長多"
```
- **參數**: 爆量倍數門檻（書中僅用「異常大量」字眼，未給精確數字，需自訂，可參考 R-TREND-12 的3～5倍/2倍作為參考起點）；均線排列條件（MA10>MA20且皆向上、突破MA60）為書中明確描述
- **可回測**: 部分（爆量門檻需自訂數字才能精確回測；均線排列與MA60突破條件可直接回測）
- **信心**: 62/100 中（整體流程方向與均線排列條件描述清楚＝高；「爆量」量化倍數本節未給具體數字＝拉低信心，綜合評為中）
