# 多頭趨勢進場訊號（回後買上漲與盤整突破）

- **Rule ID**: R-TREND-06
- **名稱**: 多頭趨勢進場訊號（回後買上漲與盤整突破）
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.050、p.067（見 P02-C2-多頭趨勢總論.md）
- **解讀**: 書中多頭進場口訣為「①回後買上漲、②盤整的突破」。「回後買上漲」：多頭趨勢中上漲一波後回檔修正，只要不跌破前一個轉折低點，出現帶量中長紅K線上漲，收盤同時突破5日均線與前一日最高點，即為進場買點。「盤整的突破」：多頭上漲一波後走勢橫向盤整，出現帶量中長紅K線收盤突破盤整的上頸線，即為進場買點。書中並補充：成功率較高的多頭上漲當天買點條件為「股價上漲、有大量、實體紅K線、收盤突破昨高」。這兩個訊號都必須建立在多頭趨勢已確認（R-TREND-03）或即將確認的前提上，才是有效的順勢進場點。
- **可程式化**: 是
- **所需資料**: 收盤價、最高價、最低價、成交量、5日均線、盤整上頸線（R-TREND-05）、多頭趨勢判定結果（R-TREND-03）、K線實體漲幅
- **計算公式**:
```
# 訊號1：回後買上漲
if is_bull_trend and 回檔未跌破前一個轉折低點 (Low_pullback > prev_bottom):
    if Close[t] > MA5[t] and Close[t] > High[t-1] and Volume[t] > Volume[t-1] \
       and (Close[t] - Open[t]) / Open[t] >= 中長紅K門檻(預設2%):
        signal = "回後買上漲"

# 訊號2：盤整的突破
if 前一趨勢為多頭 and is_consolidation (見 R-TREND-05):
    if Close[t] > upper_neckline and Volume[t] > 前均量*攻擊量倍數(預設1.3) \
       and (Close[t] - Open[t]) / Open[t] >= 中長紅K門檻(預設2%):
        signal = "盤整的突破"
```
- **參數**: 中長紅K實體漲幅門檻（書中本節未直接給數字，借用同章短線做多SOP的「紅K實體棒>2%」作為代理，預設2%）；攻擊量倍數（借用同章SOP的「較前一日增加1.3倍以上，2倍以上更強」，預設1.3）
- **可回測**: 是
- **信心**: 63/100 中（訊號名稱、方向與觸發條件描述明確＝高；但「帶量」「中長紅K」的精確量化門檻本節未直接給出，需借用書中後段短線做多SOP的數字作為代理門檻，故綜合信心中）
