# 多頭短線選股與停損停利SOP

- **Rule ID**: R-TREND-14
- **名稱**: 多頭短線選股與停損停利SOP
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.067-068（見 P02-C2-多頭趨勢總論.md）
- **解讀**: 整合書中「短線做多選股SOP」與「短線做多獲利方程式」，是一套完整的多頭順勢短線交易檢核清單。選股條件（6要件）：①趨勢須符合頭頭高底底高多頭架構（R-TREND-03）；②日線MA10、MA20多頭排列且方向向上（若季線仍在股價上方且下彎，股價上漲靠近時會有壓力）；③股價收盤須在MA10、MA20之上，並判斷目前處於初升、主升或末升段以決定操作時程；④成交量方面，打底時要有大量，發動攻擊時當天量須比前一日增加1.3倍以上（2倍以上力道更強，沒放量仍可能上漲但後續要補量）；⑤進場K線要價漲、量增、紅K實體棒漲幅>2%；⑥指標參考：MACD綠柱縮短或紅柱延長、KD黃金交叉向上多排。停損停利方程式：停損設在進場中長紅K線最低點下方5%～7%，收盤跌破停損即出場；收盤出現「頭頭低」即出場（見R-TREND-08）；未跌破停損且獲利未達5%則續抱；獲利超過10%且收盤跌破MA5則出場；獲利超過20%，或連續3天以上急漲後出現大量長黑K強覆蓋／長黑吞噬，則當天出場。
- **可程式化**: 部分（趨勢、均線排列、股價位置、量能倍數、K線實體%、停損停利%數皆可程式化；MACD/KD指標訊號本身屬於指標分類，此處僅整合引用，實際判斷邏輯需參照指標分類規則庫）
- **所需資料**: 收盤價、MA5、MA10、MA20、MA60（季線）、成交量、5日均量、K線實體漲幅、多頭趨勢判定結果（R-TREND-03）、MACD、KD指標
- **計算公式**:
```
# 選股條件（進場前置檢核）
entry_ready = (
    is_bull_trend                                    # 見 R-TREND-03
    and MA10 > MA20 and slope(MA10) > 0 and slope(MA20) > 0
    and Close > MA10 and Close > MA20
    and Volume[t] >= 1.3 * Volume[t-1]                # 攻擊量門檻，2倍以上更強
    and (Close[t]-Open[t]) / Open[t] > 0.02            # 紅K實體棒 > 2%
)
# 指標輔助條件（可選）：MACD綠柱縮短或紅柱延長；KD黃金交叉向上多排

if entry_ready:
    entry_price = Close[t]
    stop_loss = Low(進場中長紅K) * (1 - 停損比例)      # 停損比例 5%~7%

# 停損停利判斷（持倉期間逐日檢查）
if Close[t] < stop_loss:
    action = "跌破停損，出場"
elif 出現頭頭低 (見 R-TREND-08):
    action = "收盤出現頭頭低，出場"
elif profit_pct(Close[t], entry_price) > 0.10 and Close[t] < MA5[t]:
    action = "獲利超過10%且跌破MA5，出場"
elif profit_pct(Close[t], entry_price) > 0.20:
    action = "獲利超過20%，當天出場"
elif 連續3天以上急漲 and 出現大量長黑K強覆蓋或長黑吞噬:
    action = "當天出場"
else:
    action = "續抱"
```
- **參數**: 停損比例＝5%～7%（書中明確數字）；攻擊量倍數＝1.3倍（2倍以上更強，書中明確數字）；進場紅K實體漲幅門檻＝2%（書中明確數字）；停利觸發＝獲利>10%且跌破MA5、或獲利>20%、或連續3天以上急漲＋大量長黑K（皆為書中明確數字）
- **可回測**: 是
- **信心**: 92/100 高（書中對進場、停損、停利各環節皆給出具體數字：1.3倍/2倍量、2%實體漲幅、5%～7%停損、10%/20%停利門檻、3天急漲）
