# 空頭短線選股與停損停利SOP

- **Rule ID**: R-TREND-15
- **名稱**: 空頭短線選股與停損停利SOP
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.093-094（見 P02-C3-空頭趨勢總論.md）
- **解讀**: 與多頭短線選股與停損停利SOP（R-TREND-14）完全鏡射對稱。選股條件（6要件）：①趨勢須符合頭頭低底底低空頭架構（R-TREND-04）；②日線MA10、MA20空頭排列且方向向下（若季線仍在股價下方且上揚，股價下跌靠近時會有支撐）；③股價收盤須在MA10、MA20之下，並判斷處於初跌、主跌或末跌段以決定操作時程；④成交量方面，做頭時要有大量（代表主力出貨），發動下跌時黑K實體或跳空黑K須有大量配合（當天量比前一日增加1.3倍以上），若未放量仍可能下跌，但後續容易出現恐慌性大量賣出；⑤進場K線要價跌、黑K實體棒跌幅>2%；⑥指標參考：MACD紅柱縮短或綠柱延長、KD死亡交叉向下空排。停損停利方程式：做空進場位置為收盤跌破5日均線並跌破前一日最低點；停損設在進場中長黑K線最高點上方5%～7%；收盤突破停損即出場；收盤出現「底底高」即回補（見R-TREND-09）；未突破停損且獲利未達5%則續抱；獲利超過10%且收盤突破MA5則回補；獲利超過20%，或連續3天以上急跌後出現大量長紅K強覆蓋／長紅吞噬，則當天回補。
- **可程式化**: 部分（趨勢、均線排列、股價位置、量能倍數、K線實體%、停損停利%數皆可程式化；MACD/KD指標訊號屬於指標分類，此處僅整合引用）
- **所需資料**: 收盤價、MA5、MA10、MA20、MA60（季線）、成交量、K線實體跌幅、空頭趨勢判定結果（R-TREND-04）、MACD、KD指標
- **計算公式**:
```
# 選股條件（進場前置檢核）
entry_ready = (
    is_bear_trend                                     # 見 R-TREND-04
    and MA10 < MA20 and slope(MA10) < 0 and slope(MA20) < 0
    and Close < MA10 and Close < MA20
    and Volume[t] >= 1.3 * Volume[t-1]
    and (Open[t]-Close[t]) / Open[t] > 0.02             # 黑K實體棒 > 2%
)
# 指標輔助條件（可選）：MACD紅柱縮短或綠柱延長；KD死亡交叉向下空排

if entry_ready and Close[t] < MA5[t] and Close[t] < Low[t-1]:
    entry_price = Close[t]
    stop_loss = High(進場中長黑K) * (1 + 停損比例)      # 停損比例 5%~7%

# 停損停利判斷（持倉期間逐日檢查）
if Close[t] > stop_loss:
    action = "突破停損，回補"
elif 出現底底高 (見 R-TREND-09):
    action = "收盤出現底底高，回補"
elif profit_pct_short(Close[t], entry_price) > 0.10 and Close[t] > MA5[t]:
    action = "獲利超過10%且突破MA5，回補"
elif profit_pct_short(Close[t], entry_price) > 0.20:
    action = "獲利超過20%，當天回補"
elif 連續3天以上急跌 and 出現大量長紅K強覆蓋或長紅吞噬:
    action = "當天回補"
else:
    action = "續抱"
```
- **參數**: 停損比例＝5%～7%（書中明確數字）；放量倍數＝1.3倍（書中明確數字）；進場黑K實體跌幅門檻＝2%（書中明確數字）；停利觸發＝獲利>10%且突破MA5、或獲利>20%、或連續3天以上急跌＋大量長紅K（皆為書中明確數字）
- **可回測**: 是
- **信心**: 92/100 高（書中對進場、停損、停利各環節皆給出具體數字，與多頭SOP對稱且同樣明確）
