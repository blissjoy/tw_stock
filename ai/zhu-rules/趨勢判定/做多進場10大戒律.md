# 做多進場10大戒律

- **Rule ID**: R-TREND-16
- **名稱**: 做多進場10大戒律
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.069（見 P02-C2-多頭趨勢總論.md）
- **解讀**: 書中列出做多進場時應避免的10種情境，性質是「反向過濾清單」——只要命中其中一條，就不應該進場做多。這10條戒律涵蓋趨勢位置（盤底未過月線、盤整區內、一般空頭反彈）、K線位置（上漲第3根以上追高、連續急漲大量長紅K高檔、進場位置是價漲的黑K）、指標背離（量價背離＋KD高檔＋乖離過大＋均線開口大）、週線壓力、與回檔結構（跌破月線再上漲未過月線、跌破前低再上漲）。這條規則本質上是多條獨立過濾條件的組合，許多條件依賴其他規則（例如「盤整區內」依賴R-TREND-05、「一般空頭的反彈」依賴R-TREND-04）作為輸入。
- **可程式化**: 部分（多數條件可轉換為可程式化的量化過濾條件，但部分條件（如「量價背離」「均線開口大」「乖離過大」）需要額外定義量化門檻，書中本節未逐一給出精確數字）
- **所需資料**: 收盤價、MA5/MA10/MA20/MA60、週線資料、K線實體與位置、成交量、KD指標、乖離率、多頭/空頭/盤整趨勢判定結果（R-TREND-03/04/05）
- **計算公式**:
```
avoid_long = False
reasons = []

if 盤整打底尚未突破MA60(季線):
    avoid_long = True; reasons.append("盤底未過月線")
if 目前是上漲第3根以上K線 (連續上漲天數 >= 3):
    avoid_long = True; reasons.append("追高風險")
if 量價背離 and KD處於高檔 and 乖離率過大(需自訂門檻) and 均線開口過大(需自訂門檻):
    avoid_long = True; reasons.append("量價背離＋指標過熱")
if 週線遇壓力區間附近:
    avoid_long = True; reasons.append("週線壓力")
if 回檔曾跌破MA20 and 反彈上漲未重新站回MA20:
    avoid_long = True; reasons.append("回檔跌破月線未站回")
if 回檔跌破前一個轉折低點 and 再上漲:
    avoid_long = True; reasons.append("跌破前低再上漲，架構已破壞")
if is_consolidation:                                   # 見 R-TREND-05
    avoid_long = True; reasons.append("盤整區間不進場")
if is_bear_trend and 目前是空頭中的反彈:                 # 見 R-TREND-04
    avoid_long = True; reasons.append("一般空頭反彈不做多")
if 連續急漲(需自訂天數門檻) and 出現大量長紅K高檔:
    avoid_long = True; reasons.append("連續急漲大量高檔")
if 當前K線為價漲但收黑K (Close > Close[t-1] but Close < Open):
    avoid_long = True; reasons.append("進場位置是價漲的黑K")

can_enter_long = not avoid_long
```
- **參數**: 追高天數門檻＝3根以上（書中明確）；乖離率過大、均線開口過大、連續急漲天數等門檻書中未給精確數字，需自訂
- **可回測**: 部分（多數條件可回測；「量價背離」「乖離過大」「均線開口大」需先自訂量化門檻才能精確回測）
- **信心**: 62/100 中（10條戒律的方向與邏輯描述清楚＝高；但其中3～4條涉及「背離」「乖離過大」「均線開口大」等模糊描述，書中未給明確數字門檻＝拉低信心，綜合評為中）
