# 盤整趨勢判定與型態

- **Rule ID**: R-TREND-05
- **名稱**: 盤整趨勢判定與型態
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.097-099、p.101（見 P02-C4-盤整趨勢總論.md）
- **解讀**: 當走勢既不符合多頭「頭頭高底底高」架構，也不符合空頭「頭頭低底底低」架構時，即判定為盤整趨勢（依道氏理論：橫向盤整時波浪的頭部與底部會相互交叉）。書中列出4種盤整型態：①三角收斂盤整（頭愈來愈低、底愈來愈高，向中間收斂）、②矩形盤整（頭大致等高、底大致等低）、③上升直角三角盤整（底逐漸墊高、頭大致等高）、④下降直角三角盤整（頭逐漸走低、底大致等低）。盤整期間的操作原則：不進場交易、畫出上頸線（盤整區間頭部連線）與下頸線（盤整區間底部連線）、等待其中一條頸線被突破/跌破才確認盤整結束的方向。若盤整區間上下幅度小於15%，不宜在盤整區間中頻繁進出（風險報酬比太差）。盤整期間若要操作，原則是延續前一個趨勢方向：前面是多頭則盤整區間內先買後賣，前面是空頭則先賣後買。
- **可程式化**: 部分（核心判定「非多頭且非空頭＝盤整」與15%寬度門檻可完全程式化；4種型態的細分類需要對上頸線、下頸線做斜率估計，書中僅用示意圖呈現，無量化標準，屬工程判斷）
- **所需資料**: R-TREND-03（多頭判定結果）、R-TREND-04（空頭判定結果）、盤整區間內的轉折頭/底序列、收盤價、成交量
- **計算公式**:
```
is_consolidation = (not is_bull_trend) and (not is_bear_trend)   # 見 R-TREND-03 / R-TREND-04

# 頸線定義
upper_neckline = max(Head 在盤整區間內的所有轉折頭部)
lower_neckline = min(Bottom 在盤整區間內的所有轉折底部)
range_pct = (upper_neckline - lower_neckline) / lower_neckline

if range_pct < 0.15:
    建議: 不宜在此盤整區間中進出交易

# 盤整結束判定
if Close[t] > upper_neckline and 帶量長紅K:
    盤整結束，方向轉為多頭（若前趨勢為空頭則是反轉；若前趨勢為多頭則是續勢上漲）
if Close[t] < lower_neckline and 帶量長黑K:
    盤整結束，方向轉為空頭（若前趨勢為多頭則是反轉；若前趨勢為空頭則是續勢下跌）

# 盤整期間操作方向（順勢原則）
if 前一趨勢 == "多頭":
    盤整區間操作 = "先買（近下頸線）後賣（近上頸線）"
elif 前一趨勢 == "空頭":
    盤整區間操作 = "先賣（近上頸線）後買（近下頸線）"

# 型態分類（需自行實作頸線斜率估計，書中僅示意，無量化標準）
if slope(upper_neckline) < 0 and slope(lower_neckline) > 0: 型態 = "三角收斂"
elif slope(upper_neckline) ≈ 0 and slope(lower_neckline) ≈ 0: 型態 = "矩形"
elif slope(upper_neckline) ≈ 0 and slope(lower_neckline) > 0: 型態 = "上升直角三角"
elif slope(upper_neckline) < 0 and slope(lower_neckline) ≈ 0: 型態 = "下降直角三角"
```
- **參數**: 盤整寬度門檻 = 15%（書中明確數字，低於此不宜進出）；型態斜率分類的「約等於0」容許誤差需自訂
- **實作決定（2026-07-22，工程補充）**：`is_consolidation`與頸線直接沿用R-SR-03/R-SR-04的決定，重用`detect_consolidation()`與R-TREND-03/04的多空排除判斷，不另立偵測邏輯。「約等於0」的斜率容許誤差查證外部資料未找到公認統一慣例（4種盤整子型態的斜率分類本身就是書中示意圖的定性描述，非行業標準指標），故此參數保留為工程估計值（採相對於股價的斜率千分位，如|slope|/price < 0.1%視為走平），不宣稱有外部來源佐證，如實標註。
- **可回測**: 是（核心判定與15%門檻可回測；4種型態細分類可回測但精確度依頸線擬合演算法而定）
- **信心**: 62/100 中（盤整核心定義與15%寬度門檻明確＝高；但4種盤整型態僅以示意圖呈現、無量化斜率標準＝中，綜合評為中）
