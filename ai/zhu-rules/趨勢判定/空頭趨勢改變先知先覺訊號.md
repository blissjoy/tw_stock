# 空頭趨勢改變先知先覺訊號

- **Rule ID**: R-TREND-09
- **名稱**: 空頭趨勢改變先知先覺訊號
- **分類**: 趨勢判定
- **原文與頁碼**: 上冊 p.082-085、p.093-094（見 P02-C3-空頭趨勢總論.md）
- **解讀**: 與多頭趨勢改變先知先覺訊號（R-TREND-08）完全鏡射對稱。在空頭趨勢已確認（頭頭低底底低，R-TREND-04）的前提下，只要出現「反彈突破前一個高點」（即頭頭高）或「下跌未創新低就轉漲」（即底底高）任一種現象，即代表空頭方向已出現改變的早期訊號，應先視為進入盤整，做短線的空單要回補。書中短線做空獲利方程式也明確列出「收盤出現底底高，回補」為具體出場條件。
- **可程式化**: 是
- **所需資料**: 轉折波頭/底序列（R-TREND-01輸出）、空頭趨勢判定狀態（R-TREND-04）
- **計算公式**:
```
if is_bear_trend_previously_confirmed:
    if Head[-1] > Head[-2]:          # 反彈突破前高，出現「頭頭高」
        warning = "空頭出現改變（頭頭高），趨勢轉入盤整，短線空單回補"
    elif Bottom[-1] > Bottom[-2]:    # 下跌未破前低就轉漲，出現「底底高」
        warning = "空頭出現改變（底底高），趨勢轉入盤整，短線空單回補"
    else:
        bear_trend_intact = True     # 空頭架構仍維持
```
- **參數**: 無額外可調參數（純比較最近轉折頭/底的高低）
- **可回測**: 是
- **信心**: 91/100 高（書中明確列出2種先知先覺情況，並在短線做空獲利方程式中直接寫明「收盤出現底底高，回補」為具體可執行規則）
