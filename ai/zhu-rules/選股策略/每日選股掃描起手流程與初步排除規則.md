# 每日選股掃描起手流程與初步排除規則

- **Rule ID**: R-SCREEN-16
- **名稱**: 每日選股掃描起手流程與初步排除規則
- **分類**: 選股策略
- **原文與頁碼**: `P10-C5-股市交易工作重點.md`（下冊 p.299，「每日選股掃描的起手流程」）
- **解讀**: 每天收盤後，把當日漲幅≥3.5%的強勢股，以及跌幅≥3.5%的弱勢股全部看過一遍，且至少檢視當日排行前50檔（不只看前幾名）。接著初步排除以下三類股票，不列入候選：
  1. 線形不好、看不出多空方向的股票。
  2. 成交量太小的股票（量小不易大量買進，買進後也不易賣出）。
  3. 股價太低的股票（容易遇到地雷股）。
  - 本規則是 [[特別報價選股法]] 的姊妹流程（同屬收盤後例行掃描），差異在於本規則以「漲跌幅百分比」而非排行榜名次作為起手篩選依據，且排除條件聚焦在流動性與地雷股風險，而非量價絕對值門檻。
- **可程式化**: 部分（漲跌幅門檻、排行檢視範圍可完全程式化；「線形不好看不出方向」屬定性判斷，可用趨勢判定分類規則的「盤整」狀態作為代理；「成交量太小」「股價太低」書中未給精確數字，需自訂或沿用 [[特別報價選股法]] 的1,000張／5元門檻作為參考）
- **所需資料**: 當日漲跌幅、當日排行榜（漲幅/跌幅排序）、成交量、收盤價
- **計算公式**:
```
function daily_scan_and_prefilter(all_stocks, t):
    # 起手篩選：當日漲跌幅絕對值 >= 3.5%
    movers = [s for s in all_stocks if abs(daily_return(s, t)) >= 0.035]

    # 至少檢視排行前50檔（不只看前幾名）
    top_gainers = rank_by_return(all_stocks, t, direction="up")[:50]
    top_losers = rank_by_return(all_stocks, t, direction="down")[:50]
    scan_pool = set(movers) | set(top_gainers) | set(top_losers)

    # 初步排除
    candidates = []
    for s in scan_pool:
        if trend_state(s, t) == "盤整" and not has_clear_directional_bias(s, t):
            continue  # 排除1：線形不明確
        if s.volume < min_volume_threshold:  # 門檻書中未給，可參考特別報價選股法的1,000張
            continue  # 排除2：成交量太小
        if s.close_price < min_price_threshold:  # 門檻書中未給，可參考特別報價選股法的5元
            continue  # 排除3：股價太低
        candidates.append(s)

    return candidates
```
- **參數**: 漲跌幅起手門檻3.5%（書中明確）；排行檢視範圍至少前50檔（書中明確）；成交量與股價門檻本節未給精確數字，可沿用 [[特別報價選股法]] 的1,000張／5元作為預設代理
- **可回測**: 部分（漲跌幅與排行門檻可回測；三項排除條件中僅「線形不明確」需借用趨勢判定規則量化，成交量/股價門檻需自訂）
- **信心**: 75/100 中高（起手篩選的3.5%門檻與前50檔範圍書中明確；三項排除條件為定性描述，精確數字需借用書中其他章節或自訂）
