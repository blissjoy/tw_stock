# 緩步上漲後大量長紅K攻擊位置

- **Rule ID**: R-SCREEN-14
- **名稱**: 緩步上漲後大量長紅K攻擊位置
- **分類**: 選股策略
- **原文與頁碼**: `P10-C4-底部強勢反轉圖形.md`（下冊 p.295-296，「緩步上漲後的攻擊位置」）
- **解讀**: 股價已處於「緩步上漲」趨勢一段時間，特徵是溫和、階梯式墊高，非急漲、也非橫盤。書中原文：「緩步上漲後，突然出現大量長紅K，發動攻擊」。取點方式：觀察緩漲趨勢線／均線多頭排列是否持續，作為判斷目前處於「緩步上漲階段」的依據（圖上以「轉強買進」標示攻擊發動點）。進場觸發條件：緩步上漲過程中，某日突然出現「大量長紅K」，即視為攻擊訊號，觸發條件為當日K棒為長紅實體，且成交量達「爆天量」等級（明顯高於近期均量的異常巨量），可作為緩漲轉強攻擊的量化訊號。圖上另於後續相對高點標示「賣出」箭頭，屬波段操作出場示意，非本節辨識重點，僅供對照參考。
- **可程式化**: 是
- **所需資料**: 日K線OHLCV、成交量、均線組（判斷是否維持多頭排列的緩漲趨勢）
- **計算公式**:
```
function slow_rally_attack_signal(stock, t):
    # 緩步上漲階段：均線多頭排列持續一段時間、漲勢平緩（非急漲）
    slow_rally = count_bullish_aligned_MA(stock, t) >= 3 \
                 and is_gradual_uptrend(stock, t, lookback=20)  # 排除急漲/橫盤，需自訂斜率區間

    if not slow_rally:
        return False

    is_long_red_k = is_long_red_body(stock, t)  # 長紅實體K棒判定，見K棒型態分類規則
    huge_volume = Volume[t] >= 3 * average_volume(stock, t, lookback=20)  # 爆天量，倍數書中未給精確數字

    return is_long_red_k and huge_volume
```
- **參數**: 「爆天量」倍數書中未給精確數字，此處以「近20日均量的3倍以上」作為預設代理門檻（高於一般爆大量的2倍基準，因書中用詞「爆天量」語意強度更高），實作時建議以歷史案例校準；「緩步上漲」的斜率區間書中未量化
- **可回測**: 是（核心「大量長紅K攻擊訊號」可回測；「緩步上漲階段」的判定需先量化斜率/時長門檻）
- **信心**: 74/100 中高（進場觸發邏輯與文字定義明確，2個範例佐證；但「爆天量」與「緩步上漲」的精確量化門檻書中未給數字，需自訂代理參數）
