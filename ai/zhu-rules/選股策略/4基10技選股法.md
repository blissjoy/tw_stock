# 4基10技選股法

- **Rule ID**: R-SCREEN-05
- **名稱**: 4基10技選股法
- **分類**: 選股策略
- **原文與頁碼**: `P10-C3-投機目的技術分析選股策略.md`（下冊 p.269-271，「4基10技選股法」；範例安國8054，圖表10-3-1、10-3-2）
- **解讀**: 在 [[六六大順選股法]] 的技術面架構之外，加入4項基本面／籌碼面參考資料，合計「4基＋10技」共14項檢核，並提供一份可套用的「選股評量表格」。此法適合用於候選股鎖定後的「深度建檔」與最終進場決策確認，而非大範圍篩選工具。
  - **4項基本參考資料**：
    1. 產業類股：選當下市場熱門的產業類股。
    2. 股本：50億元以下的中小型股，較受主力喜愛。
    3. 營收：營收要成長，並查看盈餘獲利及預估全年績效。
    4. 籌碼：三大法人買賣超需為買超方向。
  - **10條技術面篩選條件（均分日線／週線兩層次檢視）**：
    1. 趨勢：多頭／空頭／盤整（日線、週線，見 [[頭頭高底底高多頭趨勢判定]]、[[頭頭低底底低空頭趨勢判定]]、[[盤整趨勢判定與型態]]）。
    2. 位置：打底、反轉、初升、主升、末升、起漲、上漲中、高檔、回檔、支撐、壓力、盤頭、初跌、主跌、末跌、起跌、下跌中、低檔、反彈（日線、週線）。
    3. 均線：3線多排／4線多排／6線多排／3線空排／4線空排／6線空排（日線、週線，見 [[均線多頭排列]]、[[均線空頭排列]]）。
    4. K線：日、週K線的強弱變化。
    5. 成交量：打底量、起漲量、高檔量、遇壓量、頭部量、起跌量、低檔量、支撐量、換手量、騙線量（日線、週線，見 [[成交量分類與倍數門檻定義]]）。
    6. 型態：底部型態、頭部型態、中繼型態（日線、週線）。
    7. 支撐壓力：向上的壓力、向下的支撐（日線、週線，見 [[支撐壓力]] 分類各規則）。
    8. 指標：MACD、KD（日線、週線）。
    9. 背離：量價背離、指標背離（日線、週線）。
    10. 獲利目標與操作定法：短中長線目標、短中長線定法（依日線操作）。
  - 書中實例（安國8054，2023/9/15）：股本9.79億元（符合50億以下）、週線底底高並突破MA20/MA60、日線多頭確認並突破MA20/MA60、週日均線多排向上、週MACD零軸上黃金交叉、外資連3日買超，最終策略為待突破壓力38.3元後進場做多，資金分配20張（10張短線守MA5、10張長線守MA20），長線目標價67.8元（約獲利77%）。此為完整套用14項檢核並產出具體操作策略的範例。
- **可程式化**: 部分（4項基本面條件與10項技術面條件中的均線多排、支撐壓力突破、MACD/KD方向、成交量分類均可程式化；但「位置」「型態」「獲利目標與操作定法」等項目多屬定性研判，需人工介入或搭配其他規則量化）
- **所需資料**: 股本、產業類別、月營收年增率、三大法人買賣超、日/週K線OHLCV、日/週均線組（MA5/10/20/60/120/240）、日/週MACD、日/週KD、支撐壓力價位
- **計算公式**:
```
function si_ji_shi_ji_check(stock, t):
    # 4項基本面
    fund_ok = {
        "股本": stock.capital <= 5_000_000_000,  # 50億元以下
        "營收成長": stock.revenue_yoy(t) > 0,
        "法人籌碼": stock.institutional_net_buy(t) > 0,
        "產業熱度": stock.industry in current_hot_industries()  # 需另建產業強度排名
    }

    # 10項技術面（日線＋週線各檢查一次，此處以單一週期示意）
    tech_ok = {
        "趨勢": trend_state(stock, t, "日") in ["多頭"] and trend_state(stock, t, "週") in ["多頭", "盤整轉多"],
        "均線多排": count_bullish_aligned_MA(stock, t, "日") >= 3,
        "支撐壓力": near_key_support_or_resistance(stock, t),
        "指標": macd_zero_axis_golden_cross(stock, t, "週") or kd_golden_cross(stock, t, "日"),
        "無背離": not check_divergence(stock, t)
    }

    pass_fund = sum(fund_ok.values())
    pass_tech = sum(tech_ok.values())
    return {"基本面符合項數": pass_fund, "技術面符合項數": pass_tech,
            "建議進場": pass_fund >= 3 and pass_tech >= 4}
```
- **參數**: 股本門檻50億元；均線多排分級3/4/6線；其餘位置/型態/背離項目為條件式布林檢核，無額外數值參數
- **可回測**: 部分（股本、營收年增率、均線多排、法人買超等可回測；「位置」「型態」「獲利目標」等定性項目需先量化才能納入回測）
- **信心**: 90/100 高（4項基本面條件與部分技術面條件書中給出明確數字門檻如「50億元以下」；10項技術面條件多數需搭配其他章節規則量化，但條列清楚，安國實例完整佐證整體流程）
