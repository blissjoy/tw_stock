# 淘汰法選股排除規則

- **Rule ID**: R-SCREEN-06
- **名稱**: 淘汰法選股排除規則
- **分類**: 選股策略
- **原文與頁碼**: `P10-C3-投機目的技術分析選股策略.md`（下冊 p.273-279，「淘汰法選股票」）
- **解讀**: 反向思考的選股法：與其找「好股票」的條件，不如先淘汰技術面不好的股票。作為選股流程中的「濾網」，用在候選名單已產生之後，逐條排除高風險／趨勢不明／騙量／基本面技術面背離的個股。書中列出11種應避開做多的狀況，任一項成立即應排除該候選股：
  1. **沒走出底部要避開**：趨勢還沒完成，均線還沒有多排。
  2. **重壓不過、跌破MA5要避開**：短趨勢已完成、均線多排，但向上有重壓未突破，且股價已跌破MA5（範例：力山1515）。
  3. **上漲一波後趨勢不明確要避開**：有重壓、下有支撐，處於區間震盪的股票不可做多（範例：尤克2338）。
  4. **沒有量能要避開**：上漲行進中成交量明顯縮小，或出現量價背離（見 [[成交量分類與倍數門檻定義]]、[[多頭上漲量價背離三型態]]）。
  5. **大幅上漲過高要避開**：股價已上漲達1倍（100%）以上、且呈現盤整趨勢，要立刻出場（範例：虹光2380、技嘉2376）。
  6. **遇壓力大量長黑要避開**：在壓力線附近多次出現大量長黑K（範例：世紀3551，見 [[大量長黑K的二分之一價壓力規則]]）。
  7. **MACD或KD指標背離要避開**：股價呈現頭頭低，且同時出現MACD或KD指標背離，要立刻出場（範例：敬鵬2355，底背離案例）。
  8. **三大法人連續賣超要避開**：下跌時出現爆大量長黑K，或連續3根黑K大量賣壓。
  9. **頻頻爆大量但股價不漲要避開**：多次出現爆量但股價原地不動（範例：標準5414、平新1905）。
  10. **看不懂的股票要避開**：走勢型態、位置、量價關係無法明確判讀時直接放棄（範例：抓寶9950）。
  11. **有基本面、沒有技術面要避開**：即使基本面佳，若技術面未給出明確多頭訊號，仍不應進場（範例：金像電2368）。
  - 書中提醒：把不好的條件都淘汰掉之後，剩下能操作的好股票其實不多，但這正是重點——淘汰法的價值在於提高整體操作勝率，而非追求候選股數量。
  - 正面對照組：安國（8054）週線／日線走勢圖（圖10-3-3、10-3-4）未觸犯上述11條，可作為「淘汰後留下」的範例。
- **可程式化**: 部分（第2、4、5、8、9項可具體量化為布林條件；第1、3、6、7項需搭配其他規則的中間產出（均線多排狀態、指標背離判定）；第10、11項屬於「訊號不明確／人工判讀」性質，難以完全程式化，實作上可用「條件同時滿足度過低」代理判斷）
- **所需資料**: 日K線OHLCV、均線組（MA5/10/20/60/120/240）、成交量、MACD、KD、三大法人買賣超、支撐壓力價位、波段起漲點價格
- **計算公式**:
```
function should_exclude(stock, t):
    reasons = []

    # 1. 沒走出底部
    if count_bullish_aligned_MA(stock, t) < 3:
        reasons.append("未走出底部：均線未多排")

    # 2. 重壓不過、跌破MA5
    if Close[t] < MA5[t] and has_unbroken_resistance_above(stock, t):
        reasons.append("重壓不過且跌破MA5")

    # 3. 上漲一波後趨勢不明確（區間震盪，見盤整判定）
    if trend_state(stock, t) == "盤整" and has_prior_rally(stock, t):
        reasons.append("上漲一波後轉區間震盪，趨勢不明")

    # 4. 沒有量能／量價背離
    if is_volume_shrinking_in_uptrend(stock, t) or check_divergence(stock, t, kind="價量"):
        reasons.append("量能不足或量價背離")

    # 5. 大幅上漲過高後轉盤整
    swing_gain = (Close[t] - swing_low_price(stock, t)) / swing_low_price(stock, t)
    if swing_gain >= 1.0 and trend_state(stock, t) == "盤整":
        reasons.append("波段漲幅達100%以上且轉盤整")

    # 6. 壓力區重複出現大量長黑K
    if count_big_black_k_near_resistance(stock, t, lookback=10) >= 2:
        reasons.append("壓力區重複出現大量長黑K")

    # 7. MACD/KD指標背離
    if make_lower_low(stock, t) and check_divergence(stock, t, kind="指標"):
        reasons.append("價格創低但指標未創低（背離）")

    # 8. 三大法人連續賣超 或 連3黑K爆量
    if institutional_net_sell_streak(stock, t) >= 3 or (consecutive_black_k(stock, t) >= 3 and is_big_vol_MA5(stock, t)):
        reasons.append("三大法人連續賣超或連續大量黑K")

    # 9. 頻頻爆量但股價不漲
    if count_big_vol_days(stock, t, lookback=20) >= 3 and abs(price_change_pct(stock, t, lookback=20)) < 0.03:
        reasons.append("頻繁爆量但股價原地不動")

    # 10. 看不懂（訊號矛盾度代理指標，需人工複核，此處僅為近似）
    if signal_ambiguity_score(stock, t) > threshold_ambiguous:
        reasons.append("走勢/量價訊號不明確，建議人工複核")

    # 11. 有基本面沒技術面
    if stock.fundamentals_good and trend_state(stock, t) != "多頭":
        reasons.append("基本面佳但技術面未給出明確多頭訊號")

    return {"exclude": len(reasons) > 0, "reasons": reasons}
```
- **參數**: 波段漲幅門檻100%（第5項）；連續黑K門檻3根（第8項）；爆量次數門檻與觀察窗（第9項，書中未給精確數字，建議自訂如20日內≥3次）；其餘項目多為布林條件組合，無單一數值參數
- **可回測**: 部分（第2、4、5、8、9項可回測；第10項依賴主觀判讀代理指標，回測結果穩健性較低）
- **信心**: 75/100 中高（11條排除條件書中逐一明確條列並各附反面案例；其中第5項的「100%漲幅」為書中明確數字，其餘多數項目為條件式定性描述，量化門檻需部分自訂，故整體評為中高）
