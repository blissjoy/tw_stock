# 盤中選股設定法

- **Rule ID**: R-SCREEN-08
- **名稱**: 盤中選股設定法
- **分類**: 選股策略
- **原文與頁碼**: `P10-C3-投機目的技術分析選股策略.md`（下冊 p.280，「盤中利用『選股設定』法」）
- **解讀**: 利用看盤軟體（書中示範富邦e01系統）盤中即時選股條件設定功能，捕捉盤中即時強勢訊號，用於盤中即時捕捉當沖或隔日沖題材股的異常訊號，與 [[特別報價選股法]]（收盤後例行篩選）互補——一個是盤中即時掃描，一個是收盤後排行榜複盤。書中列出9種可設定的盤中選股條件：
  1. 爆量股。
  2. 開高強勢指標股。
  3. 即將漲停。
  4. 股價翻紅。
  5. 連續漲停打開。
  6. 連續跌停打開。
  7. 1分鐘內漲幅2%以上。
  8. 連續3筆外盤大量成交。
  9. 收盤前1小時精選次日續漲的機會股。
  - 本節書中僅列出9種條件名稱，未附圖表與個股案例，各條件的精確判定門檻（除第7項「2%」與第8項「3筆」外）未進一步量化。
- **可程式化**: 部分（第7、8項有明確數字可程式化；其餘7項條件名稱明確但「爆量」「開高強勢」「即將漲停」「精選次日續漲」等判定門檻書中未給出具體數字，需自訂或依看盤軟體既有邏輯實作）
- **所需資料**: 盤中即時Tick成交量、即時股價、漲跌停價、開盤價、內外盤成交筆數與量
- **計算公式**:
```
function intraday_screen_signals(tick_data, t):
    signals = {}

    # 1. 爆量股（門檻需自訂，如盤中量能相對過去同時段均量的倍數）
    signals["爆量股"] = intraday_volume(t) >= volume_threshold_intraday  # 門檻書中未給，需自訂

    # 2. 開高強勢指標股（門檻需自訂）
    signals["開高強勢"] = open_price(t) > prev_close(t) * open_strength_threshold  # 門檻書中未給

    # 3. 即將漲停
    signals["即將漲停"] = current_price(t) >= limit_up_price(t) * near_limit_threshold  # 門檻書中未給，常用如99%

    # 4. 股價翻紅
    signals["股價翻紅"] = (current_price(t) > prev_close(t)) and (open_price(t) <= prev_close(t))

    # 5. 連續漲停打開
    signals["連續漲停打開"] = was_limit_up(t-1) and current_price(t) < limit_up_price(t)

    # 6. 連續跌停打開
    signals["連續跌停打開"] = was_limit_down(t-1) and current_price(t) > limit_down_price(t)

    # 7. 1分鐘內漲幅2%以上（書中明確數字）
    signals["1分鐘急漲"] = (current_price(t) - price_1min_ago(t)) / price_1min_ago(t) >= 0.02

    # 8. 連續3筆外盤大量成交（書中明確筆數，量能門檻需自訂）
    signals["連續外盤大量"] = consecutive_ask_side_large_trades(t) >= 3

    # 9. 收盤前1小時精選次日續漲機會股（需綜合前述訊號 + 收盤前時段濾網，門檻書中未給）
    signals["尾盤續漲候選"] = (t.time >= "13:00") and (sum(signals.values()) >= composite_threshold)

    return signals
```
- **參數**: 1分鐘漲幅門檻2%（書中明確）；連續外盤大量成交筆數門檻3筆（書中明確）；其餘各條件（爆量倍數、開高強度、即將漲停距離、尾盤綜合門檻）書中未給數字，需依實務或回測自訂
- **可回測**: 部分（第7、8項可回測；其餘條件需先自訂門檻才能回測，回測結果對自訂參數敏感）
- **信心**: 32/100 低（僅第7、8項有明確數字；其餘7種條件書中僅列名稱，無文字說明判定標準，也無圖表案例佐證）
