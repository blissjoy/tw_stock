# 特別報價選股法

- **Rule ID**: R-SCREEN-07
- **名稱**: 特別報價選股法
- **分類**: 選股策略
- **原文與頁碼**: `P10-C3-投機目的技術分析選股策略.md`（下冊 p.280，「特別報價選股法」）
- **解讀**: 利用看盤軟體（書中示範富邦e01系統）每日收盤後的「特別報價」排行榜作為選股來源，再逐步過濾，適合每日收盤後的例行選股流程，從全市場排行榜快速縮小範圍到可管理的鎖股清單。操作步驟（依序執行）：
  1. 每天收盤查看當日「特別報價」的前100檔強勢股、放大量股，以及弱勢股。
  2. 去除成交量少於1,000張的股票。
  3. 去除股價低於5元的股票。
  4. 先看日線走勢圖，初步挑選符合做多條件的股票。
  5. 注意是否出現同類股群聚上榜的現象（類股輪動訊號）。
  6. 發覺強勢題材股的出現。
  7. 對初步挑選出的日線股票，再看週線仔細研判。
  8. 建立「鎖股資料夾」，並按強弱優先順序排列。
  9. 每日檢視「鎖股資料夾」，做汰換或更新排序。
  - 本法未附個別公司深入案例分析，書中僅提供富邦e01系統排行榜截圖示範（強勢股／弱勢股各前100名）。
- **可程式化**: 部分（步驟1-3的量價門檻可完全程式化；步驟4-9涉及「符合做多條件」「仔細研判」「強弱優先順序」等定性判斷，需串接其他技術面規則如 [[六六大順選股法]] 或趨勢判定分類規則才能量化）
- **所需資料**: 當日漲跌幅排行（前100檔強勢股/放大量股/弱勢股）、當日成交量、當日收盤價、類股分類、日線與週線K線
- **計算公式**:
```
function special_quote_screen(daily_ranking_top100):
    candidates = daily_ranking_top100

    # 步驟2：成交量門檻
    candidates = [s for s in candidates if s.volume >= 1000]  # 單位：張

    # 步驟3：股價門檻
    candidates = [s for s in candidates if s.close_price >= 5]  # 單位：元

    # 步驟4：日線初步篩選（需搭配趨勢/K線規則，如六六大順選股法）
    candidates = [s for s in candidates if daily_chart_bullish_setup(s)]

    # 步驟5：類股群聚偵測
    sector_cluster = detect_sector_clustering(candidates)

    # 步驟7：週線複核
    candidates = [s for s in candidates if weekly_chart_confirms(s)]

    # 步驟8-9：鎖股資料夾維護
    watch_list = rank_by_strength(candidates)
    return watch_list, sector_cluster
```
- **參數**: 成交量門檻1,000張；股價門檻5元
- **可回測**: 部分（量價門檻可回測；日線/週線「符合做多條件」需先定義具體規則才可回測）
- **信心**: 60/100 中（步驟1-3有明確數字門檻；步驟4-9為流程性定性描述，未附個股案例佐證判讀標準）
