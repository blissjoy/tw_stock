# 六六大順選股法

- **Rule ID**: R-SCREEN-04
- **名稱**: 六六大順選股法
- **分類**: 選股策略
- **原文與頁碼**: `P10-C3-投機目的技術分析選股策略.md`（下冊 p.267-269，「六六大順選股法」）
- **解讀**: 以「6大構面」作為選股的總檢核表，逐檔股票依固定順序檢查，用來快速判斷個股目前處於多頭／空頭／盤整的哪個階段、是否具備做多或做空的完整條件。研判順序固定為：①趨勢→②位置→③K線轉折→④均線→⑤成交量→⑥指標。
  1. **趨勢方向**：多頭（頭頭高、底底高，見 [[頭頭高底底高多頭趨勢判定]]）→順勢做多；空頭（頭頭低、底底低，見 [[頭頭低底底低空頭趨勢判定]]）→順勢做空；盤整（見 [[盤整趨勢判定與型態]]）→退出觀察，等突破/跌破。
  2. **當下位置**：多頭段內細分打底／初升段／主升段／末升段等；空頭段內細分做頭／初跌段／主跌段／末跌段等；盤整段內細分開始盤整／盤整型態／盤整末端／盤整突破或跌破。
  3. **K線轉折**：多頭關鍵K線（多頭確認、型態確認、多頭起漲、突破高點、過壓力K線、大量長紅K線），價量線條件為價漲、量增（大於昨日1.3倍）、線實（中長實體紅K收盤確認）；空頭關鍵K線（空頭確認、型態確認、空頭起跌、跌破低點、跌破支撐K線、大量長黑K線），價量線條件為價跌、有量無量均可、線實（中長實體黑K線收盤確認）。盤整突破確認＝價漲、量增、中長實體紅K收盤確認；盤整跌破確認＝價跌、有量無量均可、中長實體黑K收盤確認。
  4. **均線架構**：多頭最少3線多排、方向向上、股價在月線之上（見 [[均線多頭排列]]、[[短多做多成功條件（3線多頭排列）]]）；空頭最少3線空排、方向向下、股價在月線之下（見 [[均線空頭排列]]、[[短空做空成功條件（3線空頭排列）]]）；盤整則均線雜亂無章，退出觀察。
  5. **成交量**：多頭為價漲量增、價跌量縮；空頭通常為價漲量縮、價跌量增（但非必然）；兩者皆須留意起漲量與壓力量的關係、高檔或壓力前爆量「非福」、量能定義須看日後價格走勢回頭確認、並留意價量背離（見 [[成交量分類與倍數門檻定義]]）。
  6. **指標觀察**：觀察高檔或低檔是否出現黃金交叉或死亡交叉（見 [[黃金交叉]]、[[死亡交叉]]），均線多頭排列向上／空頭排列向下，以及指標是否與價格背離。
- **可程式化**: 是（六大構面本身可各自轉為布林條件並組合成綜合評分，但「位置」細分類別多屬定性描述，實作時需簡化為有限狀態集合）
- **所需資料**: 日K線OHLCV、5條以上均線（MA5/10/20/60/120/240）、MACD、KD指標、5日均量
- **計算公式**:
```
function liu_liu_da_shun_score(stock, t):
    score = {}
    # ①趨勢
    score.trend = trend_state(stock, t)  # "多頭"/"空頭"/"盤整"，見趨勢判定分類規則
    if score.trend == "盤整":
        return {"result": "退出觀察", "detail": score}

    # ②位置（簡化為多頭/空頭段內的相對位置分類，實作可用ATR/波段高低點百分位近似）
    score.position = classify_wave_position(stock, t, score.trend)

    # ③K線轉折
    if score.trend == "多頭":
        score.kbar_ok = (Volume[t] > 1.3 * Volume[t-1]) and is_mid_long_red_body(t)
    else:
        score.kbar_ok = is_mid_long_black_body(t)  # 空頭有量無量皆可，量增下殺力道更強

    # ④均線架構
    score.ma_ok = (count_bullish_aligned_MA(t) >= 3 and Close[t] > MA20[t]) if score.trend == "多頭" \
             else (count_bearish_aligned_MA(t) >= 3 and Close[t] < MA20[t])

    # ⑤成交量
    score.volume_ok = check_volume_price_relationship(stock, t, score.trend)  # 含價量背離檢查

    # ⑥指標
    score.indicator_ok = check_golden_death_cross(stock, t) and not check_divergence(stock, t)

    pass_count = sum([score.kbar_ok, score.ma_ok, score.volume_ok, score.indicator_ok])
    return {"result": "候選" if pass_count >= 3 else "觀察", "detail": score}
```
- **參數**: K線量能倍數門檻1.3倍（見 [[成交量分類與倍數門檻定義]] 的攻擊量定義）；均線多排最少3線
- **可回測**: 是
- **信心**: 88/100 高（六大構面與研判順序書中明確條列，關鍵量化門檻如1.3倍量能、3線多排均有明確數字；「位置」細分類別為定性描述，需自行離散化）
