# 做多環境四大前提規則

- **Rule ID**: R-SCREEN-17
- **名稱**: 做多環境四大前提規則
- **分類**: 選股策略
- **原文與頁碼**: `P10-C5-股市交易工作重點.md`（下冊 p.299，「進場做多的前提」）
- **解讀**: 四項條件需同時具備，才符合「做多」的環境前提，屬於「由上而下（Top-down）」的多頭環境確認流程：
  1. 大盤站上月線，呈多頭格局：股價在月線（MA20）之上，且月線本身上揚。
  2. 類股要多頭，且該類股表現要比大盤更強；從所有類股中挑出當前最強勢的前3類股。
  3. 從龍頭股與強勢類股中，找出該類股裡最強勢的個股1～3檔作為做多標的。
  4. 個股要有未來可能持續發酵的題材加持，書中舉例：AI、6G光纖、節能工業、國防工業等（題材強度可作為加分項而非硬性條件）。
  - 本規則與 [[做空環境四大前提規則]] 呈鏡像對稱結構，兩者共用「大盤→類股→個股→題材」四層由上而下的篩選骨架，可作為選股引擎的「環境濾網」，置於個股層級技術面選股法（如 [[六六大順選股法]]、[[4基10技選股法]]）之前執行。
- **可程式化**: 是（前3項可完全量化；第4項題材屬性需仰賴題材資料庫或新聞NLP標記，屬加分項而非必要條件）
- **所需資料**: 大盤指數日K線與MA20、各類股指數日K線與強度排名、個股所屬類股歸屬、個股相對強度排名、題材標籤資料
- **計算公式**:
```
function long_environment_check(market_index, sector_index_list, stock, t):
    # (a) 大盤站上月線且月線上揚
    market_ok = (market_index.close(t) > market_index.MA20(t)) and (MA20_slope(market_index, t) > 0)

    # (b) 個股所屬類股強度排名進入全市場前3名
    sector_rank = rank_sectors_by_strength(sector_index_list, t)
    sector_ok = stock.sector in sector_rank[:3]

    # (c) 個股在該強勢類股中屬於龍頭股或強度前1~3名
    intra_sector_rank = rank_stocks_within_sector(stock.sector, t)
    stock_ok = stock in intra_sector_rank[:3]

    # (d) 題材加分項（非必要條件）
    theme_bonus = stock.theme in current_hot_themes()  # 如AI/6G光纖/節能工業/國防工業等

    long_env_pass = market_ok and sector_ok and stock_ok  # 前3項為必要條件
    return {"做多環境成立": long_env_pass, "題材加分": theme_bonus}
```
- **參數**: 類股強度排名取前3名；個股在類股內強度排名取前1~3檔；MA20方向作為大盤多頭確認依據
- **可回測**: 是
- **信心**: 89/100 高（四項條件書中逐一明確列出，且第1-3項均給出具體排名數字「前3類股」「前1~3檔個股」）
