# 緩漲上升軌道線突破大量長紅K做多

- **Rule ID**: R-SCREEN-15
- **名稱**: 緩漲上升軌道線突破大量長紅K做多
- **分類**: 選股策略
- **原文與頁碼**: `P10-C4-底部強勢反轉圖形.md`（下冊 p.296-297，「緩漲上升突破上升軌道做多」）
- **解讀**: 股價處於「緩漲」型態：漲勢平緩、非急拉。取點方式：沿緩漲過程中股價的相對低點，畫出一條向上傾斜的「上升軌道線」（見 [[上升軌道線畫法（切線平移取中間高點）]]），作為觀察股價是否加速偏離原本緩漲節奏的依據。書中原文：「在緩漲行進過程中，當大量長紅K突破緩漲的上升軌道線，可進場做多」。進場觸發條件：當股價出現大量長紅K，向上突破原本緩漲的上升軌道線（即股價脫離原有軌道傾斜角度、轉為加速上攻）時進場，觸發條件為K棒收盤價突破上升軌道線（非僅盤中觸及，須收盤站上），且當日量能明顯放大。此規則與 [[突破上升軌道線壓力的大量長紅K]] 的K棒定義相通，可視為同一K線型態在「底部強勢反轉圖形」情境下的應用案例。
- **可程式化**: 是
- **所需資料**: 日K線OHLCV、成交量、緩漲段相對低點（用於畫上升軌道線）
- **計算公式**:
```
function slow_rally_channel_breakout(stock, t):
    channel = draw_ascending_channel_line(stock, t)  # 沿緩漲低點畫上升軌道線，見上升軌道線畫法規則
    if channel is None:
        return False

    close_above_channel = Close[t] > channel.upper_line_value(t)  # 收盤價突破軌道線（非僅盤中觸及）
    is_long_red_k = is_long_red_body(stock, t)
    big_volume = Volume[t] >= 2 * average_volume(stock, t, lookback=20)  # 大量，沿用成交量分類倍數門檻

    return close_above_channel and is_long_red_k and big_volume
```
- **參數**: 大量／爆天量倍數書中未給精確數字，沿用 [[成交量分類與倍數門檻定義]] 的2倍基準量作為預設代理；軌道線畫法參數見 [[上升軌道線畫法（切線平移取中間高點）]]
- **可回測**: 是
- **信心**: 88/100 高（辨識條件、取點方式與進場觸發邏輯書中文字明確描述，2個範例附具體低點價位佐證；量能倍數雖未精確量化，但可穩定沿用既有的成交量分類規則）
