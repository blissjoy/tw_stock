# 做空環境四大前提規則

- **Rule ID**: R-SCREEN-18
- **名稱**: 做空環境四大前提規則
- **分類**: 選股策略
- **原文與頁碼**: `P10-C5-股市交易工作重點.md`（下冊 p.299，「進場做空的前提」）
- **解讀**: 與 [[做多環境四大前提規則]] 呈鏡像對稱的四項條件，需同時具備才符合「做空」的環境前提：
  1. 大盤在月線之下，呈空頭格局：股價在月線之下，且月線向下彎。
  2. 類股要空頭，且要挑類股中最弱的前3類股。
  3. 從最弱勢類股中，挑出該類股裡最弱的個股1～3檔作為做空標的。
  4. 鎖定「漲多且領先開始下跌」的個股，或是沒有基本面支撐、純粹靠炒作而超漲後又下跌的股票。
  - 與做多前提相同，本規則屬於「由上而下」環境濾網，應置於個股層級的空頭技術面選股法（見 [[空頭趨勢進場訊號]]）之前執行。
- **可程式化**: 是（前3項可完全量化；第4項需結合「前波漲幅排名」與「率先轉弱」的相對時間判斷，或基本面資料判斷是否為炒作性超漲）
- **所需資料**: 大盤指數日K線與MA20、各類股指數日K線與強度排名、個股所屬類股歸屬、個股相對弱勢排名、個股前波漲幅與轉弱時間點、基本面資料（用於判斷是否無基本面支撐）
- **計算公式**:
```
function short_environment_check(market_index, sector_index_list, stock, t):
    # (a) 大盤在月線之下且月線下彎
    market_ok = (market_index.close(t) < market_index.MA20(t)) and (MA20_slope(market_index, t) < 0)

    # (b) 個股所屬類股強度排名進入全市場最弱後3名
    sector_rank = rank_sectors_by_strength(sector_index_list, t)  # 由強到弱排序
    sector_ok = stock.sector in sector_rank[-3:]

    # (c) 個股在該弱勢類股中屬於最弱勢前1~3名
    intra_sector_rank = rank_stocks_within_sector(stock.sector, t)  # 由強到弱排序
    stock_ok = stock in intra_sector_rank[-3:]

    # (d) 前波漲幅大且領先轉弱，或無基本面支撐的炒作性超漲
    lead_turn_weak = (prior_swing_gain(stock, t) > lead_turn_weak_threshold) and (turned_weak_before_market(stock, t))
    speculative_overextended = (not stock.fundamentals_supported) and (prior_swing_gain(stock, t) > lead_turn_weak_threshold)
    target_bonus = lead_turn_weak or speculative_overextended

    short_env_pass = market_ok and sector_ok and stock_ok
    return {"做空環境成立": short_env_pass, "優先目標加分": target_bonus}
```
- **參數**: 類股強度排名取最弱後3名；個股在類股內弱勢排名取最弱前1~3檔；「前波漲幅大」的量化門檻（lead_turn_weak_threshold）書中未給精確數字，需自訂
- **可回測**: 是（前3項可回測；第4項「前波漲幅大」門檻需自訂後才可完整回測）
- **信心**: 89/100 高（四項條件與做多前提對稱，書中明確列出具體排名數字「最弱前3類股」「最弱1~3檔個股」；僅第4項的「漲多」程度未量化）
