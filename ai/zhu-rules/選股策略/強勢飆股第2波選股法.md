# 強勢飆股第2波選股法

- **Rule ID**: R-SCREEN-10
- **名稱**: 強勢飆股第2波選股法
- **分類**: 選股策略
- **原文與頁碼**: `P10-C3-投機目的技術分析選股策略.md`（下冊 p.289，「『強勢飆股』第2波選股法」）
- **解讀**: 鎖住當下熱門題材、且股價已強勢上漲「第1波」到高檔的股票，等待該股回檔修正後，再進場搶「第2波」的上漲。適合強勢多頭市場中的熱門題材股、已確認第1波漲勢的個股，用於漲勢中段的加碼或轉單進場，而非底部進場。書中定性描述此法「簡單易勝」，但**未附額外數字門檻**（回檔幅度、回檔天數、第2波攻擊量倍數等均未量化）。書中範例（股票代號6405、4190，2024/04/03附近）均呈現「第1波上漲、多頭確認→回檔→第2波攻擊量進場」的走勢型態。
- **可程式化**: 部分（第1波多頭確認、回檔修正、第2波攻擊量進場的邏輯骨架可轉為條件式流程；但「回檔修正」的幅度／天數門檻、「攻擊量」的具體倍數，書中未給數字，需借用其他規則（如 [[成交量分類與倍數門檻定義]] 的攻擊量1.2-1.3倍MA5）或自行以歷史資料校準）
- **所需資料**: 日K線OHLCV、成交量、均線組（MA5/10/20/60）、波段高低點、熱門題材／類股分類
- **計算公式**:
```
function second_wave_screen(stock, t):
    # 第1波：確認已出現一段多頭上漲並到達相對高檔
    wave1_confirmed = trend_state(stock, t, lookback="第1波起漲點") == "多頭" \
                       and is_at_relative_high(stock, t)

    if not wave1_confirmed:
        return None

    # 回檔修正：第1波高點後股價回落（幅度/天數門檻書中未給，需自訂）
    pullback_ok = detect_pullback(stock, t, min_pct=pullback_min_pct, max_pct=pullback_max_pct)

    if not pullback_ok:
        return None

    # 第2波進場：回檔後再度出現攻擊量的紅K，且題材仍熱門
    wave2_entry = is_attack_vol(stock, t) and Close[t] > Close[t-1] \
                  and stock.theme in current_hot_themes()

    return "第2波進場" if wave2_entry else "等待第2波確認"
```
- **參數**: 回檔幅度區間 pullback_min_pct / pullback_max_pct（書中未給數字，需自訂，常見實務參考如回檔至前波漲幅的1/3～1/2，但非書中原文數字）；第2波攻擊量門檻可沿用 [[成交量分類與倍數門檻定義]] 的攻擊量（1.2-1.3倍MA5）
- **可回測**: 部分（骨架可回測，但因回檔幅度/天數門檻未量化，回測結果高度依賴自訂參數，穩健性待驗證）
- **信心**: 30/100 低（書中僅提供定性描述與2檔股票代號範例（因OCR辨識限制，個股中文名稱未能確認），無任何回檔幅度、天數或攻擊量倍數的量化門檻）
