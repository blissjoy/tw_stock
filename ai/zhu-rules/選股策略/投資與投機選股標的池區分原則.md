# 投資與投機選股標的池區分原則

- **Rule ID**: R-SCREEN-01
- **名稱**: 投資與投機選股標的池區分原則
- **分類**: 選股策略
- **原文與頁碼**: `P10-C1-建立對股市的正確認識.md`（下冊 p.261，「投資與投機的差異」第三點「選股標的不同」）
- **解讀**: 操作股票前須先確認「目的」是投資還是投機，兩者的選股母體與方法完全不同：
  1. **投資**：以獲取長期股息／股利為目的，採「長線持有」，以基本面為選股依據（合理價格區間內承接），股票池鎖定「台灣50」「台灣中型100」成分股等基本面佳、未來發展穩定的大型權值股。基本面查核項目見 [[基本面選股六大類查核清單]]。
  2. **投機**：不以領股息為目的，採「長線短做」，在股價上漲階段入場擷取短期利潤，以技術面為選股依據，標的池以中小型股為主，挑選技術線形強勢的個股。技術面選股法見本分類其餘規則（六六大順選股法、4基10技選股法等）。
  - 本規則的意義在於「先分流」：程式化選股引擎啟動前，應先讓使用者或策略設定選擇「投資模式」或「投機模式」，兩模式各自套用不同的母體篩選條件與後續規則集，避免用投機的技術面規則去篩選投資型的權值股母體，或反之。
- **可程式化**: 部分（母體篩選條件本身可程式化，如「是否為台灣50/中型100成分股」「股本規模區分大型股/中小型股」；但「基本面佳」「線形強勢」等具體判準需依賴其他規則，本規則僅提供分流架構）
- **所需資料**: 台灣50／台灣中型100成分股名單、個股股本規模
- **計算公式**:
```
function select_universe(mode):
    if mode == "投資":
        universe = stocks_in(TW50_index) ∪ stocks_in(TW_MidCap100_index)
        # 後續套用基本面查核清單（見 R-SCREEN-03）做進一步篩選
        return universe
    elif mode == "投機":
        universe = all_listed_stocks WHERE market_cap_or_股本 NOT IN 大型權值股區間
        # 後續套用技術面選股法（六六大順、4基10技等，見本分類其餘規則）
        return universe
```
- **參數**: mode（"投資"／"投機"，二選一的策略模式開關）；中小型股的股本區間書中未在本章給出，另見 [[4基10技選股法]] 的「股本50億元以下」門檻
- **可回測**: 否（本規則僅為母體分流架構，非可直接產生買賣訊號的規則，需搭配後續章節規則才可回測）
- **信心**: 60/100 中（分流原則書中明確主張，但「投資母體=台灣50/中型100」「投機母體=中小型股」為定性描述，未給出精確的市值/股本切分數字）
