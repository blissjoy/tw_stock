# 未卜先知選股法

- **Rule ID**: R-SCREEN-09
- **名稱**: 未卜先知選股法
- **分類**: 選股策略
- **原文與頁碼**: `P10-C3-投機目的技術分析選股策略.md`（下冊 p.282-287，「未卜先知選股法」）
- **解讀**: 核心邏輯是股票由空頭反轉多頭的過程中，底部會出現一定的路徑規律；提早辨識這些底部型態變化，就能提早「鎖股」，準備在多頭趨勢正式起漲時進場，適用於長線布局或波段短線操作，屬於相對積極、需承擔較高不確定性的選股方式。書中列出5種底部鎖股觀察樣態，每種皆分「鎖股條件」與「進場觸發條件」兩階段：
  1. **底部盤整鎖股**：趨勢反轉多頭前，常先在低檔出現爆大量後上漲，再次下跌時收盤無法跌破前低支撐線，形成底部盤整→開始鎖股。當大量紅K線突破底部盤整區時：3線多排向上開始做短多，4線多排向上開始規劃做長多（範例：AMAX-KY 6913、遠雄2504）。
  2. **底部2個月均線糾結鎖股做多**：空頭低檔經過2個月不漲不跌、上下小幅度震盪（均線糾結，見 [[均線糾結辨識]]）→開始鎖股。當大量紅K上漲突破盤整區→開始做短多或長多（範例：茂矽2277）。
  3. **空頭反彈突破月線**：空頭強力反彈，突破月線（MA20）壓力，短時間內股價維持在月線之上整理→開始鎖股。當月線上揚、大量紅K上漲突破橫盤或盤整→開始做短多或長多（範例：燦星-DR 912000）。
  4. **空頭反彈突破前高**：空頭低檔大量強力反彈，突破前高→開始鎖股。當第2支腳出現位置在月線之上、出現大量長紅K上漲、月線上揚→開始做短多或長多（範例：雙成3324）。
  5. **出現強勢型態**：空頭低檔經過一段時間走勢，出現強勢底部型態（如頭肩底、圓弧底）→開始鎖股。當大量長紅K上漲突破型態頸線／確認位置→開始做短多或長多（範例：立昌6111、安國1339）。此觀察與 [[型態頸線量測目標價支撐壓力規則]] 的頸線突破邏輯相通。
  - 5種觀察樣態的共同結構：先有「底部路徑訊號」成立鎖股，再等「大量紅K突破關鍵價位」才正式觸發進場，兩階段缺一不可。
- **可程式化**: 部分（各觀察的「進場觸發」條件（大量紅K突破關鍵價位、均線多排分級）可程式化；「鎖股」階段的底部型態辨識（如頭肩底、圓弧底、均線糾結時長）需依賴其他型態辨識規則或人工複核輔助）
- **所需資料**: 日K線OHLCV、成交量、均線組（MA5/10/20/60/120/240）、盤整區間高低點、前波高點、型態頸線價位
- **計算公式**:
```
function weibu_xianzhi_lockstock(stock, t):
    results = {}

    # 觀察一：底部盤整鎖股
    if not broke_prior_low(stock, t) and in_range_consolidation(stock, t):
        lock1 = True
        if big_red_k_breakout(stock, t, level=consolidation_upper(stock, t)):
            results["觀察一"] = "短多" if count_bullish_aligned_MA(stock, t) >= 3 else None
            if count_bullish_aligned_MA(stock, t) >= 4:
                results["觀察一"] = "長多"

    # 觀察二：底部2個月均線糾結
    if ma_tangled(stock, t, min_months=2):  # 見均線糾結辨識
        if big_red_k_breakout(stock, t, level=consolidation_upper(stock, t)):
            results["觀察二"] = "短多或長多"

    # 觀察三：空頭反彈突破月線
    if Close[t] > MA20[t] and consolidating_above_MA20(stock, t):
        if MA20_slope(stock, t) > 0 and big_red_k_breakout(stock, t, level=consolidation_upper(stock, t)):
            results["觀察三"] = "短多或長多"

    # 觀察四：空頭反彈突破前高
    if broke_prior_high(stock, t):
        second_leg_above_ma20 = second_leg_position(stock, t) > MA20[t]
        if second_leg_above_ma20 and big_red_k(stock, t) and MA20_slope(stock, t) > 0:
            results["觀察四"] = "短多或長多"

    # 觀察五：出現強勢型態（頭肩底/圓弧底等）
    pattern = detect_bottom_pattern(stock, t)  # 頭肩底/圓弧底偵測，需另建型態辨識模組
    if pattern is not None:
        if big_red_k_breakout(stock, t, level=pattern.neckline):
            results["觀察五"] = "短多或長多"

    return results
```
- **參數**: 均線糾結最短時長2個月（觀察二明確門檻）；均線多排分級3線=短多、4線=長多（觀察一）；其餘「大量紅K」「大量強力反彈」未給精確倍數，可沿用 [[成交量分類與倍數門檻定義]] 的爆大量（2倍MA5量）作為預設代理門檻
- **可回測**: 部分（有明確門檻的觀察一、二可回測；觀察三、四、五依賴型態辨識與相對位置判斷，量化後可回測，但實作複雜度較高）
- **信心**: 62/100 中（5種觀察各有清楚的兩階段邏輯與個股範例佐證，但「大量」「強力反彈」等量能強度多數未給精確倍數，需借用其他規則的量能分類補齊）
