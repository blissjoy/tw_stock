# 基本面選股六大類查核清單

- **Rule ID**: R-SCREEN-03
- **名稱**: 基本面選股六大類查核清單
- **分類**: 選股策略
- **原文與頁碼**: `P10-C2-投資目的基本面選股條件.md`（下冊 p.263，「基本面選股的6大類參考資料」；案例示範見 p.263-265，鴻海2317）
- **解讀**: 以投資為目的選股時，作者列出6大類應查核的基本面資料維度，性質上是一份「盡職調查（due diligence）欄位清單」，而**非可直接套用的數字篩選公式**：
  1. **公司基本資料**：股本、營業項目、公司董事等。
  2. **公司經營績效**：營業額、獲利率、年增率、淨值、本益比、股東權益報酬率（ROE）等。
  3. **公司財務結構**：財務報表（三表）分析。
  4. **公司長年獲利盈餘分配**：歷年股利分配紀錄。
  5. **公司價值評估**：綜合經營管理、產業發展、獲利情形，研究公司應有的合理價值，判斷現行股價是否合理，並規劃長期投資。
  6. **經營者理念**：了解公司經營者的經營理念。
  - 書中以鴻海（2317）為例展示第1、2類欄位的實際樣貌（如本益比10、股價淨值比1.64、內部持股0.24%、外資持股99.76%等），但這些是「範例展示的個股當時實際數值」，**不是選股門檻**，不可誤植為篩選標準。
  - 書中總結基本面與技術面的分工：**用基本面判斷較長期趨勢（篩選標的池），用技術面判斷短期進出時機**（見選股策略分類其餘規則、以及 [[趨勢判定]]／[[均線]]／[[量價]] 等分類）。
  - 真正帶數字門檻的基本面條件（如股本規模）出現在 [[4基10技選股法]] 的「4項基本參考資料」，不在本規則範圍內。
- **可程式化**: 部分（6大類可轉為「資料完整性檢查」的必查欄位清單，屬ETL/資料模型層面；但因書中未提供各欄位的合格區間數字，無法直接產生布林篩選條件，需另外取得門檻來源才能程式化為真正的篩選規則）
- **所需資料**: 股本、營業項目、董事會結構、營業額、獲利率、年增率、淨值、本益比、ROE、財務三表、歷年股利分配紀錄、產業發展資訊、經營者相關揭露
- **計算公式**:
```
function fundamental_data_completeness_check(stock):
    required_fields = {
        "基本資料": ["股本", "營業項目", "董事會結構"],
        "經營績效": ["營業額", "獲利率", "年增率", "淨值", "本益比", "ROE"],
        "財務結構": ["資產負債表", "損益表", "現金流量表"],
        "獲利盈餘分配": ["歷年股利紀錄"],
        "價值評估": ["合理價值估算"],  # 書中無公式，需另訂估值模型
        "經營者理念": ["經營者相關揭露"]
    }
    missing = [f for group in required_fields.values() for f in group if not stock.has(f)]
    return len(missing) == 0, missing
    # 注意：本函式僅檢查「資料是否齊全」，不做「是否合格」的判斷，
    # 因為書中本節未給出任何欄位的合格門檻數值。
```
- **參數**: 無數值門檻參數（書中本章未提供；如需真正的數字篩選條件，應改用 [[4基10技選股法]] 中的股本門檻，或自行從其他資料源訂定ROE/本益比等合格區間）
- **可回測**: 否（無篩選門檻可回測，僅為欄位盤點清單）
- **信心**: 28/100 低（書中明確僅列出「該查哪些欄位」的分類框架，未提供任何數字及格線；鴻海案例數字為個股實際值而非標準，強行套用會誤導規則庫）
