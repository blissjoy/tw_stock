# 長線波段選股與操作SOP

- **Rule ID**: R-STRATEGY-02
- **名稱**: 長線波段選股與操作SOP
- **分類**: 綜合操作SOP
- **原文與頁碼**: `P11-C2-進場實戰操作SOP.md`（下冊 p.322-323，「長線波段操作守則」，含「多頭長線選股條件」3條與「多頭長線操作SOP」8條）
- **解讀**: 長線波段操作以MA20（月均線）為核心操作依據，同時參考日線與週線走勢，分為選股與操作兩部分：
  - **選股條件（3條）**：①月K出現大量中長紅突破月線下降切線，且反彈站上MA20（可對照 [[突破下降切線分級與壓力支撐角色互換]] 判斷下降切線突破的有效性分級）；②週線呈多頭排列，均線3線多排，底部出現放大量；③日線呈多頭，均線4線多排（見 [[均線多頭排列]]），底部經多頭確認後才開始起漲（可引用 [[頭頭高底底高多頭趨勢判定]] 作為「多頭確認」的判定依據）。
  - **操作SOP（8條）**：進場、停損、頭頭低出場、續抱邏輯與短線波段（見 [[短線波段進出場20條守則]]）第1-4條完全相同（收盤突破MA5+前高+2%紅K進場、進場價5%停損、頭頭低出場、未破停損續抱）；差異在於停利依據——長線波段獲利超過10%後，改以「收盤跌破MA20」（而非短線的MA5）作為停利出場依據，容忍回檔幅度更大；獲利超過20%且連續急漲後出現大量長黑K跌破前一日低點即出場；出場後若股價自起漲點累計漲幅已超過50%，且再次符合日線進場條件，可重新進場操作長線第2波，並重複第2～7條操作邏輯；股價自起漲點累計上漲約達1倍（100%）之後，不再進行長線操作（不再新進場／加碼）。
- **可程式化**: 部分（進場、停損、停利百分比與MA20停利切換條件皆可程式化；「底部經多頭確認」「反彈站上MA20」等定性條件需依賴 [[頭頭高底底高多頭趨勢判定]]、[[均線多頭排列]] 等既有規則模組的輸出）
- **所需資料**: 日線／週線／月線收盤價、開盤價、最高價、最低價、MA5、MA20、成交量、下降切線位置、多頭趨勢判定結果（見 [[頭頭高底底高多頭趨勢判定]]）、均線多排判定結果（見 [[均線多頭排列]]）
- **計算公式**:
```
# 選股條件（進場前置檢核，三者需同時或依權重綜合成立）
long_term_candidate = (
    月K出現大量中長紅突破月線下降切線 and Close_monthly[t] > MA20_monthly[t]
    and 週線3線多排 and 週線底部曾出現放大量
    and 日線4線多排(見 [[均線多頭排列]]) and 底部已由 is_bull_trend 確認後才開始起漲
)

# 操作SOP（與短線波段第1-4條相同，見 [[短線波段進出場20條守則]]）
entry_ready = (
    Close[t] > MA5[t] and Close[t] > High[t-1]
    and (Close[t]-Open[t]) / Open[t] >= 0.02
)
if entry_ready and long_term_candidate:
    entry_price = Close[t]
    stop_loss = entry_price * 0.95
    launch_point = entry_price

profit_pct = (Close[t] - entry_price) / entry_price

if Close[t] < stop_loss:
    action = "跌破停損，出場"
elif 收盤出現頭頭低:
    action = "頭頭低，出場"
elif profit_pct >= 0.20 and 連續3天以上長紅或連續急漲 and 出現大量長黑K and Close[t] < Low[t-1]:
    action = "全部出場"
elif profit_pct >= 0.10 and Close[t] < MA20[t]:                     # 長線用MA20，非短線的MA5
    action = "獲利≥10%跌破MA20，停利出場"
else:
    action = "續抱"

# 出場後的第2波重新進場判斷
if 已出場 and (Close[t] - launch_point) / launch_point > 0.50 and entry_ready:
    action = "重新進場做長線第2波，重複第2～7條邏輯"

# 長線操作上限
if (Close[t] - launch_point) / launch_point >= 1.00:
    action = "累計漲幅達1倍，不再進行長線新進場/加碼"
```
- **參數**: 停損比例＝5%（與短線相同）；停利切換門檻＝獲利10%（切換為跌破MA20，短線波段則是跌破MA5，兩者出場均線標的不同）；大量長黑出場門檻＝獲利20%；第2波重新進場門檻＝出場後反彈幅度>50%；長線操作上限＝累計漲幅約1倍（100%）後不再新進場（皆為書中明確數字）
- **可回測**: 是
- **信心**: 74/100 中高（進場停損停利數字明確、可直接複用短線波段架構＝高；但選股條件中「底部經多頭確認」「反彈站上月線」等定性描述需組合多個既有規則模組共同判定，且第2波重新進場與1倍上限屬於較粗略的經驗法則，故整體評為中高）
