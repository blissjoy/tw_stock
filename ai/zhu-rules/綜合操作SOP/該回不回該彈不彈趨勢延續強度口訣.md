# 該回不回該彈不彈趨勢延續強度口訣

- **Rule ID**: R-STRATEGY-06
- **名稱**: 該回不回該彈不彈趨勢延續強度口訣（12口訣之5、6）
- **分類**: 綜合操作SOP
- **原文與頁碼**: `P11-C3-12個操作口訣圖形練功.md`（下冊 p.335-338）。口訣5「多頭該回不回，過高要大漲」（p.335-336，圖表11-3-13～11-3-16共4例）；口訣6「空頭該彈不彈，破低要大跌」（p.337-338，圖表11-3-17～11-3-20共4例，為口訣5的空方鏡射版本）
- **解讀**: 口訣5與口訣6為多空鏡射的一組「趨勢延續強度驗證」口訣：
  - **口訣5（多頭該回不回）**：股價處於上升趨勢，依技術面（如漲多、觸及壓力）理應出現回檔整理，但股價遲遲不回檔，顯示多方力道極強、賣壓輕微，隨後股價直接創新高並出現顯著漲勢（過高大漲）。型態常以階梯狀反覆出現：「該回不回→過高大漲→再度該回不回→再過高大漲」，可作為多頭趨勢強度的持續驗證工具，**每一次「該回不回」都是加碼或續抱的訊號，而非停利訊號**。
  - **口訣6（空頭該彈不彈）**：股價處於下降趨勢，依技術面理應出現反彈，但股價遲遲不反彈，顯示空方力道極強，隨後股價跌破前低出現顯著跌勢（破低大跌），同樣以階梯狀反覆出現。
  - **與其他規則的區別提醒**：此口訣描述的是「已持有部位在強勢延續過程中的續抱/加碼判斷」，與 [[做多進場10大戒律]] 中「連續上漲第3根以上追高不做多」的**新進場過濾**是不同情境——前者針對既有持股續抱決策，後者針對新單進場決策，兩者判斷主體不同，不可混淆套用。
  - 書中範例11-3-16（代碼3377）示範口訣1（大量不漲）與口訣5（該回不回）可能在同一走勢的不同階段先後出現，實務判讀順序建議先觀察是否僅為口訣1的短暫回檔，若確認遲遲不回檔則升級為口訣5的強勢延續判讀。
- **可程式化**: 部分（趨勢前提與「過高大漲/破低大跌」結果的方向判斷可程式化；但「該回」的觸發時機（理應回檔的判斷基準，例如上漲天數、接近壓力的距離）與「不回/不彈」的容忍觀察期間，書中僅定性描述、未給精確數字，需自訂）
- **所需資料**: 收盤價、最高價、最低價、上升/下降趨勢判定結果（見 [[頭頭高底底高多頭趨勢判定]]、[[頭頭低底底低空頭趨勢判定]]）、前波高點/低點（見 [[轉折波定義與取點演算法]]）
- **計算公式**:
```
# 口訣5：多頭該回不回，過高要大漲
def signal_5_bull_no_pullback(t):
    if not is_bull_trend:
        return None
    should_pullback = 連續上漲天數(t) >= PULLBACK_EXPECT_DAYS or 接近前高壓力(t)   # 觸發基準需自訂，書中未給精確天數/距離
    if should_pullback:
        no_pullback_observed = all(Close[t+i] >= Close[t] for i in range(1, NO_PULLBACK_WINDOW))  # 觀察期需自訂
        if no_pullback_observed and Close[t+NO_PULLBACK_WINDOW] > 前高:
            return "該回不回，過高要大漲，可續抱或加碼（非停利訊號）"

# 口訣6：空頭該彈不彈，破低要大跌（口訣5的鏡射版本）
def signal_6_bear_no_rebound(t):
    if not is_bear_trend:
        return None
    should_rebound = 連續下跌天數(t) >= REBOUND_EXPECT_DAYS or 接近前低支撐(t)
    if should_rebound:
        no_rebound_observed = all(Close[t+i] <= Close[t] for i in range(1, NO_REBOUND_WINDOW))
        if no_rebound_observed and Close[t+NO_REBOUND_WINDOW] < 前低:
            return "該彈不彈，破低要大跌"
```
- **參數**: 書中未定義「該回/該彈」的精確觸發條件（連續上漲/下跌天數門檻、接近壓力/支撐的距離）與「不回/不彈」的容忍觀察天數，屬定性判讀口訣，程式化時須自訂上述門檻（可參考 [[上升切線支撐與力道遞減規則]]／[[下降切線壓力與力道遞減規則]] 的角度強弱判斷作為輔助基準）
- **可回測**: 部分（方向與結果驗證可回測；「該回/該彈」觸發時機與容忍期間需先自訂參數）
- **信心**: 60/100 中（多空鏡射邏輯清楚、範例合計8例、可與口訣1/2構成同一走勢不同階段的判讀順序，實務脈絡完整；但核心觸發時機與容忍期間書中未給精確數字，需自訂參數，故評為中）
