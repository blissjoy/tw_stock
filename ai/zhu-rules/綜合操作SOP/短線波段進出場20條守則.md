# 短線波段進出場20條守則

- **Rule ID**: R-STRATEGY-01
- **名稱**: 短線波段進出場20條守則
- **分類**: 綜合操作SOP
- **原文與頁碼**: `P11-C2-進場實戰操作SOP.md`（下冊 p.320-322，「短線波段操作守則」20條，含開頭總則p.320-321之停損5%基準）
- **解讀**: 作者明言這20條是「全書操作紀律的總彙整」，是短線波段做多從進場到出場的完整檢核清單。可歸納為7組邏輯：
  1. **進場**（第1條）：收盤突破MA5、突破前一日K線高點、當日漲幅≥2%紅K實體棒、MA20向上、KD的K值向上、成交量增加或與均量相當皆可。前提須先符合多頭架構，可引用 [[頭頭高底底高多頭趨勢判定]] 與 [[均線多頭排列]] 作為進場前置條件。
  2. **停損**（第2、4條）：停損設在進場價之下5%，以收盤價（非盤中）跌破才觸發；只要未跌破停損價即續抱。
  3. **趨勢反轉出場**（第3條）：收盤出現「頭頭低」代表趨勢改變即出場——此處僅取「頭頭低」單一特徵作提前示警，與 [[頭頭低底底低空頭趨勢判定]] 要求「頭頭低＋底底低同時成立」才判定完整空頭架構不同，屬更寬鬆、更早期的出場觸發條件，須注意兩者判定邏輯的差異。
  4. **停利基準隨獲利分級切換**（第5、6、7、8條）：獲利<10%時跌破MA5不出場；獲利≥10%後改以「收盤跌破MA5」為停利出場依據；獲利≥20%（或連續3天以上長紅K急漲）後，若出現大量長黑K但未跌破前一日K線低點，先減碼二分之一，次日開低走低則出清剩餘部位；若當日大量長黑K已收盤跌破前一日低點，直接全部出場。
  5. **進場/加碼位置的K棒與量能過濾**（第9、10、11、12、13、14條）：大量紅K突破MA5但K值仍向下時應觀望，待K值轉向上再買進；進場當根紅K若上影線超過實體二分之一，暫緩追進，改看次日再決定；遇壓後黑K若跌破前一日紅K實體二分之一（「強覆蓋」，可對照 [[高檔長黑覆蓋]] 的強弱覆蓋分級）且K值下彎，可先賣出二分之一部位；高檔爆量長上影線視為壓力，若新進場點與原高點差距未達5%須等突破上影線高點才可再買進，差距達10%且未跌破月線則仍可回後買上漲、但未過前高即見大量黑K須立刻出場；高檔連續2日爆量後回檔已連3根黑K，則暫停回後買上漲操作。
  6. **持股中的容忍區間**（第15、16條）：黑K跌破MA5但跌幅<1%、量縮、MA20仍向上時當日續抱，次日續跌才賣；橫盤整理期間跌破MA5仍續抱，須跌破整個橫盤區間才賣。
  7. **加碼與週線/季線配合**（第17、18、19、20條）：低檔起漲後遇橫盤（5%以內）或小跌（未觸停損），量增紅K可加碼，加碼單各自依紀律設停損停利；波段漲幅≥10%後不宜再加碼；週線接近壓力且日線黑K跌破MA5，多單先出場；底部反轉、多頭月線上揚做多、股價上漲突破季線，但若季線本身向下，一旦回檔跌破MA5即出場，不受10%漲幅門檻保護（即季線走弱時出場基準提前收緊）。
- **可程式化**: 部分（進場價量條件、停損停利百分比、K棒上影線比例、強覆蓋比例、加碼與週線配合的數字條件皆可程式化；KD的K值方向、週線壓力位置判讀需搭配技術指標與支撐壓力分類規則庫共同運作）
- **所需資料**: 收盤價、開盤價、最高價、最低價、MA5、MA20、MA60（季線）、成交量、5日均量、KD指標K值、週線資料、頭頭低判定結果（見 [[頭頭低底底低空頭趨勢判定]]）
- **計算公式**:
```
# 進場
entry_ready = (
    is_bull_trend                                        # 見 [[頭頭高底底高多頭趨勢判定]]
    and Close[t] > MA5[t] and Close[t] > High[t-1]
    and (Close[t]-Open[t]) / Open[t] >= 0.02              # 2%以上紅K實體
    and slope(MA20) > 0
    and slope(KD_K) > 0
    and (Volume[t] >= Volume_avg[t] or Volume[t] > Volume[t-1])
)
if entry_ready:
    entry_price = Close[t]
    stop_loss = entry_price * 0.95

# 持倉期間逐日狀態機
profit_pct = (Close[t] - entry_price) / entry_price

if Close[t] < stop_loss:
    action = "跌破停損，出場"
elif 收盤出現頭頭低:                                       # 早期示警，非完整空頭架構
    action = "頭頭低，趨勢改變出場"
elif profit_pct >= 0.20 and 出現大量長黑K:
    if Close[t] < Low[t-1]:
        action = "跌破前一日低點，全部出場"
    else:
        action = "減碼二分之一"
        if 次日開低走低:
            action = "出清剩餘部位"
elif profit_pct >= 0.10 and Close[t] < MA5[t]:
    action = "獲利≥10%跌破MA5，停利出場"
elif profit_pct < 0.10 and Close[t] < MA5[t]:
    if (MA5[t] - Close[t]) / MA5[t] < 0.01 and 量縮 and slope(MA20) > 0:
        action = "跌幅<1%且量縮，續抱觀察次日"
    else:
        action = "續抱（未達10%不因跌破MA5出場）"
else:
    action = "續抱"

# 加碼判斷（另一條獨立分支，可與上方出場邏輯並行檢查）
can_add_position = (
    profit_pct < 0.10                                    # 波段漲幅≥10%後不宜加碼
    and (橫盤整理 and 累計漲幅 <= 0.05 or 小跌未觸及停損)
    and 量增 and 是紅K
)
# 加碼單各自比照上述同一套停損停利邏輯獨立管理

# 季線走弱時的加嚴出場（第20條，覆蓋第5條的10%保護）
if 底部反轉多頭月線上揚做多 and Close[t] > MA60[t] and slope(MA60) < 0:
    if Close[t] < MA5[t]:
        action = "季線向下，跌破MA5即出場（不受10%漲幅保護）"
```
- **參數**: 停損比例＝5%（進場價）；停利基準切換門檻＝獲利10%（切換為跌破MA5出場）；大量長黑減碼門檻＝獲利20%或連續3天以上急漲；上影線暫緩進場門檻＝上影線＞實體二分之一；強覆蓋比例＝跌破前一日紅K實體二分之一；壓力測試差距門檻＝5%（須等突破上影線高點）／10%（可回後買上漲）；回後買上漲暫停條件＝高檔連2日爆量＋回檔連3黑；小跌容忍＝跌幅<1%且量縮；加碼容忍橫盤幅度＝5%以內；停止加碼門檻＝波段漲幅≥10%（皆為書中明確數字）
- **可回測**: 是
- **信心**: 90/100 高（20條中多數皆有明確百分比、根數或天數條件，可直接轉換為量化規則；僅KD的K值方向與週線壓力位置屬於需搭配指標/支撐壓力模組的定性判斷，故「可程式化」評為部分而非全部，但不影響整體信心）
