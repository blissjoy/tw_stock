# 多頭大量不漲與空頭大量不跌回檔反彈口訣

- **Rule ID**: R-STRATEGY-04
- **名稱**: 多頭大量不漲與空頭大量不跌回檔反彈口訣（12口訣之1、2）
- **分類**: 綜合操作SOP
- **原文與頁碼**: `P11-C3-12個操作口訣圖形練功.md`（下冊 p.329-333）。口訣1「多頭大量不漲，股價要回檔（當日或後數日）」（p.329，圖表11-3-1～11-3-4共4例）；口訣2「空頭大量不跌，股價要反彈（當日或後數日）」（p.331-333，圖表11-3-5～11-3-10共6例，為口訣2的空方鏡射版本）
- **解讀**: 口訣1與口訣2為多空鏡射的一組量價背離判讀口訣，是全書「口訣總複習」中量價背離型回檔/反彈過濾的核心：
  - **口訣1（多頭大量不漲）**：股價處於多頭上升趨勢中（見 [[頭頭高底底高多頭趨勢判定]]），某交易日成交量明顯放大（見 [[成交量分類與倍數門檻定義]]）但股價漲幅有限或收黑，代表上方賣壓未消化，股價會在當日或後數日出現回檔。此型態可在盤整區反覆測試前高時多次出現，也可能出現在波段最高點成為頭部反轉警訊，須搭配所在位置（盤整中段 vs 波段高點）判斷是短暫回檔還是趨勢反轉。
  - **口訣2（空頭大量不跌）**：股價處於空頭下降趨勢中（見 [[頭頭低底底低空頭趨勢判定]]），某交易日成交量放大但股價跌幅有限或收紅，代表買盤進場承接，股價會在當日或後數日出現反彈。若「大量不跌」在同一走勢反覆出現且最終伴隨「多頭確認」標記，代表下跌動能持續衰竭，趨勢可能轉為反轉向上。
  - 兩者皆屬於此規則庫中「量價背離型回檔/反彈過濾」訊號，可作為判斷波段途中爆量不漲/不跌只是正常拉回或反彈、不必因此提早出場或提早停損的過濾器，用於嵌入短線波段守則（見 [[短線波段進出場20條守則]]）與飆股續抱判斷（見 [[飆股選股與技術面操作守則]]）的執行節點。
- **可程式化**: 部分（趨勢前提、成交量放大皆可程式化；「不漲/不跌」與「回檔/反彈」的精確幅度、以及是否演變為趨勢反轉頭部/底部訊號，書中僅定性描述、未給精確數字門檻，需自訂）
- **所需資料**: 收盤價、開盤價、成交量、5日均量、多頭/空頭趨勢判定結果（見 [[頭頭高底底高多頭趨勢判定]]、[[頭頭低底底低空頭趨勢判定]]）
- **計算公式**:
```
# 口訣1：多頭大量不漲
def signal_1_bull_high_vol_no_rise(t):
    if not is_bull_trend:                                  # 見 [[頭頭高底底高多頭趨勢判定]]
        return None
    is_big_vol = is_attack_vol[t] or is_big_vol_MA5[t]      # 見 [[成交量分類與倍數門檻定義]]
    is_no_rise = (Close[t] - Close[t-1]) / Close[t-1] < NO_RISE_THRESHOLD or 收黑(t)   # 需自訂門檻，書中未給精確%
    if is_big_vol and is_no_rise:
        signal = "大量不漲，預期當日或後數日回檔"
        if t 位於波段最高點附近:
            signal += "；若於波段最高點出現，須留意頭部反轉警訊"
        return signal

# 口訣2：空頭大量不跌（口訣1的鏡射版本）
def signal_2_bear_high_vol_no_fall(t):
    if not is_bear_trend:                                  # 見 [[頭頭低底底低空頭趨勢判定]]
        return None
    is_big_vol = is_attack_vol[t] or is_big_vol_MA5[t]
    is_no_fall = (Close[t] - Close[t-1]) / Close[t-1] > -NO_FALL_THRESHOLD or 收紅(t)  # 需自訂門檻
    if is_big_vol and is_no_fall:
        signal = "大量不跌，預期當日或後數日反彈"
        if 同一走勢中重複出現 and 最終出現多頭確認:
            signal += "；下跌動能衰竭，可能轉為打底反轉"
        return signal
```
- **參數**: 大量放大門檻沿用 [[成交量分類與倍數門檻定義]]（攻擊量1.2~1.3倍MA5、爆大量2倍以上MA5）；「不漲/不跌」判定門檻（NO_RISE_THRESHOLD／NO_FALL_THRESHOLD）書中未給精確數字，需自訂（建議可先以收黑/收紅或漲跌幅<2%作為預設）
- **可回測**: 部分（量能放大條件可回測；不漲/不跌與回檔/反彈的精確幅度需先自訂門檻才能精確回測）
- **信心**: 62/100 中（多空鏡射邏輯與趨勢前提描述清楚，範例合計10例、為書中12口訣中範例數量較多的一組，顯示實務可靠度不低；但精確量能倍數與「不漲/不跌」判定閾值書中未明文給出數字，需自訂參數，故評為中）
