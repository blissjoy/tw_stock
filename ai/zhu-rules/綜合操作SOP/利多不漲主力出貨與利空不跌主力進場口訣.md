# 利多不漲主力出貨與利空不跌主力進場口訣

- **Rule ID**: R-STRATEGY-05
- **名稱**: 利多不漲主力出貨與利空不跌主力進場口訣（12口訣之3、4）
- **分類**: 綜合操作SOP
- **原文與頁碼**: `P11-C3-12個操作口訣圖形練功.md`（下冊 p.334）。口訣3「多頭利多不漲，主力出貨做頭」（圖表11-3-11～11-3-12共2例）；口訣4「空頭利空不跌，主力進場築底」——**書中僅以文字與口訣3並列於同一頁，並未提供口訣4專屬編號圖例**，圖表11-3-11、11-3-12皆標註為口訣3的圖形範例；口訣4可視為與口訣3共用同一組判讀邏輯的鏡射情境，實務上可套用口訣3的範例圖形，將多空方向對調理解。
- **解讀**: 口訣3與口訣4為多空鏡射的一組「消息面與價格背離」判讀口訣：
  - **口訣3（多頭利多不漲）**：個股處於多頭高檔區間，傳出利多消息，但股價公布後未能同步上漲（利多不漲），代表主力正藉利多出貨（利用市場樂觀情緒出脫持股）而非續抱做多，隨後型態轉為「空頭確認」（見 [[頭頭低底底低空頭趨勢判定]]），趨勢由多轉空形成頭部。書中範例（代碼1310、1312）皆在高點標示「利多不漲」，緊接出現「空頭確認」轉折線。
  - **口訣4（空頭利空不跌）**：個股處於空頭低檔區間，傳出利空消息，但股價公布後未能同步下跌（利空不跌），代表主力正藉利空進場低接（利用市場悲觀情緒吸籌）而非續殺出場，股價於低檔完成築底，為後續反轉向上鋪墊。**因書中無獨立圖例，此判讀邏輯為依口訣3文字定義推論的鏡射版本，程式化時信心應低於有明確圖例佐證的口訣3。**
  - 兩者核心邏輯相同：K棒/股價對消息面的「反常無反應」，代表主力正在利用消息面情緒進行反向操作（利多出貨／利空進貨），須以「消息公布後股價的實際反應」而非消息本身的多空屬性作為判讀依據。
- **可程式化**: 部分（「股價未同步上漲/下跌」的技術面判定可程式化；但「利多/利空消息」的判定與時間戳屬於基本面/消息面資料，不在本規則庫的量價技術指標範圍內，需額外接入新聞或公告資料源作為外部輸入才能完整程式化）
- **所需資料**: 收盤價、開盤價、成交量、利多/利空消息事件時間戳（外部消息面資料，需另行接入）、空頭/多頭趨勢反轉判定結果（見 [[頭頭低底底低空頭趨勢判定]]、[[頭頭高底底高多頭趨勢判定]]）
- **計算公式**:
```
# 口訣3：多頭利多不漲，主力出貨做頭
def signal_3_bull_good_news_no_rise(t, news_flag):
    # news_flag 為外部輸入：t 日是否公布利多消息（需接入消息面資料源，書中未給量化定義）
    if not (t 位於多頭高檔區間 and news_flag == "利多"):
        return None
    is_no_rise = Close[t] <= Close[t-1]                       # 利多公布後股價未同步上漲
    if is_no_rise:
        signal = "利多不漲，疑似主力出貨"
        if 隨後出現空頭確認:                                    # 見 [[頭頭低底底低空頭趨勢判定]]
            signal += "，確認做頭反轉"
        return signal

# 口訣4：空頭利空不跌，主力進場築底（口訣3的鏡射版本，書中無獨立圖例佐證）
def signal_4_bear_bad_news_no_fall(t, news_flag):
    if not (t 位於空頭低檔區間 and news_flag == "利空"):
        return None
    is_no_fall = Close[t] >= Close[t-1]                       # 利空公布後股價未同步下跌
    if is_no_fall:
        signal = "利空不跌，疑似主力進場築底（鏡射推論，書中無獨立圖例）"
        return signal
```
- **參數**: 書中無明確數字門檻，純文字定性描述，無精確百分比或天數條件；`news_flag` 消息面輸入需另行定義來源與判定標準
- **可回測**: 否（需要外部消息面時間戳資料，書中未提供量化判斷公式，回測前須自行接入新聞/公告資料庫並定義「利多/利空」的判定標準）
- **信心**: 18/100 低（口訣3有2個圖例佐證但仍只是純文字定性描述，無精確數字；口訣4完全無獨立圖例、屬鏡射推論；且兩者皆依賴書中未涵蓋的外部消息面資料，是全書12口訣中程式化難度最高、信心最低的一組，來源筆記亦將其列為次優先開發項目）
