# 關前爆量與高低檔久盤突破跌破口訣

- **Rule ID**: R-STRATEGY-09
- **名稱**: 關前爆量與高低檔久盤突破跌破口訣（12口訣之11、12）
- **分類**: 綜合操作SOP
- **原文與頁碼**: `P11-C3-12個操作口訣圖形練功.md`（下冊 p.346-350）。口訣11「關前放大量，股價不漲要回檔」（p.346-348，圖表11-3-35～11-3-40共6例，為12口訣中範例數量最多的一組）；口訣12「上漲高檔久盤必跌，下跌低檔久盤必漲」（p.349-350，圖表11-3-41～11-3-44共4例）
- **解讀**: 口訣11與口訣12是判斷停利/停損最後時機的把關口訣：
  - **口訣11（關前放大量，不漲要回檔）**：股價接近前面某個關鍵壓力價位（前高，「關前」，見 [[轉折高點壓力與突破後轉支撐規則]]），股價於接近前高處出現爆大量（「關前爆量」，見 [[成交量分類與倍數門檻定義]]）但未能有效上漲（「大量不漲」），代表上方套牢籌碼/賣壓仍重、尚未消化完畢，訊號成立後股價會回檔整理。此型態常在同一走勢中重複出現於多次測試同一前高的時點，須耐心等待爆量後股價真正站穩並突破，才視為壓力解除；若反覆測試後始終無法放量突破，最終可能轉為「空頭確認」、「大跌」，即壓力關卡久攻不下轉為反轉向下的頭部型態。此型態與口訣1（多頭大量不漲）邏輯相近，差異在於口訣11特別強調「位置」須是已知的前高關卡，而非任意位置的爆量不漲。
  - **口訣12（高檔久盤必跌，低檔久盤必漲）**：股價於相對高檔區間長時間橫盤整理（久盤，可搭配 [[盤整趨勢判定與型態]] 或 [[密集盤整區區間支撐壓力規則]] 辨識整理區間）卻未能向上突破，整理時間越久代表多方力道逐漸衰竭，最終容易向下跌破整理區間並出現顯著跌勢；股價於相對低檔區間長時間橫盤整理，最終容易向上突破整理區間並出現顯著漲勢。核心判讀邏輯：盤整持續時間越長，代表多空籌碼交換消化越徹底，一旦整理區間的上緣或下緣被有效突破/跌破，累積能量容易加大釋放，形成較大幅度的後續走勢，而非小幅假突破。書中範例11-3-44（代碼1742）同一張圖完整示範口訣12的多空兩種情境前後接續發生（低檔久盤突破大漲後轉為高檔久盤再度跌破大跌）。
  - 兩口訣皆屬本規則庫中「停利/停損時機的最後把關」訊號，且範例數量在12口訣中並列或接近最多，顯示這兩個型態在實務中出現頻率最高、最值得優先掌握。
- **可程式化**: 部分（前高關卡位置、成交量放大條件、久盤區間可分別引用既有支撐壓力與盤整判定規則程式化；「不漲」與「久盤」的精確幅度與天數門檻，書中未給精確數字，需自訂）
- **所需資料**: 收盤價、最高價、最低價、成交量、前高關卡位置（見 [[轉折高點壓力與突破後轉支撐規則]]）、盤整區間辨識結果（見 [[盤整趨勢判定與型態]]、[[密集盤整區區間支撐壓力規則]]）
- **計算公式**:
```
# 口訣11：關前放大量，不漲要回檔
def signal_11_resistance_high_vol_no_rise(t, 前高):
    近前高 = abs(Close[t] - 前高) / 前高 <= NEAR_RESISTANCE_PCT     # 門檻需自訂，書中未給精確%
    is_big_vol = is_attack_vol[t] or is_big_vol_MA5[t]              # 見 [[成交量分類與倍數門檻定義]]
    is_no_rise = Close[t] < 前高                                    # 未有效突破前高
    if 近前高 and is_big_vol and is_no_rise:
        signal = "關前爆量不漲，先回檔"
        if 反覆測試前高多次仍未放量突破:
            signal = "壓力久攻不下，轉為空頭確認、大跌"
        return signal

# 口訣12：高檔久盤必跌，低檔久盤必漲
def signal_12_long_consolidation_breakout(t):
    整理區間 = 盤整區間辨識(t)                                       # 見 [[盤整趨勢判定與型態]]
    整理天數 = 整理區間.天數
    is_long_enough = 整理天數 >= LONG_CONSOLIDATION_DAYS             # 天數門檻需自訂，書中未給精確天數
    if not is_long_enough:
        return None
    if 整理區間.位置 == "高檔" and Close[t] < 整理區間.下緣:
        return "跌破久盤，預期大跌"
    if 整理區間.位置 == "低檔" and Close[t] > 整理區間.上緣:
        return "突破久盤，預期大漲"
```
- **參數**: 「關前」距離門檻（NEAR_RESISTANCE_PCT）與「久盤」天數門檻（LONG_CONSOLIDATION_DAYS）書中未給精確數字，屬定性判讀口訣，程式化時需自訂（可參考 [[密集盤整區區間支撐壓力規則]]、[[盤整區上下頸線支撐壓力規則]] 中已定義的盤整辨識邏輯作為輸入基準）；成交量放大門檻沿用 [[成交量分類與倍數門檻定義]]
- **參考來源查證（2026-07-22，工程補充）**：「關前」距離門檻查無新的外部公認慣例，沿用R-SR-13已查證的2%心理關卡容許誤差作為內部一致代理值。「久盤」天數門檻沿用R-SCREEN-09已查證(書中明確)的均線糾結最短時長2個月(約40個交易日)作為代理值，非另行查證之新來源，僅求規則庫內部一致。
- **可回測**: 部分（前高關卡、成交量放大、盤整區間突破/跌破皆可回測；「關前」距離門檻與「久盤」天數門檻需先自訂才能精確回測）
- **信心**: 62/100 中（方向邏輯明確、範例數量在12口訣中並列最多（口訣11共6例）顯示實務出現頻率最高、可靠度不低；但「關前」距離與「久盤」天數缺乏書中精確數字，需自訂並搭配既有支撐壓力/盤整規則，故整體評為中）
