# 飆股選股與技術面操作守則

- **Rule ID**: R-STRATEGY-03
- **名稱**: 飆股選股與技術面操作守則
- **分類**: 綜合操作SOP
- **原文與頁碼**: `P11-C2-進場實戰操作SOP.md`（下冊 p.323-327，「飆股交易實戰守則」，含「如何選到飆股」8條、「飆股完全依技術面操作」5條（第5條再細分量能5級距）、「飆股的3種變化」，以及圖表11-2-4～11-2-7共4組個股操作範例）
- **解讀**: 分為選股、續抱依據、波段變化型態三部分：
  - **如何選到飆股（選股8條）**：曾有飆漲紀錄者優先；偏好中小型股（主力較易介入拉抬）；圖形要件為長期盤整（≥2個月）＋左低右高（型態內至少2支腳）＋股價過前高（過頸線）向上發動＋過頸線後拉回測試再發動＋第1段飆漲後經短期整理再度發動；雙重底大量突破且均線4線多排（見 [[均線多頭排列]]）；均線糾結後大量紅K上漲（見 [[均線糾結辨識]]）；大量紅K突破原空頭長期下降切線、站上月線後再現盤整大量紅K突破（見 [[突破下降切線分級與壓力支撐角色互換]]）；所有飆股發動點都須配合成交量同步放大（見 [[成交量分類與倍數門檻定義]]）確認；每日逐一檢視強勢股，符合特徵者立即列入鎖股觀察名單。
  - **飆股完全依技術面操作（續抱四要件＋量能5級距）**：續抱四要件——只要未跌破上升趨勢線（見 [[上升軌道線畫法（切線平移取中間高點）]]、[[上升切線支撐與力道遞減規則]]）、未跌破前2日低點、未跌破3日均線、未出現黑K，即可續抱，四者須同時成立；量能5級距決定續抱／減碼／出清：①無量飆漲續抱；②量稍放大股價仍穩續抱；③量放大且價格大幅震盪先出脫一半；④量大增仍收紅，保守者可先出場或再觀察次日；⑤量大增但開高走低收黑K，代表高點已到須迅速賣出。
  - **飆股的3種變化**：起飆拉回再飆、長黑K洗盤再飆、續勢整理再飆。統計上80%的飆股有2波上漲、10%有3波上漲、10%僅1波即結束，可作為分批停利部位規劃的機率參考，而非個股必然發生的規則。
  - **個股操作範例圖表（圖表11-2-4～11-2-7）**：4組實例共用同一套「飆股進場→回後買上漲加碼→分波停利→長黑跌破5日均線出場」邏輯，其中2組（1911、1416）明確標註第2波停利觸發參考值分別為「獲利超過30%＋大量長黑K」與「獲利超過60%＋大量長黑K」，兩者並列顯示此為案例觀察區間值而非條文明定的固定停利門檻；最終出場訊號皆為「長黑跌破5日均線後股價一路下跌」。
- **可程式化**: 部分（續抱四要件、量能5級距、成交量同步放大條件皆可程式化；「左低右高2支腳」「長期盤整2個月」等型態辨識需搭配型態辨識模組；飆股波段機率分布（2波80%/3波10%/1波10%）為統計參考值，非個股層級可直接判斷的條件；第2波停利30%～60%為案例觀察區間，非固定公式）
- **所需資料**: 收盤價、開盤價、最高價、最低價、成交量、上升趨勢線位置（見 [[上升軌道線畫法（切線平移取中間高點）]]）、MA3、MA5、均線多排判定結果（見 [[均線多頭排列]]）、盤整區間長度、頸線位置
- **計算公式**:
```
# 選股（部分條件需型態辨識模組配合，此處列出可直接量化者）
surge_candidate = (
    曾有歷史飆漲紀錄
    and 屬中小型個股
    and 長期盤整天數 >= 40                              # 約2個月交易日
    and 型態內轉折腳數 >= 2 and 左低右高
    and Close[t] > 頸線價 and 過頸線時放量               # 見 [[成交量分類與倍數門檻定義]]
)

# 續抱四要件（技術面完全依循，不摻雜主觀判斷）
def hold_surge_stock(t):
    cond1 = not 跌破上升趨勢線(t)                        # 見 [[上升軌道線畫法（切線平移取中間高點）]]
    cond2 = Close[t] >= min(Low[t-1], Low[t-2])          # 未跌破前2日低點
    cond3 = Close[t] >= MA3[t]                           # 未跌破3日均線
    cond4 = 是紅K(t) or Close[t] >= Open[t]               # 未出現黑K
    return cond1 and cond2 and cond3 and cond4

# 量能5級距判斷（在續抱四要件之外，進一步依量能決定減碼幅度）
def volume_tier_action(t):
    if 無量飆漲(t):
        return "續抱"
    elif 量稍放大 and 股價仍穩(t):
        return "續抱"
    elif 量放大 and 價格大幅震盪(t):
        return "出脫二分之一"
    elif 量大增 and 收紅(t):
        return "保守者先出場，或觀察次日"
    elif 量大增 and 開高走低 and 是黑K(t):
        return "迅速賣出"

# 波段次數機率參考（非個股條件，僅供停利規劃參考）
surge_wave_distribution = {"2波": 0.80, "3波": 0.10, "1波": 0.10}

# 分波停利參考（案例觀察區間，非固定門檻）
if profit_pct >= 0.30 and 出現大量長黑K:                  # 30%~60%為區間參考值，需自訂個股適用門檻
    action = "第2波停利參考觸發，觀察是否出清"
if Close[t] < MA5[t] and 前一根為大量長黑K:
    action = "長黑跌破5日均線，確認離場"
```
- **參數**: 長期盤整門檻＝2個月以上；型態腳數＝至少2支腳；續抱四要件＝趨勢線／前2日低／3日均線／無黑K皆不破；量能5級距＝無量／稍放大／放大震盪／大增收紅／大增開高走低黑K；飆股波段機率＝2波80%、3波10%、1波10%（書中明確數字）；第2波停利參考值＝30%～60%（案例觀察區間，非固定門檻，書中明確標註為區間值建議）
- **可回測**: 部分（續抱四要件與量能5級距可直接回測；型態選股條件與波段機率統計屬於統計/型態層級參考，需另建型態辨識模組才能精確回測）
- **信心**: 62/100 中（續抱四要件與量能5級距條件明確、可程式化程度最高＝高；但選股型態條件、飆股波段機率與停利區間值皆屬經驗參考而非精確公式，故整體評為中）
