# 盤整區上下頸線支撐壓力規則

- **Rule ID**: R-SR-04
- **名稱**: 盤整區上下頸線支撐壓力規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C2-走勢圖中的支撐壓力位置.md（下冊 p.045-046，6-2節「盤整區上下頸線的支撐與壓力」）
- **解讀**: 與[[密集盤整區區間支撐壓力規則]]取「整個區塊」不同，本規則取的是盤整區內部上下震盪邊界的**兩條連線**本身：下頸線＝連接盤整區內至少兩個以上次低點所畫出的線；上頸線＝連接盤整區內至少兩個以上次高點所畫出的線。取這個價位的具體做法：股價下跌碰到下頸線時，該線提供支撐；股價上漲碰到上頸線時，該線提供壓力。頸線本身被突破/跌破後的角色轉換，併入[[密集盤整區區間支撐壓力規則]]中「整個盤整區」突破上沿/跌破下沿的轉換邏輯，本規則不重複定義。
- **可程式化**: 部分（頸線的連線演算法需自訂，書中僅描述概念未給精確連線方法）
- **所需資料**: 已框定的盤整區間內每日OHLC
- **計算公式**:
```
function fit_neckline(zone_bars, side):
    # side = "下頸線"：取盤整區內各次局部低點 (local minima of Low)
    # side = "上頸線"：取盤整區內各次局部高點 (local maxima of High)
    points = extract_local_extrema(zone_bars, side)
    if len(points) < 2:
        return None  # 資料不足以連線
    neckline = linear_fit(points)   # 簡化可直接取兩點連線；書中範例多近似水平線
    return neckline   # 回傳 (slope, intercept)，可推算任一日的頸線價位

function neckline_price(neckline, t):
    return neckline.slope * t + neckline.intercept

# 支撐/壓力判定：
if Low[t] <= neckline_price(下頸線, t):
    信號 = "觸及下頸線，具支撐參考"
if High[t] >= neckline_price(上頸線, t):
    信號 = "觸及上頸線，具壓力參考"
```
- **參數**: 局部極值判定的窗口大小（書中未給，需自訂）
- **實作決定（2026-07-22，工程補充）**：沿用R-SR-03的決定，上下頸線直接取`detect_consolidation()`回傳的`upper_neckline`/`lower_neckline`（盤整區間內最高/最低價），不另外實作局部極值連線演算法——書中原文本身也說範例「多近似水平線」，用區間極值當作頸線是合理簡化，且避免引入無外部佐證的新窗口參數。
- **可回測**: 是
- **信心**: 60/100 中（概念明確，但連線演算法細節書中未給，屬合理推斷實作）
