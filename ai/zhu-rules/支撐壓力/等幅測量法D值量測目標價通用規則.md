# 等幅測量法D值量測目標價通用規則

- **Rule ID**: R-SR-19
- **名稱**: 等幅測量法D值量測目標價通用規則
- **分類**: 支撐壓力
- **原文與頁碼**: 工程補充延伸規則。核心「量出法」概念取材自 P06-C3-走勢圖上看不見的支撐與壓力.md（下冊 p.90-93，「型態目標價」一節：頭肩底/圓弧底/N字底/三重底突破頸線後，`目標價 = 頸線 +（頸線−型態最低點）`；頭肩頂/圓弧頂/倒N字頭/三重頂跌破頸線後，`目標價 = 頸線 −（型態最高點−頸線）`；書中該節本身也註明是依示意圖箭頭對稱長度倒推、非文字算式）。**本規則要補的缺口，是把「頸線導向」的量出法抽象成更通用的「用任一前一波段A→B的幅度，從突破/跌破點C等距外推目標價」版本，書中本身沒有以此更抽象的形式獨立命名或給出算式**，屬工程補充延伸；抽象化的必要性可見規則庫中 [[突破ABC上漲圖]]（R-CLASSIC-31）與 [[跌破ABC下跌圖]]（R-CLASSIC-14）兩條規則皆各自在「解讀」欄註明「等幅測量法尚無獨立規則對應，暫借用型態頸線規則的量測邏輯延伸表達」——本規則即是把這個一直被暫時借用的邏輯，正式獨立定義出來。
- **解讀**: 通用等幅測量法（俗稱「D值量測」）的核心公式是：取前一波段兩個端點 A、B 之間的價格幅度 `D = |B − A|`，當股價在突破點/跌破點 C 處確認突破或跌破後，把這個幅度 D 原封不動從 C 點再往同方向投射一次，得到目標價 `D值 = C ± D`（多頭突破用 C+D，空頭跌破用 C−D）。這是「把過去發生過的一段幅度，視為未來會重演一次的等量走勢」的量測假設，屬於技術分析中常見的「量出法／同幅measured move」思路的一般化表達。

  A、B、C 三點在不同情境下對應到不同的具體轉折點，本規則統一用這組通用符號描述，實際使用時依情境代入：
  - **情境1：ABC修正後突破/跌破延續**（對應 [[突破ABC上漲圖]]、[[跌破ABC下跌圖]]）：A、B 取「修正前的驅動段」起訖點（例如上漲情境中，A＝驅動段起漲點，B＝驅動段高點，即ABC修正的A點；或依規則實作彈性可取ABC修正本身A高點與B低點的幅度），C＝股價重新突破/跌破ABC修正區間並站穩切線之外的確認價位。
  - **情境2：切線突破/跌破後的滿足點預估**：A、B 取切線形成前，驅動股價畫出這條切線的那一段波段的起訖點，C＝股價突破下降切線（或跌破上升切線）當時的確認價位，套用 [[突破下降切線分級與壓力支撐角色互換]] / [[跌破上升切線分級與支撐壓力角色互換]] 判定切線已真正突破/跌破後才計算目標價。
  - **情境3：型態頸線量測（既有特例）**：[[型態頸線量測目標價支撐壓力規則]] 是本規則在「A＝型態最高/最低點，B＝頸線價，C＝頸線價（B=C）」這個特殊參數組合下的特例，兩者公式本質相同（`2×頸線−型態極值` 就是 `C+D` 或 `C−D` 展開後的結果）。

  目標價到達後的支撐壓力角色：多頭情境下，D值量測算出的上漲目標價到達後轉為壓力；空頭情境下，下跌目標價到達後轉為支撐（與 [[型態頸線量測目標價支撐壓力規則]] 的角色轉換邏輯一致）。

- **可程式化**: 部分（公式本身可完全程式化；但A、B、C三點在「情境1」「情境2」的具體取點依賴 [[轉折波定義與取點演算法]] 的ABC修正偵測或切線偵測模組是否準確，取點誤差會直接放大到目標價誤差）
- **所需資料**: 前一波段A、B兩端點價位、突破/跌破確認點C的價位、方向（多頭突破或空頭跌破）
- **計算公式**:
```
function equal_move_target(A_price, B_price, C_price, direction):
    D = abs(B_price - A_price)
    if direction == "多頭突破":
        target = C_price + D
        role_after_reach = "壓力"
    elif direction == "空頭跌破":
        target = C_price - D
        role_after_reach = "支撐"
    return target, role_after_reach

# 情境1：ABC修正後突破延續（比照 [[突破ABC上漲圖]] / [[跌破ABC下跌圖]]）
A, B, C_point = detect_ABC_and_breakout(bars)   # 見 [[轉折波定義與取點演算法]]
if 突破/跌破已依 [[突破跌破未站穩假突破假跌破通用判定規則]] 確認為真突破/真跌破:
    target, role = equal_move_target(A.price, B.price, C_point.price, direction)

# 情境2：切線突破/跌破後滿足點
A2, B2 = 切線形成前的驅動波段起訖點
C2 = 切線突破/跌破當時確認價位   # 見 [[突破下降切線分級與壓力支撐角色互換]] / [[跌破上升切線分級與支撐壓力角色互換]]
target2, role2 = equal_move_target(A2.price, B2.price, C2.price, direction)

# 情境3：頸線量測（與 [[型態頸線量測目標價支撐壓力規則]] 完全等價，B=C=頸線價）
target3, role3 = equal_move_target(pattern_extreme_price, neckline_price, neckline_price, direction)
# 展開後 = neckline_price ± (neckline_price - pattern_extreme_price) = 2*neckline_price - pattern_extreme_price
```
- **參數**: 無額外可調參數；核心自由度在於 A、B、C 三點的取法會隨情境不同（工程補充：情境1、2的A/B/C取點規則本身也未見書中逐字定義，屬合理但非唯一的類推方式，實作時可能需要依回測結果微調A/B兩端點的選取邏輯）
- **可回測**: 是（公式本身簡單可回測；準確度高度依賴A/B/C取點模組，故實際勝率需靠回測校準，不宜假設書中已驗證此通用形式的成功率）
- **信心**: 58/100 中（`目標價=頸線±型態高度`這個「等出一段既有幅度」的量出法核心技巧在書中 6-3 節有明確文字與圖表依據，可信；但把它抽象成「任意前一波段A→B」的通用D值量測版本，屬工程延伸類推，書中未見以此抽象形式獨立展開或命名，A/B/C的取點規則也非書中逐字明訂，故整體評為中，而非高）
- **底層規則依賴**: 型態頸線量測目標價支撐壓力規則、轉折波定義與取點演算法、突破跌破未站穩假突破假跌破通用判定規則、突破下降切線分級與壓力支撐角色互換、跌破上升切線分級與支撐壓力角色互換
