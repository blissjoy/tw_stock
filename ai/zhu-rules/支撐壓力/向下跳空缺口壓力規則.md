# 向下跳空缺口壓力規則

- **Rule ID**: R-SR-10
- **名稱**: 向下跳空缺口壓力規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C1-支撐壓力的重要觀念.md（下冊 p.027-028，空頭關鍵壓力空點來源之一）＋ P06-C2-走勢圖中的支撐壓力位置.md（下冊 p.073-076，6-2節「K線跳空缺口的支撐與壓力」下半部）
- **解讀**: 與[[向上跳空缺口支撐規則]]結構完全對稱、方向相反。支撐壓力來源是向下跳空造成的K線缺口，取的是**整個缺口區間**。取這個價位的具體做法：向下跳空缺口區間 = [跳空發生當根K棒的最高價, 跳空發生前一根K棒的最低價]（前提是當根最高價 < 前一根最低價，即真實存在缺口）。股價下跌後若反彈到此缺口區域，**整個缺口區間**都具壓力作用。缺口越大、跳空當天成交量越大，壓力力道越強。**永久失效規則**：缺口壓力一旦被（收盤）突破，即使日後股價拉回、回補了該缺口，該缺口也**不再具有壓力作用**。缺口被突破，代表多方力道轉強。更精細的缺口四層價位分級屬於跳空缺口專門分類的規則範圍（P09缺口篇），本規則不重複定義。
- **可程式化**: 是
- **所需資料**: 每日OHLC、成交量、缺口偵測
- **計算公式**:
```
function detect_down_gap(bars, i):
    prev_low  = bars[i-1].Low
    curr_high = bars[i].High
    if curr_high < prev_low:
        gap = {"low": curr_high, "high": prev_low, "date": bars[i].date,
               "size": prev_low - curr_high, "volume": bars[i].Volume,
               "breached": False}
        return gap
    return None

function on_price_return_to_gap(close_t, gap):
    if gap.breached:
        return "缺口已失效，不再具壓力作用（即使已回補）"
    if gap.low <= close_t <= gap.high:
        return f"股價反彈進入缺口區間，具壓力作用，強度={gap_resistance_strength(gap, avg_volume)}"
    if close_t > gap.high:
        gap.breached = True   # 收盤突破缺口上緣，永久失效
        return "缺口壓力已被突破，多方力道轉強，缺口永久失效"
```
- **參數**: 缺口大小與成交量對壓力力道的權重函數（書中僅定性描述，需自訂）
- **可回測**: 是
- **信心**: 62/100 中（缺口區間定義與永久失效規則明確；力道強度的量化權重書中未給，需自行校準）
