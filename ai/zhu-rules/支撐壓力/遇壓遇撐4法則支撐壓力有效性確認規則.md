# 遇壓遇撐4法則支撐壓力有效性確認規則

- **Rule ID**: R-SR-14
- **名稱**: 遇壓遇撐4法則支撐壓力有效性確認規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C1-支撐壓力的重要觀念.md（下冊 p.021-026，「支撐與壓力並非絕對」與「面臨支撐與壓力的可能變化」兩節）
- **解讀**: 這是全「支撐壓力」分類所有取點規則（R-SR-01~13）共用的**有效性確認濾網**，核心觀念是：**支撐與壓力都不是絕對的**，股價「碰到」某個支撐/壓力價位本身不足以判定該處真的有效，必須看「當日K棒在該價位的反應」＋「次日走勢方向確認」兩步驟，才能判定支撐/壓力是否真正成立。同一條支撐/壓力線（例如同一條均線）在不同時間點可能一次有效、一次無效，需逐次確認，不可一概而論。
  **遇壓力4法則**（股價上漲接近壓力位置時）：
  1. 當天大量中長紅K → 次日收黑K，確認「闖關前爆大量，股價不漲要回檔」，判定遇壓回檔。
  2. 當天大量長上影線K或中長黑K（盤中衝過壓力但收盤未站穩）→ 次日容易下跌，判定遇壓。
  3. 當天大量變盤線（十字線一類，多空膠著）→ 須看次日：次日收黑K才確認遇壓回檔。
  4. 當天大量中長紅K且成功突破壓力 → 仍須留意次日是否出現大量長黑K：若次日出現，屬於「假突破」機率很高，須預先做好多單停損準備。
  **遇支撐4法則**（股價下跌接近支撐位置時，與遇壓力邏輯完全對稱、方向相反）：
  1. 當天大量中長黑K → 次日收紅K，確認「過撐爆大量，股價不跌要反彈」，判定遇撐反彈。
  2. 當天大量長下影線K或中長紅K（盤中跌破支撐但收盤未失守）→ 次日容易反彈，判定遇撐。
  3. 當天大量變盤線 → 須看次日：次日收紅K才確認遇撐反彈。
  4. 當天大量中長黑K且真的跌破支撐 → 仍須留意次日是否出現大量長紅K：若次日出現，屬於「假跌破」機率很高，須預先做好空單停損準備。
  以上8條情境的共同判讀邏輯是「量＋當日K棒型態＋次日確認」三步驟：量是判斷多空力道是否真的在該價位交手的前提，K棒型態判斷當日盤中/收盤是否守住/測試該價位，次日方向則是最終確認。
- **可程式化**: 是
- **所需資料**: 每日OHLC、成交量、近N日均量（判斷是否為「大量」）、支撐/壓力參考價位（來自R-SR-01~13任一取點規則）

function confirm_resistance(bar_t, bar_next, sr_price, avg_volume_N):
    if not is_big_volume(bar_t.Volume, avg_volume_N):
        return "量能不足，暫不套用遇壓4法則"
    ctype = candle_type(bar_t)
    if ctype == "中長紅K" and bar_t.Close <= sr_price:
        if bar_next.Close < bar_next.Open:   # 次日收黑
            return "確認遇壓回檔（闖關前爆量，股價不漲要回檔）"
    elif ctype in ["長上影線K", "中長黑K"] and 盤中曾觸及或穿過 sr_price and bar_t.Close <= sr_price:
        return "遇壓，次日容易下跌"
    elif ctype == "變盤線":
        if bar_next.Close < bar_next.Open:
            return "次日收黑，確認遇壓回檔"
        else:
            return "尚未確認，持續觀察"
    elif ctype == "中長紅K" and bar_t.Close > sr_price:   # 成功突破
        if is_big_volume(bar_next.Volume, avg_volume_N) and candle_type(bar_next) == "中長黑K":
            return "高機率假突破，執行多單停損準備"
        else:
            return "突破有效，壓力轉支撐"

function confirm_support(bar_t, bar_next, sr_price, avg_volume_N):
    # 與 confirm_resistance 方向完全對稱（紅黑互換、漲跌互換）
    if not is_big_volume(bar_t.Volume, avg_volume_N):
        return "量能不足，暫不套用遇撐4法則"
    ctype = candle_type(bar_t)
    if ctype == "中長黑K" and bar_t.Close >= sr_price:
        if bar_next.Close > bar_next.Open:   # 次日收紅
            return "確認遇撐反彈（過撐爆量，股價不跌要反彈）"
    elif ctype in ["長下影線K", "中長紅K"] and 盤中曾觸及或跌破 sr_price and bar_t.Close >= sr_price:
        return "遇撐，次日容易反彈"
    elif ctype == "變盤線":
        if bar_next.Close > bar_next.Open:
            return "次日收紅，確認遇撐反彈"
        else:
            return "尚未確認，持續觀察"
    elif ctype == "中長黑K" and bar_t.Close < sr_price:   # 真的跌破
        if is_big_volume(bar_next.Volume, avg_volume_N) and candle_type(bar_next) == "中長紅K":
            return "高機率假跌破，執行空單停損準備"
        else:
            return "跌破有效，支撐轉壓力"
