# 整數心理關卡支撐壓力規則

- **Rule ID**: R-SR-13
- **名稱**: 整數心理關卡支撐壓力規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C3-走勢圖上看不見的支撐與壓力.md（下冊 p.94-95，「整數心理的支撐與壓力」）
- **解讀**: 支撐壓力來源是投資人對特定整數數字的心理關卡效應（例如「千萬富翁」「999元定價」等心理現象在股市中的映射），屬於「看不見、屬於心理面而非結構面」的支撐壓力類型。取這個價位的具體做法：股價或指數逼近「整數關卡」（個股逢百元整數，如台積電900元保衛戰；大盤逢千點整數，如20,000點、22,000點、24,000點；「7千金」等說法亦屬此類定性描述）時，將該整數位標記為潛在支撐／壓力觀察點。效果：股價來到整數關卡時，往往出現「震盪走勢」（多空反覆拉鋸），而非一次到位快速穿越。書中未給出更細的量化門檻（例如取哪個位數的倍數、距離關卡多少百分比才觸發觀察），僅以「整數」「千金」等定性描述呈現，屬於信心較低、需大量歷史案例校準的規則。
- **可程式化**: 部分（整數關卡本身容易定義，但「震盪」的判定門檻與觸發距離書中未量化，需自訂）
- **所需資料**: 股價／指數當前價格
- **計算公式**:
```
function nearest_round_level(price, round_unit):
    # round_unit 依標的規模自訂：個股常用100（百元關卡）或1000（千元關卡）；
    # 大盤指數常用1000點或10000點（書中範例含20000、22000、24000點）
    return round(price / round_unit) * round_unit

function is_near_psychological_level(price, round_unit, distance_pct_threshold):
    level = nearest_round_level(price, round_unit)
    distance_pct = abs(price - level) / level
    if distance_pct <= distance_pct_threshold:
        return True, level   # 標記為潛在整數心理關卡觀察區，預期出現震盪而非單邊突破/跌破
    return False, None
```
- **參數**: round_unit（整數關卡的位數單位，書中未給固定規則，需依標的價格區間自訂，如個股100/1000、大盤1000/10000）、distance_pct_threshold（判定「逼近關卡」的距離百分比門檻，書中未給，需自行以歷史資料校準）
- **參考來源查證（2026-07-22，工程補充）**：查證外部資料，確認「整數關卡因投資人心理習慣與停損停利單常掛在整數附近而形成支撐壓力」是普遍被提及的現象（來源：[cmoney投資小學堂](https://www.cmoney.tw/learn/course/technicals/topic/506)），但**查無「逼近關卡」的公認統一距離百分比門檻**，distance_pct_threshold=2%維持為工程估計值，非外部佐證數字。
- **可回測**: 是（但因門檻需自訂，回測結果對參數敏感）
- **信心**: 32/100 低（書中僅為定性描述，無明確倍數或距離量化門檻）
