# 突破跌破未站穩假突破假跌破通用判定規則

- **Rule ID**: R-SR-18
- **名稱**: 突破跌破未站穩假突破假跌破通用判定規則
- **分類**: 支撐壓力
- **原文與頁碼**: 綜合依據 P06-C2-走勢圖中的支撐壓力位置.md（下冊 p.031-039，「轉折波高點（頭）」「轉折波低點（底）」的突破/跌破後支撐壓力互換邏輯）、P06-C1-支撐壓力的重要觀念.md（「遇壓遇撐4法則」第4款提及「假突破」「假跌破」須預先停損準備，但未獨立展開判定條件）、P09-C5-缺口的進階應用.md（下冊 p.243-246，「真封口／假封口」三要件判斷風格，作為本規則「未站穩」條件設計的推理範本）。**未站穩的天數窗口、量能門檻等具體數字，書中沒有針對「非缺口的一般突破/跌破」給出逐字明訂數值，屬工程補充延伸**，設計時比照書中已有的缺口真假封口判定風格與[[移動平均線支撐壓力與3日站回觀察窗規則]]的3日觀察窗精神類推。
- **解讀**: 本規則填補「支撐壓力」分類的一個底層概念缺口：現有 [[轉折高點壓力與突破後轉支撐規則]]（R-SR-01）與 [[轉折低點支撐與跌破後轉壓力規則]]（R-SR-02）只定義「一旦收盤突破/跌破，該點角色互換」，並未處理「突破/跌破後很快又跌回/彈回原區間」這種騙線情境；而 [[缺口假封口判定規則]] 只處理缺口封口的真假判定，不適用於前高/前低、盤整區頸線、切線等非缺口類支撐壓力來源。本規則提供一個通用版本，供 [[突破高檔前高黑K出現大量要下跌圖]]、[[突破ABC上漲圖]]、[[跌破ABC下跌圖]]、[[底部洗盤上攻大漲圖]] 等經典型態總覽分類的複合規則引用，取代它們原本各自「暫借用」其他規則臨時拼湊表達假突破/假跌破的做法。

  **核心邏輯**：
  1. 先用 [[轉折高點壓力與突破後轉支撐規則]] / [[轉折低點支撐與跌破後轉壓力規則]] 判斷是否發生「收盤突破前高」或「收盤跌破前低」（也可套用在密集盤整區上下緣、切線、頸線等其他支撐壓力來源，凡具備「突破後應轉支撐」或「跌破後應轉壓力」性質者皆可代入）。
  2. 疊加「未能站穩」的確認條件：突破/跌破後，若股價在短期觀察窗內（工程補充，預設N=3個交易日，可依個股波動性調整）又收盤跌回原前高之下（或彈回原前低之上），且期間收盤維持在突破/跌破方向那一側的天數不足（工程補充，預設M=2日），即判定為「假突破」或「假跌破」——原本應該轉換角色的支撐/壓力並未真正轉換成功，維持原本角色（前高仍是壓力、前低仍是支撐）。
  3. 若觀察窗內都沒有跌回/彈回，或跌回/彈回前已站穩滿M日，則視為真突破/真跌破，正式套用 R-SR-01／R-SR-02 的角色互換，並可再用 [[遇壓遇撐4法則支撐壓力有效性確認規則]] 做次日層級的細部確認。
  4. 量能輔助判斷（可選，比照[[遇壓遇撐4法則支撐壓力有效性確認規則]]第4款）：突破/跌破當天量縮，或突破/跌破後的反向K棒為大量，皆提高判定為假突破/假跌破的機率。
  5. **與缺口假封口的分工**：若討論的標的本身是缺口邊界，應優先使用 [[缺口真封口判定規則]] / [[缺口假封口判定規則]] 的三要件精確版本（大量＋實體反向K＋收盤越界）；本規則是這套邏輯在「前高/前低/頸線/切線」等非缺口支撐壓力來源上的通用延伸版本，判斷粗細度略低於缺口專用規則。

- **可程式化**: 是
- **所需資料**: 依 [[轉折波定義與取點演算法]] 取得的轉折高點/低點（或盤整區上下緣、切線價位），觀察窗內每日OHLC與成交量，近N日均量
- **計算公式**:
```
function classify_breakout(prior_high, bars, breakout_day, N=3, M=2):
    # prior_high: 依 [[轉折高點壓力與突破後轉支撐規則]] 取得的前高（或其他支撐壓力來源價位）
    # breakout_day: 收盤首次突破 prior_high 的日期
    window = bars[breakout_day : breakout_day + N]
    fell_back = any(day in window where day.Close < prior_high)
    days_held_above = count(day in window where day.Close >= prior_high)

    if fell_back and days_held_above < M:
        return "假突破（未站穩）— 前高角色不變，仍為壓力，套用 [[轉折高點壓力與突破後轉支撐規則]] 原壓力邏輯"
    elif fell_back and days_held_above >= M:
        return "已站穩後才小幅拉回，視為真突破後的正常回測，非假突破"
    else:
        return "真突破，前高轉支撐，套用 [[轉折高點壓力與突破後轉支撐規則]] 角色互換"

function classify_breakdown(prior_low, bars, breakdown_day, N=3, M=2):
    # 與 classify_breakout 完全對稱（漲跌、高低互換）
    window = bars[breakdown_day : breakdown_day + N]
    bounced_back = any(day in window where day.Close > prior_low)
    days_held_below = count(day in window where day.Close <= prior_low)

    if bounced_back and days_held_below < M:
        return "假跌破（未站穩）— 前低角色不變，仍為支撐，套用 [[轉折低點支撐與跌破後轉壓力規則]] 原支撐邏輯"
    elif bounced_back and days_held_below >= M:
        return "已站穩後才小幅反彈，視為真跌破後的正常測試，非假跌破"
    else:
        return "真跌破，前低轉壓力，套用 [[轉折低點支撐與跌破後轉壓力規則]] 角色互換"
```
