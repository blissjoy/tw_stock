# 多頭回檔四大關鍵支撐買進訊號規則

- **Rule ID**: R-SR-15
- **名稱**: 多頭回檔四大關鍵支撐買進訊號規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C1-支撐壓力的重要觀念.md（下冊 p.026-027，「支撐與壓力的進出位置判斷」多頭關鍵支撐買點）
- **解讀**: 這是本書把「支撐壓力位置」與「進場訊號」結合的具體方法：在確認目前處於多頭趨勢（見[[頭頭高底底高多頭趨勢判定]]）的前提下，股價回檔遇到以下4種支撐位置之一，且**當天K棒收出止跌訊號**（紅K或長下影線），即為準備次日上漲的買點。4種支撐來源：
  1. **月線、季線支撐**（見[[移動平均線支撐壓力與3日站回觀察窗規則]]、[[均線上彎的支撐與助漲作用]]）。
  2. **上升切線支撐**（見[[上升切線支撐與力道遞減規則]]，切線畫法見切線/軌道線分類）。
  3. **前低支撐**（前一波轉折低點，見[[轉折低點支撐與跌破後轉壓力規則]]）。
  4. **下方大量跳空缺口支撐**（見[[向上跳空缺口支撐規則]]）。
  四種來源的共同判斷條件都是：**支撐位置本身＋當天在該位置收出止跌訊號K棒（紅K或長下影線）**，兩者缺一不可。單純價格碰到支撐線不足以構成買點，必須有當日K棒的止跌反應確認；更嚴謹的確認流程可再套用[[遇壓遇撐4法則支撐壓力有效性確認規則]]（尤其是「大量長下影線/中長紅K」與「大量變盤線待次日確認」兩種情境）。
- **可程式化**: 是
- **所需資料**: 多頭趨勢判定結果、當前股價相對月線/季線/上升切線/前低/跳空缺口的位置、當日K棒OHLC（判斷止跌紅K或長下影線）
- **計算公式**:
```
function is_bullish_reversal_candle(bar):
    is_red = bar.Close > bar.Open
    lower_shadow = min(bar.Open, bar.Close) - bar.Low
    body = abs(bar.Close - bar.Open)
    is_long_lower_shadow = lower_shadow > body * 1.0   # 長下影線判定，書中未給精確倍數，建議下影線長度 >= 實體1倍
    return is_red or is_long_lower_shadow

function check_bullish_support_buy_signal(trend, bar_t, support_candidates):
    if trend != "多頭趨勢":
        return None
    for support in support_candidates:
        # support ∈ {月線, 季線, 上升切線, 前低, 下方大量跳空缺口}
        if 股價當日觸及 support.price 且 is_bullish_reversal_candle(bar_t):
            return f"{support.type}支撐＋止跌訊號K棒 → 次日買進訊號候選"
    return None
```
- **參數**: 長下影線判定的下影線/實體比例門檻（書中未給精確倍數，建議預設1.0倍實體，可調整）
- **可回測**: 是
- **信心**: 90/100 高（書中4種來源與共同判斷條件描述明確一致）
