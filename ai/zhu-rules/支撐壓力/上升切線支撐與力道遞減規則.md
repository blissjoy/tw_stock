# 上升切線支撐與力道遞減規則

- **Rule ID**: R-SR-06
- **名稱**: 上升切線支撐與力道遞減規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C1-支撐壓力的重要觀念.md（下冊 p.026-027，多頭關鍵支撐買點來源之一）＋ P06-C2-走勢圖中的支撐壓力位置.md（下冊 p.053-056，6-2節「上升切線的支撐與壓力」）
- **解讀**: 支撐壓力來源是連接兩個以上上升轉折低點所畫出的上升趨勢切線本身（延長線）。切線的具體畫法屬於切線/軌道線分類，本規則不重複定義，取價方法請參照該分類的上升切線畫法規則（假定名稱 [[上升切線畫法與線不蓋線判準]]，見切線軌道線分類）。取這個價位的具體做法：股價位在切線**上方**時，將切線方程式延伸到當前日期，計算出的價位即為當日支撐參考價；股價回檔碰到此延長線時，該處提供支撐。**力道遞減規律**（務必保留的精細規則）：切線第1次被股價回檔觸及時支撐最強；第2次回檔觸及時，支撐成功率約剩一半；第3次（以後）回檔觸及則容易被跌破——書中未給精確機率數字，僅為定性強弱排序，屬於觸碰次數的遞減函數。轉換條件：上升切線支撐一旦被（收盤）跌破，股價轉到切線下方，此後股價反彈上漲碰到原切線位置時，切線角色反轉為壓力。是否真的止跌，仍須以當日K棒反應＋次日確認為準（見[[遇壓遇撐4法則支撐壓力有效性確認規則]]）。
- **可程式化**: 部分（切線本身的擬合演算法歸屬切線/軌道線分類；本規則只處理切線價位轉為支撐/壓力訊號與遞減強度的邏輯）
- **所需資料**: 上升切線方程式（斜率、截距，來自切線/軌道線分類規則）、每日OHLC、歷史觸碰次數計數
- **計算公式**:
```
trendline_price(t) = slope * t + intercept    # 由 [[上升切線畫法與線不蓋線判準]] 提供的切線方程式

touch_count = 0   # 累計股價回檔觸及此切線的次數

function on_price_touch_trendline(close_t, trendline_price_t):
    global touch_count
    if close_t 接近或觸及 trendline_price_t 且 close_t >= trendline_price_t:
        touch_count += 1
        if touch_count == 1:
            strength = "最強"
        elif touch_count == 2:
            strength = "約剩一半（次強）"
        else:  # touch_count >= 3
            strength = "偏弱，容易被跌破"
        signal = f"上升切線第{touch_count}次觸及支撐，強度：{strength}"
        # 是否真的止跌 -> 套用 [[遇壓遇撐4法則支撐壓力有效性確認規則]]

function on_breakdown(close_t, trendline_price_t):
    if close_t < trendline_price_t（收盤確認跌破）:
        角色轉換 = "上升切線由支撐轉為壓力（股價轉至切線下方後的反彈參考）"
        touch_count = 0   # 角色轉換後可重新計數
```
- **參數**: 觸碰次數強度對照表（第1次=最強，第2次≈50%，第3次以後=弱，書中僅定性描述，未給精確機率）
- **可回測**: 是
- **信心**: 60/100 中（力道遞減規律清楚但為定性描述；切線擬合本身依賴切線/軌道線分類的實作品質）
