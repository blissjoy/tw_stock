# 支撐壓力測試結果回推多空趨勢強弱規則

- **Rule ID**: R-SR-17
- **名稱**: 支撐壓力測試結果回推多空趨勢強弱規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C1-支撐壓力的重要觀念.md（下冊 p.028-029，「不同趨勢支撐、壓力的慣性變化」）
- **解讀**: 與R-SR-01~16「找出支撐壓力在哪裡」的方向相反，本規則是**用支撐壓力被觸及後的反應，回過頭來推斷目前趨勢的強弱與是否延續／改變**。多頭、空頭各5條、彼此鏡像對稱：
  **多頭趨勢（5條）**：
  1. 股價持續運行於10日均線之上、回檔不破10MA → 判定強勢多頭。
  2. 回檔不破前低支撐（見[[轉折低點支撐與跌破後轉壓力規則]]）＋維持在月線之上且月線上揚 → 多頭趨勢不變，可繼續做多。
  3. 上漲測試前高（見[[轉折高點壓力與突破後轉支撐規則]]）：突破＝多頭沒有改變；未突破反而下跌＝多頭進入盤整。
  4. 回檔跌破前低支撐＝多頭趨勢改變；此後若反彈仍無法突破前高壓力，即為多單逃命波的賣點。
  5. 回檔跌破月線支撐＝多頭趨勢轉弱；月線支撐/壓力轉換的3～5日觀察窗規則見[[移動平均線支撐壓力與3日站回觀察窗規則]]，本規則不重複定義，僅引用其結論作為趨勢轉弱的判斷依據。
  **空頭趨勢（5條，方向相反、結構對稱）**：
  1. 股價持續運行於10日均線之下、反彈不破10MA → 判定弱勢空頭。
  2. 反彈不突破前高壓力＋維持在月線之下且月線下彎 → 空頭趨勢不變，可繼續做空。
  3. 下跌測試前低：跌破＝空頭沒有改變；未跌破反而上漲＝空頭進入盤整。
  4. 反彈突破前高壓力＝空頭趨勢改變；此後若下跌仍無法跌破前低支撐，即為空單逃命波的回補點。
  5. 反彈突破月線壓力＝空頭趨勢轉強；月線觀察窗同引用[[移動平均線支撐壓力與3日站回觀察窗規則]]。
  實務應用（收尾要點）：上漲時可先評估上方壓力是否過大（過大則突破機率不高）；下跌時可先評估下方支撐力道是否很強（很強則不易被跌破），藉此預先擬定操作計畫。支撐、壓力的力量大小受「趨勢、位置、成交量」三因素影響，各不相同。
- **可程式化**: 是
- **所需資料**: MA10及其逐日位置關係、前高/前低價位（依[[轉折波定義與取點演算法]]取得）、月線位置與方向、股價OHLC
- **計算公式**:
```
function bull_trend_strength(bars, ma10, prior_low, prior_high, ma20):
    if all(Close[t] > ma10[t] for t in 回檔期間):
        return "強勢多頭"
    if Close[t] > prior_low and Close[t] > ma20[t] and MA_direction(20,t) == "上揚":
        return "多頭趨勢不變，可續做多"
    if 股價上漲測試 prior_high:
        if Close[t] > prior_high:
            return "多頭沒有改變"
        else:
            return "多頭進入盤整"
    if Close[t] < prior_low:
        趨勢狀態 = "多頭趨勢改變"
        if 之後反彈 and Close[t2] < prior_high:
            return "多單逃命波賣點"
    if Close[t] < ma20[t]:
        return "多頭趨勢轉弱（後續套用 [[移動平均線支撐壓力與3日站回觀察窗規則]] 判斷是否真轉弱）"

function bear_trend_strength(bars, ma10, prior_low, prior_high, ma20):
    # 與 bull_trend_strength 方向完全對稱
    if all(Close[t] < ma10[t] for t in 反彈期間):
        return "弱勢空頭"
    if Close[t] < prior_high and Close[t] < ma20[t] and MA_direction(20,t) == "下彎":
        return "空頭趨勢不變，可續做空"
    if 股價下跌測試 prior_low:
        if Close[t] < prior_low:
            return "空頭沒有改變"
        else:
            return "空頭進入盤整"
    if Close[t] > prior_high:
        趨勢狀態 = "空頭趨勢改變"
        if 之後下跌 and Close[t2] > prior_low:
            return "空單逃命波回補點"
    if Close[t] > ma20[t]:
        return "空頭趨勢轉強（後續套用 [[移動平均線支撐壓力與3日站回觀察窗規則]] 判斷是否真轉強）"
```
- **參數**: 無額外參數；沿用MA10/MA20與轉折高低點的既有定義
- **可回測**: 是
- **信心**: 91/100 高（書中多空各5條規則對稱清楚，可直接轉為狀態機邏輯）
