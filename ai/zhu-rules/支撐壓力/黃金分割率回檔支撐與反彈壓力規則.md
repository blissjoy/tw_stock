# 黃金分割率回檔支撐與反彈壓力規則

- **Rule ID**: R-SR-11
- **名稱**: 黃金分割率回檔支撐與反彈壓力規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C3-走勢圖上看不見的支撐與壓力.md（下冊 p.87-89，「黃金分割率的支撐與壓力」）
- **解讀**: 支撐壓力來源是波段高低點依黃金分割比例（費波那契比例）分割出的價位，屬於「走勢圖上看不見、需要計算」的支撐壓力類型，與R-SR-01~10「肉眼可見」的類型互補。取這個價位的具體做法：先取一段明確的波段兩端（波段最低點A、波段最高點B，可用[[轉折波定義與取點演算法]]取得的轉折高低點作為A、B候選），再依比例 r ∈ {0.236, 0.382, 0.5, 0.618} 計算分割價位（書中文字定義另列0.809，但兩張範例圖表實際僅標示到0.618與兩端錨點0/1.000，未見0.809被實際使用，整理時如實保留此落差）。多頭回檔情境：從波段低點A拉到波段高點B，回檔支撐價位 = B − (B − A) × r；範例顯示0.382、0.5是較常見的短波段回檔支撐候選價位，跌破後可再看0.618。空頭反彈情境：從波段高點A拉到波段低點B，反彈壓力價位 = B + (A − B) × r；範例顯示0.5、0.382是較常見的反彈壓力候選價位，突破後可再看0.618。實務可直接使用看盤軟體「黃金分割率」繪圖工具，選取波段高低點後自動畫線，不需手算。
- **可程式化**: 是
- **所需資料**: 波段起訖兩端價位（A、B，需先由波段/轉折高低點偵測模組給出）
- **計算公式**:
```
fib_ratios = [0.236, 0.382, 0.5, 0.618]   # 書中文字另提0.809，但範例圖表未使用，暫不納入預設集合

function bull_pullback_support(A, B, r):
    # A = 波段最低點, B = 波段最高點（多頭上漲段）
    return B - (B - A) * r

function bear_rebound_resistance(A, B, r):
    # A = 波段最高點, B = 波段最低點（空頭下跌段）
    return B + (A - B) * r

# 多頭回檔：計算各 r 對應支撐價，股價回檔接近該價位視為潛在支撐
support_levels = {r: bull_pullback_support(A, B, r) for r in fib_ratios}

# 空頭反彈：計算各 r 對應壓力價，股價反彈接近該價位視為潛在壓力
resistance_levels = {r: bear_rebound_resistance(A, B, r) for r in fib_ratios}

# 優先關注順序（依書中範例統計傾向，非鐵律）：
# 多頭回檔：優先看 0.382、0.5，跌破後再看 0.618
# 空頭反彈：優先看 0.5、0.382，突破後再看 0.618
```
- **參數**: fib_ratios（分割比例集合，預設[0.236, 0.382, 0.5, 0.618]，書中另提0.809可選用）、波段A/B的偵測方式（建議沿用轉折波高低點）
- **可回測**: 是
- **信心**: 58/100 中（分割公式為依書中範例圖表數字倒推所得，非書中逐字給出算式；哪個分割點會實際生效需個案驗證，非固定規律）
