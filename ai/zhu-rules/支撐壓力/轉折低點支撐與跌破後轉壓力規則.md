# 轉折低點支撐與跌破後轉壓力規則

- **Rule ID**: R-SR-02
- **名稱**: 轉折低點支撐與跌破後轉壓力規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C2-走勢圖中的支撐壓力位置.md（下冊 p.035-039，6-2節「轉折波低點（底）的支撐與壓力」）
- **解讀**: 支撐壓力來源是轉折波的最低點（底），取點方法沿用[[轉折波定義與取點演算法]]，不重新定義轉折點本身。取這個價位的具體做法：取轉折低點（底）當根K線的最低價，畫一條水平線作為支撐／壓力參考。判斷邏輯與轉折高點完全對稱、方向相反：只要股價還在該底部**上方**（尚未跌破），股價下跌接近底部價位時，此處形成支撐；一旦股價**收盤跌破**該底部，此後股價反彈上漲接近底部價位，底部反過來變成壓力。同一個轉折低點的「支撐」或「壓力」身分由股價目前相對其位置決定。是否真的止跌/止漲，同樣須以當日K棒反應＋次日確認（見[[遇壓遇撐4法則支撐壓力有效性確認規則]]）為準。
- **可程式化**: 是
- **所需資料**: 依[[轉折波定義與取點演算法]]取得的轉折低點（底）價位序列（含日期）、當前股價OHLC
- **計算公式**:
```
turning_points = compute_turning_points(bars, N)   # 見 [[轉折波定義與取點演算法]]
bottoms = [p for p in turning_points if p.type == "bottom"]

function classify_bottom_role(bottom, current_price):
    if current_price > bottom.price:
        role = "支撐"      # 股價尚未跌破底部
    elif current_price <= bottom.price and 曾經收盤跌破過 bottom.price:
        role = "壓力"      # 跌破後反彈測試
    return role

# 進場/確認仍需套用 [[遇壓遇撐4法則支撐壓力有效性確認規則]]
```
- **參數**: 繼承轉折波取點演算法的均線天期N（5/10/20日短中長線版本）
- **可回測**: 是
- **信心**: 89/100 高
