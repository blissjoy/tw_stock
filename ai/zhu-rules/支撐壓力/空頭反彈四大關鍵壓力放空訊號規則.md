# 空頭反彈四大關鍵壓力放空訊號規則

- **Rule ID**: R-SR-16
- **名稱**: 空頭反彈四大關鍵壓力放空訊號規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C1-支撐壓力的重要觀念.md（下冊 p.027-028，「支撐與壓力的進出位置判斷」空頭關鍵壓力空點）
- **解讀**: 與[[多頭回檔四大關鍵支撐買進訊號規則]]結構完全對稱、方向相反。在確認目前處於空頭趨勢（見[[頭頭低底底低空頭趨勢判定]]）的前提下，股價反彈遇到以下4種壓力位置之一，且**當天K棒收出止漲訊號**（黑K或長上影線），即為準備次日下跌的空點。4種壓力來源：
  1. **月線、季線壓力**（見[[移動平均線支撐壓力與3日站回觀察窗規則]]、[[均線下彎的壓力與助跌作用]]）。
  2. **下降切線壓力**（見[[下降切線壓力與力道遞減規則]]，切線畫法見切線/軌道線分類）。
  3. **前高壓力**（前一波轉折高點，見[[轉折高點壓力與突破後轉支撐規則]]）。
  4. **上方大量跳空缺口壓力**（見[[向下跳空缺口壓力規則]]）。
  四種來源的共同判斷條件都是：**壓力位置本身＋當天在該位置收出止漲訊號K棒（黑K或長上影線）**，兩者缺一不可。更嚴謹的確認流程可再套用[[遇壓遇撐4法則支撐壓力有效性確認規則]]（尤其是「大量長上影線/中長黑K」與「大量變盤線待次日確認」兩種情境）。
- **可程式化**: 是
- **所需資料**: 空頭趨勢判定結果、當前股價相對月線/季線/下降切線/前高/跳空缺口的位置、當日K棒OHLC（判斷止漲黑K或長上影線）
- **計算公式**:
```
function is_bearish_reversal_candle(bar):
    is_black = bar.Close < bar.Open
    upper_shadow = bar.High - max(bar.Open, bar.Close)
    body = abs(bar.Close - bar.Open)
    is_long_upper_shadow = upper_shadow > body * 1.0   # 書中未給精確倍數，建議上影線長度 >= 實體1倍
    return is_black or is_long_upper_shadow

function check_bearish_resistance_short_signal(trend, bar_t, resistance_candidates):
    if trend != "空頭趨勢":
        return None
    for resistance in resistance_candidates:
        # resistance ∈ {月線, 季線, 下降切線, 前高, 上方大量跳空缺口}
        if 股價當日觸及 resistance.price 且 is_bearish_reversal_candle(bar_t):
            return f"{resistance.type}壓力＋止漲訊號K棒 → 次日放空訊號候選"
    return None
```
- **參數**: 長上影線判定的上影線/實體比例門檻（書中未給精確倍數，建議預設1.0倍實體，可調整）
- **可回測**: 是
- **信心**: 90/100 高（書中4種來源與共同判斷條件描述明確一致）
