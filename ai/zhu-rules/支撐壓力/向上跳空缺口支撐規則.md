# 向上跳空缺口支撐規則

- **Rule ID**: R-SR-09
- **名稱**: 向上跳空缺口支撐規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C1-支撐壓力的重要觀念.md（下冊 p.026-027，多頭關鍵支撐買點來源之一）＋ P06-C2-走勢圖中的支撐壓力位置.md（下冊 p.069-072，6-2節「K線跳空缺口的支撐與壓力」上半部）
- **解讀**: 支撐壓力來源是向上跳空造成的K線缺口，取的是**整個缺口區間**（缺口上緣到下緣的價格帶），而非單一邊界價位。取這個價位的具體做法：向上跳空缺口區間 = [跳空發生前一根K棒的最高價, 跳空發生當根K棒的最低價]（前提是當根最低價 > 前一根最高價，即真實存在缺口，未被當日或次日行情立即回補）。股價上漲後若回檔跌回此缺口區域，**整個缺口區間**都具支撐作用。缺口越大，支撐力道越強；跳空當天成交量越大，支撐力道也越強。**永久失效規則（務必保留的精細規則）**：缺口支撐一旦被（收盤）跌破，即使日後股價又反彈填補、回補了該缺口，該缺口也**不再具有支撐作用**——不會因回補而恢復支撐效力。缺口被跌破，代表空方力道轉強。更精細的缺口四層價位分級（上高／上沿／下沿／下底）屬於跳空缺口專門分類的規則範圍（P09缺口篇），不在本規則重複定義，本規則僅處理缺口作為支撐壓力來源的基本二分邊界。
- **可程式化**: 是
- **所需資料**: 每日OHLC、成交量、缺口偵測（相鄰兩根K棒間的價格跳空）
- **計算公式**:
```
function detect_up_gap(bars, i):
    prev_high = bars[i-1].High
    curr_low  = bars[i].Low
    if curr_low > prev_high:
        gap = {"low": prev_high, "high": curr_low, "date": bars[i].date,
               "size": curr_low - prev_high, "volume": bars[i].Volume,
               "breached": False}
        return gap
    return None

function gap_support_strength(gap, avg_volume):
    size_factor = gap.size / gap.low                      # 缺口相對大小
    volume_factor = gap.volume / avg_volume                # 跳空當天量能相對強度
    strength = f(size_factor, volume_factor)                # 兩者越大，支撐力道越強（書中未給精確權重公式）
    return strength

function on_price_return_to_gap(close_t, gap):
    if gap.breached:
        return "缺口已失效，不再具支撐作用（即使已回補）"
    if gap.low <= close_t <= gap.high:
        return f"股價回檔進入缺口區間，具支撐作用，強度={gap_support_strength(gap, avg_volume)}"
    if close_t < gap.low:
        gap.breached = True   # 收盤跌破缺口下緣，永久失效
        return "缺口支撐已被跌破，空方力道轉強，缺口永久失效"
```
