# 下降切線壓力與力道遞減規則

- **Rule ID**: R-SR-07
- **名稱**: 下降切線壓力與力道遞減規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C1-支撐壓力的重要觀念.md（下冊 p.027-028，空頭關鍵壓力空點來源之一）＋ P06-C2-走勢圖中的支撐壓力位置.md（下冊 p.057-060，6-2節「下降切線的壓力與支撐」）
- **解讀**: 與[[上升切線支撐與力道遞減規則]]結構完全對稱、方向相反。支撐壓力來源是連接兩個以上下降轉折高點所畫出的下降趨勢切線本身（延長線）。切線的具體畫法屬於切線/軌道線分類，取價方法請參照該分類的下降切線畫法規則（假定名稱 [[下降切線畫法與線不蓋線判準]]，見切線軌道線分類）。取這個價位的具體做法：股價位在切線**下方**時，將切線方程式延伸到當前日期，計算出的價位即為當日壓力參考價；股價反彈碰到此延長線時，該處提供壓力。**力道遞減規律**：切線第1次被股價反彈觸及時壓力最強；第2次觸及時壓力成功率約剩一半；第3次（以後）觸及則容易被突破——同樣為定性強弱排序，書中未給精確機率。轉換條件：下降切線壓力一旦被（收盤）突破，股價轉到切線上方，此後股價下跌碰到原切線位置時，切線角色反轉為支撐。是否真的止漲，仍須以當日K棒反應＋次日確認為準（見[[遇壓遇撐4法則支撐壓力有效性確認規則]]）。
- **可程式化**: 部分（切線本身的擬合演算法歸屬切線/軌道線分類；本規則只處理切線價位轉為壓力/支撐訊號與遞減強度的邏輯）
- **所需資料**: 下降切線方程式（斜率、截距，來自切線/軌道線分類規則）、每日OHLC、歷史觸碰次數計數
- **計算公式**:
```
trendline_price(t) = slope * t + intercept    # 由 [[下降切線畫法與線不蓋線判準]] 提供的切線方程式
```

- **參數**: 觸碰次數強度對照表（第1次=最強，第2次≈50%，第3次以後=弱，書中僅定性描述，未給精確機率）

- **信心**: 60/100 中（力道遞減規律清楚但為定性描述；切線擬合本身依賴切線/軌道線分類的實作品質）
