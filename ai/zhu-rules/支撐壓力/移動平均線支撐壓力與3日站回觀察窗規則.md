# 移動平均線支撐壓力與3日站回觀察窗規則

- **Rule ID**: R-SR-08
- **名稱**: 移動平均線支撐壓力與3日站回觀察窗規則
- **分類**: 支撐壓力
- **原文與頁碼**: P06-C1-支撐壓力的重要觀念.md（下冊 p.026-029）＋ P06-C2-走勢圖中的支撐壓力位置.md（下冊 p.060-068，6-2節「移動平均線的支撐與壓力」）
- **解讀**: 均線作為支撐／壓力的基本機制（股價站上上揚均線得支撐並助漲、跌破下彎均線得壓力並助跌，且天期越長、角度越大力道越強）已由[[均線上彎的支撐與助漲作用]]、[[均線下彎的壓力與助跌作用]]兩條規則定義，本規則不重複定義，只補充兩項均線支撐壓力特有、且屬於「支撐壓力篇」視角的規則：
  1. **均線支撐壓力力道排序**：週期越長支撐/壓力力道越強，即 60日均線 > 20日均線(月線) > 10日均線 > 5日均線。多頭股價回檔時容易先跌破5日線，回檔到月線（MA20）或季線（MA60）才經常出現真正有效的支撐；空頭反彈對稱，容易先突破5日線，反彈到月線或季線才經常遇到真正有效的壓力。
  2. **支撐/壓力轉換的時間觀察窗（3～5日規則，務必保留的精細規則）**：多頭回檔跌破月線（MA20）支撐後，若在**3個交易日內**重新收盤站回月線之上，且月線仍維持上揚，判定月線支撐依然有效，多頭趨勢未變，仍可持續操作；若超過3日（書中另有彈性提及3～5日）未能站回，判定月線支撐轉為壓力，多頭趨勢轉弱／反轉。空頭對稱：反彈突破月線壓力後，若3個交易日內重新收盤跌破月線，且月線仍下彎，判定月線壓力依然有效，空頭趨勢不變；若超過3日未能再跌破（即持續站上月線），判定月線壓力轉為支撐，空頭趨勢轉強／反轉。
- **可程式化**: 是
- **所需資料**: 收盤價、多組均線（MA5/MA10/MA20/MA60）及其逐日方向、跌破/突破月線的事件日期
- **計算公式**:
```
strength_rank = {MA60: 4, MA20: 3, MA10: 2, MA5: 1}   # 數字越大代表支撐/壓力力道越強

function ma_support_resistance_conversion(bars, ma_period=20, window_days=3):
    breach_date = None
    for t in range(len(bars)):
        MA_t = MA(bars.Close, ma_period)[t]

        if breach_date is None:
            if bars.Close[t-1] >= MA(ma_period)[t-1] and bars.Close[t] < MA_t:
                breach_date = t   # 記錄跌破月線的日期（多頭情境；空頭情境方向相反）
        else:
            days_elapsed = t - breach_date
            if bars.Close[t] >= MA(ma_period)[t] and MA_direction(ma_period, t) == "上揚":
                # 在 window_days 內收盤站回月線之上
                if days_elapsed <= window_days:
                    result = "月線支撐依然有效，多頭趨勢未變"
                else:
                    result = "逾期站回，仍判定月線支撐已轉弱一次，需觀察後續是否延續"
                breach_date = None   # 重置
            elif days_elapsed > window_days:
                result = "逾3日未站回，月線支撐轉為壓力，多頭趨勢轉弱/反轉"
                breach_date = None

    return result

# 空頭方向對稱：突破月線壓力後，3日內再跌破 -> 壓力依然有效；逾期未再跌破（持續站上）-> 壓力轉為支撐，空頭趨勢轉強/反轉
```
- **參數**: ma_period（預設20日/月線，也可套用其他天期）、window_days（觀察窗天數，預設3日，書中另有彈性提及3～5日，可調整為5）
- **可回測**: 是
- **信心**: 91/100 高（力道排序與3日觀察窗皆為書中明確給出的具體規則）
