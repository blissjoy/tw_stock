# 洗盤假跌破誘空進貨規則

- **Rule ID**: R-SR-20
- **名稱**: 洗盤假跌破誘空進貨規則
- **分類**: 支撐壓力
- **原文與頁碼**: 綜合依據 P02-C2-多頭趨勢總論.md（上冊 p.054-058，「多頭打底量價變化的觀察與鎖股」一節：第1支腳大量反彈、第2支腳大量反彈〔黃金右腳〕的打底邏輯，及「打底盤整中出現的大量，暫時視為主力的進貨量（也稱『草叢量』）」的描述）、P06-C2-走勢圖中的支撐壓力位置.md（下冊 p.035-039，轉折低點支撐與跌破後的角色互換邏輯，作為本規則「表面上看似跌破」的對照基礎）。書中在打底章節明確描述「主力進貨」「草叢量」的概念，但**沒有在支撐壓力篇獨立定義「洗盤」這個以跌破區間下緣為手法的通用規則**，本規則庫中 [[底部洗盤上攻大漲圖]]（經典型態總覽分類）也已在其「解讀」欄自行註明「現有規則庫沒有獨立命名的『洗盤』（假跌破誘空）通用規則」——本規則即補上這個底層定義，供該複合規則與其他型態規則引用。判定門檻（跌破量能倍數、拉回天數窗口）為工程補充，非書中逐字明訂數字。
- **解讀**: 洗盤（假跌破誘空）是主力在**底部打底或盤整已接近完成**時，刻意讓股價跌破區間下緣（前低或箱型下緣），造成散戶恐慌性停損賣出、製造浮額換手的假象，但主力本身其實在低點默默承接（低接浮額），隨後股價迅速拉回區間之上。其方向與意圖都與「假突破」相反：假突破通常是主力在高檔誘多後倒貨出貨的陷阱（見 [[突破高檔前高黑K出現大量要下跌圖]]），洗盤則是主力在低檔誘空承接的進貨手法——兩者K線表面現象都是「價格短暫穿越關鍵價位後又拉回」，但背後籌碼意圖相反，需分開定義，不能用同一組判斷條件套用。

  **判定條件（工程補充，比照書中打底邏輯與量價概念設計）**：
  1. **前置條件——須已具備打底/盤整架構**：股價須先出現空頭下跌後止跌打底的架構（呼應 P02-C2「第1支腳」「第2支腳（黃金右腳）」打底邏輯，可用 [[空轉多打底量價與均線排列鎖股訊號]] 或均線觀察輔助判斷），或至少已在 [[密集盤整區區間支撐壓力規則]] 定義的箱型區間內盤整一段時間。若沒有事先的打底/盤整架構，不適用洗盤判讀，應直接視為一般跌破。
  2. **跌破區間下緣**：股價跌破 [[轉折低點支撐與跌破後轉壓力規則]]（前低）或 [[密集盤整區區間支撐壓力規則]]（箱型下緣）定義的支撐位。
  3. **跌破當根K棒的籌碼特徵（與真跌破的關鍵區別）**：跌破當根量能不爆量、甚至量縮（不同於真跌破常見的爆大量長黑），或跌破時留有長下影線（盤中跌破但收盤未真正棄守，比照 [[突破跌破未站穩假突破假跌破通用判定規則]] 的「未站穩」判準）；打底/盤整期間若曾出現「草叢量」（見 [[成交量分類與倍數門檻定義]] 第10項），可作為主力已在低檔承接的輔助佐證。
  4. **快速拉回**：跌破後短期觀察窗內（工程補充，預設N=3個交易日）收盤即重新站回區間下緣之上，拉回速度較一般假跌破更快、更明確。
  5. **後續攻擊確認（洗盤特有，區別於單純假跌破）**：拉回站上區間之後，若伴隨攻擊量或爆大量向上突破區間上緣或前高（見 [[成交量分類與倍數門檻定義]]），才完整驗證主力洗盤進貨的意圖成立，可比照 [[底部洗盤上攻大漲圖]] 視為多頭買進訊號；若拉回後量能持續萎縮、遲遲不能突破前高，則洗盤意圖尚未驗證，僅能視為普通假跌破，不宜貿然視為做多訊號。

  **與 [[突破跌破未站穩假突破假跌破通用判定規則]] 的關係**：洗盤在K線/價格層面可視為「假跌破」的一個特殊子類型，差別在於洗盤額外要求（a）發生在打底/盤整接近完成的位置、（b）跌破當根有主力承接的量能佐證（量縮或草叢量）、（c）拉回後常接續攻擊性突破——這些額外條件是洗盤與「一般假跌破」的區分依據，用來排除單純誘空出貨或雜訊震盪的可能性。

- **可程式化**: 部分（打底架構的自動辨識、草叢量的量化門檻依賴 [[空轉多打底量價與均線排列鎖股訊號]] 與 [[成交量分類與倍數門檻定義]] 前置模組是否準確；跌破/拉回本身的價量條件可完全程式化）
- **所需資料**: 前低或箱型下緣價位、打底/盤整期OHLCV序列、跌破當根OHLCV、近N日均量、拉回後續K棒OHLCV
- **計算公式**:
```
function detect_wash_out_breakdown(bars, support_price, base_confirmed, N=3, K_vol=1.0):
    # support_price: 前低或箱型下緣，見 [[轉折低點支撐與跌破後轉壓力規則]] / [[密集盤整區區間支撐壓力規則]]
    # base_confirmed: 是否已具備打底/盤整架構（呼應P02-C2打底邏輯，可用 [[空轉多打底量價與均線排列鎖股訊號]] 判斷）
    if not base_confirmed:
        return "不適用洗盤判讀（尚無打底/盤整架構前置條件），依一般跌破處理"

    breakdown_day = first day where bars.Low[t] < support_price or bars.Close[t] < support_price
    if breakdown_day is None:
        return None

    vol_signature = bars.Volume[breakdown_day] <= K_vol * MA5_vol[breakdown_day]   # 量縮或不爆量
    long_lower_shadow = has_long_lower_shadow(bars[breakdown_day])
    prior_bush_vol = any(is_bush_vol[t] for t in 打底期間)   # 見 [[成交量分類與倍數門檻定義]] 第10項

    window = bars[breakdown_day : breakdown_day + N]
    rebound_back_above = any(day in window where day.Close > support_price)

    if not rebound_back_above:
        return "尚未拉回，暫視為真跌破，套用 [[轉折低點支撐與跌破後轉壓力規則]]（支撐轉壓力）"

    if vol_signature or long_lower_shadow or prior_bush_vol:
        signal = "疑似洗盤（假跌破誘空）：跌破區間下緣但量縮/收長下影線/打底期已見草叢量，主力可能低接承接"
        # 後續攻擊確認
        if 拉回後出現攻擊量或爆大量突破區間上緣或前高（見 [[成交量分類與倍數門檻定義]]）:
            return signal + "；且拉回後爆量突破前高，洗盤進貨意圖確認，套用 [[底部洗盤上攻大漲圖]] 做多訊號"
        else:
            return signal + "；尚未出現後續攻擊性突破，洗盤意圖未完全驗證，僅列為觀察"
    else:
        return "跌破後雖有拉回，但無明顯主力承接量能特徵，依 [[突破跌破未站穩假突破假跌破通用判定規則]] 列為一般假跌破，不預設為洗盤"
```
- **參數**: N＝拉回觀察窗天數（工程補充，預設3日）；K_vol＝跌破當根量能相對MA5的倍數上限（工程補充，預設1.0倍，即不超過均量視為量縮/不爆量）；草叢量、攻擊量/爆大量倍數門檻沿用 [[成交量分類與倍數門檻定義]]
- **參考來源查證（2026-07-23，工程補充）**：查證外部資料，「洗盤/假跌破」的拉回天數與量縮倍數查無以本書用語為準的公認統一數字，但找到相近情境的參考區間：「股價短線拉回不跌破第一防守線，屬較強勢整理，維持時間通常3～6天之內」（來源：[Chief PaPa洗盤教學](https://chiefpapalearning.com/stock-shakeout-guide/)），本規則採用的N=3日落在此區間下緣，方向合理但非直接針對「洗盤觀察窗」的逐字佐證，K_vol=1.0倍量縮門檻則仍無外部數字可引用，維持工程估計值。
- **可回測**: 是
- **信心**: 30/100 低（核心「主力打底進貨、跳空跌破誘空洗盤」的概念在書中打底章節與草叢量描述中有明確文字支持，方向可信；但「洗盤」本身作為一個獨立判定規則的量化門檻——跌破當根量能上限、拉回天數窗口、後續攻擊確認的量能倍數——書中並未給出，皆屬工程補充，且洗盤與真跌破在事前difficult以量價特徵完全區分，實務上仍常須靠事後反彈速度與後續是否攻擊突破回溯確認，事前預測把握度較低）
- **底層規則依賴**: 轉折低點支撐與跌破後轉壓力規則、密集盤整區區間支撐壓力規則、突破跌破未站穩假突破假跌破通用判定規則、成交量分類與倍數門檻定義、空轉多打底量價與均線排列鎖股訊號、底部洗盤上攻大漲圖
