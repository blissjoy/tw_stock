# 低檔變盤線

- **Rule ID**: R-CANDLE-13
- **名稱**: 低檔變盤線
- **分類**: K棒型態
- **原文與頁碼**: P03-C3-K線低檔反轉3大型態訊號.md（上冊 p.189-191；原書3-3節「一、低檔變盤線的轉折變化」）
- **解讀**: 與高檔變盤線（R-CANDLE-05）完全對稱的低檔版本。股價下跌到低檔，出現書中列出的8種變盤K線之一：墓碑線、長T線（長下影線）、十字線、紡錘線、反鎚線、吊人線、漲停線、長紅線，視為止跌訊號，須持續觀察是否反轉。空頭趨勢的低檔出現，視為轉折向上反彈的變盤線；多頭趨勢的回檔低檔出現，視為準備做多的訊號。低檔爆出大量或窒息量，且出現價量背離現象、KD指標黃金交叉，更容易反轉上漲；下跌走勢遇到支撐位置時出現變盤線，反轉走升機率高。值得注意：低檔變盤訊號不一定伴隨大量出現，須以「次日股價是否上漲」作為確認條件，量能不是必要確認變數（這點與高檔變盤線的加分條件敘述方式略有差異）。書中低檔清單以「吊人線」「漲停線」「長紅線」取代高檔清單的「槌子線」「跌停線」「長黑線」，屬原書用詞差異，本規則忠實保留，不另行更動。
- **可程式化**: 部分
- **所需資料**: 開盤價、收盤價、最高價、最低價、成交量、KD指標、股價相對高低檔位置
- **計算公式**:
```
# 沿用 R-CANDLE-05 的操作型定義邏輯（鏡射版），
# 紡錘線見 R-CANDLE-24，槌子/反鎚線見 R-CANDLE-25，
# 其餘（十字線、墓碑線、長T線、漲停線、長紅線）書中未給精確數值門檻，需自訂：
is_doji       = abs(close - open) / open < DOJI_THRESHOLD
is_gravestone = is_doji_like and upper_shadow大 and lower_shadow趨近0
is_long_T     = is_doji_like and lower_shadow大 and upper_shadow趨近0
is_limit_up   = (close - prev_close) / prev_close >= LIMIT_UP_PCT
is_long_red   = is_long_red_candle(t)   # 見 R-CANDLE-21

is_reversal_candle_low = is_doji or is_gravestone or is_long_T or \
                          is_spindle(t) or is_hammer(t) or is_inverted_hammer(t) or \
                          is_limit_up or is_long_red

if 位於低檔 and is_reversal_candle_low:
    signal = "止跌訊號，須觀察次日確認"
    if 位於空頭趨勢: signal = "轉折向上反彈的變盤線"
    if 位於多頭回檔低檔: signal = "準備做多的訊號"
    if (大量 or 窒息量) and 價量背離 and KD黃金交叉:
        signal_strength = "反轉機率大增"
    # 量能非必要條件，最終仍須以次日股價是否上漲作為確認
```
- **參數**: DOJI_THRESHOLD（建議<1%）、LIMIT_UP_PCT（漲停幅度，依當時台股漲跌幅制度）
- **參考來源查證（2026-07-22，工程補充）**：與R-CANDLE-05同批查證，台股現行漲跌幅限制**10%**（2015-06-01起，來源：[Tsung's Blog](https://blog.longwin.com.tw/2018/02/taiwan-stock-change-limit-7-10-2018/)），LIMIT_UP_PCT採10%；DOJI_THRESHOLD查無外部統一慣例，採0.5%工程估計值。
- **可回測**: 是
- **信心**: 55/100 中
