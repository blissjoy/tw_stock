# 紅K線

- **Rule ID**: R-CANDLE-21
- **名稱**: 紅K線（陽線）分類與位置判讀
- **分類**: K棒型態
- **原文與頁碼**: P03-C1-K線起源與基本概念.md（上冊 p.125-128）；P03-C4-不同型態的單一K線.md（上冊 p.213-217，原書3-4節「一、實體紅K線」）
- **解讀**: 開盤價 < 收盤價 → K線為紅色（陽線）。書中用「開盤到收盤的漲幅百分比」（(收盤-開盤)/開盤價）分類紅K實體大小，**不是**用實體占最高減最低全距的比例：漲幅>6.5%為長紅K（多方力道最強）；3.5%～6.5%為中紅K（多方力道強）；<3.5%為小紅K（多方力道小強）。紅K在不同趨勢位置意義不同：出現在底部打底期間代表主力接籌碼，尚未立刻續攻；底部完成向上突破盤整時通常配合攻擊量，是多頭確認訊號；多頭上漲行進中代表多方氣勢不墜，常見惜售量縮；上漲回檔修正完成後（未跌破前低）出現紅K，代表修正結束、可把握買進機會；多頭上漲一段後的盤整末端出現大量紅K突破盤整，是攻擊訊號；多頭高檔或連續急漲後的紅K要注意成交量爆增、價量背離，可能是主力誘多出貨；多頭高檔盤頭區間出現大量紅K，是主力分批出貨訊號。空頭下跌行進中的反彈紅K代表主力誘多，不可搶進，若反彈無量則更不易反彈；空頭低檔或急跌後出現紅K止跌，或紅K突破前一根紅K最高點，視為反彈信號。
- **可程式化**: 部分
- **所需資料**: 開盤價、收盤價、成交量、股價相對趨勢位置（打底/多頭起漲/多頭行進/回檔/高檔/盤頭/空頭反彈/空頭低檔）
- **計算公式**:
```
is_red = close > open
body_pct = (close - open) / open   # 僅在紅K時適用

if is_red:
    if body_pct > 0.065:
        size = "長紅K"; strength_note = "多方力道最強"
    elif body_pct >= 0.035:
        size = "中紅K"; strength_note = "多方力道強"
    else:
        size = "小紅K"; strength_note = "多方力道小強"

# 位置判讀（需外部趨勢位置模組輸入 position）
interpretation = {
    "底部打底期間": "主力接籌碼，量增但尚不會立刻續攻",
    "底部突破盤整": "配合攻擊量，多頭確認訊號",
    "多頭上漲行進中": "多方氣勢不墜，常見惜售量縮",
    "回檔修正完成(未破前低)": "修正結束，把握買進機會",
    "多頭盤整末端大量突破": "攻擊訊號，把握買進",
    "多頭高檔或連續急漲": "注意價量背離，警惕主力誘多出貨",
    "多頭高檔盤頭區間大量": "主力分批出貨訊號",
    "空頭下跌反彈": "主力誘多，不可搶進；若無量更不易反彈",
    "空頭低檔或急跌後止跌": "反彈信號",
}[position]
```
- **參數**: 長紅K門檻>6.5%、中紅K門檻3.5%～6.5%、小紅K門檻<3.5%（皆為固定漲幅%門檻，非實體占比）
- **可回測**: 是
- **信心**: 63/100 中（實體大小分類門檻精確、信心高；趨勢位置判讀屬定性描述且依賴外部趨勢模組，故整體評為中）
