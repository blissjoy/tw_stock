# 高檔長黑貫穿

- **Rule ID**: R-CANDLE-11
- **名稱**: 高檔長黑貫穿（一路向下）
- **分類**: K棒型態
- **原文與頁碼**: P03-C2-K線高檔反轉3大型態訊號.md（上冊 p.179後段-182；原書3-2節「二」型態6「高檔『長黑貫穿』」）
- **解讀**: 由2根K線構成，中長紅（Day1）＋次日長黑（Day2）。黑K開盤即下跌（非開高，即Day2開盤價不需高於Day1收盤價，可直接低走），收盤跌破前一日中長紅K實體的最低點。與「長黑吞噬」（R-CANDLE-10）的關鍵差異：吞噬是「開高走低」（Day2開盤 ≥ Day1收盤），貫穿是「開盤即下跌」（不需開高，直接下跌破底）；但兩者共同點都是收盤跌破前一日最低點，同屬全書描述的「反轉最強」訊號。貫穿當日或前一日出現大量，反轉訊號越強。短線連續上漲或急漲、獲利達15%以上時出現長黑貫穿，單日反轉機率很高，建議持有多單「立刻」出場（措辭比長黑吞噬「出場或減碼一半」更急迫）。若1根黑K一次貫穿並跌破前面2～3根K線低點（3線反黑），反轉力道更強。高檔爆量長黑貫穿後下跌，日後再上漲到當初貫穿K線價位會形成重大壓力，屬多單逃命波不可追多，容易形成「頭頭低」頭部型態。
- **可程式化**: 是
- **所需資料**: Day1、Day2的開盤價、收盤價、最低價、成交量、前2～3根K線的最低價、近期漲幅
- **計算公式**:
```
is_open_not_higher = open[2] <= close[1]     # 開盤未開高（直接下跌或平開走低）
is_break_low        = close[2] < low[1]        # 收盤跌破前一日最低點

is_pattern = is_red(day1) and is_black(day2) and 位於高檔 and \
             is_open_not_higher and is_break_low

if is_pattern:
    signal = "高檔長黑貫穿，反轉最強訊號之一（一路向下）"
    if 貫穿當日或前一日大量:
        signal += "，反轉訊號更強"
    if 近期漲幅 >= 0.15:
        action = "建議持有多單立刻出場"    # 比長黑吞噬更急迫
    if close[2] < min(low[t-1], low[t-2], low[t-3]):
        signal += "，屬於3線反黑，反轉力道更強"
    if 日後反彈碰到貫穿K線價位:
        note = "形成重大壓力，逃命波不可追多，容易形成頭頭低頭部型態"
```
- **參數**: 15%漲幅門檻、3線反黑比較根數（預設2～3根）
- **可回測**: 是
- **信心**: 88/100 高
