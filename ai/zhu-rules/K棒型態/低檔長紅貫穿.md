# 低檔長紅貫穿

- **Rule ID**: R-CANDLE-19
- **名稱**: 低檔長紅貫穿（一路向上）
- **分類**: K棒型態
- **原文與頁碼**: P03-C3-K線低檔反轉3大型態訊號.md（上冊 p.203-205；原書3-3節「二」(6)「低檔『長紅貫穿』」）
- **解讀**: 2根K線；下跌走勢到低檔，出現轉折向上的「貫穿」紅K線。當日開高走高，收盤突破前一日黑K的最高點（比「長紅吞噬」更強——吞噬只要求覆蓋前一日實體，貫穿要求突破前一日整根K線含影線的最高點）。意義為多方當天向上貫穿，多空主控權易位。貫穿當日或前一日出現大量，反轉訊號越強。短線連續下跌或急跌、跌幅達15%以上時出現長紅貫穿K線，1日反轉機率很高，持有空單應該立刻回補。擴展型態「3線反紅」：1根紅K一次貫穿突破前面2～3根K線的高點，反轉力道更強。低檔爆量長紅貫穿K線後反彈，日後再下跌到長紅貫穿K線位置會形成重大支撐，再下跌是空單逃命波，不可追空，容易形成「底底高」的底部型態。
- **可程式化**: 是
- **所需資料**: Day1、Day2的開盤價、收盤價、最高價、成交量、前2～3根K線的最高價、近期跌幅
- **計算公式**:
```
is_open_high = open[2] >= close[1]     # 開高走高（非開低）
is_break_high = close[2] > high[1]       # 收盤突破前一日最高點（含影線）

is_pattern = is_black(day1) and is_red(day2) and 位於低檔 and is_open_high and is_break_high

if is_pattern:
    signal = "低檔長紅貫穿，最強反轉訊號之一（一路向上）"
    if 貫穿當日或前一日大量:
        signal += "，反轉訊號更強"
    if 近期跌幅 >= 0.15:
        action = "持有空單應立刻回補"
    if close[2] > max(high[t-1], high[t-2], high[t-3]):
        signal += "，屬於3線反紅，反轉力道更強"
    if 日後回檔到貫穿K線位置:
        note = "形成重大支撐；再下跌是空單逃命波，不可追空，容易形成底底高底部型態"
```
- **參數**: 15%跌幅門檻、3線反紅比較根數（預設2～3根）
- **可回測**: 是
- **信心**: 88/100 高
