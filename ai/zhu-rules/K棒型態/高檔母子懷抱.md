# 高檔母子懷抱

- **Rule ID**: R-CANDLE-09
- **名稱**: 高檔母子懷抱（不懷好意）
- **分類**: K棒型態
- **原文與頁碼**: P03-C2-K線高檔反轉3大型態訊號.md（上冊 p.174後段-176前段；原書3-2節「二」型態4「高檔『母子懷抱』」）
- **解讀**: 由2根K線構成，上漲高檔出現中長紅K（稱「母線」），次日出現不創新高也不破低的黑K線（稱「子線」，完全被母線的高低點範圍包覆）。子線可以是任意顏色/大小的K棒，只要求「不過高不破低」。上漲到明顯高檔出現母子懷抱組合，代表上漲走勢突然變不確定，多空力量拉鋸、多頭力道減弱，方向可能轉折向下，是一組止漲K線訊號，須注意次日是否轉折向下。若子線本身是變盤線（十字線、紡錘線等），反轉力道大於一般母子懷抱，屬強力反轉訊號（近似「十字孕線」的加強版）。
- **可程式化**: 是
- **所需資料**: Day1（母線）、Day2（子線）的最高價、最低價、開盤價、收盤價
- **計算公式**:
```
is_engulfed = high[2] <= high[1] and low[2] >= low[1]   # 子線完全被母線高低點包覆

is_pattern = is_red(day1) and 位於高檔 and is_engulfed

if is_pattern:
    signal = "高檔母子懷抱，止漲訊號，須觀察次日是否轉折向下"
    if is_reversal_candle(day2):    # 子線本身為變盤線（十字線/紡錘等，見R-CANDLE-05/24）
        signal = "強力反轉訊號（子線為變盤線的加強版）"
```
- **參數**: 無額外參數
- **可回測**: 是
- **信心**: 88/100 高
