# 低檔母子懷抱

- **Rule ID**: R-CANDLE-17
- **名稱**: 低檔母子懷抱（光明在望）
- **分類**: K棒型態
- **原文與頁碼**: P03-C3-K線低檔反轉3大型態訊號.md（上冊 p.199-201；原書3-3節「二」(4)「低檔『母子懷抱』」）
- **解讀**: 2根K線構成，下跌低檔出現中長黑K線（母線），次日出現不過高也不破低的紅K線（子線，高低點皆被母線實體/區間包住），子線可以是十字線／變盤線變體（母子懷抱＋十字）。意義為多空雙方力量突然呈現拉鋸，空頭下跌力道減弱，方向可能轉折向上，屬止跌訊號，須觀察次日是否轉折向上。母線的最高點被突破即為反彈確認。成交量條件：母量須大於子量（量能遞減），若出現「價漲量縮」背離，須次日放量上漲才能化解背離，否則不易反轉成功、次日容易下跌。母子懷抱2根K線的最高點與最低點是重要觀察點：向上突破最高點代表多空易位、多方轉為主動掌控；向下跌破最低點代表空方繼續主導下跌。次日開盤位置是關鍵：開高容易向上反轉，開低容易下跌。若子線本身是「變盤線」型態，反轉機率大於一般母子懷抱型態，很容易發展成「低檔晨星轉折」的3根K線組合（母子懷抱是晨星型態的雛形之一，見R-CANDLE-20）。
- **可程式化**: 是
- **所需資料**: Day1（母線）、Day2（子線）的最高價、最低價、開盤價、收盤價、成交量
- **計算公式**:
```
is_engulfed  = high[2] <= high[1] and low[2] >= low[1]   # 子線完全被母線高低點包覆
volume_shrink = volume[2] < volume[1]                      # 母量>子量

is_pattern = is_black(day1) and 位於低檔 and is_engulfed

if is_pattern:
    signal = "低檔母子懷抱，止跌訊號，須觀察次日是否轉折向上"
    if high[t] > high[1]:
        signal = "反彈確認（母線最高點被突破）"
    if not volume_shrink:  # 價漲量縮的反向情形，即量未遞減
        note = "須注意背離；若母量未大於子量，須次日放量上漲才能化解背離，否則不易反轉，次日容易下跌"
    if is_reversal_candle(day2):
        signal = "強力反轉訊號，可能發展為低檔晨星轉折（見 R-CANDLE-20）"
    if open[t] > close[2]:
        outlook = "開高，容易向上反轉"
    elif open[t] < close[2]:
        outlook = "開低，容易下跌"
```
- **參數**: 無額外參數
- **可回測**: 是
