# 黑K線

- **Rule ID**: R-CANDLE-22
- **名稱**: 黑K線（陰線）分類與位置判讀
- **分類**: K棒型態
- **原文與頁碼**: P03-C1-K線起源與基本概念.md（上冊 p.125-128）；P03-C4-不同型態的單一K線.md（上冊 p.218-221，原書3-4節「二、實體黑K線」）
- **解讀**: 開盤價 > 收盤價 → K線為黑色（陰線）。原文特別提醒：出現黑K線並不表示日後一定會繼續下跌，解讀仍須看趨勢位置。與紅K完全對稱，用「開盤到收盤跌幅%」分類：跌幅>6.5%為長黑K（空方力道最強）；3.5%～6.5%為中黑K（空方力道強）；<3.5%為小黑K（空方力道小強）。空頭低檔打底盤整期間出現黑K，代表底部尚未確認，可能是主力再洗盤，不可認為股價跌深而做多；多頭向上行進中出現黑K，通常是漲多賣壓，若多頭趨勢未變，屬正常回檔；多頭上漲到高點出現黑K，是止漲訊號，若伴隨大量或爆天量，主力出貨可能性大，須密切注意次日走勢。高檔出現頭頭低、第2個頭出現長黑K下跌，一般容易轉成空頭，有量下跌威力較大、無量也會下跌；空頭初跌段出現黑K通常出量、行情繼續下跌；空頭下跌行進中出現黑K且無止跌訊號，無論是否有量都不能低接；連續長黑K後若出現爆量或窒息量，容易反彈，當紅K收盤過前一日高點可做短線搶反彈；空頭下跌盤整中黑K跌破盤整，是做空機會。
- **可程式化**: 部分
- **所需資料**: 開盤價、收盤價、成交量、股價相對趨勢位置
- **計算公式**:
```
is_black = close < open
body_pct = (open - close) / open   # 僅在黑K時適用

if is_black:
    if body_pct > 0.065:
        size = "長黑K"; strength_note = "空方力道最強"
    elif body_pct >= 0.035:
        size = "中黑K"; strength_note = "空方力道強"
    else:
        size = "小黑K"; strength_note = "空方力道小強"

interpretation = {
    "空頭低檔打底盤整": "底部尚未確認，可能主力再洗盤，不可視為止跌做多",
    "多頭上漲行進中": "漲多賣壓，趨勢未變則屬正常回檔",
    "多頭高點": "止漲訊號，若大量/爆天量須警惕主力出貨，密切注意次日",
    "高檔頭頭低第2個頭長黑": "容易轉成空頭，有量或無量皆可能續跌",
    "空頭初跌段": "通常出量，行情繼續下跌",
    "空頭下跌行進中無止跌訊號": "不論是否有量，都不能低接",
    "連續長黑後爆量或窒息量+隔日紅K收盤過前高": "短線搶反彈訊號",
    "空頭下跌盤整跌破": "做空機會",
}[position]
```
- **參數**: 長黑K門檻>6.5%、中黑K門檻3.5%～6.5%、小黑K門檻<3.5%（固定跌幅%門檻，非實體占比）
- **可回測**: 是
- **信心**: 63/100 中（實體大小分類門檻精確、信心高；趨勢位置判讀屬定性描述且依賴外部趨勢模組，故整體評為中）
