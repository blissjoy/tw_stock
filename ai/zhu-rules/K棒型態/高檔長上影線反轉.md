# 高檔長上影線反轉

- **Rule ID**: R-CANDLE-36
- **名稱**: 高檔長上影線反轉
- **分類**: K棒型態
- **原文與頁碼**: 主要對應 `P12-C4a-15種K線多轉空祕笈圖.md` 第6種「高檔大量長上影線反轉圖」（下冊 p.381-382，圖表12-4-13、12-4-14）與第8種「連3天上漲長上影線 大敵當前主力出貨圖」（下冊 p.383-384，圖表12-4-17、12-4-18）；型態外觀的上影線倍數門檻比照 `P03-C4-不同型態的單一K線.md`「四、變盤槌子與倒鎚K線」（上冊 p.222-225，重點說明第2點「上下影線至少是實體2倍以上」）工程延伸；「高檔」的相對位置定義比照 `P03-C2-K線高檔反轉3大型態訊號.md`（上冊 p.161、p.164、p.183，三節開頭皆重申「高檔＝多頭趨勢上漲高點，或空頭趨勢反彈相對高點」）。
- **解讀**: 高檔（多頭趨勢的上漲高點，或空頭趨勢中反彈的相對高點）出現實體小、上影線長的K棒（顏色不拘），代表當日多方一度攻高，但賣壓在盤中重新把股價壓回，是短線反轉／滯漲警訊。若當日同時爆大量，反轉訊號更強烈（即「高檔大量長上影線反轉」）。操作SOP與高檔變盤線／長黑吞噬一致，有兩種變化：①當天先減碼二分之一部位；②隔日若收盤跌破當日K棒（或其大量區間）最低點，視為一日反轉確認，賣出剩餘部位。若同一壓力區連續3天都出現長上影線（每天上攻都失敗、無法站穩當日最高點），屬於同一K線家族的加強版警訊，代表主力連續在同一價位調節出貨，之後常接續出現「頭頭低」正式轉為空頭確認。
  **與既有規則的區別**：「槌子線與倒槌K線」(R-CANDLE-25) 是以形狀（下影線/上影線≥2倍實體）定義的通用規則，同時涵蓋高檔與低檔兩種情境、以「次日跳空大跌或跌破最低點」作確認條件；本規則專門補強「高檔」情境下的量能加權與出場SOP（先賣一半、隔日確認出清），並延伸涵蓋「連續多日同一價位反覆受挫」的加強版態樣，兩者互補而非重複定義。
- **可程式化**: 部分（單根K線形狀與量能條件可程式化；「連續同一壓力區」的比對需仰賴壓力位置規則，精確度中等）
- **所需資料**: 開盤價、收盤價、最高價、最低價、成交量、股價相對高低檔位置、近期壓力位置（用於連續3日加強版）
- **計算公式**:
```
body = abs(close - open)
upper_shadow = high - max(open, close)
lower_shadow = min(open, close) - low

is_long_upper_shadow = upper_shadow >= 2 * body and upper_shadow > lower_shadow   # 工程預設門檻，非書中原文明訂倍數

if is_long_upper_shadow and 位於高檔（多頭高點或空頭反彈高點）:
    signal = "高檔長上影線，短線反轉/滯漲警訊"
    if volume(t) 為大量:
        signal += "，反轉機率更高（高檔大量長上影線反轉）"
    action_A = "當天先賣出二分之一部位"
    if close[t+1] < low(t):
        action_B = "隔日跌破當日低點，一日反轉確認，賣出剩餘部位"

# 加強版：連續3天同一壓力區反覆出現長上影線
count = 0
for k in 最近3根上漲K棒:
    if upper_shadow(k) >= 2 * body(k) and near(high(k), 近期壓力位置):
        count += 1
if count == 3:
    signal = "連3日長上影，大敵當前，主力出貨警訊"
```
- **參數**: 上影線/實體倍數門檻（工程補充預設值2倍，比照槌子/倒槌定義風格，本型態原文本身未給精確數字）、大量門檻（書中未給精確值，建議1.5～2倍N日均量）、壓力區判定容忍範圍（工程補充）
