# 槌子線與倒槌K線

- **Rule ID**: R-CANDLE-25
- **名稱**: 槌子線與倒槌K線
- **分類**: K棒型態
- **原文與頁碼**: P03-C4-不同型態的單一K線.md（上冊 p.222-225；原書3-4節「四、變盤槌子與倒鎚K線」）
- **解讀**: 書中把槌子/倒槌依「顏色 x 形狀」分成4種（變盤紅K槌子、變盤黑K槌子、變盤紅K倒槌、變盤黑K倒槌），但**判斷邏輯完全相同**，圖說一致寫著「高檔易下跌，低檔易反彈」——即不因顏色或槌子/倒槌形狀不同而給予不同的多空傾向，真正決定意義的是「出現的價格位階」。外型特徵：小實體在上方或下方（槌子＝下影線長、實體在上方；倒槌＝上影線長、實體在下方），實體的紅黑不重要；下影線或上影線很長，**至少是實體的2倍以上**。強勢上漲到高點出現槌子或倒槌，次日走勢非常重要：次日向下跳空大跌，多頭結構可能反轉；次日收盤跌破槌子或倒槌線的最低點，轉折向下確認。急速下跌到低點出現槌子或倒槌，次日向上跳空大漲，空頭結構可能反轉；次日收盤突破槌子或倒槌線的最高點，轉折向上確認。若在槌子或倒槌線出現大成交量，轉折的機率更高。**重要提醒**：書中沒有十字線/墓碑線/仙人指路這些西方獨立命名，是用「槌子/倒槌」+「高檔/低檔位階」+「次日確認動作」判斷，程式化時不應套用西方陰陽線命名法分開處理。
- **可程式化**: 是
- **所需資料**: 開盤價、收盤價、最高價、最低價、成交量、次日開盤價、次日收盤價、股價相對高低檔位置
- **計算公式**:
```
body = abs(close - open)
upper_shadow = high - max(open, close)
lower_shadow = min(open, close) - low

is_hammer = lower_shadow >= 2 * body and upper_shadow < lower_shadow      # 槌子：下影線長，顏色不拘
is_inverted_hammer = upper_shadow >= 2 * body and lower_shadow < upper_shadow  # 倒槌：上影線長，顏色不拘

if is_hammer or is_inverted_hammer:
    if 位於高檔（強勢上漲後）:
        if open[t+1] 向下跳空大跌:
            signal = "多頭結構可能反轉（強確認）"
        elif close[t+1] < low(candle):
            signal = "轉折向下確認"
        if volume(candle) 為大量:
            signal += "，轉折機率更高"
    elif 位於低檔（急速下跌後）:
        if open[t+1] 向上跳空大漲:
            signal = "空頭結構可能反轉（強確認）"
        elif close[t+1] > high(candle):
            signal = "轉折向上確認"
        if volume(candle) 為大量:
            signal += "，轉折機率更高"
```
- **參數**: 影線/實體倍數門檻（固定為2倍，書中明訂）
- **可回測**: 是
- **信心**: 88/100 高
