# 高檔長黑吞噬

- **Rule ID**: R-CANDLE-10
- **名稱**: 高檔長黑吞噬（主力出貨）
- **分類**: K棒型態
- **原文與頁碼**: P03-C2-K線高檔反轉3大型態訊號.md（上冊 p.175後段-179前段；原書3-2節「二」型態5「高檔『長黑吞噬』」）
- **解讀**: 由2根K線構成，中長紅（Day1）＋次日長黑（Day2），黑K實體必須完全吞噬前一日紅K的實體（Day2開盤價 ≥ Day1收盤價 且 Day2收盤價 ≤ Day1開盤價）。當日開高走低，收盤跌破前一日最低點，賣壓出籠、多空主控權易位，是本節6型態中最強的反轉訊號之一（與長黑貫穿並列最強）。吞噬當日或前一日出現大量，反轉訊號越強。若短線連續上漲或急漲、獲利已達15%以上時出現長黑吞噬，單日反轉機率很高，建議持有多單出場或至少減碼一半。若1根黑K一次吞噬並跌破前面2～3根K線的低點，反轉力道更強，書中稱為「3線反黑」。高檔爆量長黑吞噬後下跌，日後若再上漲碰到當初吞噬K線的價位，會形成重大壓力（多單逃命波，不可追多），容易形成「頭頭低」頭部型態。
- **可程式化**: 是
- **所需資料**: Day1、Day2的開盤價、收盤價、最低價、成交量、前2～3根K線的最低價、近期漲幅
- **計算公式**:
```
is_engulf = open[2] >= close[1] and close[2] <= open[1]   # 黑K實體完全吞噬紅K實體
is_break_low = close[2] < low[1]                            # 收盤跌破前一日最低點

is_pattern = is_red(day1) and is_black(day2) and 位於高檔 and is_engulf and is_break_low

if is_pattern:
    signal = "高檔長黑吞噬，最強反轉訊號之一（主力出貨）"
    if 吞噬當日或前一日大量:
        signal += "，反轉訊號更強"
    if 近期漲幅 >= 0.15:
        action = "建議持有多單出場或至少減碼一半"
    if close[2] < min(low[t-1], low[t-2], low[t-3]):   # 一次跌破前2~3根低點
        signal += "，屬於3線反黑，反轉力道更強"
    if 日後反彈碰到吞噬K線價位:
        note = "形成重大壓力，逃命波不可追多，容易形成頭頭低頭部型態"
```
- **參數**: 15%漲幅門檻（獲利判定基準）、3線反黑比較根數（預設2～3根）
- **可回測**: 是
- **信心**: 88/100 高
