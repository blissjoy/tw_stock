# 低檔晨星

- **Rule ID**: R-CANDLE-20
- **名稱**: 低檔晨星
- **分類**: K棒型態
- **原文與頁碼**: P03-C3-K線低檔反轉3大型態訊號.md（上冊 p.205-210；原書3-3節「三、低檔晨星組合K線的轉折變化」）
- **解讀**: 3根K線：第1根（左）為中長黑K線，第2根（中）為變盤線（可以是十字線或其他變盤K線，且中間可延伸為多根小K線），第3根（右）為中長紅K線。標準型態即「黑K→變盤線→紅K」，書中列出6種變化組合（孤島晨星、母子變盤、雙星變盤、雙肩變盤、群星變盤，另有標準型態本身）。下跌低檔出現此組合，是低檔轉折向上確認的K線組合。晨星組合同時爆大量，轉折向上機率大大提高且上漲力道強；中間夾雜的變盤線數量越多，之後中長紅K線上漲時，多頭控盤的時間也越長。**關鍵失效條件**：晨星轉折組合，若上漲那根中長紅K線的低點被後續下跌的黑K線跌破，轉折向上的結構就被破壞，向上多方力道轉為向下由空方主控——此現象經常發生在空頭反彈的「晨星」中，須以「第3根紅K最低點不被後續K線跌破」作為型態有效性的判定標準，不能出現晨星就直接視為底部確認。書中範例中有2例（圖表3-3-29、3-3-33）明確出現「晨星轉折後續走空頭確認」的反轉失敗案例，程式化時應把這兩例當成失效條件的直接證據。
- **可程式化**: 部分
- **所需資料**: 左黑K、中段變盤線（1根以上）、右紅K的開盤價、收盤價、最高價、最低價、成交量、後續K棒的收盤/最低價
- **計算公式**:
```
# 中段變盤線的精確辨識依賴 R-CANDLE-13／R-CANDLE-24／R-CANDLE-25 等單根K線變盤定義
left   = 中長黑K，位於低檔
middle = 1根以上「變盤線」（十字線/紡錘線/墓碑線/長T線等止跌型K棒）
right  = 中長紅K

is_pattern = is_mid_long_black(left) and all(is_reversal_candle(k) for k in middle) and \
             is_mid_long_red(right) and 位於低檔

if is_pattern:
    signal = "低檔晨星，轉折向上確認"
    if 晨星組合同時爆大量:
        signal += "，轉折向上機率大增，上漲力道強"
    if len(middle) > 1:
        signal += "，中段變盤天數越多，多頭控盤時間越長"

    # 型態有效性須持續監控
    if any(low[k] < low[right] for k in 後續K棒):
        signal = "型態失效：紅K低點被後續黑K跌破，結構轉為空方主控（常見於空頭反彈中的晨星）"
```
- **參數**: 中段變盤線根數（不固定，1根以上）
- **可回測**: 是
- **信心**: 63/100 中（3根整體結構與失效條件定義明確，信心高；但中段「變盤線」的精確辨識依賴其他未完全量化的單根K線定義，故整體評為中）
