# 前一日高低點支撐壓力規則

- **Rule ID**: R-CANDLE-01
- **名稱**: 前一日高低點支撐壓力規則
- **分類**: K棒型態
- **原文與頁碼**: P03-C1-K線起源與基本概念.md（上冊 p.138、p.146；原書3-1節「K線基本概念」之「收盤價的意義」與「K線的支撐壓力與強弱的改變」）
- **解讀**: 每一根K棒的最高價是下一根K棒的壓力參考，最低價是下一根K棒的支撐參考。若次一根K棒收盤價向上突破前一根K棒的最高點，代表買方力量轉強；若收盤價向下跌破前一根K棒的最低點，代表賣方力量轉強。此觀念不限日K，60分K等其他週期同樣適用（用「前一根K棒」取代「前一日」）。這是全書所有K線型態判讀的最基礎邏輯，也是後續「機械化多頭/空頭K線交易規則」（R-CANDLE-32、R-CANDLE-33）的理論基礎，兩者granularity不同：本規則是單純的支撐壓力觀察，機械化交易規則是含停損的完整進出場系統。
- **可程式化**: 是
- **所需資料**: 每根K棒的最高價、最低價、收盤價
- **計算公式**:
```
prev_high = high[t-1]
prev_low  = low[t-1]

if close[t] > prev_high:
    signal = "買方力量轉強：收盤突破前一根K棒最高點"
elif close[t] < prev_low:
    signal = "賣方力量轉強：收盤跌破前一根K棒最低點"
else:
    signal = "收在前一根K棒高低點區間內，多空未表態"
```
- **參數**: 比較基準K棒數（預設N=1，即僅比較前一根K棒；可調整為比較前N根K棒的高低點包絡）
- **可回測**: 是
- **信心**: 88/100 高
