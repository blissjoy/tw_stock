# 上升三法

- **Rule ID**: R-CANDLE-26
- **名稱**: 上升三法
- **分類**: K棒型態
- **原文與頁碼**: P03-C5-不同K線組合的意義.md（上冊 p.227-229；原書3-5節「一、繼續看漲的K線組合」型態1）
- **解讀**: 至少3根K線，可能更多（中段天數不固定）。第1根為上漲的中長紅K線（起漲棒，決定後續突破基準）；中段連續1到數日的小黑K、小紅K或變盤線，形成「母子懷抱」，即這些K線的實體/價格都收在第1根中長紅K的範圍內（每根收盤價都未跌破第1根中長紅K的最低點）；最後1根再度出現中長紅K線，且收盤突破第1根中長紅K的最高點，型態才成立。是上漲趨勢中常見的中繼（續漲）訊號，多單可續抱；若出現在空頭低檔且帶大量，則代表多方力道轉強，要注意趨勢可能反轉向上。
- **可程式化**: 是
- **所需資料**: 每根K棒的開盤價、收盤價、最高價、最低價、成交量
- **計算公式**:
```
day1 = 中長紅K（起漲棒）
mid_candles = []  # 中段N根，N>=1

for k in 中段K棒（從day1之後開始）:
    if is_small_black(k) or is_small_red(k) or is_reversal_candle(k):
        if close[k] >= low[day1]:      # 不破第1根最低點（母子懷抱）
            mid_candles.append(k)
        else:
            break   # 母子懷抱條件被打破，型態不成立
    else:
        last_candle = k
        break

is_pattern = len(mid_candles) >= 1 and \
             is_mid_long_red(last_candle) and \
             close[last_candle] > high[day1]     # 收盤突破第1根最高點

if is_pattern:
    signal = "上升三法，多頭中繼續漲訊號，多單可續抱"
    if 位於空頭低檔 and 大量:
        signal = "多方力道轉強，注意趨勢可能反轉向上"
```
- **參數**: 中段最少根數N（預設N≥1，天數不固定，需以「不破低」與「突破前高」作為區間邊界判斷）
- **可回測**: 是
- **信心**: 88/100 高
