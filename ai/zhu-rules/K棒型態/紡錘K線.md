# 紡錘K線

- **Rule ID**: R-CANDLE-24
- **名稱**: 紡錘K線
- **分類**: K棒型態
- **原文與頁碼**: P03-C4-不同型態的單一K線.md（上冊 p.221-222；原書3-4節「三、實體紡錘K線」）
- **解讀**: 外型特徵為當天實體上方有壓力、下方有支撐，**上下影線都很短**，影線長度沒有超過實體棒的二分之一。**重要提醒**：本書定義的「紡錘K」與一般西方蠟燭圖術語「Spinning Top」（強調上下影線都長）**方向相反**——本書的紡錘K是短影線、小實體的概念，程式化時不可套用西方定義。分為紅K紡錘線（多空拉鋸，多方力道強）與黑K紡錘線（多空拉鋸，空方力道強）兩種。書中未對紡錘K線另外區分高檔/低檔的不同解讀規則，僅提示「次日再看K線的變化」，即紡錘K本身不足以判斷轉折，需等待次日K線確認方向。
- **可程式化**: 是
- **所需資料**: 開盤價、收盤價、最高價、最低價
- **計算公式**:
```
body = abs(close - open)
upper_shadow = high - max(open, close)
lower_shadow = min(open, close) - low

is_spindle = (upper_shadow <= body / 2) and (lower_shadow <= body / 2)

if is_spindle:
    if close > open:
        note = "紅K紡錘線，多空拉鋸，多方力道強"
    else:
        note = "黑K紡錘線，多空拉鋸，空方力道強"
    # 書中未區分高低檔解讀差異，須待次日K線確認方向
```
- **參數**: 影線/實體比例門檻（固定為1/2，書中明訂）
- **可回測**: 是
- **信心**: 88/100 高
