# 突破上升軌道線壓力的大量長紅K

- **Rule ID**: R-CANDLE-34
- **名稱**: 突破上升軌道線壓力的大量長紅K
- **分類**: K棒型態
- **原文與頁碼**: P03-C6-K線練功實戰變化.md（上冊 p.247-250；原書3-6節「四、K線活用練功」之「1. 突破上升軌道線壓力的大量長紅線」）
- **解讀**: 多頭上漲時，連接低點畫出的趨勢線稱為「上升切線」（支撐參考線）；在切線之上、於高點畫一條與上升切線平行的線，稱為「上升軌道線」，是一條壓力線。當股價突破上升軌道壓力線，視為走勢轉強訊號，往往會出現連續上漲的一波攻勢。突破當天若出現跳空上漲、帶大量的長紅K實體棒，力道最強。沿軌道區間上漲到趨勢高檔（波段第3波）才出現的突破長紅K，容易發生急漲，屬於末升段現象，此時要注意隨後出現的變盤訊號，準備停利出場（可另外畫一條「急切線」，若股價跌破急切線即停利）。上升軌道線本身只是壓力，沒有實際突破確認前不可提早進場做多；相反地，若股價上漲接近軌道線但未突破就回落，可短線逆勢做空——這是書中少數明確允許逆勢交易的規則，與其餘章節「先確認再順勢操作」的主軸不同，建議在規則庫中獨立標記管理。
- **可程式化**: 部分
- **所需資料**: 一段時間內的價格序列（用於畫上升趨勢線/軌道線）、開盤價、收盤價、最高價、最低價、成交量、波段分段（第幾波）
- **計算公式**:
```
# 上升切線：連接近期波段低點的趨勢線；上升軌道線：與上升切線平行、通過近期波段高點的線
uptrend_line = fit_line(近期波段低點序列)
rising_channel_line = uptrend_line 平移至通過近期波段高點

if close[t] > rising_channel_line(t):
    signal = "突破上升軌道線壓力，波段轉強訊號，常伴隨連續上漲"
    strength_score = 0
    if 跳空上漲: strength_score += 1
    if 大量:     strength_score += 1
    if is_long_red_candle(t): strength_score += 1   # 見 R-CANDLE-21

    if 當前波段編號 >= 3:
        signal += "；末升段現象，須留意變盤訊號，可另畫急切線做停利依據"
        note = "若股價跌破急切線即停利"
elif 股價接近軌道線但未突破 and 隨後回落:
    signal = "短線逆勢放空訊號（軌道尚未突破，不可提早做多）"
```
- **參數**: 波段分段編號（需前置的「第幾波」判斷模組）、急切線斜率（另行繪製，無固定公式）
- **參考來源查證（2026-07-22，工程補充）**：strength_score中的「大量」門檻沿用R-CANDLE-23／R-TREND-13已查證的1.5～2倍均量慣例，非本規則另行查證之新來源。
- **可回測**: 是
- **信心**: 60/100 中（突破邏輯與加權因子明確、信心高；但依賴人工畫線/波段分段等尚未量化的前置模組，故整體評為中）
