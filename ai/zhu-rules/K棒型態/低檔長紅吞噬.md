# 低檔長紅吞噬

- **Rule ID**: R-CANDLE-18
- **名稱**: 低檔長紅吞噬（主力進貨）
- **分類**: K棒型態
- **原文與頁碼**: P03-C3-K線低檔反轉3大型態訊號.md（上冊 p.201-203；原書3-3節「二」(5)「低檔『長紅吞噬』」）
- **解讀**: 2根K線；下跌走勢到低檔，出現1根長紅K，完全包覆（吞噬）前一日下跌黑K的實體——紅K實體必須完全吞噬前一根黑K線的實體棒（比「長紅覆蓋」更嚴格：覆蓋只需超過二分之一即算強覆蓋，吞噬要求100%覆蓋前一根實體）。當日開低走高，收盤突破前一日高點，多方買盤出籠，多空主控權產生易位。長紅吞噬是雙K線向上轉折訊號中最強的組合，空單要立刻回補。吞噬當日或前一日出現大量或窒息量，反轉訊號越強。擴展型態「3線反紅」：1根紅K一次吞噬突破前面2～3根黑K線的高點，反轉力道更強。低檔出現爆量長紅吞噬K線後上漲反彈一段，日後再下跌到長紅吞噬K線區間會形成重要支撐；若再跌破此支撐，是空單的逃命波，不可追空，容易形成「底底高」的底部型態。
- **可程式化**: 是
- **所需資料**: Day1、Day2的開盤價、收盤價、最高價、成交量、前2～3根K線的最高價
- **計算公式**:
```
is_engulf     = open[2] <= close[1] and close[2] >= open[1]   # 紅K實體完全吞噬黑K實體
is_break_high = close[2] > high[1]                              # 收盤突破前一日高點

is_pattern = is_black(day1) and is_red(day2) and 位於低檔 and is_engulf and is_break_high

if is_pattern:
    signal = "低檔長紅吞噬，雙K線向上轉折訊號中最強（主力進貨），空單應立刻回補"
    if 吞噬當日或前一日大量或窒息量:
        signal += "，反轉訊號更強"
    if close[2] > max(high[t-1], high[t-2], high[t-3]):   # 一次突破前2~3根高點
        signal += "，屬於3線反紅，反轉力道更強"
    if 日後下跌回測到吞噬K線區間:
        note = "形成重要支撐；若再跌破，是空單逃命波不可追空，容易形成底底高底部型態"
```
- **參數**: 3線反紅比較根數（預設2～3根）
- **可回測**: 是
- **信心**: 88/100 高
