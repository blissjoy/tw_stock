# 上漲連3紅

- **Rule ID**: R-CANDLE-28
- **名稱**: 上漲連3紅
- **分類**: K棒型態
- **原文與頁碼**: P03-C5-不同K線組合的意義.md（上冊 p.231-233；原書3-5節「一、繼續看漲的K線組合」型態3）
- **解讀**: 固定3根、連續同向：3根**連續、帶大量**的向上紅K實體棒。書中定義原文明確包含「大量」二字（「3根連續的大量向上紅K線實體棒」）——這是與「下跌連3黑」（R-CANDLE-31）定義的關鍵不對稱處，下跌連3黑的定義文字未要求「大量」，書中原文本身如此，不應自行補平對稱。3種子型態：起漲上漲連3紅（波段起漲處出現）、跳空上漲連3紅（連續跳空，威力更大）、低檔上漲連3紅（空頭低檔反彈出現，暗示打底機率高）。連續大量紅K數量越多，代表力量越大、影響時間越長；若連續紅K呈現大量跳空上漲，威力更大；底部打底的第2隻腳出現「上漲連3紅」，為強勢多頭起動訊號。多頭行進到高點出現「上漲連3紅」需特別注意兩個警示訊號：連續紅K都出現長上影線，代表大敵當前、高點不多；高檔出現上漲連3紅同時爆量或價量背離，要注意股價不漲或下跌的反轉變盤訊號。
- **可程式化**: 是
- **所需資料**: 連續3根K棒的開盤價、收盤價、成交量、上下影線
- **計算公式**:
```
is_pattern = all(is_red_candle(k) for k in [day1, day2, day3]) and \
             all(volume[k] > K * avg_volume(N) for k in [day1, day2, day3])   # 定義本身要求大量

if is_pattern:
    strength_score = 連續大量紅K根數   # 越多分數越高，代表力量越大、影響時間越長
    if all(是跳空 for k in [day2, day3]):
        signal = "跳空上漲連3紅，威力更大"
    if 位於空頭低檔反彈:
        signal = "強力反彈訊號，打底機率高"
    if 位於底部打底第2隻腳:
        signal = "強勢多頭起動訊號"
    if 位於多頭高點:
        warning = []
        if all(有長上影線 for k in [day1, day2, day3]):
            warning.append("大敵當前，高點不多，注意變盤線訊號")
        if 爆量 or 價量背離:
            warning.append("注意股價不漲或下跌的反轉變盤訊號")
```
- **參數**: 大量倍數門檻K（書中未給精確值，建議1.5～2倍N日均量）、N=均量計算天數
- **可回測**: 是
- **信心**: 88/100 高
