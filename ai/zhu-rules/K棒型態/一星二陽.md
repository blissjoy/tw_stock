# 一星二陽

- **Rule ID**: R-CANDLE-27
- **名稱**: 一星二陽
- **分類**: K棒型態
- **原文與頁碼**: P03-C5-不同K線組合的意義.md（上冊 p.229-231；原書3-5節「一、繼續看漲的K線組合」型態2）
- **解讀**: 固定3根K線。第1根為上漲的中長紅K線（第1陽）；第2根恰好1根向上的小黑、小紅或變盤線（十字星形態，即「一星」，僅此1天，與上升三法的「一到數日」不同，是兩型態最大的結構差異）；第3根再上漲的中長紅K線（第2陽），構成「二陽」。書中定義未明文要求第2根／第3根與第1根之間的高低點包夾關係（不像上升三法明確寫「母子懷抱」+「不破低」），僅描述K線顏色與方向組合。是上漲趨勢中常見的中繼（續漲）組合，多單可續抱；空頭低檔反彈的「第2隻腳」若出現帶大量的一星二陽，代表還有高點，注意趨勢容易反轉多頭。
- **可程式化**: 是
- **所需資料**: 3根K棒的開盤價、收盤價、成交量
- **計算公式**:
```
day1 = 中長紅K（第1陽）
day2 = 恰好1根（小黑K 或 小紅K 或 變盤線）
day3 = 中長紅K（第2陽）

is_pattern = is_mid_long_red(day1) and \
             (is_small_black(day2) or is_small_red(day2) or is_reversal_candle(day2)) and \
             is_mid_long_red(day3)

# 書中未明文要求高低點包夾；以下為程式化保守判斷建議，非書中原文明訂：
if is_pattern and close[day3] > high[day1]:
    strict_confirm = True

if is_pattern:
    signal = "一星二陽，多頭中繼續漲組合，多單可續抱"
    if 位於空頭低檔反彈第2隻腳 and 大量:
        signal = "還有高點，注意趨勢容易反轉多頭"
```
- **參數**: 中段星形K線固定1根（與上升三法「1~N根」不同）
- **可回測**: 是
- **信心**: 62/100 中（3根結構定義明確；但高低點包夾條件為程式化建議補充，非書中原文明訂，故整體評為中）
