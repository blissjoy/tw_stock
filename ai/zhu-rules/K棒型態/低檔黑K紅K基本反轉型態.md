# 低檔黑K紅K基本反轉型態

- **Rule ID**: R-CANDLE-14
- **名稱**: 低檔黑K紅K基本反轉型態
- **分類**: K棒型態
- **原文與頁碼**: P03-C3-K線低檔反轉3大型態訊號.md（上冊 p.192-193；原書3-3節「二」(1)「低檔2根K線轉折向上的基本型態——長黑＋長紅」）
- **解讀**: 由2根K線構成，下跌低檔出現：左邊第1根為下跌幅度達3%以上的中長黑K，右邊第2根為上漲幅度達3%以上的中長紅K，兩根都須為中長實體，可以有小的上下影線。若出現在「空頭起跌」位置要提高警覺，小心是主力假跌真拉的騙線（誘空）；若出現在「多頭走勢」中，是回檔結束訊號，若出現在盤整區，表示主力還沒有下殺的企圖。
- **可程式化**: 是
- **所需資料**: Day1、Day2的開盤價、收盤價、股價相對高低檔位置
- **計算公式**:
```
day1_down_pct = (open[1] - close[1]) / open[1]
day2_up_pct   = (close[2] - open[2]) / open[2]

is_pattern = is_black(day1) and day1_down_pct >= 0.03 and \
             is_red(day2) and day2_up_pct >= 0.03 and \
             位於下跌低檔或底部

if is_pattern:
    signal = "低檔轉折向上訊號（基本型態）"
    if 出現在空頭起跌位置:
        warning = "可能是主力假跌真拉的誘空騙線，提高警覺"
    if 出現在多頭走勢中:
        signal = "回檔結束訊號"
    if 出現在盤整區:
        signal = "主力尚無下殺企圖"
```
- **參數**: 漲跌幅門檻（預設各3%）
- **可回測**: 是
- **信心**: 88/100 高
