# 低檔長紅覆蓋

- **Rule ID**: R-CANDLE-16
- **名稱**: 低檔長紅覆蓋（旭日東升）
- **分類**: K棒型態
- **原文與頁碼**: P03-C3-K線低檔反轉3大型態訊號.md（上冊 p.197-198；原書3-3節「二」(3)「低檔『長紅覆蓋』」）
- **解讀**: 下跌低檔，出現下跌中長黑K線；次日開低盤、盤中跌破新低價，但收盤突破平盤，紅K實體進入黑K實體範圍內。覆蓋比例以50%為分界：紅K收盤價若進入黑K實體但未突破黑K實體二分之一，屬「長紅弱覆蓋」，須再觀察1天確認是否反轉；若突破黑K實體二分之一以上，屬「長紅強覆蓋」，反轉向上可能性高。兩根K線的最高點H、最低點L是重要壓力與支撐：收盤突破H容易出現一波上漲，跌破L容易出現一波下跌。覆蓋的2根K線若同時爆大量，更加容易反轉向上。
- **可程式化**: 是
- **所需資料**: Day1、Day2的開盤價、收盤價、最高價、最低價、成交量
- **計算公式**:
```
midpoint_day1 = (open[1] + close[1]) / 2     # 黑K實體中點（open1>close1）

is_open_lower   = open[2] < open[1]                  # 次日開低盤
is_new_low      = low[2] < low[1]                     # 盤中創新低
is_close_in_body = close[2] > close[1] and close[2] < open[1]   # 收盤突破平盤且進入黑K實體

is_pattern = is_black(day1) and 位於低檔 and is_open_lower and is_new_low and is_close_in_body

if is_pattern:
    if close[2] < midpoint_day1:
        strength = "長紅弱覆蓋，力道較弱，建議再觀察1天"
    elif close[2] > midpoint_day1:
        strength = "長紅強覆蓋，反轉向上可能性高"

    H = max(high[1], high[2])
    L = min(low[1], low[2])
    if close[t] > H: outlook = "容易出現一波上漲"
    if close[t] < L: outlook = "容易出現一波下跌"

    if 覆蓋2根K線同時爆大量:
        strength += "；更容易反轉向上"
```
- **參數**: 無額外參數，二分之一分界為固定比例規則
- **可回測**: 是
- **信心**: 88/100 高
