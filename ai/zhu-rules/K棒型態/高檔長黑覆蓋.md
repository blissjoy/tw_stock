# 高檔長黑覆蓋

- **Rule ID**: R-CANDLE-08
- **名稱**: 高檔長黑覆蓋（烏雲罩頂）
- **分類**: K棒型態
- **原文與頁碼**: P03-C2-K線高檔反轉3大型態訊號.md（上冊 p.171下段-174前段；原書3-2節「二」型態3「高檔『長黑覆蓋』」）
- **解讀**: 由2根K線構成，上漲高點出現中長紅K；次日開高盤、盤中創新高價（Day2最高價 > Day1最高價），但收盤跌入紅K實體內部。覆蓋（黑K）越長，轉折力道越強。黑K收盤若跌入紅K實體但未達實體二分之一（收盤價 > 紅K實體中點），屬「長黑弱覆蓋」，反轉力道較弱，建議再觀察1天；若跌破紅K實體二分之一（收盤價 < 紅K實體中點），屬「長黑強覆蓋」，可視為反轉訊號。黑K收盤若進一步跌破紅K實體最低點，型態就轉變為更強的「長黑吞噬」（R-CANDLE-10），兩型態之間有明確的強度遞進關係。覆蓋2根K線的最高點(H)、最低點(L)是重要壓力/支撐觀察點：收盤突破H容易一波上漲，收盤跌破L容易一波下跌；覆蓋後走勢常在L～H區間橫向盤整，多為做頭訊號。
- **可程式化**: 是
- **所需資料**: Day1、Day2的開盤價、收盤價、最高價、最低價、成交量
- **計算公式**:
```
midpoint_day1 = (open[1] + close[1]) / 2     # 紅K實體中點（open1<close1）

is_open_higher   = open[2] > open[1]                 # 次日開高盤
is_new_high      = high[2] > high[1]                  # 盤中創新高
is_close_in_body = close[2] < close[1] and close[2] > open[1]  # 收盤跌入紅K實體內

is_pattern = is_red(day1) and 位於高檔 and is_open_higher and is_new_high and is_close_in_body

if is_pattern:
    if close[2] > midpoint_day1:
        strength = "長黑弱覆蓋，力道較弱，建議再觀察1天"
    elif close[2] < midpoint_day1:
        strength = "長黑強覆蓋，可視為反轉訊號"

    if close[2] < open[1]:                      # 跌破紅K最低點（開盤價）
        strength = "型態升級為長黑吞噬（見 R-CANDLE-10）"

    H = max(high[1], high[2])
    L = min(low[1], low[2])
    if close[t] > H: outlook = "容易一波上漲"
    if close[t] < L: outlook = "容易一波下跌"

    if 覆蓋2根K線同時爆大量:
        strength += "；更容易反轉向下"
```
- **參數**: 無額外參數，二分之一分界為固定比例規則
- **可回測**: 是
- **信心**: 88/100 高
