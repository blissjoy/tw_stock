# 機械化空頭K線交易規則

- **Rule ID**: R-CANDLE-33
- **名稱**: 機械化空頭K線交易規則
- **分類**: K棒型態
- **原文與頁碼**: P03-C6-K線練功實戰變化.md（上冊 p.244；原書3-6節「一、K線多空操作獲利戰法」之「2. 空頭K線交易規則」）
- **解讀**: 與機械化多頭K線交易規則（R-CANDLE-32）完全鏡射對稱：進場——收盤前確認股價，跌破前一日最低點時放空；停損——以進場當日K線股價的最高點為停損價，且不要超過7%；續抱——每天收盤前檢視股價，沒有突破前一日的最高點時續抱；出場（回補）——收盤前確認股價，突破前一日最高點時出場（回補）。書中範例（空運嘉3049，2015/06/18附近）顯示同樣一段空頭趨勢中，此規則會產生多次來回進出。兩套多空規則本質上是「以前一日高低點為唯一比較基準的逐日狀態機」，可用同一函式以方向參數（多/空）統一實作。
- **可程式化**: 是
- **所需資料**: 每日開盤價、收盤價、最高價、最低價
- **計算公式**:
```
state = "空手"
entry_price = None
stop_loss = None

for t in 每個交易日（收盤前決策）:
    if state == "空手":
        if close[t] < low[t-1]:
            state = "持有空單"
            entry_price = close[t]
            stop_loss = min(high[t], entry_price * (1 + 0.07))   # 進場當日K棒最高點 與 7%停損 取更嚴格(更低)者
    elif state == "持有空單":
        if close[t] > stop_loss:
            state = "空手（觸發停損出場/回補）"
        elif close[t] > high[t-1]:
            state = "空手（突破前一日高點回補）"
        # 否則：沒有突破前一日高點 → 續抱空單
```
- **參數**: 停損上限（固定7%，書中明訂）、比較基準K棒數（預設前1日）
- **可回測**: 是
- **信心**: 89/100 高
