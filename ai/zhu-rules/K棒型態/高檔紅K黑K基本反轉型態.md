# 高檔紅K黑K基本反轉型態

- **Rule ID**: R-CANDLE-06
- **名稱**: 高檔紅K黑K基本反轉型態
- **分類**: K棒型態
- **原文與頁碼**: P03-C2-K線高檔反轉3大型態訊號.md（上冊 p.165-168；原書3-2節「二、高檔紅K＋黑K線的轉折變化」型態1「高檔2根K線轉折向下的基本型態」）
- **解讀**: 由2根K線構成，出現在上漲高檔（多頭趨勢上漲高點，或空頭反彈相對高點）：左邊為上漲幅度達3%以上的中長紅K，右邊為下跌幅度達3%以上的中長黑K，兩根皆須為中長實體，可以有小的上下影線。這是「高檔紅K＋黑K」6種型態中力道最弱的基本款，代表2根K線構成的上漲高點/頭部反轉向下訊號。若出現在多頭剛起攻階段要提高警覺，可能是主力「假攻、真下跌」的騙線；在空頭走勢中此型態代表反彈結束訊號；若出現在盤整區，代表主力尚無上攻企圖。
- **可程式化**: 是
- **所需資料**: Day1、Day2的開盤價、收盤價、股價相對高低檔位置
- **計算公式**:
```
day1_up_pct   = (close[1] - open[1]) / open[1]
day2_down_pct = (open[2] - close[2]) / open[2]

is_pattern = is_red(day1) and day1_up_pct >= 0.03 and \
             is_black(day2) and day2_down_pct >= 0.03 and \
             位於上漲高檔

if is_pattern:
    signal = "高檔轉折向下訊號（基本型態，力道弱）"
    if 出現在多頭剛起攻階段:
        warning = "可能是主力假攻真下跌的騙線，提高警覺"
    if 出現在空頭反彈:
        signal = "反彈結束訊號"
    if 出現在盤整區:
        signal = "主力尚無上攻企圖"
```
- **參數**: 漲跌幅門檻（預設各3%）
- **可回測**: 是
- **信心**: 88/100 高
