# 大量長紅K的二分之一價支撐規則

- **Rule ID**: R-CANDLE-02
- **名稱**: 大量長紅K的二分之一價支撐規則
- **分類**: K棒型態
- **原文與頁碼**: P03-C1-K線起源與基本概念.md（上冊 p.133、p.141-145、p.147-149；原書3-1節「K線『4+1』元素的意義與應用」與「大量長紅K線日後的支撐與壓力」）
- **解讀**: 朱老師在開盤、收盤、最高、最低4個元素外，額外定義「二分之一價 = (當日最高價＋當日最低價) ÷ 2」為第5元素，代表當日多空交易的平均成本。當走勢中出現「帶大量」的長紅K，且該K棒位於相對高檔或低檔時，此K棒的最高價、二分之一價、最低價會成為日後3個由強到弱的支撐價位。股價回測時：跌破最高價代表向上攻擊力道減弱（須在3～5個交易日內站回該高點之上，否則要注意轉折向下）；跌破二分之一價代表跌破當日平均成本，容易產生大量賣壓，多方氣勢已被空方破壞；跌破最低價則代表多空易位，該長紅K反而變成日後的壓力。此外，若跌破二分之一價後次日股價又收復上漲，視為「假跌破」，不算真正轉弱——這是判斷是否真轉弱的重要停看聽條件。股價漲到高檔出現大量長紅K，其二分之一價一旦被跌破，高檔做頭機率大增。
- **可程式化**: 是
- **所需資料**: 該長紅K棒的開盤價、收盤價、最高價、最低價、成交量、近N日均量、後續K棒的收盤價、股價相對高低檔位置
- **計算公式**:
```
half_price = (high_bigred + low_bigred) / 2
is_big_volume = volume_bigred > K * avg_volume(N)     # K, N 為參數，書中未給精確倍數
is_relative_high_or_low = 位於近期相對高檔或低檔       # 需搭配趨勢位置判斷模組

if is_long_red_candle(bigred) and is_big_volume and is_relative_high_or_low:
    支撐分3層（由強至弱）:
        Tier1_最強支撐 = high_bigred
        Tier2_平均成本支撐 = half_price
        Tier3_最弱支撐 = low_bigred

    後續黑K測試反應（逐日檢查）:
        if close_test < Tier1 and close_test >= Tier2:
            status = "攻擊力道減弱，須3~5個交易日內站回最高點之上，否則注意轉折向下"
        elif close_test < Tier2 and close_test >= Tier3:
            status = "跌破平均成本，容易產生大量賣壓，多方氣勢轉弱"
            if close_test < Tier2 and 股價位於高檔:
                status += "；高檔做頭機率大增，須小心反轉"
        elif close_test < Tier3:
            status = "跌破最低點，多空易位，該長紅K轉為日後壓力"
            if close[隔日] > Tier3:
                status = "假跌破，不算真正轉弱（次日已收復）"
```
- **參數**: K=大量倍數門檻（書中未給精確值，建議預設1.5～2倍N日均量）、N=均量計算天數（預設5或20日）、「相對高檔/低檔」判斷方式（建議用近期波段高低點相對位置）
- **可回測**: 是
- **信心**: 60/100 中（3層價位定義與反應邏輯清楚明確，信心高；但「大量」與「相對高檔/低檔」門檻書中未給精確數字，需自行定義，故整體評為中）
