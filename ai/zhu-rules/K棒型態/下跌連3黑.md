# 下跌連3黑

- **Rule ID**: R-CANDLE-31
- **名稱**: 下跌連3黑
- **分類**: K棒型態
- **原文與頁碼**: P03-C5-不同K線組合的意義.md（上冊 p.238-241；原書3-5節「二、繼續看跌的K線組合」型態6）
- **解讀**: 固定3根、連續同向：3根連續的向下黑K實體棒。**重要**：書中定義原文為「3根連續的向下黑K線實體棒」，**未使用「大量」字眼**，與「上漲連3紅」（R-CANDLE-28）定義明文要求大量不同——這是書中定義上明文的不對稱之處，成交量並非本型態成立的必要條件，僅在重點說明/範例配圖中反覆出現「大量」標記，屬於加分/警示訊號。3種子型態：高檔下跌連3黑、下跌跳空連3黑、下跌連3黑後續形成「頭頭低」的頭部。連續黑K呈現跳空下跌威力更大；若連續黑K出現在多頭高檔的回檔，之後若再上漲，做頭的可能性很高；多頭高檔出現「下跌連3黑」視為強力回檔，容易回檔到前面轉折低點或跌破月線，若再上漲回升卻無法突破「下跌連3黑」最高點，應視為逃命波，多單要趕快出場；頭部型態中第2個頭出現「下跌連3黑」組合，為強勢空頭起跌訊號；空頭下跌到低點出現「下跌連3黑」，連續黑K都出現長下影線是打樁探底訊號，低檔連3黑爆量或KD指標背離則要注意止跌反轉。
- **可程式化**: 是
- **所需資料**: 連續3根K棒的開盤價、收盤價、成交量、上下影線
- **計算公式**:
```
is_pattern = all(is_black_candle(k) for k in [day1, day2, day3])
# 注意：型態定義本身不要求大量，與上漲連3紅（R-CANDLE-28）不對稱，不可自行補上大量條件作為必要條件

if is_pattern:
    if all(是跳空 for k in [day2, day3]):
        signal = "跳空下跌連3黑，威力更大"
    if 位於多頭高檔回檔:
        signal = "強力回檔，容易回檔到前轉折低點或跌破月線"
        max_high_of_pattern = max(high[day1], high[day2], high[day3])
        if 後續反彈未能突破 max_high_of_pattern:
            warning = "視為逃命波，多單應趕快出場"
    if 位於頭部型態第2個頭:
        signal = "強勢空頭起跌訊號"
    if 位於空頭低點:
        if all(有長下影線 for k in [day1, day2, day3]):
            warning = "打樁探底訊號，低點不多，注意止跌回升變盤線"
        if 爆量 or KD指標背離:
            warning = "注意股價止跌或反轉變盤訊號，容易反彈"
```
- **參數**: 無「大量」門檻參數（型態定義不要求大量，僅為加分警示條件）
- **可回測**: 是
- **信心**: 88/100 高
