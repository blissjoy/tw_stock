# 高檔夜星

- **Rule ID**: R-CANDLE-12
- **名稱**: 高檔夜星
- **分類**: K棒型態
- **原文與頁碼**: P03-C2-K線高檔反轉3大型態訊號.md（上冊 p.183-186；原書3-2節「三、高檔夜星組合K線的轉折變化」）
- **解讀**: 固定（基本款）3根K線：左邊中長紅K＋中間1根（或多根）「變盤線」＋右邊中長黑K，稱為「夜星」，右邊黑K完成即代表轉折確認（型態成立的觸發點是第3根黑K收盤）。書中列出6種組合變化（標準型態、孤島夜星、母子變盤、雙星變盤、雙鴉變盤、群星變盤），差異主要在中間變盤線的根數與型態細節。夜星組合同時爆大量，轉折向下機率大大提高且下跌力道強；中間夾雜的變盤線天數越多，中長黑K下跌時空方控盤時間也越長。型態失效條件：若下跌的中長黑K的高點，被後續上漲的紅K線突破，轉折向下的結構就被破壞，空方力道轉為由多方主控——此情形常發生在「多頭回檔」情境下的夜星，須留意黑K高點是否被突破作為停損/型態失效判準。
- **可程式化**: 部分
- **所需資料**: 左紅K、中段變盤線（1根以上）、右黑K的開盤價、收盤價、最高價、最低價、成交量、後續K棒收盤/最高價
- **計算公式**:
```
# 中段變盤線的精確辨識依賴 R-CANDLE-05／R-CANDLE-24／R-CANDLE-25 等單根K線變盤定義
left  = 中長紅K，位於高檔
middle = 1根以上「變盤線」（十字線/紡錘線/墓碑線/長T線等止漲型K棒）
right = 中長黑K，收盤完成即觸發型態確認

is_pattern = is_mid_long_red(left) and all(is_reversal_candle(k) for k in middle) and \
             is_mid_long_black(right) and 位於高檔

if is_pattern:
    signal = "高檔夜星，轉折向下確認（觸發點=右邊黑K收盤）"
    if 夜星組合同時爆大量:
        signal += "，轉折向下機率大增，下跌力道強"
    if len(middle) > 1:
        signal += "，中段變盤天數越多，空方控盤時間越長"

    # 型態失效條件（需持續監控後續K棒）
    if any(high[k] > high[right] for k in 後續K棒):
        signal = "型態失效：黑K高點被後續紅K突破，結構轉為多方主控（常見於多頭回檔中的夜星）"
```
- **參數**: 中段變盤線根數（不固定，1根以上）
- **可回測**: 是
- **信心**: 63/100 中（3根整體結構與失效條件定義明確，信心高；但中段「變盤線」的精確辨識依賴其他未完全量化的單根K線定義，故整體評為中）
