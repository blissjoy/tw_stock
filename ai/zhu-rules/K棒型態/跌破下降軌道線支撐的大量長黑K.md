# 跌破下降軌道線支撐的大量長黑K

- **Rule ID**: R-CANDLE-35
- **名稱**: 跌破下降軌道線支撐的大量長黑K
- **分類**: K棒型態
- **原文與頁碼**: P03-C6-K線練功實戰變化.md（上冊 p.251-253；原書3-6節「四、K線活用練功」之「2. 跌破下降軌道線支撐的大量長黑K線」）
- **解讀**: 與突破上升軌道線壓力規則（R-CANDLE-34）完全鏡射對稱。空頭下跌時，連接高點畫出的趨勢線稱為「下降切線」（壓力參考線）；在切線之下、於低點畫一條與下降切線平行的線，稱為「下降軌道線」，是一條支撐線。當股價跌破下降軌道支撐線，視為走勢轉弱訊號，往往會出現連續下跌的一波跌勢。跌破當天若出現跳空下跌、帶大量的長黑K實體棒，力道最強。沿軌道區間下跌到趨勢低檔（波段第3波）才出現的跌破長黑K，容易發生急跌，屬於末跌段現象，注意隨後出現的變盤訊號，準備空單回補停利出場（可畫「下降急切線」，股價突破急切線即停利回補）。下降軌道線本身只是支撐，沒有實際跌破確認前不可提早進場做空；相反地，若股價下跌接近軌道線但未跌破就反彈，可短線逆勢做多。書中範例中有1檔個股（圖表3-6-17，代號2612）明確示範「跌破下降軌道後走勢後續反轉向上、重新出現多頭確認」的案例，提醒軌道跌破訊號並非保證持續下跌，仍需搭配後續K線確認。
- **可程式化**: 部分
- **所需資料**: 一段時間內的價格序列（用於畫下降趨勢線/軌道線）、開盤價、收盤價、最高價、最低價、成交量、波段分段（第幾波）
- **計算公式**:
```
downtrend_line = fit_line(近期波段高點序列)
falling_channel_line = downtrend_line 平移至通過近期波段低點

if close[t] < falling_channel_line(t):
    signal = "跌破下降軌道線支撐，波段轉弱訊號，常伴隨連續下跌"
    strength_score = 0
    if 跳空下跌: strength_score += 1
    if 大量:     strength_score += 1
    if is_long_black_candle(t): strength_score += 1   # 見 R-CANDLE-22

    if 當前波段編號 >= 3:
        signal += "；末跌段現象，須留意變盤訊號，可另畫下降急切線做回補依據"
        note = "若股價突破急切線即停利回補"
elif 股價接近軌道線但未跌破 and 隨後反彈:
    signal = "短線逆勢做多訊號（軌道尚未跌破，不可提早做空）"

# 注意：跌破後仍可能反轉向上（如書中代號2612案例），需搭配後續K線確認，不保證持續下跌
```
- **參數**: 波段分段編號（需前置的「第幾波」判斷模組）、下降急切線斜率（另行繪製，無固定公式）
- **參考來源查證（2026-07-22，工程補充）**：strength_score中的「大量」門檻沿用R-CANDLE-23／R-TREND-13已查證的1.5～2倍均量慣例，非本規則另行查證之新來源。
- **可回測**: 是
- **信心**: 62/100 中（跌破邏輯與加權因子明確、信心高；但依賴人工畫線/波段分段等尚未量化的前置模組，故整體評為中）
