# 下降三法

- **Rule ID**: R-CANDLE-29
- **名稱**: 下降三法
- **分類**: K棒型態
- **原文與頁碼**: P03-C5-不同K線組合的意義.md（上冊 p.234-236；原書3-5節「二、繼續看跌的K線組合」型態4）
- **解讀**: 與上升三法（R-CANDLE-26）完全鏡射對稱的看跌版本。至少3根K線，可能更多。第1根為下跌的中長黑K線；中段連續1到數日的小黑K、小紅K或變盤線，呈現「母子懷抱」（每根收盤價都未突破第1根中長黑K的最高點）；最後1根再度出現的中長黑K線，且收盤跌破第1根中長黑K的最低點，型態才成立。是下跌趨勢中常見的中繼（續跌）訊號，空單可續抱；若出現在多頭高檔盤整時，代表空方力道轉強，注意後續趨勢可能改變。
- **可程式化**: 是
- **所需資料**: 每根K棒的開盤價、收盤價、最高價、最低價、成交量
- **計算公式**:
```
day1 = 中長黑K
mid_candles = []

for k in 中段K棒（從day1之後開始）:
    if is_small_black(k) or is_small_red(k) or is_reversal_candle(k):
        if close[k] <= high[day1]:     # 不過第1根最高點（母子懷抱）
            mid_candles.append(k)
        else:
            break
    else:
        last_candle = k
        break

is_pattern = len(mid_candles) >= 1 and \
             is_mid_long_black(last_candle) and \
             close[last_candle] < low[day1]      # 收盤跌破第1根最低點

if is_pattern:
    signal = "下降三法，空頭中繼續跌訊號，空單可續抱"
    if 位於多頭高檔盤整:
        signal = "空方力道轉強，注意後續趨勢可能改變"
```
- **參數**: 中段最少根數N（預設N≥1，天數不固定）
- **可回測**: 是
- **信心**: 88/100 高
