# 高檔長黑遭遇

- **Rule ID**: R-CANDLE-07
- **名稱**: 高檔長黑遭遇（一日封口）
- **分類**: K棒型態
- **原文與頁碼**: P03-C2-K線高檔反轉3大型態訊號.md（上冊 p.169-172前段；原書3-2節「二」型態2「高檔『長黑遭遇』」）
- **解讀**: 由2根K線構成，上漲到明顯高檔出現中長紅K；次日開高（跳空或跳高開盤）後走低，收盤價接近前一日紅K的收盤價（即未深入紅K實體太多），等於把當天向上跳空開高的缺口封閉，故稱「一日封口」。這是本節6種高檔紅黑組合型態中力道最弱的一種，只能視為止漲，多頭仍可能續漲，建議再觀察1天確認是否真反轉。若多頭上漲中出現遭遇組合且爆大量，轉折機率大增，做多不可追多；若空頭反彈中出現遭遇組合，視為反彈結束訊號，可把握做空機會。
- **可程式化**: 部分
- **所需資料**: Day1、Day2的開盤價、收盤價、成交量
- **計算公式**:
```
is_open_higher = open[2] > close[1]                     # 次日開高
is_black_day2  = close[2] < open[2]                      # 次日收黑
is_close_near  = abs(close[2] - close[1]) <= NEAR_PCT * close[1]   # 收盤接近前日紅K收盤（門檻書中未明確給值）

is_pattern = is_red(day1) and 位於高檔 and \
             is_open_higher and is_black_day2 and is_close_near

if is_pattern:
    signal = "高檔長黑遭遇，止漲訊號（6型態中最弱），建議再觀察1天"
    if 大量:
        signal = "轉折機率大增，做多不可追多"
    if 位於空頭反彈:
        signal = "反彈結束訊號，可考慮做空"
```
- **參數**: NEAR_PCT（「收盤接近前日收盤」的容許誤差比例，書中未給精確數字，建議自訂如1%～2%）
- **參考來源查證（2026-07-22，工程補充）**：查證外部技術分析資料，未找到「一日封口」此書中特定用語的公認統一容許誤差慣例，NEAR_PCT=1%為工程估計值，非外部佐證數字。
- **可回測**: 是
- **信心**: 58/100 中（型態方向與強度定性明確；「接近」門檻需自行量化，書中未給精確數字）
