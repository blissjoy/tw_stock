# 一星二陰

- **Rule ID**: R-CANDLE-30
- **名稱**: 一星二陰
- **分類**: K棒型態
- **原文與頁碼**: P03-C5-不同K線組合的意義.md（上冊 p.236-238；原書3-5節「二、繼續看跌的K線組合」型態5）
- **解讀**: 與一星二陽（R-CANDLE-27）對稱，固定3根K線。第1根為下跌的中長黑K線（第1陰）；第2根恰好1根向下的小黑、小紅或變盤線（一星）；第3根再下跌的中長黑K線（第2陰），構成「二陰」。中間星形K線固定1根，與下降三法「中段N根可變」不同。定義文字同樣未明確寫出高低點包夾關係。是下跌趨勢中常見的中繼（續跌）組合，空單可續抱；多頭高檔回檔的「第2個頭」出現帶大量的一星二陰，代表還有低點，注意趨勢容易反轉向下。
- **可程式化**: 是
- **所需資料**: 3根K棒的開盤價、收盤價、成交量
- **計算公式**:
```
day1 = 中長黑K（第1陰）
day2 = 恰好1根（小黑K 或 小紅K 或 變盤線）
day3 = 中長黑K（第2陰）

is_pattern = is_mid_long_black(day1) and \
             (is_small_black(day2) or is_small_red(day2) or is_reversal_candle(day2)) and \
             is_mid_long_black(day3)

# 書中未明文要求高低點包夾；以下為程式化保守判斷建議，非書中原文明訂：
if is_pattern and close[day3] < low[day1]:
    strict_confirm = True

if is_pattern:
    signal = "一星二陰，空頭中繼續跌組合，空單可續抱"
    if 位於多頭高檔回檔第2個頭 and 大量:
        signal = "還有低點，注意趨勢容易反轉向下"
```
- **參數**: 中段星形K線固定1根（與下降三法「1~N根」不同）
- **可回測**: 是
- **信心**: 62/100 中（3根結構定義明確；但高低點包夾條件為程式化建議補充，非書中原文明訂，故整體評為中）
