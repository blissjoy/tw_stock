# 低檔長紅遭遇

- **Rule ID**: R-CANDLE-15
- **名稱**: 低檔長紅遭遇（一日封口）
- **分類**: K棒型態
- **原文與頁碼**: P03-C3-K線低檔反轉3大型態訊號.md（上冊 p.194-196；原書3-3節「二」(2)「低檔『長紅遭遇』」）
- **解讀**: 由2根K線構成，下跌走勢到明顯低檔，出現中長黑K線；第2天開低走高，收盤價收在接近前一天中長黑K線收盤價附近（形成「封口」）。此為6組反轉訊號中最弱的一種，只能視為止跌，可能還會續跌，最好再觀察1天確認是否反轉。空頭下跌中出現遭遇型態：若無量、次日向上，屬弱勢反彈；若急跌爆量、次日向上，屬強勢反彈（皆屬逆勢反彈，搶反彈要小心）。空頭下跌中出現遭遇組合若爆大量，轉折機率大增，做空要小心不可追空；多頭回檔中出現遭遇組合，是回檔結束訊號，若次日出量上漲，是多頭續漲訊號。
- **可程式化**: 部分
- **所需資料**: Day1、Day2的開盤價、收盤價、成交量
- **計算公式**:
```
is_open_lower = open[2] < close[1]                    # 次日開低
is_red_day2   = close[2] > open[2]                      # 次日收紅
is_close_near = abs(close[2] - close[1]) <= NEAR_PCT * close[1]   # 收盤接近前日黑K收盤

is_pattern = is_black(day1) and 位於低檔 and \
             is_open_lower and is_red_day2 and is_close_near

if is_pattern:
    signal = "低檔長紅遭遇，止跌訊號（6型態中最弱），建議再觀察1天"
    if 無量:
        signal = "弱勢反彈，逆勢搶反彈需小心"
    elif 急跌爆量:
        signal = "強勢反彈，逆勢搶反彈需小心"
    if 位於空頭下跌 and 大量:
        signal = "轉折機率大增，做空不可追空"
    if 位於多頭回檔 and 次日出量上漲:
        signal = "多頭續漲訊號"
```
- **參數**: NEAR_PCT（「收盤接近前日收盤」容許誤差比例，書中未給精確數字，建議自訂如1%～2%）
- **參考來源查證（2026-07-22，工程補充）**：與R-CANDLE-07同批查證，查無公認統一容許誤差慣例，NEAR_PCT=1%為工程估計值。
- **可回測**: 是
- **信心**: 58/100 中（型態方向與強度定性明確；「接近」門檻需自行量化）
