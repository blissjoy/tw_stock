# 高檔變盤線

- **Rule ID**: R-CANDLE-05
- **名稱**: 高檔變盤線
- **分類**: K棒型態
- **原文與頁碼**: P03-C2-K線高檔反轉3大型態訊號.md（上冊 p.161-163；原書3-2節「一、高檔變盤線的轉折變化」）
- **解讀**: 出現在高檔（多頭趨勢的上漲高點，或空頭趨勢中反彈的相對高點）的單一根「變盤線」K棒（或連續數天都出現變盤線），視為止漲訊號。書中圖表列出8種變盤K棒外觀：墓碑線、長T線、十字線、紡錘線、反錘線（紅／黑皆可）、槌子線（紅／黑皆可）、跌停線、長黑線。變盤線本身不保證次日就下跌，但所有事後回顧的轉折下跌位置幾乎都先出現變盤線——即它是必要不充分條件，須觀察次日走勢確認。若同時爆大量、出現價量背離、KD指標死亡交叉，或位於壓力位置爆量卻收盤未突破，反轉機率更高。
- **可程式化**: 部分
- **所需資料**: 開盤價、收盤價、最高價、最低價、成交量、KD指標、股價相對高低檔位置
- **計算公式**:
```
# 8種外觀中，紡錘線見 R-CANDLE-24、槌子線/反錘線(倒槌)見 R-CANDLE-25，
# 其餘4種（十字線、墓碑線、長T線、跌停線/長黑線）書中未給精確數值門檻，
# 以下為程式化時建議的操作型定義，非書中原文明訂：
is_doji        = abs(close - open) / open < DOJI_THRESHOLD              # 十字線
is_gravestone  = is_doji_like and upper_shadow大 and lower_shadow趨近0   # 墓碑線（近似倒槌變體）
is_long_T      = is_doji_like and lower_shadow大 and upper_shadow趨近0   # 長T線（近似槌子變體）
is_limit_down  = (close - prev_close) / prev_close <= -LIMIT_DOWN_PCT   # 跌停線
is_long_black  = is_long_black_candle(t)                                # 長黑線，見 R-CANDLE-22

is_reversal_candle_high = is_doji or is_gravestone or is_long_T or \
                           is_spindle(t) or is_hammer(t) or is_inverted_hammer(t) or \
                           is_limit_down or is_long_black

if 位於高檔 and is_reversal_candle_high:
    signal = "止漲訊號，須觀察次日確認"
    if 大量 and 價量背離 and KD死亡交叉:
        signal_strength = "反轉機率大增"
    if 上升走勢遇壓力位置 and 大量 and close未突破壓力:
        signal_strength = "反轉機率更高"
```
- **參數**: DOJI_THRESHOLD（十字線實體/開盤價比例門檻，書中未給，建議<1%）、LIMIT_DOWN_PCT（跌停幅度，依當時台股漲跌幅制度）、大量倍數門檻（建議1.5～2倍均量）
- **參考來源查證（2026-07-22，工程補充）**：①十字線實體門檻查證外部資料僅有「開盤收盤趨近相同」的定性描述，無公認統一數字，採0.5%為工程估計值。②台股現行漲跌幅限制為**10%**（金管會自2015-06-01起由7%放寬為10%，法規事實非TA慣例，來源：[Tsung's Blog](https://blog.longwin.com.tw/2018/02/taiwan-stock-change-limit-7-10-2018/)），LIMIT_DOWN_PCT直接採10%。③大量倍數沿用R-CANDLE-23/R-TREND-13已查證的1.5～2倍均量。
- **可回測**: 是
- **信心**: 62/100 中（型態出現位置與強化條件明確；但4種未另行定義的K棒外觀需自訂操作型定義，非書中原文精確數值）
