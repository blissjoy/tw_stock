# 均線多頭排列

- **Rule ID**: R-MA-08
- **名稱**: 均線多頭排列
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.275、p.278
- **解讀**: 這是全書均線篇被引用最多次的核心定義：當短天期均線在上、長天期均線在下，依序由上而下排列時，稱為「均線多頭排列」，標準排列為 MA5 > MA10 > MA20（＞ MA60 ＞ MA120 ＞ MA240，視觀察條數而定）。多頭排列代表每條均線對股價都同時發揮支撐與助漲作用，排列的均線條數越多（例如6線多排：5日、10日、月線、季線、半年線、年線皆呈短在上長在下），代表越多天期的持有者都處於獲利、股價長期上漲的支撐力越強、上漲時間也越持久，適合鎖股做中長多。日線、週線、月線的多頭排列判讀方式相同，只是分別對應短、中長、長線操作。務必注意：多頭排列不只要求均線數值大小順序正確，通常還要求各條均線「方向皆向上」才算有效的多頭排列（純數值順序符合但方向走平或向下者，強度較弱）。
- **可程式化**: 是
- **所需資料**: 多條不同天期均線（至少 MA5、MA10、MA20，可擴充 MA60、MA120、MA240）之逐日數值與方向
- **計算公式**:
```
# N 條均線由短到長依序為 MA_1 < MA_2 < ... （天期遞增）
IS_BULLISH_ALIGNMENT(t) =
    MA5(t) > MA10(t) > MA20(t)                       # 3 線多排（最低標準）
    # 可擴充：MA5(t) > MA10(t) > MA20(t) > MA60(t) > MA120(t) > MA240(t)  # 6 線多排

# 加強版（含方向條件，強度更高）：
IS_BULLISH_ALIGNMENT_STRICT(t) =
    IS_BULLISH_ALIGNMENT(t)
    AND MA5(t) > MA5(t-1)
    AND MA10(t) > MA10(t-1)
    AND MA20(t) > MA20(t-1)
    # 各均線方向皆向上

# 排列條數（多排線數）：
bullish_line_count(t) = 依序檢查 MA5>MA10>MA20>MA60>MA120>MA240 中，
                         從短天期開始連續滿足大於關係的均線條數
```
- **參數**: 均線組合（預設 MA5、MA10、MA20；可擴充至 MA60、MA120、MA240 做 4～6 線多排判斷）
- **可回測**: 是
- **信心**: 90/100 高
