# 葛蘭碧賣點四大法則

- **Rule ID**: R-MA-20
- **名稱**: 葛蘭碧賣點四大法則
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.301-309
- **解讀**: 葛蘭碧8大法則中做空的4個賣點，與「葛蘭碧買點四大法則」完全對稱，同樣以股價與20日均線（MA20）的相對關係判斷。賣點①盤頭反轉賣點：MA20方向由上升轉為走平甚至下彎，且股價收盤由均線上方向下跌破MA20，代表行情由漲轉跌的反轉轉折處。賣點②空頭段反彈遇壓賣點：股價在MA20下方、MA20方向向下，股價反彈到均線附近但未突破、隨後轉而下跌，代表均線發揮壓力，此賣點可在同一空頭段落中反覆出現（加碼放空）。賣點③短暫突破快速回跌賣點：MA20方向保持下彎，股價收盤短暫突破MA20後很快回跌到均線下方，代表行情仍是下跌走勢。賣點④乖離過大逆勢搶跌賣點：多頭走勢中股價連續急漲3天以上、與MA20正乖離過大（書中範例以正負乖離15%設通道帶），股價開始回檔向均線靠近，此為逆勢交易訊號，僅屬短期做空，不代表趨勢反轉。
- **可程式化**: 是
- **所需資料**: 收盤價（Close）、20日均線（MA20）及其逐日方向、股價與MA20的乖離率
- **計算公式**:
```
MA20_direction(t) = "下彎" IF MA20(t) < MA20(t-1) ELSE ("上揚" IF MA20(t) > MA20(t-1) ELSE "走平")
bias(t) = (Close[t] - MA20(t)) / MA20(t) * 100%    # 乖離率

# 賣點①：盤頭反轉賣點
SELL_SIGNAL_1(t) =
    MA20_direction(t-1) IN ["上升轉走平", "走平"]  OR  MA20_direction(t) == "下彎"
    AND Close[t-1] > MA20(t-1)      # 前一日收盤仍在均線上方
    AND Close[t] < MA20(t)           # 當日收盤向下跌破MA20

# 賣點②：空頭段反彈遇壓賣點（可在同一空頭段落中反覆出現）
SELL_SIGNAL_2(t) =
    Close[t] < MA20(t)                  # 股價在均線下方
    AND MA20_direction(t) == "下彎"
    AND MAX(High[t-2..t]) 接近 MA20(t) 但未突破 (Close[t-1] <= MA20(t-1) 附近)
    AND Close[t] < Close[t-1]            # 隨後轉而下跌

# 賣點③：短暫突破快速回跌賣點
SELL_SIGNAL_3(t) =
    MA20_direction(t) == "下彎"
    AND EXISTS day d IN [t-3, t-1]: Close[d] > MA20(d)   # 近期曾短暫突破
    AND Close[t] < MA20(t)                                 # 當日收盤已回跌至MA20下方

# 賣點④：乖離過大逆勢搶跌賣點（逆勢短打）
SELL_SIGNAL_4(t) =
    main_trend == "多頭"
    AND Close[t] > MA20(t)
    AND 連續上漲天數(t) >= 3
    AND bias(t) >= 15%                     # 正乖離超過15%（通道帶參數可調）
    AND Close[t] < Close[t-1]               # 股價開始回檔向均線靠近
    # 出場：反彈後K線突破前一日高點應停利回補（逆勢短打，非趨勢反轉）
```
- **參數**: MA 天期固定為20（原著設定）、乖離率通道帶（賣點④，預設±15%，可調整）、連續上漲天數門檻（賣點④，預設3天）
- **可回測**: 是
- **信心**: 89/100 高
