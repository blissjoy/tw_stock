# 死亡交叉

- **Rule ID**: R-MA-14
- **名稱**: 死亡交叉
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.284、p.286
- **解讀**: 股價上漲一段時間、兩條均線呈多頭排列往上；當股價開始下跌時，短天期均線先轉向下，若下跌格局持續，短天期均線就會由上往下穿越長天期均線，這個向下交叉稱為「死亡交叉」。與黃金交叉完全對稱，同樣必定是短天期均線往長天期均線方向交叉。死亡交叉本身只是一個「事件」，其多空意義必須配合當時所處的主趨勢解讀（見「黃金死亡交叉配合主趨勢判讀」規則），不能單獨視為做空訊號。
- **可程式化**: 是
- **所需資料**: 短天期均線（MA_short，如MA5）、長天期均線（MA_long，如MA10）之逐日數值
- **計算公式**:
```
IS_DEATH_CROSS(t) =
    MA_short(t-1) >= MA_long(t-1)     # 前一日短均線仍在長均線之上或相等
    AND MA_short(t) < MA_long(t)       # 當日短均線已下穿長均線
```
- **參數**: MA_short 天期（預設5）、MA_long 天期（預設10，亦可用10與20等其他組合）
- **可回測**: 是
- **信心**: 89/100 高
