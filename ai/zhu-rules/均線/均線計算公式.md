# 均線計算公式

- **Rule ID**: R-MA-01
- **名稱**: 均線計算公式
- **分類**: 均線
- **原文與頁碼**: P04-C1-均線的基本概念.md，上冊 p.257；扣抵更新公式另見 P04-C2-均線的重要功能與實戰活用.md，上冊 p.263
- **解讀**: 均線（Moving Average，簡稱 MA）由葛蘭碧（Granville Joseph）於 1960 年發表，是均線篇一切規則的計算基礎。均線是用「收盤價」計算的簡單移動平均，不是用開盤、最高、最低價，也不是用四價平均，這一點在建立資料管線與回測引擎時務必精確，否則後續所有均線相關規則（多空排列、交叉、扣抵、乖離）都會失真。均線每日隨新收盤價滾動更新，本質上是「加入最新一天、扣除最舊一天」的移動平均，此即後續「移動扣抵」規則的數學基礎。
- **可程式化**: 是
- **所需資料**: 每日收盤價（Close）之時間序列
- **計算公式**:
```
# 定義：N 日均線（SMA，簡單移動平均）
MA(N, t) = SUM(Close[t-N+1 .. t]) / N
# 例：
MA5(t)  = (Close[t-4] + Close[t-3] + Close[t-2] + Close[t-1] + Close[t]) / 5
MA10(t) = (Close[t-9] + ... + Close[t]) / 10
MA20(t) = (Close[t-19] + ... + Close[t]) / 20
MA60(t) = (Close[t-59] + ... + Close[t]) / 60

# 每日滾動更新公式（移動扣抵，等價寫法）：
MA(N, t) = MA(N, t-1) + (Close[t] - Close[t-N]) / N
# 其中 Close[t-N] 稱為「扣價」（被扣除的最舊一天收盤價）
#      Close[t]   稱為「抵價」（新加入的今日收盤價）
```
- **參數**: N（均線天期，可為 5、10、20、60、120、240 等任意正整數；週線、月線則以對應週期收盤價計算）
- **可回測**: 是
- **信心**: 90/100 高
