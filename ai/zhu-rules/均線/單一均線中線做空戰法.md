# 單一均線中線做空戰法

- **Rule ID**: R-MA-25
- **名稱**: 單一均線中線做空戰法
- **分類**: 均線
- **原文與頁碼**: P04-C3-均線戰法與實戰活用.md，上冊 p.319-320（戰法4）
- **解讀**: 與「單一均線中線做多戰法」完全對稱。使用MA10作為持股與停利依據，MA20為空頭大方向濾網，進場訊號沿用MA5判斷反彈結束。前提是趨勢空頭不變、股價在MA20之下。進場條件：空頭彈後空下跌確認點（收盤跌破MA5且收盤跌破前一日最低點）、空頭盤整跌破確認點、頭部型態確認點。進場後只要每日收盤未突破MA10就持續抱空單，收盤突破MA10（轉折上漲確認）才空單回補；前提是空頭趨勢不變、股價在MA20之下、MA10及MA20維持空頭排列。停損邏輯同R-MA-21。停利：跌幅超過10%後反彈突破MA10才回補停利，未達10%續抱；盤整小賠小賺出場條件仍參照MA5與前高；急跌末端出現爆量長紅或長下影線的處理方式與短線做空戰法相同。
- **可程式化**: 是
- **所需資料**: 收盤價、最高價、最低價、成交量、MA5、MA10、MA20、K線型態
- **計算公式**:
```
# 前提濾網
PREREQUISITE(t) = main_trend == "空頭" AND Close[t] < MA20(t)

# 進場條件（任一成立）
ENTRY(t) =
    (Close[t] < MA5(t) AND Close[t] < Low[t-1])
    OR 空頭盤整跌破確認(t)
    OR 頭部型態確認(t)
    IF PREREQUISITE(t) AND 以上任一成立: ENTER SHORT

# 持股（空單）規則（改守MA10）
HOLD SHORT WHILE Close[t] <= MA10(t) AND main_trend == "空頭" AND Close[t] < MA20(t) AND MA10(t) < MA20(t)
COVER IF Close[t] > MA10(t)   # 轉折上漲確認，停利回補

# 停損（見 R-MA-21 統一邏輯）
stop_loss = IF entry_candle_loss_pct < 5%: 進場轉折最高點 ELSE High[entry_day]

# 停利
target_10pct = entry_price * 0.90
IF current_price <= target_10pct:
    IF Close[t] > MA10(t): TAKE_PROFIT (全部回補)
ELSE:
    IF Close[t] > MA10(t): HOLD (續抱)

IF (收盤未跌破前低) AND (底底高) AND (Close[t] > MA5(t)) AND (Close[t] > High[t-1]):
    EXIT (小賠或小賺出場；此處盤整判斷仍參照MA5/前高)

IF (急跌>=3天中長黑K OR 累積獲利>=20%) AND (低檔爆量長紅(開低走高>5%) OR 低檔爆量長下影線):
    COVER 二分之一 today; IF 次日續漲: COVER 剩餘100%
IF 當天出現長紅吞噬:
    COVER 100% today
```
- **參數**: MA5（進場訊號）、MA10（持股與停利依據）、MA20（大方向濾網）、停利觸發跌幅（10%）、獲利了結追蹤門檻（20%）
- **可回測**: 是
- **信心**: 88/100 高
