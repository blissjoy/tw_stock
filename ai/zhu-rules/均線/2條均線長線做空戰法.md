# 2條均線長線做空戰法

- **Rule ID**: R-MA-29
- **名稱**: 2條均線長線做空戰法
- **分類**: 均線
- **原文與頁碼**: P04-C3-均線戰法與實戰活用.md，上冊 p.330-331（戰法8）
- **解讀**: 與「2條均線長線做多戰法」完全對稱。使用MA10與MA20雙均線交叉排列關係作為進出場依據。訊號定義：MA10、MA20死亡交叉且呈空排向下時開始做空；MA10、MA20黃金交叉且呈多排向上時做空結束（回補）。進場：於日線空頭趨勢進場位置進場（依死亡交叉空排條件成立時進場）。停損邏輯同R-MA-21。停利／出場：（1）MA10、MA20黃金交叉且多排向上為主要出場訊號；（2）波段獲利超過20%以上，收盤突破MA5即回補停利；（3）日線低檔爆量、趨勢轉多頭確認，先回補出場；（4）長線空頭股價到末跌段，或股價達頭部最高點的二分之一價，此位置進場只做短空，出現爆量容易落底。
- **可程式化**: 是
- **所需資料**: 收盤價、MA5、MA10、MA20、成交量
- **計算公式**:
```
# 進場訊號
ENTRY_SIGNAL(t) = IS_DEATH_CROSS(MA10, MA20, t) AND MA10(t) < MA20(t)   # 死亡交叉且空排向下
IF ENTRY_SIGNAL(t): ENTER SHORT

# 停損（見 R-MA-21 統一邏輯）
stop_loss = IF entry_candle_loss_pct < 5%: 進場轉折最高點 ELSE High[entry_day]

# 停利／出場
EXIT_SIGNAL(t) = IS_GOLDEN_CROSS(MA10, MA20, t) AND MA10(t) > MA20(t)   # 黃金交叉且多排向上（主要出場）
IF EXIT_SIGNAL(t): EXIT ALL (做空結束，回補)

IF 波段獲利率 > 20% AND Close[t] > MA5(t):
    TAKE_PROFIT (回補停利)

IF 日線低檔爆量 AND 趨勢轉多頭確認:
    EXIT (先回補出場)

IF 股價已下跌達頭部最高點的二分之一價 OR 進入空頭末跌段低檔:
    後續再進場僅做短空操作，出現爆量須提高警覺（容易落底）
```
- **參數**: MA10與MA20交叉組合、波段獲利門檻（20%）
- **可回測**: 是
- **信心**: 89/100 高
