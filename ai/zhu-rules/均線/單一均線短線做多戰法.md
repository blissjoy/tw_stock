# 單一均線短線做多戰法

- **Rule ID**: R-MA-22
- **名稱**: 單一均線短線做多戰法
- **分類**: 均線
- **原文與頁碼**: P04-C3-均線戰法與實戰活用.md，上冊 p.311-312（戰法1）
- **解讀**: 均線篇9種戰法之一，使用MA5作為單一進出依據，MA20作為大方向濾網。前提是趨勢多頭不變、股價在MA20之上才持續用本戰法做多。進場條件符合以下任一種確認點即可：多頭回後買上漲確認點（收盤突破MA5且收盤突破前一日最高點）、多頭盤整突破確認點、急跌反彈確認點、底部型態確認點。進場後只要每日收盤未跌破MA5就持續抱股，收盤跌破MA5（轉折下跌確認）才出場；前提是多頭趨勢不變、股價在MA20之上、MA10及MA20維持多頭排列。停損依「均線戰法停損點設定邏輯（5%分界法）」：進場紅K漲幅<5%守轉折最低點，≥5%守進場K棒最低點。停利規則：進場後即計算上漲10%價位；漲幅超過10%後回檔跌破MA5才停利，未達10%則續抱；若進入盤整出現「收盤未過前高＋頭頭低＋跌破MA5＋跌破前一日最低點」則屬小賠小賺出場；若急漲3天以上中長紅K或已獲利達20%以上，出現高檔爆量長黑（開高走低>5%）或高檔爆量長上影線，當日收盤先停利一半、次日再跌則全部停利，若直接出現長黑吞噬則當天全部停利。
- **可程式化**: 是
- **所需資料**: 收盤價、最高價、最低價、成交量、MA5、MA10、MA20、K線型態（長黑吞噬、長上影線等，需搭配K棒型態分類規則）
- **計算公式**:
```
# 前提濾網
PREREQUISITE(t) = main_trend == "多頭" AND Close[t] > MA20(t)

# 進場條件（任一成立）
ENTRY(t) =
    (Close[t] > MA5(t) AND Close[t] > High[t-1])                      # 回後買上漲確認
    OR 多頭盤整突破確認(t)                                              # K線橫盤突破
    OR 急跌反彈確認(t)
    OR 底部型態確認(t)
    IF PREREQUISITE(t) AND 以上任一成立: ENTER LONG

# 持股規則
HOLD WHILE Close[t] >= MA5(t) AND main_trend == "多頭" AND Close[t] > MA20(t) AND MA10(t) > MA20(t)
EXIT IF Close[t] < MA5(t)   # 轉折下跌確認

# 停損（見 R-MA-21 統一邏輯）
stop_loss = IF entry_candle_gain_pct < 5%: 上漲轉折最低點 ELSE Low[entry_day]

# 停利
target_10pct = entry_price * 1.10
IF current_price >= target_10pct:
    IF Close[t] < MA5(t): TAKE_PROFIT (全部)
ELSE:
    IF Close[t] < MA5(t): HOLD (續抱，不停利)

IF (收盤未過前高) AND (頭頭低) AND (Close[t] < MA5(t)) AND (Close[t] < Low[t-1]):
    EXIT (小賠或小賺出場)

IF (急漲>=3天中長紅K OR 累積獲利>=20%) AND (高檔爆量長黑(開高走低>5%) OR 高檔爆量長上影線):
    SELL 50% today; IF 次日續跌: SELL 剩餘100%
IF 當天出現長黑吞噬:
    SELL 100% today
```
- **參數**: MA5（進出依據）、MA20（大方向濾網）、MA10（排列確認）、停利觸發漲幅（10%）、獲利了結追蹤門檻（20%）、爆量長黑開高走低門檻（5%）
- **可回測**: 是
- **信心**: 88/100 高
