# 3條均線綜合戰法

- **Rule ID**: R-MA-30
- **名稱**: 3條均線綜合戰法
- **分類**: 均線
- **原文與頁碼**: P04-C3-均線戰法與實戰活用.md，上冊 p.332-336（戰法9，含9a多頭/9b空頭兩個對稱子戰法）
- **解讀**: 均線篇9種戰法中唯一採用「分批建倉／分批減碼」資金管理邏輯的戰法，使用MA5（短）、MA10（中）、MA20（長）三線依序分批進出，而非其餘8種戰法多為單筆全進全出。多頭子戰法（9a）：前提是底部日線多頭確認、均線4線多排向上時開始做多；進場張數須為3的倍數（例如3、6、9張），以便三線分批出場／加碼；停損同R-MA-21統一邏輯。停利／加碼三分之一分批：股價收盤跌破MA5，停利賣出三分之一；續跌破MA10，再賣三分之一；續跌破MA20，賣出最後三分之一出清；若多頭趨勢不變、股價收盤重新突破MA5，買回三分之一，續突破MA10再買回三分之一，續突破MA20再買回三分之一（恢復原部位）；波段獲利超過20%以上時，收盤跌破MA5直接全部停利出清（不再分批）；若進入末升段高檔或股價已上漲接近底部起漲1倍，後續再進場只做短線、守MA5停利。空頭子戰法（9b）與多頭完全鏡射：前提是底部日線空頭確認、均線4線空排向下時開始做空；進場張數同樣須為3的倍數；停利／加空三分之一分批：突破MA5回補三分之一，續突破MA10再回補三分之一，續突破MA20回補最後三分之一；若空頭趨勢不變、跌破MA5加空三分之一，續跌破MA10、MA20依序加空，恢復原空單部位；波段獲利超過20%以上時，突破MA5直接全部回補；若進入末跌段低檔或股價下跌接近頭部起跌二分之一，後續再進場只做短線操作。
- **可程式化**: 是
- **所需資料**: 收盤價、MA5、MA10、MA20、MA60（4線多排/空排確認）、持倉張數（需為3的倍數）
- **計算公式**:
```
### 9a：多頭做多
PREREQUISITE = 底部日線多頭確認 AND 均線4線多排向上 (MA5>MA10>MA20>MA60，皆向上)
ENTRY: position = 3n 張 (n為正整數，例如3、6、9)

stop_loss = IF entry_candle_gain_pct < 5%: 上漲轉折最低點 ELSE Low[entry_day]

# 分批停利
IF Close[t] < MA5(t):  SELL position/3     # 賣出三分之一
IF Close[t] < MA10(t): SELL position/3     # 再賣三分之一
IF Close[t] < MA20(t): SELL position/3     # 賣出最後三分之一，出清全部

# 分批加碼（趨勢不變時買回）
IF 多頭趨勢不變 AND Close[t] > MA5(t):  BUY_BACK position/3
IF 多頭趨勢不變 AND Close[t] > MA10(t): BUY_BACK position/3
IF 多頭趨勢不變 AND Close[t] > MA20(t): BUY_BACK position/3   # 恢復原部位

# 波段獲利保護（不分批，直接出清）
IF 波段獲利率 > 20% AND Close[t] < MA5(t):
    SELL ALL (全部停利出清，不分批)

IF 股價已上漲接近底部起漲幅度的1倍 OR 進入多頭末升段高檔:
    後續再進場僅做短線操作，改守 MA5 停利

### 9b：空頭做空（完全鏡射）
PREREQUISITE = 底部日線空頭確認 AND 均線4線空排向下 (MA5<MA10<MA20<MA60，皆向下)
ENTRY: position = 3n 張

stop_loss = IF entry_candle_loss_pct < 5%: 進場轉折最高點 ELSE High[entry_day]

# 分批停利（回補）
IF Close[t] > MA5(t):  COVER position/3
IF Close[t] > MA10(t): COVER position/3
IF Close[t] > MA20(t): COVER position/3    # 回補完畢全部空單

# 分批加空（趨勢不變時加碼）
IF 空頭趨勢不變 AND Close[t] < MA5(t):  ADD_SHORT position/3
IF 空頭趨勢不變 AND Close[t] < MA10(t): ADD_SHORT position/3
IF 空頭趨勢不變 AND Close[t] < MA20(t): ADD_SHORT position/3   # 恢復原空單部位

# 波段獲利保護（不分批，直接全部回補）
IF 波段獲利率 > 20% AND Close[t] > MA5(t):
    COVER ALL (全部停利回補，不分批)

IF 股價已下跌接近頭部起跌幅度的二分之一 OR 進入空頭末跌段低檔:
    後續再進場僅做短線操作
```
- **參數**: MA5、MA10、MA20、MA60（4線多排/空排確認）、初始進場張數必須為3的倍數、波段獲利門檻（20%）
- **可回測**: 是
- **信心**: 89/100 高
