# 2條均線長線做多戰法

- **Rule ID**: R-MA-28
- **名稱**: 2條均線長線做多戰法
- **分類**: 均線
- **原文與頁碼**: P04-C3-均線戰法與實戰活用.md，上冊 p.326-329（戰法7）
- **解讀**: 使用MA10與MA20雙均線的交叉排列關係本身作為進出場依據，不用單一均線做停利判斷。訊號定義：MA10、MA20黃金交叉且呈多排向上時開始做多；MA10、MA20死亡交叉且呈空排向下時做多結束（出場）。進場：於日線多頭趨勢進場位置進場，即上述黃金交叉多排條件成立時進場，未額外規定K棒突破條件。停損邏輯同R-MA-21。停利／出場：（1）MA10、MA20死亡交叉且空排向下為主要出場訊號；（2）波段獲利超過20%以上，收盤跌破MA5即停利；（3）日線高檔爆量、趨勢轉空頭確認，先停利出場；（4）若處於多頭末升段高檔，或股價已上漲接近底部起漲1倍，此時再進場只適合短線操作、守MA5停利。
- **可程式化**: 是
- **所需資料**: 收盤價、MA5、MA10、MA20、成交量
- **計算公式**:
```
# 進場訊號
ENTRY_SIGNAL(t) = IS_GOLDEN_CROSS(MA10, MA20, t) AND MA10(t) > MA20(t)   # 黃金交叉且多排向上
IF ENTRY_SIGNAL(t): ENTER LONG

# 停損（見 R-MA-21 統一邏輯）
stop_loss = IF entry_candle_gain_pct < 5%: 上漲轉折最低點 ELSE Low[entry_day]

# 停利／出場
EXIT_SIGNAL(t) = IS_DEATH_CROSS(MA10, MA20, t) AND MA10(t) < MA20(t)     # 死亡交叉且空排向下（主要出場）
IF EXIT_SIGNAL(t): EXIT ALL (做多結束)

IF 波段獲利率 > 20% AND Close[t] < MA5(t):
    TAKE_PROFIT (停利)

IF 日線高檔爆量 AND 趨勢轉空頭確認:
    EXIT (先停利出場)

IF 股價已上漲接近底部起漲幅度的1倍 OR 進入多頭末升段高檔:
    後續再進場僅做短線操作，改守 MA5 停利
```
- **參數**: MA10與MA20交叉組合、波段獲利門檻（20%）
- **可回測**: 是
- **信心**: 89/100 高
