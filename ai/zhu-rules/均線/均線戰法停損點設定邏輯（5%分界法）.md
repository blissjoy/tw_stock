# 均線戰法停損點設定邏輯（5%分界法）

- **Rule ID**: R-MA-21
- **名稱**: 均線戰法停損點設定邏輯（5%分界法）
- **分類**: 均線
- **原文與頁碼**: P04-C3-均線戰法與實戰活用.md，上冊 p.311-336（貫穿9種均線戰法的統一停損邏輯，書末總結明確指出「可直接抽象成統一函式的核心規則」）
- **解讀**: 書中9種均線戰法（單一均線短/中/長線做多做空、2條均線長線做多做空、3條均線綜合戰法）的停損設定邏輯完全一致，可抽象為統一規則：以「進場當根K棒的漲跌幅是否達5%」為分界。做多時，若進場紅K漲幅小於5%，停損點設在該筆進場後「上漲轉折的最低點」（即後續拉回時最新出現的相對低點）；若進場紅K漲幅達5%以上（代表當根K棒波動已經很大），停損點直接設在「進場紅K當根最低點」。做空時邏輯完全鏡射：進場黑K跌幅小於5%，停損點設在「進場轉折的最高點」；跌幅達5%以上，停損點設在「進場黑K當根最高點」。此規則的意義是：當進場K棒本身波動已經很大（≥5%），代表當根K棒本身的高低點已具有足夠的統計意義可直接作為停損依據；若進場K棒波動較小（<5%），則需要等待後續價格走出「轉折點」才有意義，避免停損設得過近而被雜訊掃出。
- **可程式化**: 是
- **所需資料**: 進場當根K棒之開高低收（Open, High, Low, Close）、進場後續K棒的高低點（用於判斷「轉折點」）
- **計算公式**:
```
# 做多方向
entry_candle_gain_pct = (Close[entry_day] - Open[entry_day]) / Open[entry_day] * 100%

IF entry_candle_gain_pct < 5%:
    stop_loss_long = 進場後「上漲轉折的最低點」
    # 定義：進場後股價持續上漲過程中，最近一次拉回不創新低前的相對低點
    # （即最新一個「轉折低點」，隨走勢推進動態上移）
ELSE:  # entry_candle_gain_pct >= 5%
    stop_loss_long = Low[entry_day]     # 進場紅K當根最低點

# 做空方向（完全鏡射）
entry_candle_loss_pct = (Open[entry_day] - Close[entry_day]) / Open[entry_day] * 100%

IF entry_candle_loss_pct < 5%:
    stop_loss_short = 進場後「下跌轉折的最高點」
    # 定義：進場後股價持續下跌過程中，最近一次反彈不創新高前的相對高點
ELSE:  # entry_candle_loss_pct >= 5%
    stop_loss_short = High[entry_day]   # 進場黑K當根最高點
```
- **參數**: 漲跌幅分界門檻（固定5%，原著給定值）
- **可回測**: 是
- **信心**: 89/100 高
