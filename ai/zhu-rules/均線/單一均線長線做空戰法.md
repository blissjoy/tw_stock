# 單一均線長線做空戰法

- **Rule ID**: R-MA-27
- **名稱**: 單一均線長線做空戰法
- **分類**: 均線
- **原文與頁碼**: P04-C3-均線戰法與實戰活用.md，上冊 p.325-327（戰法6）
- **解讀**: 與「單一均線長線做多戰法」完全對稱。使用MA20作為進出判斷依據，MA60作為長期趨勢方向轉折判斷依據。操作守則（表4-3-11）：股價跌破MA20且在MA60之下時做空空單（前提是MA10/MA20兩線死亡交叉、空排下彎）；股價站上MA20但仍在MA60之下時回補空單、空手觀望。選股條件：日線需空頭確認且MA10/MA20/MA60空排向下；週線需趨勢空頭確認且週MA10/MA20空排向下。進場條件：日線空頭趨勢確認；頭部空頭趨勢確認、MA10/MA20空排向下、股價在MA20之下、第1次反彈回測未突破原下降切線，股價下跌跌破MA5及昨日K線最低點時進場。停損邏輯同R-MA-21。操作程序：（1）日線符合空頭短線條件先進場，嚴格停損；（2）日線短線下跌一波、均線4線空排、獲利後改做長線，若週線也符合空頭、週均線3線空排，改守MA20操作；（3）波段獲利超過20%以上，收盤突破MA5回補停利；（4）日週月線空頭不變，可依日線空頭條件持續進場，嚴格停損；（5）日線低檔爆量、趨勢轉多頭確認（可能尚未突破MA20或MA60），先回補停利出場，之後若日線再符合空頭短線條件再重新進場；（6）中長線空頭股價到末跌段，或股價達頭部最高點的二分之一價（容易是空頭趨勢低點），此位置進場只做短空，出現爆量容易落底、須提高警覺。
- **可程式化**: 部分（「原下降切線」需搭配切線分類規則，其餘均線與價格條件可完全程式化）
- **所需資料**: 收盤價、MA5、MA10、MA20、MA60（日線與週線）、成交量
- **計算公式**:
```
# 操作守則
IF Close[t] < MA20(t) AND Close[t] < MA60(t) AND death_cross(MA10, MA20) AND MA10(t)<MA20(t)<...空排下彎:
    ACTION = "做空空單"
ELIF Close[t] > MA20(t) AND Close[t] < MA60(t):
    ACTION = "回補空單，空手觀望"

# 選股條件
SELECT_STOCK =
    日線空頭確認 AND MA10(t)<MA20(t)<MA60(t) 皆向下
    AND 週線趨勢空頭確認 AND 週MA10(t)<週MA20(t) 皆向下

# 進場
ENTRY(t) =
    日線空頭趨勢確認 AND 均線4線空排 (MA5<MA10<MA20<MA60)
    AND 股價第1次反彈回測未突破原下降切線
    AND Close[t] < MA5(t) AND Close[t] < Low[t-1]

# 停損（見 R-MA-21 統一邏輯）
stop_loss = IF entry_candle_loss_pct < 5%: 進場轉折最高點 ELSE High[entry_day]

# 操作程序（多時間框架停利切換）
stage = "短線"
IF stage=="短線" AND 均線4線空排 AND 已獲利 AND 週線空頭 AND 週均線3線空排:
    stage = "長線"; stop_basis = MA20
IF 波段獲利率 > 20%:
    IF Close[t] > MA5(t): TAKE_PROFIT (回補停利)
IF 日線低檔爆量 AND 趨勢轉多頭確認:
    EXIT (先回補停利，未必已突破MA20/MA60)
    WAIT 再次符合空頭短線條件才重新進場
IF 股價已下跌達頭部最高點的二分之一價 OR 進入末跌段低檔:
    stage = "短線（提高警覺，出現爆量容易落底）"
```
- **參數**: MA5、MA10、MA20、MA60（日線與週線）、波段獲利門檻（20%）
- **可回測**: 部分（依賴切線判斷模組是否已實作；核心均線邏輯可先行回測）
- **信心**: 58/100 中
