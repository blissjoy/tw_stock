# 黃金交叉

- **Rule ID**: R-MA-13
- **名稱**: 黃金交叉
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.284-285
- **解讀**: 股價下跌一段時間、兩條均線呈空頭排列往下；當股價止跌上漲時，短天期均線會先轉向上，若上漲格局持續，短天期均線就會由下往上穿越長天期均線，這個向上交叉稱為「黃金交叉」。由於短天期均線對價格變化的反應速度比長天期均線快（見「均線反應速度與權重遞減」規則），交叉必定是短天期均線往長天期均線方向交叉，不會反過來。書中做短線常用組合為短均線MA5、長均線MA10。黃金交叉本身只是一個「事件」，其多空意義必須配合當時所處的主趨勢解讀（見「黃金死亡交叉配合主趨勢判讀」規則），不能單獨視為買進訊號。
- **可程式化**: 是
- **所需資料**: 短天期均線（MA_short，如MA5）、長天期均線（MA_long，如MA10）之逐日數值
- **計算公式**:
```
IS_GOLDEN_CROSS(t) =
    MA_short(t-1) <= MA_long(t-1)     # 前一日短均線仍在長均線之下或相等
    AND MA_short(t) > MA_long(t)       # 當日短均線已上穿長均線
```
- **參數**: MA_short 天期（預設5）、MA_long 天期（預設10，亦可用10與20等其他組合）
- **可回測**: 是
- **信心**: 89/100 高
