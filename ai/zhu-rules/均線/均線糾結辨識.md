# 均線糾結辨識

- **Rule ID**: R-MA-16
- **名稱**: 均線糾結辨識
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.288-289
- **解讀**: 當股價在狹幅區間盤整5～6天，MA5會逐漸走平；盤整延長到10～12天，MA10也會走平；盤整更久，MA20也趨向平緩，於是可觀察到5日、10日、20日三條均線互相靠攏糾結在一起，稱為「均線糾結」；若盤整時間拉得更長，季線甚至更長天期均線也會加入糾結（例如6線糾結：5日、10日、20日、60日、120日、240日均線）。均線糾結代表這段期間買進者的平均成本相近，且累積了大量成交能量，糾結時間越長、糾結均線條數越多，代表能量累積越大，一旦突破往往有較大漲跌幅。此規則本身是「狀態辨識」，用於觸發後續的「均線糾結向上突破做多SOP」或「均線糾結向下跌破做空SOP」。
- **可程式化**: 是
- **所需資料**: MA5、MA10、MA20（可擴充MA60、MA120、MA240）之逐日數值
- **計算公式**:
```
# 以均線間的相對乖離幅度衡量是否「靠攏糾結」
MA_spread(t) = (MAX(MA5(t), MA10(t), MA20(t)) - MIN(MA5(t), MA10(t), MA20(t))) / Close[t]

IS_MA_CONVERGED(t, threshold) =
    MA_spread(t) <= threshold          # threshold 例如 2%～3%，可依個股波動調整
    AND ABS(MA5(t) - MA5(t-1)) / Close[t] 很小   # 均線本身也趨於走平

convergence_line_count(t) = 依此邏輯擴大檢查 MA60、MA120、MA240 是否也落在同一乖離範圍內，
                             計算共有幾條均線同時糾結（3線糾結、4線糾結……6線糾結）
```
- **參數**: threshold（判定「靠攏」的乖離幅度門檻，書中未給出明確數值，需回測校準，建議初始值2%～3%）、參與判斷的均線條數（3條或6條）
- **可回測**: 是
- **信心**: 58/100 中
