# 均線下彎的壓力與助跌作用

- **Rule ID**: R-MA-07
- **名稱**: 均線下彎的壓力與助跌作用
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.272-274
- **解讀**: 當均線方向向下時，對股價發揮三大作用，與均線上彎完全對稱：（1）壓力作用——股價反彈到均線時容易再度轉跌；（2）助跌作用——股價反彈突破均線後，只要均線仍向下，會把股價拉回均線下方；（3）減弱反彈紅K的上漲力道——空頭中出現中長紅K反彈，若均線仍下彎，可減緩續漲力道。強度同樣受均線天期（越長越強）與下彎角度（越陡越強）影響。此規則是判斷「股價反彈是否會在均線附近受阻」的核心依據，也是葛蘭碧賣點②③與均線戰法「空單持股規則」（未突破均線則續抱空單）的理論基礎。
- **可程式化**: 部分
- **所需資料**: 收盤價（Close）、均線（MA(N)）及其逐日方向、均線斜率（角度）
- **計算公式**:
```
MA_direction(N, t) = "下彎" IF MA(N,t) < MA(N,t-1) ELSE ("上揚" IF MA(N,t) > MA(N,t-1) ELSE "走平")

IF MA_direction(N, t) == "下彎":
    IF Close[t] 反彈接近 MA(N,t) 但未突破 (Close[t] <= MA(N,t)):
        判定：均線發揮「壓力作用」，再度轉跌機率較高
    ELIF Close[t] > MA(N,t) 但前一日 Close 曾低於 MA(N,t) （即剛突破均線）:
        判定：均線發揮「助跌作用」，股價可能被拉回均線下方
    ELIF 當日為中長紅K且 Close[t] > Open[t]:
        判定：均線下彎可「減弱紅K上漲力道」，續漲動能有限

# 強度排序：
strength(N1) > strength(N2)  IF N1 > N2   # 天期越長，作用越強
strength ∝ |slope(MA(N))|                  # 下彎角度越陡，作用越強
```
- **參數**: N（均線天期，MA5/MA10/MA20 常用比較組）
- **可回測**: 是
- **信心**: 58/100 中
