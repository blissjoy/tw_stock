# 盤整均線交錯不進場

- **Rule ID**: R-MA-12
- **名稱**: 盤整均線交錯不進場
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.283；均線鈍化缺點另見 P04-C1-均線的基本概念.md，上冊 p.259
- **解讀**: 股價處於盤整階段時，短、中、長天期均線會相互上下交叉穿梭，不呈現規則的由上而下排列，即所謂「均線交錯」或「乱七八糟排列」，這正是均線篇 4-1 提到的均線缺點之一：「趨勢區間盤整時，均線上下變動反覆，無法用均線判斷方向與進出」。書中給出的最佳策略是：此時不要介入操作，應等待盤整完成，直到均線重新呈現多頭排列或空頭排列時再進場。此規則可作為其他進出場規則的「濾網」，用於過濾均線排列混亂、訊號不可靠的盤整期。
- **可程式化**: 是
- **所需資料**: MA5、MA10、MA20（可擴充 MA60）之逐日數值
- **計算公式**:
```
IS_MA_TANGLED(t) =
    NOT IS_BULLISH_ALIGNMENT(t)     # 非多頭排列 (MA5>MA10>MA20)
    AND NOT IS_BEARISH_ALIGNMENT(t)  # 且非空頭排列 (MA5<MA10<MA20)
    # 即均線大小順序不符合任一規則排列，視為交錯盤整

IF IS_MA_TANGLED(t):
    action = "不進場，觀望等待均線重新呈現多頭或空頭排列"
```
- **參數**: 均線組合（預設 MA5、MA10、MA20）
- **可回測**: 是
- **信心**: 58/100 中
