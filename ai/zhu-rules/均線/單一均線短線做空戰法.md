# 單一均線短線做空戰法

- **Rule ID**: R-MA-23
- **名稱**: 單一均線短線做空戰法
- **分類**: 均線
- **原文與頁碼**: P04-C3-均線戰法與實戰活用.md，上冊 p.313-314（戰法2）
- **解讀**: 與「單一均線短線做多戰法」完全對稱。使用MA5作為單一進出依據，MA20作為大方向濾網。前提是趨勢空頭不變、股價在MA20之下才持續用本戰法做空。進場條件符合以下任一種確認點即可：空頭彈後空下跌確認點（收盤跌破MA5且收盤跌破前一日最低點）、空頭盤整跌破確認點、急漲回檔確認點、頭部型態確認點。進場後只要每日收盤未突破MA5就持續抱空單，收盤突破MA5（轉折上漲確認）才回補；前提是空頭趨勢不變、股價在MA20之下、MA10及MA20維持空頭排列。停損：進場黑K跌幅<5%守轉折最高點，≥5%守進場K棒最高點。停利：進場後即計算下跌10%價位；跌幅超過10%後反彈突破MA5才回補停利，未達10%則續抱；若進入盤整出現「收盤未跌破前低＋底底高＋突破MA5＋突破前一日最高點」則屬小賠小賺出場；若急跌3天以上中長黑K或已獲利達20%以上，出現低檔爆量長紅（開低走高>5%）或低檔爆量長下影線，當日收盤先回補一半、次日再漲則全部回補，若直接出現長紅吞噬則當天全部回補。
- **可程式化**: 是
- **所需資料**: 收盤價、最高價、最低價、成交量、MA5、MA10、MA20、K線型態（長紅吞噬、長下影線等）
- **計算公式**:
```
# 前提濾網
PREREQUISITE(t) = main_trend == "空頭" AND Close[t] < MA20(t)

# 進場條件（任一成立）
ENTRY(t) =
    (Close[t] < MA5(t) AND Close[t] < Low[t-1])                       # 彈後空下跌確認
    OR 空頭盤整跌破確認(t)
    OR 急漲回檔確認(t)
    OR 頭部型態確認(t)
    IF PREREQUISITE(t) AND 以上任一成立: ENTER SHORT

# 持股（空單）規則
HOLD SHORT WHILE Close[t] <= MA5(t) AND main_trend == "空頭" AND Close[t] < MA20(t) AND MA10(t) < MA20(t)
COVER IF Close[t] > MA5(t)   # 轉折上漲確認

# 停損（見 R-MA-21 統一邏輯）
stop_loss = IF entry_candle_loss_pct < 5%: 進場轉折最高點 ELSE High[entry_day]

# 停利
target_10pct = entry_price * 0.90
IF current_price <= target_10pct:
    IF Close[t] > MA5(t): TAKE_PROFIT (全部回補)
ELSE:
    IF Close[t] > MA5(t): HOLD (續抱，不回補)

IF (收盤未跌破前低) AND (底底高) AND (Close[t] > MA5(t)) AND (Close[t] > High[t-1]):
    EXIT (小賠或小賺出場)

IF (急跌>=3天中長黑K OR 累積獲利>=20%) AND (低檔爆量長紅(開低走高>5%) OR 低檔爆量長下影線):
    COVER 50% today; IF 次日續漲: COVER 剩餘100%
IF 當天出現長紅吞噬:
    COVER 100% today
```
- **參數**: MA5（進出依據）、MA20（大方向濾網）、MA10（排列確認）、停利觸發跌幅（10%）、獲利了結追蹤門檻（20%）、爆量長紅開低走高門檻（5%）
- **可回測**: 是
- **信心**: 88/100 高
