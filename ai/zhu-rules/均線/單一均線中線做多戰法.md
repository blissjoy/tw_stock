# 單一均線中線做多戰法

- **Rule ID**: R-MA-24
- **名稱**: 單一均線中線做多戰法
- **分類**: 均線
- **原文與頁碼**: P04-C3-均線戰法與實戰活用.md，上冊 p.315-318（戰法3）
- **解讀**: 使用MA10作為持股與停利依據，MA20為多頭大方向濾網，但進場訊號仍沿用MA5判斷回檔結束。前提是趨勢多頭不變、股價在MA20之上。進場條件（較短線戰法少了「急跌反彈確認點」一項）：多頭回後買上漲確認點（收盤突破MA5且收盤突破前一日最高點）、多頭盤整突破確認點、底部型態確認點。進場後只要每日收盤未跌破MA10就持續抱股，收盤跌破MA10（轉折下跌確認）才停利出場；前提同樣是多頭趨勢不變、股價在MA20之上、MA10及MA20維持多頭排列。停損邏輯同R-MA-21統一規則。停利規則與短線戰法結構相同，但持股基準改為MA10：漲幅超過10%後回檔跌破MA10才停利，未達10%續抱；盤整小賠小賺出場條件仍參照MA5與前低；急漲末端出現爆量長黑或長上影線的處理方式與短線戰法相同。
- **可程式化**: 是
- **所需資料**: 收盤價、最高價、最低價、成交量、MA5、MA10、MA20、K線型態
- **計算公式**:
```
# 前提濾網
PREREQUISITE(t) = main_trend == "多頭" AND Close[t] > MA20(t)

# 進場條件（任一成立，注意：進場訊號仍用MA5判斷回檔結束）
ENTRY(t) =
    (Close[t] > MA5(t) AND Close[t] > High[t-1])
    OR 多頭盤整突破確認(t)
    OR 底部型態確認(t)
    IF PREREQUISITE(t) AND 以上任一成立: ENTER LONG

# 持股規則（改守MA10）
HOLD WHILE Close[t] >= MA10(t) AND main_trend == "多頭" AND Close[t] > MA20(t) AND MA10(t) > MA20(t)
EXIT IF Close[t] < MA10(t)   # 轉折下跌確認，停利出場

# 停損（見 R-MA-21 統一邏輯）
stop_loss = IF entry_candle_gain_pct < 5%: 上漲轉折最低點 ELSE Low[entry_day]

# 停利
target_10pct = entry_price * 1.10
IF current_price >= target_10pct:
    IF Close[t] < MA10(t): TAKE_PROFIT (全部)
ELSE:
    IF Close[t] < MA10(t): HOLD (續抱)

IF (收盤未過前高) AND (頭頭低) AND (Close[t] < MA5(t)) AND (Close[t] < Low[t-1]):
    EXIT (小賠或小賺出場；此處盤整判斷仍參照MA5/前低)

IF (急漲>=3天中長紅K OR 累積獲利>=20%) AND (高檔爆量長黑(開高走低>5%) OR 高檔爆量長上影線):
    SELL 50% today; IF 次日續跌: SELL 剩餘100%
IF 當天出現長黑吞噬:
    SELL 100% today
```
- **參數**: MA5（進場訊號）、MA10（持股與停利依據）、MA20（大方向濾網）、停利觸發漲幅（10%）、獲利了結追蹤門檻（20%）
- **可回測**: 是
- **信心**: 88/100 高
