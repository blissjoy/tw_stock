# 均線反映持有成本與獲利判斷

- **Rule ID**: R-MA-02
- **名稱**: 均線反映持有成本與獲利判斷
- **分類**: 均線
- **原文與頁碼**: P04-C1-均線的基本概念.md，上冊 p.258
- **解讀**: 均線的理論基礎是「成本理論」：N 日均線代表最近 N 天買進者的平均持有成本。均線上升表示市場持有成本走高，下降表示持有成本走低。由此可推導出一個可程式化的判斷：只要比較「當天收盤價」與「當天 N 日均線值」的相對位置，就能推算最近 N 天買進者平均是賺是賠——這是解讀均線支撐/壓力、助漲/助跌等後續規則的根本邏輯基礎。
- **可程式化**: 是
- **所需資料**: 收盤價（Close）、N 日均線（MA(N)）
- **計算公式**:
```
IF Close[t] > MA(N, t):
    近 N 日買進者「平均而言」處於獲利狀態（均線代表的平均成本 < 目前收盤價）
ELIF Close[t] < MA(N, t):
    近 N 日買進者「平均而言」處於虧損狀態（均線代表的平均成本 > 目前收盤價）
ELSE:
    近 N 日買進者平均成本與目前股價相當（損益兩平）
```
- **參數**: N（均線天期，預設可用 5、10、20，依觀察族群而定，無固定值）
- **可回測**: 是
- **信心**: 89/100 高
