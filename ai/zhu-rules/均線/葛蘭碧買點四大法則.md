# 葛蘭碧買點四大法則

- **Rule ID**: R-MA-19
- **名稱**: 葛蘭碧買點四大法則
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.295-301
- **解讀**: 葛蘭碧（Granville Joseph）於1960年發表均線理論，並提出以股價走勢與20日均線（MA20）相對關係判斷的8大法則，其中做多的4個買點如下（與做空4個賣點合稱「葛蘭碧8大法則」，賣點見「葛蘭碧賣點四大法則」規則）。買點①落底反轉買點：MA20方向由下降轉為走平甚至上揚，且股價收盤由均線下方向上突破MA20，代表行情由跌轉漲的反轉轉折處。買點②多頭段回測支撐買點：股價在MA20上方、MA20方向向上，股價回檔到均線附近但未跌破、隨後轉而上漲，代表均線發揮支撐，此買點可在同一波多頭段落中反覆多次出現（加碼買點）。買點③短暫跌破快速收復買點：MA20方向保持上揚，股價收盤短暫跌破MA20後很快回漲到均線上方，代表行情仍是上漲走勢（與買點②的差異在於買點③是股價曾實際跌破均線但迅速拉回站上，買點②則僅回檔至均線附近未真正跌破）。買點④乖離過大逆勢搶反彈買點：空頭走勢中股價連續急跌3天以上、與MA20乖離過大（書中範例以正負乖離15%設通道帶），股價開始反彈向均線靠近，此為逆勢交易訊號，僅屬短期買進，不代表趨勢反轉，須嚴格短打（K線跌破前一日低點應停利出脫）。
- **可程式化**: 是
- **所需資料**: 收盤價（Close）、20日均線（MA20）及其逐日方向、股價與MA20的乖離率
- **計算公式**:
```
MA20_direction(t) = "上揚" IF MA20(t) > MA20(t-1) ELSE ("下彎" IF MA20(t) < MA20(t-1) ELSE "走平")
bias(t) = (Close[t] - MA20(t)) / MA20(t) * 100%    # 乖離率

# 買點①：落底反轉買點
BUY_SIGNAL_1(t) =
    MA20_direction(t-1) IN ["下降轉走平", "走平"]  OR  MA20_direction(t) == "上揚"
    AND Close[t-1] < MA20(t-1)      # 前一日收盤仍在均線下方
    AND Close[t] > MA20(t)           # 當日收盤向上突破MA20

# 買點②：多頭段回測支撐買點（可在同一波多頭段落中反覆出現）
BUY_SIGNAL_2(t) =
    Close[t] > MA20(t)                  # 股價在均線上方
    AND MA20_direction(t) == "上揚"
    AND MIN(Low[t-2..t]) 接近 MA20(t) 但未跌破 (Close[t-1] >= MA20(t-1) 附近)
    AND Close[t] > Close[t-1]            # 隨後轉而上漲

# 買點③：短暫跌破快速收復買點
BUY_SIGNAL_3(t) =
    MA20_direction(t) == "上揚"
    AND EXISTS day d IN [t-3, t-1]: Close[d] < MA20(d)   # 近期曾短暫跌破
    AND Close[t] > MA20(t)                                 # 當日收盤已收復站上MA20

# 買點④：乖離過大逆勢搶反彈買點（逆勢短打）
BUY_SIGNAL_4(t) =
    main_trend == "空頭"
    AND Close[t] < MA20(t)
    AND 連續下跌天數(t) >= 3
    AND bias(t) <= -15%                    # 負乖離超過15%（通道帶參數可調）
    AND Close[t] > Close[t-1]               # 股價開始反彈向均線靠近
    # 出場：反彈後K線跌破前一日低點應停利出脫（逆勢短打，非趨勢反轉）
```
- **參數**: MA 天期固定為20（原著設定）、乖離率通道帶（買點④，預設±15%，可調整）、連續下跌天數門檻（買點④，預設3天）
- **可回測**: 是
- **信心**: 89/100 高
