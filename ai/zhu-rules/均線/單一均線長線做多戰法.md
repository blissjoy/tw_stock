# 單一均線長線做多戰法

- **Rule ID**: R-MA-26
- **名稱**: 單一均線長線做多戰法
- **分類**: 均線
- **原文與頁碼**: P04-C3-均線戰法與實戰活用.md，上冊 p.322-324（戰法5）
- **解讀**: 使用MA20作為進出判斷依據，MA60作為長期趨勢方向轉折判斷依據。操作守則（表4-3-8）：股價站上MA20且在MA60之上時買進多單（前提是MA10/MA20兩線黃金交叉、多排上揚）；股價跌破MA20但仍在MA60之上時賣出多單、空手觀望。選股條件涵蓋多時間框架：日線需多頭確認且MA10/MA20/MA60多排向上；週線需趨勢多頭確認且週MA10/MA20多排向上；月線需大量突破下降切線等條件。進場條件：日線多頭底部趨勢確認、均線4線多排；底部多頭趨勢確認、MA10/MA20多排向上、股價在MA20之上、第1次回測未跌破MA20，股價上漲突破MA5及昨日K線最高點時進場。停損邏輯同R-MA-21。操作程序含多時間框架切換的停利邏輯：（1）日線符合多頭短線條件先進場，嚴格停損；（2）日線短線上漲一波、均線4線多排、已獲利後改做長線，若週線也符合多頭、週均線3線多排，改守MA20操作；（3）波段獲利超過20%以上且股價與MA20乖離>20%，收盤跌破MA10停利；（4）日週月線多頭不變，可依日線多頭條件持續進場，嚴格停損；（5）日線高檔爆量、趨勢轉空頭確認（可能尚未跌破MA20或MA60），先停利出場，之後若日線再符合多頭短線條件再重新進場；（6）多頭進入末升段高檔，或股價已上漲接近底部起漲1倍，此時再進場只適合短線操作、改守MA5停利。
- **可程式化**: 部分（月線「大量突破下降切線」需搭配切線分類規則，其餘均線與價格條件可完全程式化）
- **所需資料**: 收盤價、MA5、MA10、MA20、MA60（日線與週線）、成交量
- **計算公式**:
```
# 操作守則
IF Close[t] > MA20(t) AND Close[t] > MA60(t) AND golden_cross(MA10, MA20) AND MA10(t)>MA20(t)>...多排上揚:
    ACTION = "買進多單"
ELIF Close[t] < MA20(t) AND Close[t] > MA60(t):
    ACTION = "賣出多單，空手觀望"

# 選股條件
SELECT_STOCK =
    日線多頭確認 AND MA10(t)>MA20(t)>MA60(t) 皆向上
    AND 週線趨勢多頭確認 AND 週MA10(t)>週MA20(t) 皆向上

# 進場
ENTRY(t) =
    日線多頭底部趨勢確認 AND 均線4線多排 (MA5>MA10>MA20>MA60)
    AND 股價第1次回測MA20未跌破
    AND Close[t] > MA5(t) AND Close[t] > High[t-1]

# 停損（見 R-MA-21 統一邏輯）
stop_loss = IF entry_candle_gain_pct < 5%: 上漲轉折最低點 ELSE Low[entry_day]

# 操作程序（多時間框架停利切換）
stage = "短線"
IF stage=="短線" AND 均線4線多排 AND 已獲利 AND 週線多頭 AND 週均線3線多排:
    stage = "長線"; stop_basis = MA20
IF 波段獲利率 > 20% AND bias(Close, MA20) > 20%:
    IF Close[t] < MA10(t): TAKE_PROFIT
IF 日線高檔爆量 AND 趨勢轉空頭確認:
    EXIT (先停利，未必已跌破MA20/MA60)
    WAIT 再次符合多頭短線條件才重新進場
IF 股價已上漲達底部起漲幅度的1倍 OR 進入末升段高檔:
    stage = "短線"; stop_basis = MA5   # 改守MA5停利
```
- **參數**: MA5、MA10、MA20、MA60（日線與週線）、波段獲利門檻（20%）、乖離門檻（20%）
- **可回測**: 部分（依賴切線突破判斷模組是否已實作；核心均線邏輯可先行回測）
- **信心**: 58/100 中
