# 均線反應速度與權重遞減

- **Rule ID**: R-MA-03
- **名稱**: 均線反應速度與權重遞減
- **分類**: 均線
- **原文與頁碼**: P04-C1-均線的基本概念.md，上冊 p.258-259
- **解讀**: 不同天期均線對股價漲跌的反應速度不同：天期越短，單一交易日漲跌對均線的影響權重越大（N 日均線中，單日漲跌的權重＝1/N），因此短期均線（如 MA5）只要股價連跌 1、2 天就可能轉彎向下；長期均線（如 MA20）因單日權重僅 1/20，需要更長時間、更持續的價格變化才會改變方向。這是全書解釋「均線多頭排列瓦解順序」「短均線先轉彎、長均線後轉彎」「移動扣抵可預判轉折」等規則的數學根本原因，也是判斷均線支撐力/壓力強度（天期越長強度越強）的理論基礎。
- **可程式化**: 是
- **所需資料**: 各天期均線（MA5、MA10、MA20、MA60…）之逐日數值序列
- **計算公式**:
```
# 單一交易日漲跌對 N 日均線的影響權重
weight(N) = 1 / N
# 權重越大（N 越小），均線對最新一日價格變化越敏感，轉向所需天數越少
# 可用於量化比較：同樣的價格衝擊下，不同天期均線預期反應速度
sensitivity_rank: MA5 > MA10 > MA20 > MA60 > MA120 > MA240
# 應用：判斷「短均線已轉彎但長均線尚未轉彎」是否為正常現象（是），
#       以及估算長天期均線約需連續幾日同向價格變化才會轉向
```
- **參數**: N（均線天期）
- **可回測**: 是
- **信心**: 89/100 高
