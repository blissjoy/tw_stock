# 均線空頭排列

- **Rule ID**: R-MA-09
- **名稱**: 均線空頭排列
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.279、p.282
- **解讀**: 與均線多頭排列完全相反的定義：當長天期均線在上、短天期均線在下，依序由上而下排列時，稱為「均線空頭排列」，標準排列為 MA240 > MA120 > MA60 > MA20 > MA10 > MA5。空頭排列代表每條均線對股價都同時發揮壓力與助跌作用，排列的均線條數越多（6線空排：5日、10日、月線、季線、半年線、年線皆呈長在上短在下），代表股價長期下跌壓力越強、下跌時間也越持久，適合鎖股做中長空。同多頭排列一樣，通常要求各條均線「方向皆向下」才算有效的空頭排列。
- **可程式化**: 是
- **所需資料**: 多條不同天期均線（至少 MA5、MA10、MA20，可擴充 MA60、MA120、MA240）之逐日數值與方向
- **計算公式**:
```
IS_BEARISH_ALIGNMENT(t) =
    MA5(t) < MA10(t) < MA20(t)                       # 3 線空排（最低標準）
    # 可擴充：MA5(t) < MA10(t) < MA20(t) < MA60(t) < MA120(t) < MA240(t)  # 6 線空排

IS_BEARISH_ALIGNMENT_STRICT(t) =
    IS_BEARISH_ALIGNMENT(t)
    AND MA5(t) < MA5(t-1)
    AND MA10(t) < MA10(t-1)
    AND MA20(t) < MA20(t-1)
    # 各均線方向皆向下

bearish_line_count(t) = 依序檢查 MA5<MA10<MA20<MA60<MA120<MA240 中，
                         從短天期開始連續滿足小於關係的均線條數
```
- **參數**: 均線組合（預設 MA5、MA10、MA20；可擴充至 MA60、MA120、MA240 做 4～6 線空排判斷）
- **可回測**: 是
- **信心**: 90/100 高
