# 均線糾結向下跌破做空SOP

- **Rule ID**: R-MA-18
- **名稱**: 均線糾結向下跌破做空SOP
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.293-294
- **解讀**: 與「均線糾結向上突破做多SOP」完全對稱。均線糾結後，若股價出現放量下跌、跌破均線糾結區的中長黑K線，會立刻形成多條均線方向向下的空頭排列，口訣為「跌破均線糾結的向下黑K線，是起跌的開始」。書中8步驟操作SOP：（1）平常每天收盤後檢視弱勢股，看到當天中長黑K跌破均線糾結，準備隔天進場做空；（2）發動後一般會有一段跌幅，第2、3天若沒有連續大跌仍可進場做空；（3）若起跌第2天就跌停鎖死，不追空跌停板，改等第1段飆跌後反彈再跌時切入，賺第2段飆跌（多數飆跌有2段下跌期）；（4）停損設在當天進場K線最高點，若進場為小黑K，則以反彈5%為停損；（5）下跌後用「智慧K線交易法」操作：每天收盤未突破前一天K線最高點就續抱空單；（6）每天下午1點20分檢視，若收盤確認突破前一天K線最高點，掛市價單回補；（7）回補後若仍維持空頭、再次出現收盤跌破前一天K線最低點的下跌黑K，可再次進場做空，重複步驟4～6；（8）下跌一段後若再度盤整、均線再次糾結，同樣把握再次向下跌破的做空機會。
- **可程式化**: 是
- **所需資料**: 收盤價、最高價、最低價、成交量、MA5/MA10/MA20（糾結辨識）
- **計算公式**:
```
# 觸發條件（進場）
ENTRY_TRIGGER(t) =
    IS_MA_CONVERGED(t-1)                                 # 前一日仍處於均線糾結狀態
    AND Close[t] < MIN(Low[t-N_converge .. t-1])           # 收盤跌破糾結區間低點
    AND Volume[t] > Volume[t-1] * volume_multiplier         # 放量
    AND IS_LONG_BLACK_CANDLE(t)                              # 中長黑K線

# 若進場當天即跌停鎖死，改為等待：
IF is_limit_down(t):
    WAIT for 第1段飆跌後反彈再次下跌的訊號才進場（賺第2段飆跌）

# 停損
IF entry_candle_loss_pct < threshold_small (小黑K):
    stop_loss = Close[entry_day] * (1 + 0.05)     # 反彈5%停損
ELSE:
    stop_loss = High[entry_day]                     # 進場K線最高點

# 持股（空單）與出場（智慧K線交易法）
HOLD SHORT WHILE Close[t] <= High[t-1]      # 每日收盤未突破前一日最高點則續抱空單
AT 13:20 daily:
    IF Close[t] > High[t-1]:
        COVER at market price（市價單回補）

# 再進場
IF 回補後反彈仍維持空頭趨勢
   AND 之後出現新的下跌黑K，且 Close < Low[t-1]:
        重新進場做空，重複停損/持股/出場規則
```
- **參數**: volume_multiplier（放量倍數門檻，建議回測校準，初始可用1.5～2倍）、停損反彈幅度（預設5%）、檢視時間點（每日13:20）
- **可回測**: 是
- **信心**: 58/100 中
