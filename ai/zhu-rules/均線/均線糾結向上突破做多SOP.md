# 均線糾結向上突破做多SOP

- **Rule ID**: R-MA-17
- **名稱**: 均線糾結向上突破做多SOP
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.290-292
- **解讀**: 均線糾結後，若股價出現放量上漲、且突破均線糾結區的中長紅K線，代表短天期均線最先反應上揚，後續中長天期均線依序上揚形成多頭排列，構成強勢上漲態勢，口訣為「突破均線糾結的向上紅K線，是起漲的開始」。書中提供完整8步驟操作SOP：（1）平常每天收盤後檢視強勢股，看到當天放量上漲、突破均線糾結的中長紅K線，準備隔天進場；（2）發動後一般會有一段漲幅，第2、3天若沒有大漲仍可進場；（3）若起漲第2天就飆漲停，不追漲停板，改等第1段飆漲後回檔再漲時切入，賺第2段飆漲（多數飆股有2段飆漲）；（4）停損設在當天進場K線最低點，若進場為小紅K，則以下跌5%為停損；（5）上漲後用「智慧K線交易法」操作：每天收盤未跌破前一天K線最低點就續抱；（6）每天下午1點20分檢視，若收盤確認跌破前一天K線最低點，掛市價單賣出；（7）賣出後若仍維持多頭、再次出現收盤突破前一天K線最高點的上漲紅K，可再次進場，重複步驟4～6；（8）上漲一段後若再度盤整、均線再次糾結，同樣把握再次向上突破的做多機會。
- **可程式化**: 是
- **所需資料**: 收盤價、最高價、最低價、成交量、MA5/MA10/MA20（糾結辨識）
- **計算公式**:
```
# 觸發條件（進場）
ENTRY_TRIGGER(t) =
    IS_MA_CONVERGED(t-1)                                 # 前一日仍處於均線糾結狀態
    AND Close[t] > MAX(High[t-N_converge .. t-1])          # 收盤突破糾結區間高點
    AND Volume[t] > Volume[t-1] * volume_multiplier         # 放量（成交量倍數門檻）
    AND IS_LONG_RED_CANDLE(t)                                # 中長紅K線

# 若進場當天即漲停鎖死，改為等待：
IF is_limit_up(t):
    WAIT for 第1段飆漲後回檔再次上漲的訊號才進場（賺第2段飆漲）

# 停損
IF entry_candle_gain_pct < threshold_small (小紅K):
    stop_loss = Close[entry_day] * (1 - 0.05)     # 下跌5%停損
ELSE:
    stop_loss = Low[entry_day]                     # 進場K線最低點

# 持股與出場（智慧K線交易法）
HOLD WHILE Close[t] >= Low[t-1]        # 每日收盤未跌破前一日最低點則續抱
AT 13:20 daily:
    IF Close[t] < Low[t-1]:
        SELL at market price（市價單賣出）

# 再進場
IF 賣出後回檔仍維持多頭趨勢
   AND 之後出現新的上漲紅K，且 Close > High[t-1]:
        重新進場，重複停損/持股/出場規則
```
- **參數**: volume_multiplier（放量倍數門檻，書中未給精確數值，建議回測校準，初始可用1.5～2倍）、停損下跌幅度（預設5%）、檢視時間點（每日13:20）
- **可回測**: 是
- **信心**: 58/100 中
