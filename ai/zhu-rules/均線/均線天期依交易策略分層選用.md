# 均線天期依交易策略分層選用

- **Rule ID**: R-MA-04
- **名稱**: 均線天期依交易策略分層選用
- **分類**: 均線
- **原文與頁碼**: P04-C1-均線的基本概念.md，上冊 p.259-260
- **解讀**: 均線的優點是可以過濾雜訊、掌握總體走勢，但長天期均線落後、盤整時容易鈍化失真，因此書中建議依交易策略的時間長度選用不同天期均線：長期策略用週線的 10 週、20 週均線；中期策略用日線的 20 日、60 日均線；短期策略用日線的 3 日、5 日、10 日均線；超短期當沖用 1 分、5 分、15 分、60 分均線。這條規則本身不是進出場訊號，而是「參數選擇指引」，用於決定後續其他均線規則（多頭排列、葛蘭碧法則、均線戰法等）在不同交易時間框架下應套用哪一組均線天期參數。
- **可程式化**: 部分
- **所需資料**: 交易週期設定（日線/週線/分鐘線）、交易策略時間長度分類
- **計算公式**:
```
FUNCTION select_ma_periods(holding_horizon):
    IF holding_horizon == "長期策略"（數月）:
        RETURN 週線 MA10, MA20
    ELIF holding_horizon == "中期策略"（數星期）:
        RETURN 日線 MA20, MA60
    ELIF holding_horizon == "短期策略"（幾天）:
        RETURN 日線 MA3, MA5, MA10
    ELIF holding_horizon == "當沖策略":
        RETURN 分線 MA1min, MA5min, MA15min, MA60min
```
- **參數**: holding_horizon（交易時間框架分類：長期/中期/短期/當沖）
- **可回測**: 否（本身是參數選擇規則，非可獨立產生買賣訊號的規則，但可作為其他規則的參數輸入用於回測）
- **信心**: 58/100 中
