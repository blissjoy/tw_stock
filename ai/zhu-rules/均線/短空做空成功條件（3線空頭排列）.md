# 短空做空成功條件（3線空頭排列）

- **Rule ID**: R-MA-11
- **名稱**: 短空做空成功條件（3線空頭排列）
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.279-281
- **解讀**: 與短多做多成功條件完全對稱。在頭部盤頭完成的空頭股常常「不好操作」，多因均線尚未整理成3線空頭排列，或季線（MA60）仍在上升、尚有支撐力道。做短空要成功，需同時滿足：（1）波浪型態符合「頭頭低、底底低」的空頭架構；（2）均線至少完成3線空頭排列（MA5、MA10、MA20），且三條均線方向都向下；（3）股價收盤跌破20日均線。三個條件同時成立時，做空才容易成功獲利；建議等到完成3線或4線空頭排列時再進場。
- **可程式化**: 部分（波浪型態「頭頭低、底底低」需先有波段高低點辨識演算法支援，屬於另一分類「趨勢判定」的規則；均線與收盤價條件可完全程式化）
- **所需資料**: 收盤價（Close）、MA5、MA10、MA20、MA60、波段高低點序列（頭部/底部標記）
- **計算公式**:
```
IS_SHORT_TERM_BEARISH_SETUP(t) =
    wave_pattern(t) == "頭頭低_底底低"        # 依賴外部波浪型態判斷規則（趨勢判定分類）
    AND MA5(t) < MA10(t) < MA20(t)            # 3 線空頭排列
    AND MA5(t) < MA5(t-1)
    AND MA10(t) < MA10(t-1)
    AND MA20(t) < MA20(t-1)                    # 3 條均線方向皆向下
    AND Close[t] < MA20(t)                     # 收盤跌破 20 日均線

# 加強版（4線空排，成功率更高）：
IS_SHORT_TERM_BEARISH_SETUP_STRONG(t) =
    IS_SHORT_TERM_BEARISH_SETUP(t)
    AND MA5(t) < MA10(t) < MA20(t) < MA60(t)
    AND MA60(t) < MA60(t-1)
```
- **參數**: 均線組合固定為 MA5、MA10、MA20（加強版含 MA60）
- **可回測**: 部分（依賴波浪型態辨識模組是否已實作；若暫不含波浪型態條件，可先用均線＋跌破MA20兩項條件做簡化版回測）
- **信心**: 58/100 中
