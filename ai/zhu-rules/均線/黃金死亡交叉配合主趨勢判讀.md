# 黃金死亡交叉配合主趨勢判讀

- **Rule ID**: R-MA-15
- **名稱**: 黃金死亡交叉配合主趨勢判讀
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.287-288
- **解讀**: 書中特別強調，黃金交叉與死亡交叉「必須配合當時所處的主趨勢（多頭或空頭）解讀，不能只要出現黃金交叉就做多、死亡交叉就做空」。在多頭趨勢中：黃金交叉＝回檔修正結束後的反轉上漲，是買進參考位置；死亡交叉＝上漲一段後要回檔，是獲利賣出參考位置（出多單，非做空）。在空頭趨勢中：死亡交叉＝空頭反彈後繼續下跌，是做空參考位置；黃金交叉＝空頭下跌一波後的反彈，是獲利回補參考位置（回補空單，非做多）。也就是說，同一個交叉事件在不同主趨勢下的操作意涵完全不同，這是判斷交叉訊號時不可省略的前置條件。
- **可程式化**: 是
- **所需資料**: 主趨勢方向（多頭/空頭，可由均線多頭排列/空頭排列規則或波浪型態規則判定）、黃金交叉事件、死亡交叉事件
- **計算公式**:
```
FUNCTION interpret_cross(main_trend, cross_event):
    IF main_trend == "多頭":
        IF cross_event == "黃金交叉":
            RETURN "買進參考位置（回檔結束、反轉上漲）"
        ELIF cross_event == "死亡交叉":
            RETURN "獲利賣出參考位置（出多單，回檔修正）"
    ELIF main_trend == "空頭":
        IF cross_event == "死亡交叉":
            RETURN "做空參考位置（反彈結束、續跌）"
        ELIF cross_event == "黃金交叉":
            RETURN "獲利回補參考位置（回補空單，反彈修正）"
    RETURN "無明確訊號"
```
- **參數**: main_trend 判定方式（可用均線多頭/空頭排列規則 R-MA-08／R-MA-09，或另一分類「趨勢判定」的波浪型態規則）
- **可回測**: 是
- **信心**: 89/100 高
