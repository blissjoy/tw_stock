# 移動扣抵預判均線轉向與交叉時點

- **Rule ID**: R-MA-05
- **名稱**: 移動扣抵預判均線轉向與交叉時點
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.263-267
- **解讀**: 均線每日更新的公式是「今日均值 ＝（過去 N 天收盤價總和－最前面一天收盤價＋今日收盤價）÷ N」，其中被扣除的最舊一天收盤價稱為「扣價」，新加入的今日收盤價稱為「抵價」，這個過程稱為「移動扣抵」。由於「扣價」是歷史上已經確定的數值（未來某一天將被扣除的收盤價，在今天就已知），因此可以事先算出未來每一天的扣抵值，並與目前股價或預估股價比較，在不看未來走勢的情況下預判均線是否即將轉彎：只要收盤價大於扣抵值，均線會上彎；小於扣抵值，均線會下彎。將此邏輯逐日往未來推算多天，還能估算兩條均線何時會交叉（多頭排列或空頭排列大約何時完成）、以及盤整還需要多久時間，是全書「未卜先知」判斷均線走向最重要的工具。
- **可程式化**: 是
- **所需資料**: 收盤價（Close）歷史序列、N 日均線（MA(N)）
- **計算公式**:
```
# 扣抵值：未來第 t+k 天將被扣除的收盤價，其實就是「今天」已知的歷史值
offset_value(N, t, k) = Close[t + k - N]   # k=1,2,3... 為未來第 k 個交易日

# 單日轉向預判：
FOR future day (t+k):
    IF Close[t+k] > offset_value(N, t, k):
        MA(N) 在第 t+k 天會上彎（數值較前一日增加）
    ELIF Close[t+k] < offset_value(N, t, k):
        MA(N) 在第 t+k 天會下彎（數值較前一日減少）
    ELSE:
        MA(N) 走平

# 交叉時點估算：對短均線 MA(Ns) 與長均線 MA(Nl) 分別逐日推算未來 k 天的
# offset_value 序列，搭配對股價的假設（例如維持不跌／不漲），
# 逐日模擬 MA(Ns, t+k) 與 MA(Nl, t+k)，找出兩線數值相等或交叉的最早 k，
# 即為預估的黃金交叉／死亡交叉發生時點；也可估算多頭/空頭排列完成所需天數，
# 或盤整（均線糾結）預期還會持續幾天。
```
- **參數**: N（均線天期）、Ns/Nl（短/長均線天期組合，如 10 與 20）、k（預判未來天數）
- **可回測**: 是
- **信心**: 89/100 高
