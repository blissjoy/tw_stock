# 均線上彎的支撐與助漲作用

- **Rule ID**: R-MA-06
- **名稱**: 均線上彎的支撐與助漲作用
- **分類**: 均線
- **原文與頁碼**: P04-C2-均線的重要功能與實戰活用.md，上冊 p.270-272
- **解讀**: 當均線方向向上時，會對股價發揮三大作用：（1）支撐作用——股價回檔到均線時容易止跌回升；（2）助漲作用——股價回檔跌破均線後，只要均線仍向上，會把股價拉回站上均線、續漲；（3）減弱下跌黑K的下殺力道——多頭中出現中長黑K回檔，若均線仍上揚，可減緩續跌力道。這三大作用的強度受兩個條件影響：均線天期越長、作用越強（MA20 支撐力 > MA10 > MA5）；同天期均線上彎角度越大、作用越強。此規則是判斷「股價拉回是否會在均線附近止穩」的核心依據，也是葛蘭碧買點②③與均線戰法「持股規則」（未跌破均線則續抱）的理論基礎。
- **可程式化**: 部分
- **所需資料**: 收盤價（Close）、均線（MA(N)）及其逐日方向（上揚/走平/下彎）、均線斜率（角度）
- **計算公式**:
```
MA_direction(N, t) = "上揚" IF MA(N,t) > MA(N,t-1) ELSE ("下彎" IF MA(N,t) < MA(N,t-1) ELSE "走平")

IF MA_direction(N, t) == "上揚":
    IF Close[t] 回檔接近 MA(N,t) 但未跌破 (Close[t] >= MA(N,t)):
        判定：均線發揮「支撐作用」，止跌回升機率較高
    ELIF Close[t] < MA(N,t) 但前一日 Close 曾高於 MA(N,t) （即剛跌破均線）:
        判定：均線發揮「助漲作用」，股價可能被拉回站上均線
    ELIF 當日為中長黑K且 Close[t] < Open[t]:
        判定：均線上揚可「減弱黑K下殺力道」，續跌動能有限

# 強度排序（僅供比較，非嚴格數值）：
strength(N1) > strength(N2)  IF N1 > N2   # 天期越長，作用越強
strength ∝ |slope(MA(N))|                  # 上彎角度（斜率）越大，作用越強
```
- **參數**: N（均線天期，MA5/MA10/MA20 常用比較組）
- **可回測**: 是
- **信心**: 58/100 中
