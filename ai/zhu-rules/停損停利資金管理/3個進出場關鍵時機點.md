# 3個進出場關鍵時機點

- **Rule ID**: R-RISK-06
- **名稱**: 3個進出場關鍵時機點
- **分類**: 停損停利資金管理
- **原文與頁碼**: P12-C3-股市贏家2大交易守則.md，下冊 p.368、p.372-373（「贏家的5個永遠認知」總綱、「守則2：牢牢抓住3個時機點」：時機點1進場、時機點2停利、時機點3停損）
- **解讀**: 本規則整理書中「股市贏家2大交易守則」之守則2，把交易全流程收斂成3個最關鍵的決策時機點，是連接「選股/進場條件（其他分類）」「勝率方程式（R-RISK-03）」「停損方法（R-RISK-01）」的操作層總結：
  1. **時機點1：勝率高的交易機會才出手**（p.372）：策略優劣取決於「執行後的整體勝率」與「累積績效」，不能要求每一筆單獨交易都獲利，重點是整體資金能否持續成長；具備一套高勝率選股方法＋穩健操作策略，經時間反覆驗證，理想績效會隨交易次數增加而愈明顯（呼應大數法則）。進場條件應限定在歷史高勝率的型態/位置，而非隨機或無把握出手——可對應 `P12-C1-高勝率方程式.md` 所列做多/做空各6種高勝率進場型態（共通要素：①均線多排(4線多排/3-4線糾結) ②實體K棒且漲跌幅門檻2% ③爆大量 ④收盤價確認突破/跌破關鍵價位），以及趨勢判定分類的 `R-TREND-16 做多進場10大戒律`、`R-TREND-17 做空進場10大戒律`（反向過濾清單，命中任一條則不進場，性質互補：一個定義「該進場的高勝率位置」，一個定義「不該進場的位置」）。
  2. **時機點2：獲得最大利潤才停利**（p.372）：不可能要求自己每次都賣在最高點，在適當時機獲利了結是重要課題；技術面可預先評估上漲的壓力位置，並配合成交量與K線表現判斷股價是否已走到一波轉折關鍵點。停利判斷應結合「預估壓力位置＋成交量變化＋K線轉折訊號」三項條件綜合確認，而非單一指標；固定一套停利紀律可用來驗證當天訊號是「真訊號」還是「假訊號」，才能穩定賣在相對高點。停利邏輯應標準化、可重複驗證。此處的「預估壓力位置」對應支撐壓力分類的各項規則（例如 `R-SR-01` 轉折高點壓力、`R-SR-12` 型態頸線量測目標價等）。
  3. **時機點3：接受做錯時的合理停損**（p.373）：不可能要求每一筆交易都絕對成功，因此風險控制是必要條件；快速停損、擺脫虧損股票，將資金收回並重新投入高勝率的強勢股票，可以在短時間內反敗為勝，同時不影響整體績效。停損後應立即將資金轉入符合高勝率進場條件的新標的，而非停留觀望——停損動作與資金再配置應視為同一套流程的兩個步驟，而不是停損後現金閒置。此時機點的具體停損價位邏輯見 `R-RISK-01 停損5種設法總覽`。

  此外，本節開篇「贏家的5個永遠認知」（p.368-369）中與此3個時機點直接呼應的兩條：「永遠只操作符合技術分析高勝率條件的股票」（＝時機點1）、「永遠相信技術分析，紀律操作」（貫穿3個時機點的執行紀律要求），本規則一併納入作為總綱背景，不重複拆出為獨立規則。

  三個時機點合觀，是一套完整的「進場（高勝率過濾）→ 停利（多條件確認真假訊號）→ 停損（快速停損並立即資金再配置）」操作閉環，也是全書停損停利邏輯在「操作紀律」層面的最終總結：勝率方程式（R-RISK-03）提供數學期望值框架，停損方法總覽（R-RISK-01）提供價位設定邏輯，本規則則提供「在交易的哪個時間點、依據什麼條件組合做決策」的流程骨架。
- **可程式化**: 部分（時機點1的「高勝率型態過濾」與時機點3的「停損後資金再配置」可程式化為明確的規則鏈；時機點2「停利的真假訊號判斷」涉及K線轉折訊號的主觀辨識成分，需依賴K棒型態分類規則的量化輸出才能部分程式化，書中本節未給出停利的精確數字門檻，需引用其他章節如 R-TREND-14/15 的10%/20%停利數字）
- **所需資料**: 高勝率進場型態判定結果（均線排列、K棒實體漲跌幅、成交量倍數、關鍵價位突破/跌破，見P12-C1與均線/K棒型態/趨勢判定分類）、支撐壓力位置（見支撐壓力分類）、K線轉折訊號（見K棒型態分類）、停損價位（見R-RISK-01）、資金可用餘額
- **計算公式**:
```
# 時機點1：進場過濾
FUNCTION 是否為高勝率進場機會(候選股):
    RETURN 符合任一高勝率型態(候選股)          # 見P12-C1做多/做空6型態
       AND NOT 命中10大戒律(候選股)             # 見 R-TREND-16 / R-TREND-17

# 時機點2：停利判斷（三條件綜合確認）
FUNCTION 是否應停利(持股):
    已達預估壓力位置 = 股價 >= 持股.預估壓力位置       # 見支撐壓力分類
    成交量異常變化 = 量能顯示轉弱訊號(持股)
    K線轉折訊號 = 出現轉折K線型態(持股)                # 見K棒型態分類
    RETURN 已達預估壓力位置 AND (成交量異常變化 OR K線轉折訊號)

# 時機點3：停損與資金再配置（同一流程的兩步驟）
FUNCTION 執行停損與再配置(持股, 停損價):
    IF Close[t] 觸發停損條件(持股, 停損價):        # 見 R-RISK-01
        平倉(持股)
        可用資金 += 持股.平倉金額
        新標的 = 篩選高勝率進場機會(可用股票池)     # 立即尋找新的高勝率標的，資金不閒置
        IF 新標的存在:
            準備進場(新標的)
```
- **參數**: 本規則不新增獨立數字參數，所引用的數字門檻（10大戒律的追高天數3根、高勝率型態的漲跌幅2%門檻、停利10%/20%等）皆定義於被引用的原規則中
- **可回測**: 部分（時機點1、3的規則鏈可回測；時機點2因「真假訊號判斷」的質化成分，需先將K線轉折訊號量化後才能完整回測）
- **信心**: 62/100 中（3個時機點的邏輯架構與方向書中明確條列＝高；但時機點2「停利真假訊號判斷」與時機點1「高勝率型態」的部分細節數字定義在其他章節/分類，本節正文本身未重複給出精確數字，屬於整合型規則，故信心評為中）
