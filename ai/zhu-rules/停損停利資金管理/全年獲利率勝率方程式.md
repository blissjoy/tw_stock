# 全年獲利率勝率方程式

- **Rule ID**: R-RISK-03
- **名稱**: 全年獲利率勝率方程式
- **分類**: 停損停利資金管理
- **原文與頁碼**: P12-C1-高勝率方程式.md，下冊 p.353-355（「股票輸贏關鍵是勝率」「年獲利20%的勝率方程式」「提高2倍年獲利的進階方式」及圖表12-1-1「5種贏家獲利方程式」對照表）
- **解讀**: 本規則是全書資金管理／績效試算的核心數學公式，把「勝率」「停損%」「停利%」「交易次數」四個變數整合為單一年化報酬率公式：
  > **全年獲利率 ＝（勝場次數 × 每次停利%）－（敗場次數 × 每次停損%）**
  書中特別強調：勝率單獨不足以判斷策略優劣，必須與停損/停利幅度一起計算期望值——操作10次賺9次、輸1次看似穩賺，但若輸的那1次把先前累積獲利全部吐回甚至倒賠，仍是白忙一場；停損5%是硬性上限、不能擴大，若無法絕對做到，1次重大虧損就可能瓦解所有累積成果。

  書中以「初學者基準方程式」為起點：全年交易20次、勝率50%（10勝10敗）、停損5%、停利7% → 全年獲利率 = 10×7% − 10×5% = 70% − 50% = **20%**。並提出3個可分別調整的槓桿方向（其餘條件不變，停損停利仍為5%/7%）：①**提高勝率**至60%（12勝8敗）→ 12×7%−8×5%=84%−40%=**44%**；②**增加操作次數**至全年30次、勝率仍50%（15勝15敗）→ 15×7%−15×5%=105%−75%=**30%**；③**增加每次操作獲利**改做長波段，每次獲利拉高到15%、全年操作12次、勝率50%（6勝6敗）→ 6×15%−6×5%=90%−30%=**60%**。

  達成「年獲利2成」基準方程式的4個操作重點（p.354）：①不需要頻繁交易（每月僅約1.7次）；②勝率只要達1/2（多頭順勢做多、空頭順勢做空即容易達成）；③絕對嚴格執行停損5%；④停利可以擴大（達7%初始目標時，若無危險訊號且趨勢續強，可上調停利目標，屬「移動停利/讓利潤奔跑」的雛形邏輯）。

  **圖表12-1-1「5種贏家獲利方程式」對照表**（p.355，本筆記已訂正原書一處數字矛盾，見下方★訂正說明）：

  | 方式 | 適用個人條件 | 全年交易次數 | 勝敗率 | 停利率 | 停損率 | 年獲利率 |
  |---|---|---|---|---|---|---|
  | 1 | 無法看盤或初學者 | 20次 | 50%（10勝10敗） | 7% | 5% | 20% |
  | 2 | 學習技術分析、提高選股能力者 | 20次 | 60%（12勝8敗） | 7% | 5% | 44% |

  ★**訂正說明**：原書表格第3列印刷為「30次／50%（**10勝10敗**）」，但10+10=20與「全年交易次數30次」矛盾，且與正文算式「(15次×7%)－(15次×5%)=105%－75%=30%」對照，應為「**15勝15敗**」，判斷是原書表格的印刷誤植。本規則已依正文算式訂正為15勝15敗，後續程式化與回測請使用訂正後數字，勿照搬原書表格印刷值。
- **可程式化**: 是
- **所需資料**: 全年（或指定期間）總交易次數、勝場次數、敗場次數（或勝率）、每次停利百分比（可為固定值或平均值）、每次停損百分比（可為固定值或平均值）
- **計算公式**:
```
FUNCTION 全年獲利率(總交易次數, 勝率, 停利率, 停損率):
    勝場次數 = 總交易次數 * 勝率
    敗場次數 = 總交易次數 * (1 - 勝率)
    全年獲利率 = (勝場次數 * 停利率) - (敗場次數 * 停損率)
    RETURN 全年獲利率

# 書中5組範例參數與結果（可作為單元測試基準值）
方案1 = 全年獲利率(20, 0.50, 0.07, 0.05)   # = 20%
方案2 = 全年獲利率(20, 0.60, 0.07, 0.05)   # = 44%
方案3 = 全年獲利率(30, 0.50, 0.07, 0.05)   # = 30%（已訂正為15勝15敗）
方案4 = 全年獲利率(30, 0.60, 0.07, 0.05)   # = 66%
方案5 = 全年獲利率(12, 0.5833, 0.20, 0.05) # ≈ 115%

# 反推應用：已知目標年化報酬率與可行的交易次數/停損停利率，反推所需最低勝率
FUNCTION 所需最低勝率(目標年獲利率, 總交易次數, 停利率, 停損率):
    # 全年獲利率 = N*p*停利率 - N*(1-p)*停損率  =>  解 p
    RETURN (目標年獲利率 / 總交易次數 + 停損率) / (停利率 + 停損率)
```
- **參數**: 停損硬性上限＝5%（此方程式的前提假設，書中強調不可擴大，若停損無法控制在5%，公式失效）；停利基準＝7%（可依情境調整，書中5組範例分別使用7%/7%/7%/7%/20%）；交易次數與勝率為使用者輸入變數
- **可回測**: 是（公式本身純數學計算，可直接用歷史交易紀錄的實際勝率/停利率/停損率代入驗證年化報酬率）
- **信心**: 94/100 高（核心公式與5組範例數字書中皆逐字給出且有完整算式可交叉驗證；唯一例外已於上方訂正說明中列出並修正）
