# 3階段資金規模與獲利目標管理

- **Rule ID**: R-RISK-04
- **名稱**: 3階段資金規模與獲利目標管理
- **分類**: 停損停利資金管理
- **原文與頁碼**: P12-C2-技術分析操作的目標管理.md，下冊 p.363-366（「晉升千萬富翁的目標設定」「達成目標4大重要關鍵」「3階段設定達成長期贏家目標」）
- **解讀**: 本規則整合書中「目標管理」的完整框架，分三層：
  1. **千萬富翁複利目標**（p.363）：以10萬元本金、每年獲利100%（賺1倍）複利計算，第7年即可達成千萬目標（10萬→20萬→40萬→80萬→160萬→320萬→640萬→1,280萬）。關鍵在於「1年是否能賺到100%獲利」，可逐層拆解為可執行的短週期報酬率門檻：年獲利100% → 每月約8.33%（12個月）→ 每週約2.1%（每月4週）→ 每2週獲利4.2%即視為完成年獲利1倍目標的基本操作單位。設定目標的意義在於有方向可執行、能定期檢討，並幫助克制「不願認賠」「貪婪不捨賣」的人性弱點。若目標本身無法達成，不輕易更動目標，而是檢討並改變操作方法。
  2. **達成目標4大重要關鍵**（p.364-365）：①**停利比停損重要**——停損須嚴格執行以免虧損擴大，但要達成獲利目標必須確實執行停利，觸及停利條件即應出場了結，不可因期待更高點而延遲；②**短線比長線重要**——短線操作能快速獲利、目標易掌握，除非出現大波段機會，否則應把握每個能快速獲利的短線操作機會（走勢轉飆股則切換飆股戰法，強勢波段股可長線短做，皆以快速獲利為第一目標）；③**強勢比漲勢重要**——目標管理設有時間限制，須選擇強勢股操作，持股於觀察週期（1週或半個月）內漲幅低於5%的牛步多頭股票，應換成強勢股操作；④**紀律操作比選股重要**——即使選到好股票，仍須嚴格執行整套操作紀律（選股條件、進場位置、停損、續抱、停利），任一環節破例即視為紀律失守，才能完成「每一次大賺小賠」的交易。
  3. **3階段資金規模／持股檔數／獲利目標設定**（p.365-366）：書中依操作熟練度分3階段循序漸進，每階段練習時間皆為6個月：

  | 階段 | 資金規模 | 持股檔數 | 是否用槓桿 | 6個月獲利目標 | 每月獲利目標 | 全年獲利目標 |
  |---|---|---|---|---|---|---|
  | 初階（累積經驗、驗證方法） | 50萬元 | 1～2檔 | 否 | 12% | 2% | 24% |
  | 進階（熟練方法、提高目標） | 80萬元 | 1～3檔 | 否 | 30% | 5% | 60% |
  | 終極（高勝率操作模式） | 100萬元（開立融資戶） | 2～5檔 | 是（加大槓桿） | 50% | 8.33% | 100% |

     - 初階目標：練習紀律、不逆勢操作、不用槓桿；6個月完成後需定出個人交易方法（選股條件：基本面/技術面/籌碼面；進場條件：多空各自基本進場條件；交易策略：練習短線交易）。
     - 進階目標：熟練個人操作方法、累積應變經驗；6個月完成後需修正交易方法（熟練主流強勢股選股、熟練多空雙向操作、熟練各種交易策略分配資金建立波段操作能力、完成情緒穩定訓練）。
     - 終極目標：建立高勝率操作模式、長短波段搭配；6個月完成後需固定交易方法（以主流強勢股操作、達到紀律機械式操作的「無我境界」、分配資金多元化金融商品操作如期指選擇權、研究基本面提高大波段獲利能力）。
  終極階段全年獲利目標100%，正好對應第1層「千萬富翁複利目標」中「年獲利100%、第7年達千萬」的起始假設，三層邏輯互相銜接：階段化練習（本規則）→ 每年翻倍的複利假設 → 長期財富目標。
- **可程式化**: 部分（資金規模、持股檔數區間、獲利目標百分比皆為明確數字可程式化為檢核門檻；但「熟練選股/進場/交易策略」「情緒穩定訓練」「無我境界」等質化里程碑無法量化，僅能作為人工檢核清單，不可程式化為自動判斷）
- **所需資料**: 使用者目前所處練習階段（初階/進階/終極）、目前投入資金、目前持股檔數、當期（月/半年/年）已實現報酬率、槓桿使用情況
- **計算公式**:
```
階段設定 = {
    "初階": {資金規模: 500000, 持股檔數上限: (1,2), 用槓桿: False, 半年目標: 0.12, 月目標: 0.02, 年目標: 0.24},
    "進階": {資金規模: 800000, 持股檔數上限: (1,3), 用槓桿: False, 半年目標: 0.30, 月目標: 0.05, 年目標: 0.60},
    "終極": {資金規模: 1000000, 持股檔數上限: (2,5), 用槓桿: True,  半年目標: 0.50, 月目標: 0.0833, 年目標: 1.00},
}

FUNCTION 檢核階段合規(當前階段, 目前資金, 目前持股檔數, 已用槓桿):
    設定 = 階段設定[當前階段]
    警示 = []
    IF 目前持股檔數 > 設定.持股檔數上限[1]:
        警示.append("持股檔數超過本階段上限，違反集中操作原則")
    IF 已用槓桿 AND 設定.用槓桿 == False:
        警示.append("本階段不應使用槓桿(融資)")
    RETURN 警示

FUNCTION 千萬富翁複利路徑(本金=100000, 年報酬率=1.00, 年數=8):
    資金軌跡 = [本金]
    FOR i IN range(年數):
        資金軌跡.append(資金軌跡[-1] * (1 + 年報酬率))
    RETURN 資金軌跡   # 第7年（index=7）達成約1,280萬

# 達成目標4大關鍵（質化紀律檢核，供人工/半自動檢查使用）
FUNCTION 紀律檢核清單(交易紀錄):
    RETURN {
        "停利是否確實執行": 交易紀錄.觸及停利條件時是否延遲出場 == False,
        "是否優先短線快速獲利": ...,   # 質化，無法完全量化
        "強勢股篩選": 持股週期漲幅(1週或2週) >= 0.05,   # 低於5%應換股
        "全流程紀律": all([選股條件紀律, 進場位置紀律, 停損紀律, 續抱紀律, 停利紀律]),
    }
```
- **參數**: 3階段資金規模＝50萬/80萬/100萬元（書中明確）；持股檔數上限＝1-2/1-3/2-5檔（書中明確）；6個月獲利目標＝12%/30%/50%（書中明確）；牛步股換股門檻＝觀察週期(1週或半個月)漲幅<5%（書中明確）；千萬富翁複利假設＝年報酬100%、8年翻至約2,560萬、第7年達千萬（書中明確）
- **可回測**: 部分（資金規模、持股檔數、獲利目標百分比可直接回測比對；「4大關鍵」中的質化紀律項目（如「短線比長線重要」「紀律操作比選股重要」）需搭配其他量化規則的輸出才能間接驗證，本身不是可獨立回測的數值條件）
- **信心**: 92/100 高（資金規模、持股檔數、獲利目標百分比、複利路徑書中皆有明確數字表格與逐年試算；4大關鍵為質化原則描述，數字部分明確、質化部分無法進一步量化，但不影響整體信心評級）
