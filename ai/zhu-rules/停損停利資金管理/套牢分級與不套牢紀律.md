# 套牢分級與不套牢紀律

- **Rule ID**: R-RISK-05
- **名稱**: 套牢分級與不套牢紀律
- **分類**: 停損停利資金管理
- **原文與頁碼**: P12-C3-股市贏家2大交易守則.md，下冊 p.369-371（「守則1：手上絕對不要有套牢的股票」：何謂被套牢、如何不讓股票被套牢）
- **解讀**: 本規則整理書中「股市贏家2大交易守則」之守則1，是對「停損紀律為何必須嚴格執行」的心理面與量化面補強，分三層：
  1. **套牢的成因與心理**（p.369）：套牢源於「不願賠小錢賣出」「期待明天或後天能漲」「不賣就不算賠」「堅信終有一天會解套」等僥倖心態，長期套牢會造成心理壓力並使資金卡死、無法周轉。
  2. **套牢分級定義**（p.369）：以持有虧損股票的「未實現虧損百分比」分級：
     - 賠損達**10%**仍持有 → **輕度套牢**
     - 賠損達**20%**仍持有 → **中度套牢**
     - 賠損達**30%**仍持有 → **重度套牢**
     此分級可作為部位健康度的警示分級指標，與 `R-RISK-01`（停損5種設法總覽）中「絕對停損情境④跌幅超過10%絕對要停損」相互呼應——理論上嚴格執行10%停損紀律不應讓部位進入輕度套牢以上的狀態，本分級的實務意義在於：一旦部位已經超過10%仍未出場（代表停損紀律已經失守），提供分級警示以避免虧損持續擴大到中度、重度套牢。
  3. **套牢後果與不被套牢的做法**（p.370-371）：
     - **套牢後果**：短期（3、5年）不一定能解套；公司經營不善甚至下市可能血本無歸；資金卡死失去流動性。書中以鴻海（2317）、大立光（3008）歷史高點買進為例：鴻海2000/3高點375元買進，約23年5個月後（約2023/8）才解套；2007/6高點300元買進，套牢達15年2個月；大立光2017/8高點6,075元買進套牢達6年，2018/8高點5,530元套牢達5年，具體佐證買在高點套牢的時間成本可能長達數年甚至十幾二十年。
     - **積極正面思考**：停損之後才有機會重新賺錢；小賠比大賠容易快速反敗為勝；當下小賠賣出可避開後續快速下跌風險（判斷錯誤仍可伺機買回）；在較高價位小賠賣出，待止跌反轉後可用更低價買回；保留資金彈性比死守套牢部位重要。
     - **不被套牢的3個積極做法**（p.371）：①嚴格執行停損紀律，不因單筆虧損金額大小而例外；②**每天檢視手上股票，跌幅超過5%即列為「警示股」，準備賣出**；③追求「無債一身輕」，手上都是賺錢的股票。
  本規則與 `R-RISK-01` 的差異在於：R-RISK-01 定義「如何設定停損價與何時觸發停損」（事前/事中的價位邏輯），本規則則是「停損紀律失守後的分級警示機制＋每日盤中5%跌幅預警機制」（事中/事後的部位健康監控邏輯），兩者互補。
- **可程式化**: 是（套牢分級門檻與每日警示股門檻皆為明確百分比數字，可直接程式化為每日持股健康度掃描規則）
- **所需資料**: 每一持股的買進成本價、目前收盤價（計算未實現虧損百分比）、每日收盤價與前一交易日/持有期間最高價（計算單日或持有期間跌幅）
- **計算公式**:
```
FUNCTION 套牢分級(買進成本價, 目前收盤價):
    未實現虧損百分比 = (買進成本價 - 目前收盤價) / 買進成本價

    IF 未實現虧損百分比 >= 0.30:
        RETURN "重度套牢"
    ELIF 未實現虧損百分比 >= 0.20:
        RETURN "中度套牢"
    ELIF 未實現虧損百分比 >= 0.10:
        RETURN "輕度套牢"
    ELSE:
        RETURN "未套牢"

FUNCTION 每日警示股掃描(持股清單):
    警示名單 = []
    FOR EACH 股票 IN 持股清單:
        單日跌幅 = (股票.前一日收盤價 - 股票.今日收盤價) / 股票.前一日收盤價
        IF 單日跌幅 > 0.05:
            警示名單.append(股票, 原因="單日跌幅超過5%，列為警示股準備賣出")
    RETURN 警示名單
```
- **參數**: 套牢分級門檻＝10%/20%/30%（書中明確，輕度/中度/重度）；每日警示股跌幅門檻＝5%（書中明確）
- **可回測**: 是
- **信心**: 93/100 高（套牢分級三道門檻與每日5%警示門檻書中皆有明確數字；鴻海/大立光套牢案例為書中具體歷史佐證）
