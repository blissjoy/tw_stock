# 資金分配與部位上限管理原則

- **Rule ID**: R-RISK-02
- **名稱**: 資金分配與部位上限管理原則
- **分類**: 停損停利資金管理
- **原文與頁碼**: P11-C1-進場前的準備工作.md，下冊 p.315-319（「投入資金的分配原則」：資金規劃重要原則、隨大盤走勢調整資金比例）；P12-C3-股市贏家2大交易守則.md，下冊 p.368-369（「贏家的5個永遠認知」之集中火力、汰弱換強）
- **解讀**: 本規則整合書中所有與「資金應該投入多少、放在幾檔股票」相關的原則，分三層：
  1. **資金規劃5項基本原則**（p.315）：①投入股市的資金**不可超過個人可運用資金的二分之一（50%）**；②千萬不可借貸資金買股票（不可使用槓桿借貸/融資以外的借款）；③投入前要充分了解股市風險；④股市行情再好也不可全部投入買到滿倉，**最多投入資金的9成**（永遠保留至少10%現金）；⑤有獲利時要收回部分獲利作為準備金，不全數再投入。
  2. **隨大盤走勢動態調整資金比例**（p.316-319）：資金投入比例並非固定不變，而是隨大盤（或個股）所處的多空趨勢階段動態增減——愈接近趨勢確立初期或深度空頭，持股比例愈低；愈接近中長期多頭確認、均線多頭排列（4線多排），持股比例愈高。書中以2022年加權指數為實例（圖表11-1-17）：2022/1/21確認轉空頭時資金控制在5成之內；2022/3/7長黑K跌破支撐、均線4線空頭排列下彎、形成中期空頭時，資金減至3成之內；2022/4/11長期空頭、2022/6/13跌破切線時，資金持續維持在2成低檔水位；日後趨勢轉多頭時再逐步增加資金投入。整體區間大致為：深度空頭2成 → 中期空頭3成 → 空頭確認4成 → 多頭確認初期3成 → 中期多頭（4線多排）4～8成 → 高檔頭頭低/末升段轉弱回降至4～5成。此原則與「最多9成」的整體滿倉上限相呼應，屬趨勢確認後的加碼區間。
  3. **持股檔數集中與汰弱換強原則**（P12-C3, p.368-369）：贏家5個永遠認知中，與資金/部位配置直接相關的兩條——「永遠集中火力：資金不分散買太多檔股票，集中操作2～5檔，便於管理與專注觀察」、「永遠汰弱換強：手中只留強勢上漲的股票，弱勢股應優先換出」。此持股檔數區間（2～5檔）與 P12-C2「3階段資金規模」（見 `R-RISK-04`）中終極階段的持股檔數上限一致，可視為長期穩定的部位集中度原則。
  這三層原則共同構成一個「總曝險上限（50%可運用資金）→ 單一時點滿倉上限（已投入資金的90%）→ 依大盤多空階段動態調整的實際投入比例（2成～8成區間）→ 個股層級集中2～5檔並汰弱換強」的資金控管金字塔。
- **可程式化**: 部分（總曝險上限50%、滿倉上限90%、持股檔數2～5檔、大盤多空階段對應的資金比例區間可程式化為明確的數字門檻；但「大盤多空階段」本身的判定需依賴趨勢判定分類規則的輸出（如4線多排/空排、跌破切線等），「汰弱換強」的「弱勢」判斷也需額外定義量化的相對強弱指標，書中未給出精確公式）
- **所需資料**: 個人可運用總資金、目前已投入股市金額、目前持股檔數、大盤（加權指數）當前多空趨勢階段判定（多頭確認/中期多頭/4線多排/空頭確認/中期空頭/長期空頭/跌破切線等，見趨勢判定分類）、個股相對強弱排序
- **計算公式**:
```
可運用總資金上限 = 個人總資金 * 50%          # 投入股市資金不可超過可運用資金一半

# 單一時點部位上限（永遠保留至少10%現金）
已投入股市資金上限 = 可運用總資金上限 * 90%

# 依大盤多空階段動態調整實際投入比例（區間對照，數字為書中圖表近似值）
FUNCTION 大盤階段對應資金比例(market_phase):
    IF market_phase == "長期空頭(深度下跌末段)":       RETURN 20%
    ELIF market_phase == "中期空頭(4線空排/跌破支撐)":  RETURN 30%
    ELIF market_phase == "空頭確認初期":                RETURN 40%
    ELIF market_phase == "多頭確認初期":                RETURN 30%
    ELIF market_phase == "中期多頭(逐步轉強)":           RETURN 40%~60%
    ELIF market_phase == "多頭確認且4線多排":            RETURN 60%~80%
    ELIF market_phase == "高檔頭頭低/末升段轉弱":        RETURN 40%~50%

目標投入金額 = 可運用總資金上限 * 大盤階段對應資金比例(market_phase)
目標投入金額 = min(目標投入金額, 已投入股市資金上限)   # 不得超過90%滿倉上限

# 持股檔數與汰弱換強
IF 持股檔數 > 5:
    警示 = "超過集中操作上限(2~5檔)，應精簡持股"
FOR EACH 持股 IN 持股清單:
    IF 持股相對強弱排序 == 最弱 AND 持股檔數 >= 上限:
        動作 = "優先換出，汰弱換強"

# 獲利準備金原則
IF 本次交易已實現獲利 > 0:
    提領比例 = 自訂(書中未給精確數字，僅原則性建議"收回部分獲利")
    準備金 += 已實現獲利 * 提領比例
    可再投入資金 = 已實現獲利 * (1 - 提領比例)
```
- **參數**: 股市曝險資金上限＝可運用總資金×50%（書中明確）；單一時點滿倉上限＝已投入資金×90%（書中明確）；大盤多空階段資金比例區間＝2成～8成（書中圖表近似值，非精確門檻）；持股集中檔數＝2～5檔（書中明確）；獲利提領比例＝書中僅原則性提及「收回部分獲利」，無精確數字，需自訂
- **可回測**: 部分（50%/90%資金上限與2～5檔持股檔數可直接回測；「隨大盤階段動態調整比例」需先有可程式化的大盤多空階段判定模組才能完整回測，書中圖表數字屬近似參考值而非精確公式）
- **信心**: 60/100 中（50%可運用資金上限、90%滿倉上限、2～5檔集中持股、不可借貸槓桿等原則書中皆有明確數字＝高；但「隨大盤走勢調整資金比例」的區間數字是本筆記依書中圖表示意整理的近似值，非書中逐字給出的精確公式，且「汰弱換強」與「獲利提領比例」僅為原則性描述缺乏精確門檻，綜合評為中）
