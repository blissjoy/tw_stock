# 下降切線畫法與線不蓋線判準

- **Rule ID**: R-LINE-02
- **名稱**: 下降切線畫法與線不蓋線判準
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C1-切線的基本概念.md（上冊 p.340；原書5-1節「一、切線的正確畫法」之「2. 下降切線的畫法」，圖表5-1-3、5-1-4）
- **解讀**: 下降切線是連接兩個「頭頭低」轉折高點所畫出、向右下方延伸的直線，作用與上升切線對稱（標示方向、預警轉折、提供壓力）。畫法步驟：①取第1個轉折高點A（K棒最高價，**含上影線**）；②往右找下一個轉折高點B，且B必須比A低（「頭頭低」：後點價格 < 前點價格）；③用直線連接A、B並向右無限延伸，即得到1條下降切線；④畫線正確性檢查：A、B之間所有K棒（含影線）都必須落在切線下方或切線上，只要有任何一根K棒被切線切過/壓到，此切線視為畫錯，須重新選點（「線不蓋線」原則，與上升切線同一標準，方向相反）。取點所需的轉折高點序列，沿用 [[轉折波定義與取點演算法]] 的輸出；「頭頭低」的比較邏輯與 [[頭頭低底底低空頭趨勢判定]] 所用的「頭頭低」概念一致，但此處只需最新2個高點依序走低即可畫線，不要求同時滿足「底底低」條件。走勢推進、出現更新的頭頭低兩點時，切線需用最新一組重新畫（見 [[隨機下降次切線動態更新畫法]]）。
- **可程式化**: 是
- **所需資料**: 轉折高點序列（含上影線最高價、日期，來自 [[轉折波定義與取點演算法]]）、K棒OHLC（含最高最低價，用於線不蓋線檢查）
- **計算公式**:
```
function draw_down_tangent_line(tops, bars):
    # tops: 依時間順序排列的轉折高點序列 [(date, high_incl_wick), ...]，來自轉折波演算法
    best_line = None
    for i from len(tops)-2 down to 0:
        A = tops[i]
        B = tops[i+1]
        if B.high >= A.high:
            continue   # 不滿足「頭頭低」，跳過此配對

        line = fit_line_through(A, B)   # 通過A、B兩高點（含上影線）的直線方程 y = f(date)

        # 線不蓋線檢查：A、B之間所有K棒（含影線）不可被切線切過/壓到
        valid = True
        for bar in bars_between(A.date, B.date):
            if bar.high > line(bar.date):
                valid = False
                break

        if valid:
            best_line = line   # 取最新（最靠右）一組通過檢查的頭頭低兩點畫線
            break

    return best_line   # 若為 None，代表近期找不到合法的頭頭低配對，暫無下降切線可畫
```
- **參數**: 轉折點取點所用均線天期 N（沿用 [[轉折波定義與取點演算法]] 之 N，預設5日，10/20日可選）
