# 下降急切線畫法與回補應用

- **Rule ID**: R-LINE-08
- **名稱**: 下降急切線畫法與回補應用
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C1-切線的基本概念.md（上冊 p.351-352；原書5-1節「二、切線的種類」之「6. 下降急切線」，圖表5-1-25、5-1-26）
- **解讀**: 與 [[上升急切線畫法與停利應用]] 完全對稱。下降急切線是為短期內連續數天「急跌」而畫的超短期切線，畫法與一般下降切線相同（連接高點、含上影線、線不蓋線），但取樣區間僅限急跌那幾天。應用時機：①急跌、飆跌、末跌段連續下跌、倒V形反轉位置；②短線做空回補的重要參考線，一般作為「空單回補」的參考點；③也可用於上下幅度大的盤整區進出。操作邏輯：股價急跌期間沿下降急切線下行，一旦突破急切線，視為空單短線回補停利訊號。此規則也是 [[下降軌道線畫法（切線平移取中間低點）]] 中「末跌段警訊」情境下的回補防守線工具。
- **可程式化**: 部分
- **所需資料**: K棒OHLC（含上影線最高價）、急跌區間判定（如連續N天累積跌幅超過門檻，或連續無收紅反彈）
- **計算公式**:
```
function draw_down_rapid_tangent_line(bars, rapid_window):
    # rapid_window: 判定為「急跌段」的K棒區間（例如連續N天累積跌幅 > 門檻，或連續多天無明顯反彈）
    highs_in_window = [(bar.date, bar.high) for bar in rapid_window]
    if len(highs_in_window) < 2:
        return None   # 急跌區間內高點不足2個，無法畫線

    A, B = highs_in_window[0], highs_in_window[-1]   # 取急跌段內第1個與最新高點（亦可用區間內任意2個依序走低的高點）
    line = fit_line_through(A, B)

    valid = all(bar.high <= line(bar.date) for bar in rapid_window)  # 線不蓋線檢查，區間更短、角度通常更陡
    if not valid:
        return None

    return line

function check_rapid_line_cover(line, latest_bar):
    if latest_bar.close > line(latest_bar.date):
        return "突破下降急切線：空單短線回補停利訊號"
    return "持有中"
```
- **參數**: 急跌區間判定門檻（如N日累計跌幅百分比，書中未給固定數值，需另行設定，建議預設N=3~5日）
- **參考來源查證（2026-07-22，工程補充）**：與R-LINE-07同批查證，「急漲/急跌N日判定」查無公認統一通用天數或幅度門檻；維持N=3~5日為工程估計值，非外部佐證數字（詳見R-LINE-07的查證說明）。
- **可回測**: 部分（急跌區間的量化門檻書中僅以文字描述、無具體數字，需自行設定參數後才可完整回測）
- **信心**: 58/100 中（畫線邏輯與一般切線一致、信心高；但「急跌段」的量化判定門檻需人工設定參數）
