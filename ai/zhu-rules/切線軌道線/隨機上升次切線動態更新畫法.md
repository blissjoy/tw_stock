# 隨機上升次切線動態更新畫法

- **Rule ID**: R-LINE-05
- **名稱**: 隨機上升次切線動態更新畫法
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C1-切線的基本概念.md（上冊 p.345；原書5-1節「二、切線的種類」之「2. 隨機上升切線（上升次切線）」，圖表5-1-12）
- **解讀**: 在 [[原始上升切線與盤整反轉判定]] 之後，多頭趨勢持續上漲，往後每出現一組新的「底底高」低點，就依 [[上升切線畫法與線不蓋線判準]] 的相同步驟重新取最新2個低點連線，稱為「隨機上升切線」或「上升次切線」，是動態更新、非固定一條。走勢每往前推進，切線就用最新一組低點重畫，通常角度會越來越陡（斜率遞增），代表走勢逐步加速；此角度變化本身也是判斷多頭動能強弱的依據（見 [[上升切線支撐力道遞減與角度強弱判斷]]）。畫法與上升切線完全相同，差異僅在於「每次都取最新一組底底高兩點」而非固定使用原始那一組。
- **可程式化**: 是
- **所需資料**: 轉折低點序列（來自 [[轉折波定義與取點演算法]]，持續更新）、既有的上升切線清單（含原始切線與歷次次切線，用於角度比較）
- **計算公式**:
```
function update_up_sub_tangent_lines(bottoms, bars, existing_lines):
    # bottoms 依時間順序持續增長；existing_lines 為目前已畫出的切線歷史（依畫出順序排列）
    latest_pair_index = len(bottoms) - 2
    A, B = bottoms[latest_pair_index], bottoms[latest_pair_index + 1]

    if B.low <= A.low:
        return existing_lines   # 尚未出現新的「底底高」，不更新

    if not passes_line_not_covering_check(A, B, bars):
        return existing_lines   # 新配對不合法（線蓋線），維持沿用前一條切線

    new_line = fit_line_through(A, B)
    existing_lines.append(new_line)   # 新增1條隨機（次）切線

    if len(existing_lines) >= 2:
        slope_change = slope(new_line) - slope(existing_lines[-2])
        trend_strength = "走勢轉強（角度變陡）" if slope_change > 0 else "走勢轉弱（角度趨緩）"

    return existing_lines
```
- **參數**: 轉折點取點所用均線天期 N（沿用 [[轉折波定義與取點演算法]]，預設5日）
- **可回測**: 是
- **信心**: 91/100 高（原書明確定義「往後任何2個低點連接的切線都稱為隨機/次切線」，並以圖表示範多條切線隨走勢推進、角度遞增的過程）
