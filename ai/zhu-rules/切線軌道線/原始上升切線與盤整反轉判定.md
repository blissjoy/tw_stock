# 原始上升切線與盤整反轉判定

- **Rule ID**: R-LINE-03
- **名稱**: 原始上升切線與盤整反轉判定
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C1-切線的基本概念.md（上冊 p.341-344；原書5-1節「二、切線的種類」之「1. 原始上升切線」，圖表5-1-5~5-1-11）
- **解讀**: 「原始上升切線」＝在空頭下跌過程中，走勢沒有跌破前面低點就轉向上、第一次出現「底底高」兩個轉折低點時，依 [[上升切線畫法與線不蓋線判準]] 的步驟連接這第1組底底高低點所畫出的切線。此線會成為後續盤整區的「下頸線」，其後走勢有以下分支：①盤整向上突破原始上升切線＝多頭確認（反轉成多頭趨勢）；②盤整向下跌破原始上升切線＝空頭盤整被跌破、空頭續跌；③若已判定反轉成多頭，多頭第1次回檔卻跌破原始上升切線＝反轉失敗，空頭續跌，此情形書中稱為「空頭ABC反彈修正結束」。判斷「是否為原始」的關鍵是：自空頭趨勢確認以來，第一組通過線不蓋線檢查的底底高低點配對。
- **可程式化**: 是
- **所需資料**: 轉折低點序列（來自 [[轉折波定義與取點演算法]]）、當前趨勢狀態（多/空/盤整，來自 [[頭頭低底底低空頭趨勢判定]] 或 [[盤整趨勢判定與型態]]）、後續K棒收盤價（判斷突破/跌破）
- **計算公式**:
```
function identify_original_up_tangent(bottoms, bars, trend_state):
    # 僅在近期趨勢狀態為空頭（見 頭頭低底底低空頭趨勢判定）時才尋找「原始」切線
    if trend_state != "bearish":
        return None

    # 找出空頭確認之後，第一組滿足「底底高」且通過線不蓋線檢查的低點配對
    for i from 0 to len(bottoms)-2:
        A, B = bottoms[i], bottoms[i+1]
        if B.low > A.low and passes_line_not_covering_check(A, B, bars):
            original_line = fit_line_through(A, B)
            break
    if original_line is None:
        return None

    # 後續分支判定：以 original_line 作為盤整區下頸線
    status = "watching"
    for bar in bars_after(B.date):
        if bar.close > original_line(bar.date):
            status = "多頭確認：盤整向上突破原始上升切線"
            break
        elif bar.close < original_line(bar.date):
            status = "空頭續跌：盤整向下跌破原始上升切線（下頸線失守）"
            break

    # 若先前已判定多頭確認，之後第1次回檔卻又跌破 original_line
    if status.startswith("多頭確認") and later_pullback_breaks_below(original_line, bars):
        status = "反轉失敗：空頭ABC反彈修正結束，空頭續跌"

    return {"line": original_line, "status": status}
```
- **參數**: 轉折點取點所用均線天期 N（沿用 [[轉折波定義與取點演算法]]，預設5日）
- **可回測**: 是
- **信心**: 92/100 高（原書對「第1次底底高」定義明確，並以4種走勢演變情境搭配圖表逐一示範）
