# 原始下降切線與盤整反轉判定

- **Rule ID**: R-LINE-04
- **名稱**: 原始下降切線與盤整反轉判定
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C1-切線的基本概念.md（上冊 p.347-350；原書5-1節「二、切線的種類」之「4. 原始下降切線」，圖表5-1-16~5-1-22）
- **解讀**: 「原始下降切線」＝在多頭上漲過程中，走勢沒有突破前面高點就轉向下、第一次出現「頭頭低」兩個轉折高點時，依 [[下降切線畫法與線不蓋線判準]] 的步驟連接這第1組頭頭低高點所畫出的切線。此線會成為後續盤整區的「上頸線」，其後走勢有以下分支（與原始上升切線完全對稱，方向相反）：①盤整向下跌破原始下降切線＝空頭確認（反轉成空頭趨勢）；②盤整向上突破原始下降切線＝多頭盤整被突破、多頭續漲；③若已判定反轉成空頭，空頭第1次反彈卻突破原始下降切線＝反轉失敗，多頭續漲，書中稱為「多頭ABC回檔修正結束」。判斷「是否為原始」的關鍵是：自多頭趨勢確認以來，第一組通過線不蓋線檢查的頭頭低高點配對。
- **可程式化**: 是
- **所需資料**: 轉折高點序列（來自 [[轉折波定義與取點演算法]]）、當前趨勢狀態（多/空/盤整，來自 [[頭頭高底底高多頭趨勢判定]] 或 [[盤整趨勢判定與型態]]）、後續K棒收盤價（判斷突破/跌破）
- **計算公式**:
```
function identify_original_down_tangent(tops, bars, trend_state):
    # 僅在近期趨勢狀態為多頭（見 頭頭高底底高多頭趨勢判定）時才尋找「原始」切線
    if trend_state != "bullish":
        return None

    # 找出多頭確認之後，第一組滿足「頭頭低」且通過線不蓋線檢查的高點配對
    for i from 0 to len(tops)-2:
        A, B = tops[i], tops[i+1]
        if B.high < A.high and passes_line_not_covering_check(A, B, bars):
            original_line = fit_line_through(A, B)
            break
    if original_line is None:
        return None

    # 後續分支判定：以 original_line 作為盤整區上頸線
    status = "watching"
    for bar in bars_after(B.date):
        if bar.close < original_line(bar.date):
            status = "空頭確認：盤整向下跌破原始下降切線（上頸線失守）"
            break
        elif bar.close > original_line(bar.date):
            status = "多頭續漲：盤整向上突破原始下降切線"
            break

    # 若先前已判定空頭確認，之後第1次反彈卻又突破 original_line
    if status.startswith("空頭確認") and later_rebound_breaks_above(original_line, bars):
        status = "反轉失敗：多頭ABC回檔修正結束，多頭續漲"

    return {"line": original_line, "status": status}
```
- **參數**: 轉折點取點所用均線天期 N（沿用 [[轉折波定義與取點演算法]]，預設5日）
- **可回測**: 是
- **信心**: 92/100 高（原書邏輯與原始上升切線完全對稱，並附4種走勢演變情境及多組圖表範例）
