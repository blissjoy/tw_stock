# 上升急切線畫法與停利應用

- **Rule ID**: R-LINE-07
- **名稱**: 上升急切線畫法與停利應用
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C1-切線的基本概念.md（上冊 p.345-346；原書5-1節「二、切線的種類」之「3. 上升急切線」，圖表5-1-14、5-1-15）
- **解讀**: 上升急切線是為短期內連續數天「急漲」而畫的超短期切線，畫法與一般上升切線相同（連接低點、含下影線、線不蓋線），但取樣區間僅限急漲那幾天，遠短於一般切線或次切線的取點跨度。應用時機：①急漲、飆股、末升段噴出、V形反轉位置；②短線進出的重要參考線，一般作為「多方獲利賣出」的參考點；③也可用於上下幅度大的盤整區進出。操作邏輯：股價急漲期間沿急切線上行，一旦跌破急切線，視為短線停利/出場訊號。此規則也是 [[上升軌道線畫法（切線平移取中間高點）]] 中「末升段警訊」情境下的停利防守線工具。
- **可程式化**: 部分
- **所需資料**: K棒OHLC（含下影線最低價）、急漲區間判定（如連續N天累積漲幅超過門檻，或連續無收黑回檔）
- **計算公式**:
```
function draw_up_rapid_tangent_line(bars, rapid_window):
    # rapid_window: 判定為「急漲段」的K棒區間（例如連續N天累積漲幅 > 門檻，或連續多天無明顯回檔）
    lows_in_window = [(bar.date, bar.low) for bar in rapid_window]
    if len(lows_in_window) < 2:
        return None   # 急漲區間內低點不足2個，無法畫線

    A, B = lows_in_window[0], lows_in_window[-1]   # 取急漲段內第1個與最新低點（亦可用區間內任意2個依序墊高的低點）
    line = fit_line_through(A, B)

    valid = all(bar.low >= line(bar.date) for bar in rapid_window)  # 線不蓋線檢查，區間更短、角度通常更陡
    if not valid:
        return None

    return line

function check_rapid_line_exit(line, latest_bar):
    if latest_bar.close < line(latest_bar.date):
        return "跌破上升急切線：短線停利/出場訊號"
    return "持有中"
```
- **參數**: 急漲區間判定門檻（如N日累計漲幅百分比，書中未給固定數值，需另行設定，建議預設N=3~5日）
- **參考來源查證（2026-07-22，工程補充）**：查證外部資料，「急漲/急跌N日判定」在台股實務討論中**查無公認統一的通用天數或幅度門檻**；較接近的參考數據是台股單日跌幅逾5%後平均約3.8個交易日出現短線止跌（來源：[StockFeel股感](https://www.stockfeel.com.tw/%E5%8F%B0%E8%82%A1-%E5%A4%A7%E8%B7%8C-%E5%9B%9E%E6%B8%AC/)），但該數據描述的是「單次急跌後的止跌天數」而非「急漲/急跌段本身的判定門檻」，情境不完全對應，僅作為參考背景。維持書中原有N=3~5日建議值為工程估計值，非外部佐證數字。
- **可回測**: 部分（急漲區間的量化門檻書中未明確給出固定數字，需自行設定參數後才可完整回測）
- **信心**: 58/100 中（畫線邏輯與一般切線一致、信心高；但「急漲段」的量化判定門檻書中僅以文字描述、無具體數字，需人工設定參數）
