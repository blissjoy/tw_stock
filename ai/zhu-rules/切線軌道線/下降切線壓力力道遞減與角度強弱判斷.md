# 下降切線壓力力道遞減與角度強弱判斷

- **Rule ID**: R-LINE-10
- **名稱**: 下降切線壓力力道遞減與角度強弱判斷
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C1-切線的基本概念.md（上冊 p.356-357；原書5-1節「四、下降切線的重要功能與作用」之「1. 下降切線有壓力作用」「2. 下降切線角度顯示下跌的強弱」，圖表5-1-34、5-1-37、5-1-38）
- **解讀**: ①壓力力道遞減：與 [[上升切線支撐力道遞減與角度強弱判斷]] 對稱——股價反彈觸及同一條下降切線的次數越多，壓力有效機率越低，第1次反彈觸線壓力最強、第2次約50%成功率、第3次以後容易被突破。②角度顯示強弱：**此處方向性與上升切線非鏡射對稱，需特別留意**——書中原文與配圖5-1-34明確寫「下降切線斜率（角度）的變化，斜率越來越大，走勢越來越弱；反之，斜率越來越小，走勢越來越強」，即下降切線角度變陡（下跌速度加快、類似加速趕底）反而判定為空頭動能「轉弱」（意指接近竭盡、接近反轉/急跌末端）；角度趨緩則判定為空頭「轉強」（意指空頭延續力道更強）。此用詞已對照原書兩處（標題與內文）確認一致，非單次誤植，但建議在實際使用前人工複核紙本書再次確認，避免直接套用上升切線的鏡射邏輯（角度變陡＝走勢變強）。
- **可程式化**: 是
- **所需資料**: 下降切線歷史序列（依時間先後排列，來自R-LINE-02/06）、每次股價觸線（反彈至切線附近）的次數記錄、K棒收盤價
- **計算公式**:
```
function resistance_strength_by_touch_count(touch_count):
    # touch_count: 股價反彈觸及「同一條」下降切線的累計次數
    if touch_count == 1:
        return "壓力最強"
    elif touch_count == 2:
        return "壓力約50%成功率"
    else:  # touch_count >= 3
        return "容易被突破"

function angle_strength_down(line_history):
    # line_history: 依時間先後排列的下降切線序列（原始切線 + 歷次隨機次切線）
    # 注意：方向性與上升切線非鏡射對稱，見「解讀」欄位說明，按書中原文字面實作
    results = []
    for i from 1 to len(line_history)-1:
        abs_slope_prev = abs(slope(line_history[i-1]))
        abs_slope_curr = abs(slope(line_history[i]))
        if abs_slope_curr > abs_slope_prev:
            results.append("走勢轉弱（下降角度變陡，按書中原文字面）")
        elif abs_slope_curr < abs_slope_prev:
            results.append("走勢轉強（下降角度趨緩，按書中原文字面）")
        else:
            results.append("走勢持平")
    return results
```
- **參數**: 觸線容許誤差範圍（判斷「是否觸及切線」的價格緩衝百分比，書中未給固定數字，建議預設±0.5%~1%）
- **參考來源查證（2026-07-22，工程補充，本次因R-SR-06/07重用本規則邏輯而補查）**：與R-LINE-09同批查證，觸碰次數遞減機率查無公認統一數字；主流國際技術分析文獻對觸碰次數與支撐/壓力強度的看法與本書相反（觸碰越多次通常視為越強而非越弱，來源：[微股力ScanTrader](https://scantrader.com/article/0172eb887bcf00001503000000000000)），僅供對照，不影響本規則忠實依原書邏輯實作的既有判斷。
