# 跌破上升切線分級與支撐壓力角色互換

- **Rule ID**: R-LINE-11
- **名稱**: 跌破上升切線分級與支撐壓力角色互換
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C1-切線的基本概念.md（上冊 p.354-355；原書5-1節「三、上升切線的重要功能與作用」之「3. 上升切線被跌破，多頭轉弱」「4. 上升切線被跌破，支撐變壓力」，圖表5-1-31、5-1-32）
- **解讀**: 兩條規則皆以 [[隨機上升次切線動態更新畫法]] 所累積的切線歷史（原始切線＋歷次次切線，依畫出時間排序）為輸入。①跌破分級：跌破「最新（最後）一條」上升切線＝多頭上升減弱訊號，可能要回檔修正；若股價再往下跌破「更早（前面）一條」上升切線＝警示等級升高，可能整個上升趨勢要結束或反轉，跌破的切線條數越多，警示等級越高。②角色互換：任何一條上升切線一旦被跌破，其支撐角色即轉換為壓力角色——該切線（含其向右延長線）在後續反彈時視為壓力位。此規則與角度強弱判斷（[[上升切線支撐力道遞減與角度強弱判斷]]）常一併使用：角度轉弱＋跌破切線，是多頭轉弱的複合確認訊號。
- **可程式化**: 是
- **所需資料**: 上升切線歷史序列（依畫出時間排序，最新的在最後，來自R-LINE-01/05）、K棒收盤價
- **計算公式**:
```
function classify_break_and_role_swap(line_history, latest_bar):
    # line_history: 依時間先後排列的上升切線序列，index越大代表越新
    broken_lines = []
    for i from len(line_history)-1 down to 0:
        line = line_history[i]
        if latest_bar.close < line(latest_bar.date):
            broken_lines.append(line)
        else:
            break   # 由新到舊檢查，遇到未跌破的線即停止（該線之前的更早切線本輪不需再檢查）

    if len(broken_lines) == 0:
        return {"status": "未跌破，多頭結構維持"}
    elif len(broken_lines) == 1:
        status = "跌破最新一條上升切線：回檔修正警訊（多頭轉弱）"
    else:
        status = f"連續跌破{len(broken_lines)}條上升切線（含更早的切線）：趨勢反轉警訊，警示等級隨跌破條數升高"

    # 角色互換：所有被跌破的切線，其延長線自此在後續反彈時視為壓力線
    for line in broken_lines:
        line.role = "resistance"   # 原本role = "support"

    return {"status": status, "broken_lines": broken_lines}
```
- **參數**: 無額外可調參數（純粹比較收盤價與各條切線延長線位置）
- **可回測**: 是
- **信心**: 92/100 高（書中原文明確描述「跌破最後的切線→回檔修正；再跌破前面的切線→趨勢結束或反轉」及「支撐變壓力」的角色互換邏輯，並有圖表5-1-31、5-1-32逐步範例）
