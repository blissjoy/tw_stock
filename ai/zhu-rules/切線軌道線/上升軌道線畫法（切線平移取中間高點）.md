# 上升軌道線畫法（切線平移取中間高點）

- **Rule ID**: R-LINE-14
- **名稱**: 上升軌道線畫法（切線平移取中間高點）
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C2-軌道線的畫法與實戰應用.md（上冊 p.369-371；原書5-2節「一、上升軌道線的畫法與實戰應用」，圖表5-2-1~5-2-5）
- **解讀**: 上升軌道線是在上升切線的基礎上「多畫一條平行線」而成，前提是股價處於多頭上漲階段。畫法步驟：①先依 [[上升切線畫法與線不蓋線判準]] 取出畫上升切線所用的兩個轉折低點A、B，連接畫出上升切線（此線角色為支撐）；②在A、B兩點「之間」（即A日期 < t < B日期的區間內），找出這段走勢的相對最高點M（K棒最高價，**含上影線**）；③過M點畫一條與步驟①的上升切線**斜率相同（平行）**的直線，即為「上升軌道線」；④上升軌道線會落在上升切線的上方，兩線同斜率、平行，形成一個向上傾斜的通道，K線在通道內震盪上行，上升軌道線角色為壓力線（與切線本身的支撐角色相對）。股價突破上升軌道線視為走勢轉強、可能接續一波連續上漲攻勢；突破當天K棒特徵、末升段警訊、突破前的逆勢操作等訊號應用細節，見已建立的 [[突破上升軌道線壓力的大量長紅K]]（K棒型態分類），本規則聚焦於軌道線本身的幾何畫法，不重複定義訊號判斷部分。
- **可程式化**: 是
- **所需資料**: 上升切線（來自 [[上升切線畫法與線不蓋線判準]]，含所連接的兩低點A、B）、A、B之間所有K棒的最高價（含上影線）
- **計算公式**:
```
function draw_up_channel_line(tangent_line, A, B, bars):
    # tangent_line: 依R-LINE-01畫出的上升切線，A、B為其所連接的兩個轉折低點
    # 在A、B之間找出相對最高點（含上影線）
    bars_between_AB = bars_between(A.date, B.date)
    M = max(bars_between_AB, key=lambda bar: bar.high)   # M為中間最高點（含上影線）

    # 過M點畫一條與tangent_line斜率相同（平行）的直線
    channel_line = parallel_line_through_point(
        slope=slope(tangent_line),
        point=(M.date, M.high)
    )

    # 校驗：軌道線應位於切線上方（否則取點有誤，需重新檢查A、B或M）
    assert channel_line(A.date) >= tangent_line(A.date)

    return channel_line   # 通道上緣壓力線，與tangent_line構成平行通道

function check_channel_breakout(channel_line, latest_bar):
    if latest_bar.close > channel_line(latest_bar.date):
        return "突破上升軌道線壓力，訊號判斷詳見 [[突破上升軌道線壓力的大量長紅K]]"
    return "尚未突破，軌道線仍為壓力"
```
- **參數**: 無額外可調參數（幾何構圖僅依賴上游切線A、B兩點與其間K棒最高價；「末升段」波段編號等進階應用參數見K棒型態分類對應規則）
- **可回測**: 是
- **信心**: 92/100 高（原書明確給出「取上升切線兩低點間的最高點、過該點畫平行線」的具體構圖邏輯，並附4組實戰範例圖表核對）
