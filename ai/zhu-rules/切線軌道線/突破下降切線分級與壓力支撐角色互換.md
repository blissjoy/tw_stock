# 突破下降切線分級與壓力支撐角色互換

- **Rule ID**: R-LINE-12
- **名稱**: 突破下降切線分級與壓力支撐角色互換
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C1-切線的基本概念.md（上冊 p.357-358；原書5-1節「四、下降切線的重要功能與作用」之「3. 下降切線被突破，空頭轉強」「4. 下降切線被突破，壓力變支撐」，圖表5-1-37、5-1-38）
- **解讀**: 與 [[跌破上升切線分級與支撐壓力角色互換]] 對稱。①突破分級：突破「最新（最後）一條」下降切線，內文操作邏輯為「可能只是反彈修正」；若股價再往上突破「更早（前面）一條」下降切線＝警示等級升高，可能空頭趨勢要結束或反轉，突破的切線條數越多，警示等級越高。②角色互換：任何一條下降切線一旦被突破，其壓力角色即轉換為支撐角色——該切線（含其向右延長線）在後續回檔時視為支撐位。**用詞提醒**：書中本節小標題與內文字面皆寫「下降切線被突破，空頭轉強」，與上升切線對應段落「上升切線被跌破，多頭轉弱」用詞不對稱（一般直覺「突破壓力」應偏多方轉強、空頭轉弱）；已核對原書掃描頁兩處用詞一致均為「強」，無法排除原書排版/校對疏漏，**建議對照紙本書再次確認**。此標籤疑義不影響本規則的操作邏輯本身（突破分級＋壓力變支撐），僅供人工複核時留意。
- **可程式化**: 是
- **所需資料**: 下降切線歷史序列（依畫出時間排序，最新的在最後，來自R-LINE-02/06）、K棒收盤價
- **計算公式**:
```
function classify_breakout_and_role_swap(line_history, latest_bar):
    # line_history: 依時間先後排列的下降切線序列，index越大代表越新
    broken_lines = []
    for i from len(line_history)-1 down to 0:
        line = line_history[i]
        if latest_bar.close > line(latest_bar.date):
            broken_lines.append(line)
        else:
            break   # 由新到舊檢查，遇到未突破的線即停止

    if len(broken_lines) == 0:
        return {"status": "未突破，空頭結構維持"}
    elif len(broken_lines) == 1:
        status = "突破最新一條下降切線：反彈修正訊號（書中標題用詞：空頭轉強，待覆核，見解讀欄）"
    else:
        status = f"連續突破{len(broken_lines)}條下降切線（含更早的切線）：趨勢反轉訊號，警示等級隨突破條數升高"

    # 角色互換：所有被突破的切線，其延長線自此在後續回檔時視為支撐線
    for line in broken_lines:
        line.role = "support"   # 原本role = "resistance"

    return {"status": status, "broken_lines": broken_lines}
```
- **參數**: 無額外可調參數（純粹比較收盤價與各條切線延長線位置）
- **可回測**: 是
- **信心**: 65/100 中（突破分級＋角色互換的操作邏輯明確、與上升切線完全對稱，信心高；但「空頭轉強」用詞標籤本身有疑義待對照紙本覆核，故整體評為中）
