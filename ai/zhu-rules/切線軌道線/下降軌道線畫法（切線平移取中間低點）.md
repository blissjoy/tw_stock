# 下降軌道線畫法（切線平移取中間低點）

- **Rule ID**: R-LINE-15
- **名稱**: 下降軌道線畫法（切線平移取中間低點）
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C2-軌道線的畫法與實戰應用.md（上冊 p.372-374；原書5-2節「二、下降軌道線的畫法與實戰應用」，圖表5-2-6~5-2-10）
- **解讀**: 與 [[上升軌道線畫法（切線平移取中間高點）]] 完全對稱，前提是股價處於空頭下跌階段。畫法步驟：①先依 [[下降切線畫法與線不蓋線判準]] 取出畫下降切線所用的兩個轉折高點A、B，連接畫出下降切線（此線角色為壓力）；②在A、B兩點「之間」，找出這段走勢的相對最低點M（K棒最低價，**含下影線**）；③過M點畫一條與步驟①的下降切線**斜率相同（平行）**的直線，即為「下降軌道線」；④下降軌道線會落在下降切線的下方，兩線同斜率、平行，形成一個向下傾斜的通道，K線在通道內震盪下行，下降軌道線角色為支撐線（與切線本身的壓力角色相對）。股價跌破下降軌道線視為走勢轉弱、可能接續一波連續下跌走勢；跌破當天K棒特徵、末跌段警訊、跌破前的逆勢操作等訊號應用細節，見已建立的 [[跌破下降軌道線支撐的大量長黑K]]（K棒型態分類），本規則聚焦於軌道線本身的幾何畫法，不重複定義訊號判斷部分。
- **可程式化**: 是
- **所需資料**: 下降切線（來自 [[下降切線畫法與線不蓋線判準]]，含所連接的兩高點A、B）、A、B之間所有K棒的最低價（含下影線）
- **計算公式**:
```
function draw_down_channel_line(tangent_line, A, B, bars):
    # tangent_line: 依R-LINE-02畫出的下降切線，A、B為其所連接的兩個轉折高點
    # 在A、B之間找出相對最低點（含下影線）
    bars_between_AB = bars_between(A.date, B.date)
    M = min(bars_between_AB, key=lambda bar: bar.low)   # M為中間最低點（含下影線）

    # 過M點畫一條與tangent_line斜率相同（平行）的直線
    channel_line = parallel_line_through_point(
        slope=slope(tangent_line),
        point=(M.date, M.low)
    )

    # 校驗：軌道線應位於切線下方（否則取點有誤，需重新檢查A、B或M）
    assert channel_line(A.date) <= tangent_line(A.date)

    return channel_line   # 通道下緣支撐線，與tangent_line構成平行通道

function check_channel_breakdown(channel_line, latest_bar):
    if latest_bar.close < channel_line(latest_bar.date):
        return "跌破下降軌道線支撐，訊號判斷詳見 [[跌破下降軌道線支撐的大量長黑K]]"
    return "尚未跌破，軌道線仍為支撐"
```
- **參數**: 無額外可調參數（幾何構圖僅依賴上游切線A、B兩點與其間K棒最低價；「末跌段」波段編號等進階應用參數見K棒型態分類對應規則）
- **可回測**: 是
- **信心**: 92/100 高（原書明確給出「取下降切線兩高點間的最低點、過該點畫平行線」的具體構圖邏輯，並附4組實戰範例圖表核對，其中一例（代號2612）並提醒跌破後仍可能反轉，需搭配後續K線確認）
