# 隨機下降次切線動態更新畫法

- **Rule ID**: R-LINE-06
- **名稱**: 隨機下降次切線動態更新畫法
- **分類**: 切線軌道線
- **原文與頁碼**: P05-C1-切線的基本概念.md（上冊 p.351；原書5-1節「二、切線的種類」之「5. 隨機下降切線（下降次切線）」）
- **解讀**: 與 [[隨機上升次切線動態更新畫法]] 完全對稱。在 [[原始下降切線與盤整反轉判定]] 之後，空頭趨勢持續下跌，往後每出現一組新的「頭頭低」高點，就依 [[下降切線畫法與線不蓋線判準]] 的相同步驟重新取最新2個高點連線，稱為「隨機下降切線」或「下降次切線」，動態更新、非固定一條。角度變化可作為空頭動能強弱的參考（見 [[下降切線壓力力道遞減與角度強弱判斷]]，注意該規則角度方向性與上升切線非鏡射對稱，需另行留意）。
- **可程式化**: 是
- **所需資料**: 轉折高點序列（來自 [[轉折波定義與取點演算法]]，持續更新）、既有的下降切線清單（含原始切線與歷次次切線，用於角度比較）
- **計算公式**:
```
function update_down_sub_tangent_lines(tops, bars, existing_lines):
    latest_pair_index = len(tops) - 2
    A, B = tops[latest_pair_index], tops[latest_pair_index + 1]

    if B.high >= A.high:
        return existing_lines   # 尚未出現新的「頭頭低」，不更新

    if not passes_line_not_covering_check(A, B, bars):
        return existing_lines   # 新配對不合法（線蓋線），維持沿用前一條切線

    new_line = fit_line_through(A, B)
    existing_lines.append(new_line)   # 新增1條隨機（次）切線

    if len(existing_lines) >= 2:
        slope_change = abs(slope(new_line)) - abs(slope(existing_lines[-2]))
        # 注意：下降切線角度與強弱的方向性見 R-LINE-10，與上升切線非鏡射對稱
        trend_strength = "走勢轉弱（下降角度變陡）" if slope_change > 0 else "走勢轉強（下降角度趨緩）"

    return existing_lines
```
- **參數**: 轉折點取點所用均線天期 N（沿用 [[轉折波定義與取點演算法]]，預設5日）
- **可回測**: 是
- **信心**: 90/100 高（原書明確定義與上升次切線對稱的「隨機下降切線」概念）
