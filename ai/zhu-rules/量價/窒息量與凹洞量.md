# 窒息量與凹洞量（空頭末跌段力竭反轉訊號）

- **Rule ID**: R-VOLPRICE-07
- **名稱**: 窒息量與凹洞量（空頭末跌段力竭反轉訊號）
- **分類**: 量價
- **原文與頁碼**: `P07-C3-不同趨勢位置的量價關係.md`（下冊 p.140-142，「六、空頭下跌量的變化」之「2. 空頭下跌中的爆大量」第4、5點）
- **解讀**: 空頭下跌過程中，繼下跌大量長黑K之後可能出現力竭訊號：
  1. **窒息量**：下跌大量長黑K，次日續跌，且成交量縮小到該大量的二分之一以下，稱為窒息量。
  2. **凹洞量**：窒息量次日如果出現放大量的紅K反彈，容易強力反彈；這連續3天（大量長黑K → 窒息量 → 放量紅K反彈，型態近似K線的「晨星」組合）的成交量變化組合，稱為「凹洞量」，屬強力反彈訊號。
- **可程式化**: 是
- **所需資料**: 收盤價、開盤價、最高價、最低價、成交量
- **計算公式**:
```
下跌大量長黑K = (K線顏色[t]=="黑") and (K線實體較長) and (Volume[t] 明顯放大)

# 窒息量
if 下跌大量長黑K:
    if (Close[t+1] < Close[t]) and (Volume[t+1] <= 0.5 * Volume[t]):
        is_suffocation_vol = True   # 窒息量，第 t+1 日

        # 凹洞量：窒息量次日出現放量紅K反彈
        if (K線顏色[t+2]=="紅") and (Volume[t+2] > Volume[t+1]) and (Close[t+2] > Open[t+2]):
            is_pothole_vol_pattern = True   # 凹洞量（3日組合：t, t+1, t+2）
            signal = "強力反彈訊號"
```
- **參數**: 窒息量門檻＝次日量縮小至前一根大量長黑K的0.5倍以下（書中明確數字）；凹洞量為窒息量後次日的放量紅K反彈，天數固定為3日組合
- **可回測**: 是
- **信心**: 92/100 高（窒息量的0.5倍門檻與凹洞量的3日K線組合順序在原文中皆有明確數字與描述）
