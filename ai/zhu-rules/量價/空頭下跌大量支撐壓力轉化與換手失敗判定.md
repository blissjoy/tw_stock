# 空頭下跌大量支撐壓力轉化與換手失敗判定

- **Rule ID**: R-VOLPRICE-06
- **名稱**: 空頭下跌大量支撐壓力轉化與換手失敗判定
- **分類**: 量價
- **原文與頁碼**: `P07-C3-不同趨勢位置的量價關係.md`（下冊 p.140-142，「六、空頭下跌量的變化」之「2. 空頭下跌中的爆大量」，不含窒息量/凹洞量部分，另見 [[窒息量與凹洞量]]）
- **解讀**: 空頭確認開始下跌後（空頭確認的爆大量判定已由 [[空頭確認爆大量與做空時機判斷]] 涵蓋，本規則不重複），下跌過程中出現大量長黑K，須觀察：
  1. 下跌大量長黑K代表當天空方力道強，次日容易繼續下跌。
  2. 該大量長黑K的**高點是壓力線、低點是支撐線**：後續高點被突破 → 多方力量轉強、反彈確認；後續低點被跌破 → 空方換手失敗、持續下跌。
  3. 下跌大量長黑K後，若後續出現大量紅K反彈，反彈結束再下跌，且不破前面第1支腳低點，形成不破底的「底底高」第2支腳，則前面第1支腳的下跌大量黑K，可回溯視為**主力進貨量**（此結果與打底流程銜接，完整的打底兩支腳與均線鎖股條件見 [[空轉多打底量價與均線排列鎖股訊號]]）。
- **可程式化**: 是
- **所需資料**: 收盤價、最高價、最低價、成交量
- **計算公式**:
```
下跌大量長黑K = (K線顏色[t]=="黑") and (K線實體較長) and (Volume[t] 明顯放大)

if 下跌大量長黑K:
    resistance = High[t]
    support = Low[t]
    if 後續某日 Close > resistance (or High 突破 resistance):
        signal = "多方力量轉強，反彈確認"
    elif 後續某日 Close < support (or Low 跌破 support):
        signal = "空方換手失敗，持續下跌"

    # 回溯標記為進貨量（打底第1支腳）
    if 後續出現大量紅K反彈 and 反彈結束後再下跌 and (回檔低點 > support):   # 不破前低，底底高
        retro_label[t] = "主力進貨量（打底第1支腳，銜接 R-TREND-10 打底流程）"
```
- **參數**: 無具體倍數門檻（「大量」延用 [[成交量分類與倍數門檻定義]] 的爆大量門檻，或 [[高檔爆量三分類判讀]] 的相對前一日門檻，本節未再另訂新數字）
- **可回測**: 是
- **信心**: 90/100 高（支撐/壓力轉化邏輯、換手失敗判定條件與回溯標記為進貨量的觸發條件在原文中描述明確）
