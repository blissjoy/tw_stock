# 空頭轉折關鍵大量K線操作對應

- **Rule ID**: R-VOLPRICE-10
- **名稱**: 空頭轉折關鍵大量K線操作對應
- **分類**: 量價
- **原文與頁碼**: `P07-C3-不同趨勢位置的量價關係.md`（下冊 p.146-147，「七、K線量價綜合總結」空頭轉折成交量觀察5條）
- **解讀**: 與 [[多頭轉折關鍵大量K線操作對應]] 鏡射對稱，但本節內容是書中對「空頭起跌量能」的唯一完整描述（P07-C3並未另立獨立小節專講空頭起跌的成交量，不同於多頭有專門的「三、多頭起漲的成交量」章節），因此本規則不重複引用其他規則，完整收錄5條：
  1. 起跌（彈後下跌或盤整跌破）「大量」黑K後下跌，最高點是關鍵點，被突破則做空者停損出場。
  2. 起跌「大量」黑K後下跌，次日「大量」紅K上漲，此連續2根K線的最高點與最低點是關鍵點：跌破紅K最低點續跌 → 該紅K視為「誘多出貨、殺低洗盤騙線」；突破黑K高點 → 停損出場。
  3. 下跌低檔「大量」黑K下跌，最高點是關鍵點，被突破則反彈；下跌低檔「大量」黑K下跌，次日開高走高或開低走高，則反彈。
  4. 低檔連續急跌「暴大量」（末跌段現象），股價只能下跌、不能上漲，一旦出現紅K上漲，容易急漲或反轉趨勢。
  5. 低檔股價續創新低、成交量遞增，稱為「量價背離」，代表下跌賣壓力道增加（原文用語為「下跌買盤力道增加」，依上下文應理解為賣壓增加後買盤逐漸承接的力道變化）；當股價出現紅K不跌，容易反彈或打底。
- **可程式化**: 是
- **所需資料**: 開盤價、收盤價、最高價、最低價、成交量
- **計算公式**:
```
# 1. 起跌大量黑K最高點停損
起跌大量黑K = (K線顏色[t]=="黑") and (Volume[t] 明顯放大) and (t 位於彈後下跌起點 or 盤整跌破點)
if 起跌大量黑K and Close[t+1] > High[t]:
    signal = "做空者停損出場"

# 2. 起跌大量黑K次日大量紅K，關鍵點判定
if 起跌大量黑K and (K線顏色[t+1]=="紅") and (Volume[t+1] 明顯放大):
    if Close[t+2] < Low[t+1]:
        signal = "續跌，紅K視為誘多出貨/殺低洗盤騙線"
    elif Close[t+2] > High[t]:
        signal = "突破黑K高點，停損出場"

# 3. 低檔大量黑K反彈判定
低檔大量黑K = (t 位於下跌低檔) and (K線顏色[t]=="黑") and (Volume[t] 明顯放大)
if 低檔大量黑K:
    if Close[t+1] > High[t]:
        signal = "反彈"
    次日開高走高 = Open[t+1] > Close[t] and Close[t+1] > Open[t+1]
    次日開低走高 = Open[t+1] < Close[t] and Close[t+1] > Open[t+1]
    if 次日開高走高 or 次日開低走高:
        signal = "反彈"

# 4. 末跌段暴大量只跌不漲
if 低檔連續急跌 and Volume[t] 暴大量:
    if (K線顏色[t]=="紅") and 股價當日上漲:
        signal = "末跌段暴大量出現紅K，容易急漲或反轉趨勢"

# 5. 低檔量價背離
低檔創新低 = Close[t] < min(Close[近期])
量增 = Volume[t] > MA_Volume(近期)
if 低檔創新低 and 量增:
    state = "量價背離（低檔）"
    if (K線顏色[t+1]=="紅") and not 股價續跌:
        signal = "容易反彈或打底"
```
- **參數**: 「大量/暴大量」門檻延用 [[成交量分類與倍數門檻定義]]；其餘為書中明確描述的K線型態與隔日反應條件，無額外倍數參數
- **可回測**: 是
- **信心**: 91/100 高（5條在原文中皆有明確K線顏色、關鍵點與後續反應的文字描述，且為書中對空頭起跌量能的唯一完整段落，不與其他規則重疊）
