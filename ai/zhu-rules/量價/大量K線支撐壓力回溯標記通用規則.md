# 大量K線支撐壓力回溯標記通用規則

- **Rule ID**: R-VOLPRICE-08
- **名稱**: 大量K線支撐壓力回溯標記通用規則
- **分類**: 量價
- **原文與頁碼**: `P07-C3-不同趨勢位置的量價關係.md`（下冊 p.143，「七、K線量價綜合總結」基礎定義1-7點）
- **解讀**: 本規則是全書量價分析的統一收斂骨架，可通用於多頭、空頭、盤整三種趨勢狀態，把「任一大量K線」依後續價格是否突破或跌破，回溯標記為「進貨量＋未來支撐」或「出貨量＋未來壓力」：
  1. 紅K的「成交量」為當日買方追買量（買方可能是主力，也可能是散戶）。
  2. 黑K的「成交量」為當日賣方追賣量（賣方可能是主力，也可能是散戶）。
  3. 多頭「大成交量」後**上漲突破** → 攻擊進貨量，也是日後**支撐量**。
  4. 多頭「大成交量」後**下跌跌破** → 出貨量，也是日後**壓力量**。
  5. 空頭「大成交量」後**上漲突破** → 搶反彈進貨量，也是日後**支撐量**。
  6. 空頭「大成交量」後**下跌跌破** → 恐慌出貨量，也是日後**壓力量**。
  7. 盤整區「大成交量」為換手累積量：**上漲突破** → 換手成功進貨量，也是日後支撐量；**下跌跌破** → 換手失敗出貨量，也是日後壓力量。
  核心邏輯：任一次「大成交量K線」都可依後續價格是否向上突破或向下跌破，回溯標記為「進貨量＋未來支撐」或「出貨量＋未來壓力」，與所處趨勢狀態（多頭/空頭/盤整）無關，僅方向標籤名稱不同。
- **可程式化**: 是
- **所需資料**: 開盤價、收盤價、最高價、最低價、成交量、趨勢狀態標籤（多頭/空頭/盤整）
- **計算公式**:
```
def classify_big_volume_bar(t, trend_state):
    # trend_state ∈ {"多頭", "空頭", "盤整"}
    is_red = Close[t] > Open[t]
    is_big_vol = Volume[t] 明顯放大（延用 [[成交量分類與倍數門檻定義]] 門檻）
    if not is_big_vol:
        return None

    bar_high, bar_low = High[t], Low[t]
    breakout = any(Close[t+1:] > bar_high)   # 之後某日收盤向上突破
    breakdown = any(Close[t+1:] < bar_low)   # 之後某日收盤向下跌破

    label_map = {
        "多頭": {"突破": "攻擊進貨量／未來支撐", "跌破": "出貨量／未來壓力"},
        "空頭": {"突破": "搶反彈進貨量／未來支撐", "跌破": "恐慌出貨量／未來壓力"},
        "盤整": {"突破": "換手成功進貨量／未來支撐", "跌破": "換手失敗出貨量／未來壓力"},
    }
    if breakout and not breakdown:
        return label_map[trend_state]["突破"], bar_low   # 回傳標籤與支撐價位
    elif breakdown and not breakout:
        return label_map[trend_state]["跌破"], bar_high   # 回傳標籤與壓力價位
    else:
        return "待突破/跌破結果確認", None
```
- **參數**: 無具體倍數門檻，「大成交量」延用 [[成交量分類與倍數門檻定義]]（相對MA5）或 [[高檔爆量三分類判讀]]（相對前一日）的既有門檻
- **可回測**: 是
- **信心**: 91/100 高（回溯標記邏輯在原文中以7條並列句型明確列出，且明確指出此邏輯可通用於多頭/空頭/盤整三種狀態）
