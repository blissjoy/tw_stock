# 多頭轉折關鍵大量K線操作對應

- **Rule ID**: R-VOLPRICE-09
- **名稱**: 多頭轉折關鍵大量K線操作對應
- **分類**: 量價
- **原文與頁碼**: `P07-C3-不同趨勢位置的量價關係.md`（下冊 p.146，「七、K線量價綜合總結」多頭轉折成交量觀察5條）
- **解讀**: 多頭趨勢中，圍繞「大量K線」的5個操作對應點，把量能規則收斂為具體的停損/加碼/回檔判斷。其中第1、2點的判定邏輯已由 [[多頭起漲量能判讀]] 完整涵蓋（起漲攻擊量最低點停損、假跌破洗盤vs誘多騙線），第4點已由 [[多頭高檔爆量停利訊號]] 涵蓋（末升段暴大量只漲不跌），第5點已由 [[多頭上漲量價背離三型態]] 涵蓋（價漲量縮），本規則僅重新彙整並補充第3點（原文未被其他規則涵蓋的高檔關鍵點回檔判定）：
  1. 起漲（回後上漲或盤整突破）「大量」紅K後上漲，最低點是關鍵點，被跌破則停損出場。（見 [[多頭起漲量能判讀]]）
  2. 起漲「大量」紅K後上漲，次日「大量」黑K下跌，此連續2根K線的最高點與最低點是關鍵點：突破黑K最高點續漲 → 該黑K為洗盤；跌破紅K低點 → 停損出場，該紅K視為「誘多騙線」。（見 [[多頭起漲量能判讀]]）
  3. **（本規則新增）** 高檔「大量」紅K上漲，最低點是關鍵點，跌破紅K低點則回檔；高檔「大量」黑K下跌，次日開低走低或開高走低，則回檔。
  4. 高檔連續急漲「暴大量」（末升段現象），股價只能上漲、不能下跌，一旦出現黑K下跌，容易急跌或反轉趨勢。（見 [[多頭高檔爆量停利訊號]]）
  5. 高檔股價創新高、成交量遞減，稱為「量價背離」，代表上漲買力道遞減；當股價出現黑K不漲，容易回檔或盤頭。（見 [[多頭上漲量價背離三型態]]）
- **可程式化**: 是
- **所需資料**: 開盤價、收盤價、最高價、最低價、成交量
- **計算公式**:
```
# 第3點（本規則新增，未被其他規則涵蓋）：高檔關鍵點回檔判定
高檔大量紅K = (t 位於高檔) and (K線顏色[t]=="紅") and (Volume[t] 明顯放大)
if 高檔大量紅K and Close[t+1] < Low[t]:
    signal = "跌破紅K最低點，回檔"

高檔大量黑K = (t 位於高檔) and (K線顏色[t]=="黑") and (Volume[t] 明顯放大)
if 高檔大量黑K:
    次日開低走低 = Open[t+1] < Close[t] and Close[t+1] < Open[t+1]
    次日開高走低 = Open[t+1] > Close[t] and Close[t+1] < Open[t+1]
    if 次日開低走低 or 次日開高走低:
        signal = "回檔"

# 第1、2、4、5點：分別呼叫對應規則
# 第1、2點 -> 見 [[多頭起漲量能判讀]] 的「起漲攻擊量」「起漲後假跌破（洗盤判定）」「突破起漲假突破出貨」
# 第4點 -> 見 [[多頭高檔爆量停利訊號]] 情境②連續爆量滯漲
# 第5點 -> 見 [[多頭上漲量價背離三型態]] 第1型「價漲量縮」
```
- **參數**: 「大量」門檻延用 [[成交量分類與倍數門檻定義]]；第3點的「開低走低/開高走低」為書中明確描述的隔日K線型態，無額外倍數參數
- **可回測**: 是
- **信心**: 90/100 高（5個對應點在原文中皆有明確K線顏色、關鍵點與後續反應的文字描述；第3點雖為原文獨有內容，但描述同樣具體明確）
