# 壓力支撐區大量K線隔日應對規則

- **Rule ID**: R-VOLPRICE-11
- **名稱**: 壓力支撐區大量K線隔日應對規則
- **分類**: 量價
- **原文與頁碼**: `P07-C3-不同趨勢位置的量價關係.md`（下冊 p.148，「七、K線量價綜合總結」壓力/支撐位置的量價應對4條）
- **解讀**: 股價來到既有壓力或支撐位置時，若當天出現「大量」K線但收盤未能突破/跌破，且次日走勢確認弱勢，則應依方向回檔或反彈：
  1. 上漲遇到壓力「大量」紅K，收盤沒有突破壓力，次日黑K股價不漲，要回檔。
  2. 上漲遇到壓力「大量」黑K或上影線，收盤沒有突破壓力，次日股價不漲，要回檔。
  3. 下跌遇到支撐「大量」黑K，收盤沒有跌破支撐，次日紅K股價不跌，要反彈。
  4. 下跌遇到支撐「大量」紅K或下影線，收盤沒有跌破支撐，次日股價不跌，要反彈。
  4條規則的共同結構：在壓力/支撐位置出現「大量」K線但收盤未能突破/跌破 → 次日K線顏色與漲跌方向確認弱勢/強勢 → 觸發回檔或反彈。壓力/支撐位置的定義與此前的「大量K線支撐壓力」（見 [[大量K線支撐壓力回溯標記通用規則]]）為同一套支撐壓力來源，本規則聚焦在「遇壓/遇撐當天未突破跌破」情境下的次日應對。
- **可程式化**: 是
- **所需資料**: 開盤價、收盤價、最高價、最低價、成交量、壓力價位、支撐價位
- **計算公式**:
```
# 1&2. 上漲遇壓力，未突破 -> 回檔
遇壓大量K = (Close[t] < 壓力價位) and (High[t] 接近或觸及壓力價位) and (Volume[t] 明顯放大) and \
            (K線顏色[t]=="紅" or K線顏色[t]=="黑" or 有上影線[t])
if 遇壓大量K and Close[t] < 壓力價位:   # 收盤未突破
    if (K線顏色[t+1]=="黑") and Close[t+1] <= Close[t]:   # 次日不漲
        signal = "回檔"

# 3&4. 下跌遇支撐，未跌破 -> 反彈
遇撐大量K = (Close[t] > 支撐價位) and (Low[t] 接近或觸及支撐價位) and (Volume[t] 明顯放大) and \
            (K線顏色[t]=="黑" or K線顏色[t]=="紅" or 有下影線[t])
if 遇撐大量K and Close[t] > 支撐價位:   # 收盤未跌破
    if (K線顏色[t+1]=="紅") and Close[t+1] >= Close[t]:   # 次日不跌
        signal = "反彈"
```
- **參數**: 「大量」門檻延用 [[成交量分類與倍數門檻定義]]；壓力/支撐價位需由趨勢判定或大量K線回溯標記規則（[[大量K線支撐壓力回溯標記通用規則]]）提供，本節未另訂新數字
- **可回測**: 是
- **信心**: 90/100 高（4條規則的觸發條件（K線顏色、上下影線、次日漲跌方向）在原文中皆有明確文字描述）
