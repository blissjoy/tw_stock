# 多頭上漲量價背離三型態（價漲量縮／價平量增／價漲量平）

- **Rule ID**: R-VOLPRICE-04
- **名稱**: 多頭上漲量價背離三型態（價漲量縮／價平量增／價漲量平）
- **分類**: 量價
- **原文與頁碼**: `P07-C3-不同趨勢位置的量價關係.md`（下冊 p.117-123，「二、多頭上漲量價背離」）
- **解讀**: 多頭連續上漲中，「價漲量增」是正常的多頭價量配合；但若出現以下3種情形，代表走勢可能出現變化，須做好股價下跌準備（考慮停利出場或減碼）：
  1. **價漲量縮**：股價持續創高，但成交量反而縮小，稱為量價背離，反映追價買盤力道轉弱；當股價出現黑K下跌，表示漲多要回檔。若量縮之後又出現量增上漲，則背離未必立即反轉，須惜售。
  2. **價平量增**：股價橫盤無法上漲，但成交量增加，反映買賣力道拉鋸；**位置決定意義完全相反**——出現在底部視為主力進貨訊號（偏多）；出現在高檔/末升段視為潛在出貨訊號（偏空，代表「量大不漲」，可能是主力放利多出貨、正在做頭），須待後續下跌後的價量關係確認。
  3. **價漲量平**：股價續創高，但量能未隨之放大（相對近期量能持平或略降），是股價「止漲」的徵兆，反映主力拉抬意願不高、追買意願降低、獲利了結賣壓增加；若後續未能重新出現量增，反轉下跌風險升高。
  三者皆屬「多頭高檔量價關係轉弱」的警訊型態，差別在於量與價各自呈現的方向組合不同，判斷時須留意所在位置（底部 vs. 高檔/末升段）會反轉部分結論（尤其價平量增）。
- **可程式化**: 是
- **所需資料**: 收盤價、成交量、近期波段高點/低點（可用 [[轉折波定義與取點演算法]] 取得）、趨勢位置（底部/高檔/末升段）
- **計算公式**:
```
# 1. 價漲量縮
price_new_high = Close[t] > max(Close[前波高點:t-1])
vol_shrink = Volume[t] < Volume[前波高點]
if price_new_high and vol_shrink:
    state = "量價背離：價漲量縮"
    if K線顏色[t or t+1] == "黑" and Close < Open:
        signal = "漲多回檔訊號確認，觸發減碼/停利評估"
    elif 後續出現量增上漲:
        signal = "背離未必立即反轉，惜售"

# 2. 價平量增（位置決定方向）
price_flat = not(創新高) and not(破前低)   # 橫盤
vol_expand = Volume[t] > MA_Volume(近期)
if price_flat and vol_expand:
    if 位置 == "底部":
        signal = "主力進貨訊號（偏多）"
    elif 位置 in ("高檔", "末升段"):
        signal = "潛在出貨訊號（偏空），待後續下跌後價量關係確認"

# 3. 價漲量平
price_new_high2 = Close[t] > max(Close[近期])
vol_flat = Volume[t] <= MA_Volume(近期) * 容許區間   # 未隨價放大
if price_new_high2 and vol_flat:
    state = "止漲徵兆"
    if 後續未再出現量增:
        signal = "反轉下跌風險升高"
```
- **參數**: 無具體倍數門檻，「量縮/量增/量平」皆以近期量能水準相對比較判定，實作時可用近期均量或前波高點量作為比較基準（書中未給精確數字，需自訂比較窗口）
- **可回測**: 部分（型態辨識邏輯可程式化並回測；「量縮/量增/量平」的相對比較窗口與容許區間需自訂數值才能精確回測）
- **信心**: 60/100 中（三型態的判定邏輯與「位置決定價平量增方向」的規則描述清楚＝高；但缺乏具體倍數/百分比門檻，需自訂比較基準，拉低整體信心）
