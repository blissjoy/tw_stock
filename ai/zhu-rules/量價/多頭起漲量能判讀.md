# 多頭起漲量能判讀

- **Rule ID**: R-VOLPRICE-05
- **名稱**: 多頭起漲量能判讀
- **分類**: 量價
- **原文與頁碼**: `P07-C3-不同趨勢位置的量價關係.md`（下冊 p.124-128，「三、多頭起漲的成交量」）
- **解讀**: 多頭上漲一波、回檔修正量縮之後再開始放量上漲，當天出現的大量可能是攻擊量或出貨量，起漲量經常影響後續走勢強弱。前提條件：上漲一小波段後主力清洗籌碼致股價下跌，回檔價跌量急縮、接近重要支撐出現止跌K線訊號，且未跌破前一個低點、未跌破月線，確認多頭趨勢不變，股價再度放量以大量紅K上漲。起漲量的5個判讀重點：
  1. **起漲攻擊量**：起漲大量紅K後，次日股價續漲 → 標記為攻擊量（偏多），或是主力在高檔調節賣出的量再進場買回。
  2. **起漲誘多出貨**：起漲大量紅K，次日股價下跌且跌破該紅K最低點 → 標記為誘多出貨量；此情形尤其容易發生在多頭趨勢的高檔或末升段。
  3. **起漲量縮續漲**：起漲紅K未放大量，次日成交量回補（補量）且續漲 → 視為健康多頭延續，非警訊（起漲不一定要放量，補量續漲也算健康）。
  4. **起漲後假跌破（洗盤判定）**：大量起漲後續幾天股價下跌，但未跌破起漲紅K最低點，之後再度放量上攻 → 前段下跌回溯認定為主力洗盤動作，非出貨。
  5. **突破起漲假突破出貨**：大量紅K向上突破盤整起漲，若次日出現大量黑K下跌並跌破該突破起漲紅K最低點 → 原紅K判定為假突破，這2根K線的大量合計視為出貨量。
- **可程式化**: 是
- **所需資料**: 收盤價、開盤價、最高價、最低價、成交量、MA20（月均線）、前一個轉折低點
- **計算公式**:
```
# 前提：回檔洗盤後再度放量上漲的大量紅K
起漲大量紅K = (K線顏色[t]=="紅") and (Volume[t] 明顯放大，可用[[成交量分類與倍數門檻定義]]的攻擊量/爆大量門檻)

# 1. 起漲攻擊量
if 起漲大量紅K and Close[t+1] > Close[t]:
    signal = "起漲攻擊量，偏多"

# 2. 起漲誘多出貨
if 起漲大量紅K and Close[t+1] < Close[t] and Low[t+1] < Low[t]:
    signal = "誘多出貨量"
    if 位置 in ("高檔", "末升段"):
        note = "高檔/末升段起漲更容易出現此情形"

# 3. 起漲量縮續漲
起漲紅K未放量 = (K線顏色[t]=="紅") and (Volume[t] 未達攻擊量門檻)
if 起漲紅K未放量 and Volume[t+1] > Volume[t] and Close[t+1] > Close[t]:
    signal = "量縮起漲、次日補量續漲，健康多頭延續"

# 4. 起漲後假跌破（洗盤判定）
if 起漲大量紅K:
    後續數日下跌 = all(Close[t+1:t+k] < Close[t+1:t+k-1])   # 連續下跌
    未跌破紅K最低點 = min(Low[t+1:t+k]) > Low[t]
    之後再放量上攻 = Volume[t+k] 明顯放大 and Close[t+k] > Close[t+k-1]
    if 後續數日下跌 and 未跌破紅K最低點 and 之後再放量上攻:
        signal = "前段下跌回溯認定為主力洗盤，非出貨"

# 5. 突破起漲假突破出貨
大量紅K向上突破盤整 = 起漲大量紅K and (Close[t] > 盤整區間高點)
if 大量紅K向上突破盤整:
    if (K線顏色[t+1]=="黑") and (Volume[t+1] 明顯放大) and (Low[t+1] < Low[t]):
        signal = "假突破，兩根K線合計大量標記為出貨量"
```
- **參數**: 起漲大量的具體倍數沿用 [[成交量分類與倍數門檻定義]] 的攻擊量（1.2～1.3倍MA5）/爆大量（2倍以上MA5）門檻；假跌破洗盤判定的「連續數日下跌」天數書中未給精確天數，需自訂
- **可回測**: 是
- **信心**: 91/100 高（5個判讀重點的觸發條件、關鍵價位（起漲紅K最低點/最高點）與後續K線顏色/漲跌方向在原文中描述明確）
