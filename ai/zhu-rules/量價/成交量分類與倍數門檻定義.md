# 成交量分類與倍數門檻定義

- **Rule ID**: R-VOLPRICE-01
- **名稱**: 成交量分類與倍數門檻定義
- **分類**: 量價
- **原文與頁碼**: `P07-C2-各種成交量的定義.md`（下冊 p.103-104）；「草叢量」補充見 `P07-C3-不同趨勢位置的量價關係.md`（下冊 p.131）
- **解讀**: 本規則是全書量價分析的共用詞彙表，把「成交量」拆解為10種相對分類，多數以「基本量」（個股5日平均成交量 MA5）為比較基準，換算到不同個股會得到不同的絕對張數，因此規則以倍數表示而非固定張數。注意：本規則的「爆大量」是以 **MA5** 為基準（2倍以上），另有以「前一日量」為基準的爆量判定用於高檔量能分類，兩者比較基準不同、不可混用，見 [[高檔爆量三分類判讀]]。10種分類：
  1. **基本量**：5日平均成交量（MA5），所有分類的比較原點。
  2. **攻擊量**：當日量約為基本量的1.2～1.3倍，且當日股價上漲；即一般所稱的「放大量」，常見於多頭起漲位置。
  3. **爆大量**：當日量約為基本量的2倍以上；方向意涵依所在「位置」而定——起漲位置偏多，高檔/壓力位置易為出貨訊號（需另搭配位置變數判斷，見 [[高檔爆量三分類判讀]]）。
  4. **止跌量**：下跌走勢中，成交量急縮至基本量的一半（約0.5倍）以下，且後續股價不再創新低。
  5. **進貨量**：出現攻擊量或爆大量，且股價同步上漲，視為主力吸籌訊號。
  6. **換手量**：高檔出現一根大量K線後，3個交易日內股價突破該K線最高點，代表強勢多頭續漲。
  7. **出貨量**：出現攻擊量或爆大量後，股價轉為下跌或趨勢反轉，視為主力賣壓訊號。
  8. **調節量**：高檔大量K線後股價回檔修正，經過較長時間後再上漲並突破該K線最高點（主力洗盤性質，非真出貨）；與換手量概念相近，差別在於突破所需天數（換手量3日內，調節量允許較長回檔期）。
  9. **搶反彈量**：空頭連續下跌至低檔出現大量，若後續不再破低，並出現紅K突破前一日高點，視為短線反彈買點（非趨勢反轉確認）。
  10. **草叢量**（補充定義，來自P07-C3）：底部打底盤整期間出現的相對大量，統稱「草叢量」，暫時視為主力進貨訊號，須配合後續股價突破盤整壓力才能確認趨勢反轉。完整的打底兩支腳流程與均線鎖股條件已由 [[空轉多打底量價與均線排列鎖股訊號]] 涵蓋，本規則僅補充這個量能命名。
- **可程式化**: 是
- **所需資料**: 成交量、5日均量（MA5成交量）、收盤價、開盤價、最高價、最低價
- **計算公式**:
```
MA5_vol[t] = SMA(Volume, 5)[t]   # 基本量

# 攻擊量
is_attack_vol[t] = (Volume[t] >= 1.2*MA5_vol[t]) and (Volume[t] <= 1.3*MA5_vol[t]) and (Close[t] > Close[t-1])
# 上限1.3倍為書中舉例區間，非硬性上限，實作可放寬為 >=1.2*MA5_vol 且 < 2*MA5_vol 再與爆大量互斥判斷

# 爆大量（相對MA5，方向仍需搭配位置判斷）
is_big_vol_MA5[t] = Volume[t] >= 2 * MA5_vol[t]

# 止跌量
is_stop_fall_vol[t] = (Volume[t] <= 0.5*MA5_vol[t]) and (處於下跌段) and (後續K線不再跌破前低)

# 進貨量 / 出貨量（方向依後續股價判定，屬於回溯標記）
is_accumulation_vol[t] = (is_attack_vol[t] or is_big_vol_MA5[t]) and (Close[t] > Close[t-1])
is_distribution_vol[t] = (is_attack_vol[t] or is_big_vol_MA5[t]) and (後續股價下跌或趨勢反轉)

# 換手量：高檔大量K線的最高價，3日內被突破
is_handover_vol[t] = is_big_vol_MA5[t] and any(High[t+1:t+4] > High[t]) and (t 位於高檔)

# 調節量：高檔大量K線後回檔修正，之後（較長時間後）突破其最高點
is_adjustment_vol[t] = is_big_vol_MA5[t] and (t 位於高檔) and (回檔修正後才突破High[t]，突破時點晚於3日窗口)

# 搶反彈量
is_rebound_grab_vol[t] = (連續下跌至低檔) and is_big_vol_MA5[t] and (後續不再破低) and (次幾日內出現紅K突破前一日高點)

# 草叢量（打底盤整期）
is_bush_vol[t] = (t 位於打底盤整區間) and (Volume[t] 相對區間均量偏大)
```
- **參數**: 攻擊量1.2～1.3倍MA5；爆大量2倍以上MA5；止跌量0.5倍以下MA5；換手量突破窗口3個交易日
- **可回測**: 是
- **信心**: 92/100 高（書中對主要6項給出明確倍數與天數門檻；「進貨量/出貨量/調節量/草叢量」的方向標記需搭配後續價格回溯確認，屬於書中明確描述的判定邏輯而非額外假設，故仍評為高）
