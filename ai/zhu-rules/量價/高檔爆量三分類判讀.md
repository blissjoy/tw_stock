# 高檔爆量三分類判讀（調節量／換手量／出貨量）

- **Rule ID**: R-VOLPRICE-03
- **名稱**: 高檔爆量三分類判讀（調節量／換手量／出貨量）
- **分類**: 量價
- **原文與頁碼**: `P07-C3-不同趨勢位置的量價關係.md`（下冊 p.111-116，「一、多頭上漲高檔爆量」）
- **解讀**: 多頭上漲3～4天之後出現爆大量（本節門檻為**當日量 ≥ 前一日量的2倍**，注意此基準與 [[成交量分類與倍數門檻定義]] 中「相對MA5」的爆大量不同，兩種爆量偵測需並存，不可混用），當天出現的大量須進一步判定屬於下列3種之一：
  1. **調節量**（主力洗盤性質，非真出貨）：高檔爆大量後回檔，同時符合：①回檔價跌量急縮，賣壓不重；②接近重要支撐（月均線、前低支撐、向上跳空缺口、大量上漲紅K）時出現止跌K線訊號（變盤線、中長紅K）；③未跌破前一個轉折低點、未跌破月均線，股價重新開始上漲。調節量最容易出現在多頭趨勢的初升段與主升段。
  2. **換手量**（偏多延續，常見於強勢股/飆股）：高檔大量下跌黑K，次日小跌或不跌，接著股價突破該大量黑K的最高點，強勢多頭繼續上漲；被突破的那根大量黑K即稱為換手量，換手成功後大多會再急速上漲一波。
  3. **出貨量**（空頭風險訊號，二擇一情境）：①多頭末升段的高檔盤整區出現大量，該盤整區間之後被向下跌破，空頭趨勢確認；②連續急速上漲的飆股，高檔爆大量後僅隔1個交易日即反轉快速下跌。
- **可程式化**: 是
- **所需資料**: 收盤價、開盤價、最高價、最低價、成交量、MA20（月均線）、前一個轉折低點（可用 [[轉折波定義與取點演算法]] 取得）
- **計算公式**:
```
is_high_vol_prevday[t] = Volume[t] >= 2 * Volume[t-1]   # 高檔爆量（相對前一日）

# 調節量
def is_adjustment_vol(t):
    if not (是多頭 and t 位於初升段或主升段高檔 and is_high_vol_prevday[t]):
        return False
    後續回檔 = 找到 t 之後股價開始回檔的區間
    cond1 = 回檔期間量急縮 (回檔均量 < 一般水準)
    cond2 = 回檔價格接近(MA20 or 前低支撐 or 向上跳空缺口 or 前一根大量上漲紅K) and 出現止跌K線(變盤線/中長紅K)
    cond3 = (回檔低點 > 前一個轉折低點) and (回檔期間收盤未跌破MA20) and (回檔後股價重新上漲)
    return cond1 and cond2 and cond3

# 換手量
def is_handover_vol(t):
    if not (K線顏色[t] == "黑" and is_high_vol_prevday[t] and t 位於高檔):
        return False
    cond1 = Low[t+1] >= Low[t] * (1-容許小跌幅度)   # 次日小跌或不跌
    cond2 = any(Close[t+1:] > High[t])                # 隨後突破該黑K最高點
    return cond1 and cond2

# 出貨量（二擇一）
def is_distribution_vol(t):
    情境a = (t 位於末升段高檔盤整區) and (盤整區間之後被向下跌破)
    情境b = (飆股急速上漲) and is_high_vol_prevday[t] and (Low[t+1] < Low[t] and 反轉快速下跌，僅隔1日)
    return 情境a or 情境b
```
- **參數**: 高檔爆量門檻＝前一日量的2倍以上；換手量「次日不跌」的容許小跌幅度、出貨量情境b的「僅隔1日」皆為書中明確描述
- **可回測**: 是
- **信心**: 92/100 高（3種分類的判定條件、比較基準（前一日量2倍）與情境二擇一的門檻在原文中皆有明確文字描述）
