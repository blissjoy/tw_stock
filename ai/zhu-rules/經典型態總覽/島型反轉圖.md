# 島型反轉圖

- **Rule ID**: R-CLASSIC-32
- **名稱**: 島型反轉圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第17種（下冊 p.413-414，圖表12-4-77～79）
- **解讀**: 股價下跌末端先向下跳空缺口，之後在低檔一小段區間橫向震盪（形成一個孤立的價格「島嶼」），再向上跳空缺口離開，前後兩個缺口方向相反且缺口間有明顯價格落差，即為「島型反轉」，屬於強烈的低檔反轉訊號。本型態直接對應既有規則 [[低檔島型反轉規則]]，並與型態19「缺口之上續漲圖」互為關聯（一個講缺口後續漲，一個講缺口對缺口的反轉結構）。
- **可程式化**: 是
- **所需資料**: 前後兩個缺口的價位、震盪區間OHLCV、成交量
- **計算公式**:
```
signal = [[低檔島型反轉規則]].check(price_series)
```
- **參數**: 缺口方向相反判定、震盪區間長度門檻（沿用 [[低檔島型反轉規則]] 既有參數）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 低檔島型反轉規則、向下跳空缺口四層價位分級壓力規則、向上跳空缺口四層價位分級支撐規則、成交量分類與倍數門檻定義
