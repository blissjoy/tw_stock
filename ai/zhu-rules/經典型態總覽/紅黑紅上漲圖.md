# 紅黑紅上漲圖

- **Rule ID**: R-CLASSIC-23
- **名稱**: 紅黑紅上漲圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第8種（下冊 p.402-403，圖表12-4-55～57）
- **解讀**: 「紅黑紅」三根K線組合（多方夾擊）——中間一根黑K被前後兩根紅K夾擊（實體被前後紅K完全包覆或接近吞噬），代表多方力道持續壓制空方，屬於上漲過程中的續漲訊號而非反轉警訊，需在多頭排列格局中（[[均線多頭排列]]）出現才有效。與型態11「黑紅黑續跌圖」互為多空鏡像對稱K線組合。**規則庫覆蓋提醒**：與型態11相同，現有K棒型態規則庫沒有獨立命名的「紅黑紅」三日續勢組合規則，本規則以基本紅黑K定義＋均線排列組合表達。
- **可程式化**: 部分（三日組合形狀可程式化，但「紅黑紅續勢」缺乏獨立底層K線組合規則）
- **所需資料**: 連續3根K棒的開盤價、收盤價、最高最低價、均線排列狀態
- **計算公式**:
```
d1, d2, d3 = 最近三根K
is_red_black_red = is_red(d1) and is_black(d2) and is_red(d3)
is_black_contained = d2.最高價 <= d1.最高價 and d2.最低價 >= d1.最低價   # 黑K實體被夾擊/接近吞噬

if is_red_black_red and is_black_contained and [[均線多頭排列]].check(近期):
    signal = "紅黑紅多方夾擊，續漲訊號（非反轉警訊）"
```
- **參數**: 無額外參數（依三日開收盤與均線排列狀態判定）
- **可回測**: 是
- **信心**: 35/100 低（本次比對未在既有K棒型態筆記中找到「紅黑紅」逐字命名段落，屬推論對應，與型態11同屬財富篇新增口訣）
- **底層規則依賴**: 紅K線、黑K線、均線多頭排列、多頭趨勢進場訊號
