# 突破ABC上漲圖

- **Rule ID**: R-CLASSIC-31
- **名稱**: 突破ABC上漲圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第16種（下冊 p.412-413，圖表12-4-74～76）
- **解讀**: 股價上漲過程中出現ABC型修正（A點高點、B點拉回、C點次高不創新高，形成一個小幅下降切線），當股價「突破下降切線」（[[下降切線畫法與線不蓋線判準]]）且「突破買點」有效站上A點高點，即確認修正結束、原漲勢延續，並可用D值（AB段差距）等距量測目標價。為型態14「跌破ABC下跌圖」的多空鏡像對稱版本，同樣涉及等幅測量法的規則庫覆蓋缺口（見型態14備註）。
- **可程式化**: 部分（ABC轉折點與突破切線的偵測可程式化，但等幅測量目標價缺乏獨立底層規則，精確度中等）
- **所需資料**: ABC三波轉折點、下降切線、成交量
- **計算公式**:
```
A, B, C = [[轉折波定義與取點演算法]].detect_ABC_correction(近期修正, 方向="上漲修正")
down_line = [[下降切線畫法與線不蓋線判準]](A, C)

if close > down_line(today) and close > A.高點 and volume >= 大量門檻:
    D = abs(A.高點 - B.低點)
    target = A.高點 + D   # 等幅測量法（借用 [[型態頸線量測目標價支撐壓力規則]] 量測邏輯）
    signal = "突破ABC修正下降切線與A點高點，多頭續漲確認，目標價=" + target
```
- **參數**: D值計算起訖點（A、B兩點）、大量門檻
- **可回測**: 是
- **信心**: 58/100 中（ABC修正與切線突破邏輯清楚，但等幅測量法本身未在切線章節找到逐字對應規則，屬推論延伸）
- **底層規則依賴**: 轉折波定義與取點演算法、下降切線畫法與線不蓋線判準、突破下降切線分級與壓力支撐角色互換、型態頸線量測目標價支撐壓力規則、多頭趨勢進場訊號
