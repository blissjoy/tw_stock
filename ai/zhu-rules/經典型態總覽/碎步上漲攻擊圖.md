# 碎步上漲攻擊圖

- **Rule ID**: R-CLASSIC-20
- **名稱**: 碎步上漲攻擊圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第5種（下冊 p.398-399，圖表12-4-47～49）
- **解讀**: 股價脫離低檔後，並非急漲，而是以「左低右高」的一根根小紅K「碎步緩漲」方式沿MA20（月線）之上慢慢墊高，符合 [[轉折波定義與取點演算法]] 的左低右高定義與 [[均線上彎的支撐與助漲作用]]，直到某一天出現大量長紅K向上突破整理區才轉為進場攻擊訊號，對應 [[大量長紅K進場位置篩選規則]]。範例另標示後續噴出段可能出現爆大量長上影線、變盤線等末端轉弱訊號，屬獲利了結觀察點而非否定前段攻擊有效性。
- **可程式化**: 是
- **所需資料**: 近期小紅K序列、MA20、突破日OHLCV與成交量
- **計算公式**:
```
is_left_low_right_high = [[轉折波定義與取點演算法]].check_left_low_right_high(近期小紅K序列)
is_above_ma20 = close > MA20 and [[均線上彎的支撐與助漲作用]].is_active(MA20)

if is_left_low_right_high and is_above_ma20:
    if is_red(今日K) and volume(今日K) >= 大量門檻 and close > 整理區高點:
        result = [[大量長紅K進場位置篩選規則]].check(今日K)
        signal = "碎步緩漲後大量長紅突破，攻擊買進點"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 轉折波定義與取點演算法、均線上彎的支撐與助漲作用、大量長紅K進場位置篩選規則、多頭趨勢進場訊號
