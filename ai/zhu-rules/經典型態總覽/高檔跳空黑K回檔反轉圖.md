# 高檔跳空黑K回檔反轉圖

- **Rule ID**: R-CLASSIC-07
- **名稱**: 高檔跳空黑K回檔反轉圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第7種（下冊 p.382-383，圖表12-4-15、12-4-16）
- **解讀**: 股價於高檔頭部區出現向下跳空的黑K棒且伴隨爆大量，對應 [[漲多回檔向下跳空缺口規則]] 與 [[向下跳空缺口壓力規則]]。流程為：頭 → 跳空黑K → 跌破MA20（依 [[均線下彎的壓力與助跌作用]] 判定月線由支撐轉壓力）→ 頭頭低 → 空頭確認。需同時滿足「跳空跌破前低（底底低）」＋「反彈不過前高（頭頭低）」兩條件才升級為空頭反轉確認，KD指標同步走弱可作輔助確認（[[KD黃金死亡交叉依趨勢判讀]]）。
- **可程式化**: 是
- **所需資料**: 當日/前日OHLCV、成交量、MA20、KD指標
- **計算公式**:
```
is_gap_down = [[漲多回檔向下跳空缺口規則]].detect(today, yesterday) and volume >= 大量門檻
is_break_ma20 = close < MA20 and [[均線下彎的壓力與助跌作用]].is_active(MA20)
is_lower_high = [[頭頭低底底低空頭趨勢判定]].check_first_lower_high()
is_lower_low = [[頭頭低底底低空頭趨勢判定]].check_first_lower_low()

if is_gap_down and is_break_ma20 and is_lower_high and is_lower_low:
    signal = "高檔跳空黑K回檔反轉，空頭確認，[[空頭趨勢進場訊號]]"
    if [[KD黃金死亡交叉依趨勢判讀]].is_bearish_cross(近期):
        signal += "，KD同步走弱，確認強化"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 漲多回檔向下跳空缺口規則、向下跳空缺口壓力規則、均線下彎的壓力與助跌作用、頭頭低底底低空頭趨勢判定、空頭趨勢進場訊號、KD黃金死亡交叉依趨勢判讀
