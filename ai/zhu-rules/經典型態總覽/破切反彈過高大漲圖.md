# 破切反彈過高大漲圖

- **Rule ID**: R-CLASSIC-17
- **名稱**: 破切反彈過高大漲圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第2種（下冊 p.394-395，圖表12-4-39、12-4-40）
- **解讀**: 股價原處於下降切線（[[下降切線畫法與線不蓋線判準]]）之下，先「破切」（向上突破下降切線，依 [[突破下降切線分級與壓力支撐角色互換]] 判定切線由壓力轉為支撐），拉回測試後再出現「過高」（突破前波反彈高點），隨後爆大量向上大漲。「過高」點須伴隨爆大量才是有效買進訊號。與型態15「跌破下降軌道線大跌圖」互為多空鏡像對稱操作。
- **可程式化**: 是
- **所需資料**: 下降切線、前波反彈高點、成交量
- **計算公式**:
```
down_line = [[下降切線畫法與線不蓋線判準]](前波高點序列)
is_break_line = close > down_line(today)
retest_ok = [[突破下降切線分級與壓力支撐角色互換]](down_line).支撐有效

prior_bounce_high = 前波反彈高點
if is_break_line and retest_ok and close > prior_bounce_high and volume >= 大量門檻:
    signal = "破切＋過高＋爆量，多頭確認，[[多頭趨勢進場訊號]]"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 下降切線畫法與線不蓋線判準、突破下降切線分級與壓力支撐角色互換、多頭趨勢進場訊號、成交量分類與倍數門檻定義
