# 連3天上漲長上影線 大敵當前主力出貨圖

- **Rule ID**: R-CLASSIC-08
- **名稱**: 連3天上漲長上影線 大敵當前主力出貨圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第8種（下冊 p.383-384，圖表12-4-17、12-4-18）
- **解讀**: 股價連續3天上漲，但每天都留下長上影線（壓力測試失敗、當天高點無法守住），連續3天在同一壓力區反覆受挫，對應 [[轉折高點壓力與突破後轉支撐規則]] 定義的壓力位置，是主力趁高調節出貨的訊號，之後出現頭頭低即正式空頭確認。與型態6同屬「長上影線＝主力出貨」K線家族，共享同一個規則庫覆蓋缺口（見型態6備註：現有規則庫無獨立命名的「長上影線」高檔反轉規則）。
- **可程式化**: 部分（連續3日長上影線的形狀偵測可程式化，但「連續受挫於同一壓力區」須仰賴壓力位置規則的模糊比對，精確度中等）
- **所需資料**: 連續3日OHLCV、近期壓力位置
- **計算公式**:
```
count = 0
for K in 最近3根上漲K棒:
    is_long_upper_shadow = upper_shadow(K) >= 2 * body(K)
    is_at_resistance = near(K.high, [[轉折高點壓力與突破後轉支撐規則]](近期壓力))
    if is_long_upper_shadow and is_at_resistance:
        count += 1

if count == 3:
    signal = "連3日長上影，大敵當前，主力出貨警訊"
    if [[頭頭低底底低空頭趨勢判定]].check_first_lower_high():
        signal += "，空頭確認"
```
- **參數**: 上影線倍數門檻（預設2倍實體）、壓力區判定容忍範圍
- **可回測**: 是
- **信心**: 60/100 中（三日組合的壓力測試邏輯明確，但底層「長上影線」型態未在K棒型態分類找到獨立逐字對應規則）
- **底層規則依賴**: 轉折高點壓力與突破後轉支撐規則、高檔變盤線、頭頭低底底低空頭趨勢判定、多頭高檔爆量停利訊號
