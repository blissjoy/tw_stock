# 缺口之上續漲圖

- **Rule ID**: R-CLASSIC-19
- **名稱**: 缺口之上續漲圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第4種（下冊 p.397-398，圖表12-4-44～46）
- **解讀**: 股價向上跳空留下缺口後，缺口之上的整理區間若不破缺口（缺口成為支撐，依 [[向上跳空缺口四層價位分級支撐規則]]／[[向上跳空缺口支撐規則]]），可在缺口之上找大量訊號作為買點進場；範例顯示缺口之上盤堅過程中即使交錯出現爆大量長黑、長上影線、變盤線等訊號，股價仍能持續在缺口之上續漲，核心關鍵是缺口本身形成的支撐區未被跌破，可搭配 [[缺口之上見長紅K強勢整理規則]] 判讀。
- **可程式化**: 是
- **所需資料**: 缺口價位、缺口後走勢OHLCV、成交量
- **計算公式**:
```
gap = [[向上跳空缺口四層價位分級支撐規則]].detect(today, yesterday)
is_not_covered = all(close(後續K) > gap.lower_bound for 後續K in K_list)   # 缺口不破

if gap and is_not_covered:
    if [[缺口之上見長紅K強勢整理規則]].check(當日) and volume >= 大量門檻:
        signal = "缺口之上出現大量買點，續漲確認"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 向上跳空缺口四層價位分級支撐規則、向上跳空缺口支撐規則、缺口之上見長紅K強勢整理規則、成交量分類與倍數門檻定義、多頭趨勢進場訊號
