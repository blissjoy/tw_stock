# 均線糾結紅K突破圖

- **Rule ID**: R-CLASSIC-30
- **名稱**: 均線糾結紅K突破圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第15種（下冊 p.411，圖表12-4-72、12-4-73）
- **解讀**: 多條均線（如MA5、MA10、MA20、MA60）長期糾結纏繞在一起（代表多空僵持、方向未明，依 [[均線糾結辨識]]），當出現爆大量紅K突破糾結區的買進訊號，帶動均線由糾結轉為多頭排列發散，即為強力多頭起漲訊號。本型態直接對應既有規則 [[均線糾結向上突破做多SOP]]，此複合型態卡即為該SOP在33種贏家圖像收尾章節的完整型態卡收錄。
- **可程式化**: 是
- **所需資料**: 多條均線糾結區間、突破日OHLCV與成交量
- **計算公式**:
```
signal = [[均線糾結向上突破做多SOP]].check(price_series, ma_lines)
```
- **參數**: 均線糾結判定門檻、大量門檻（沿用 [[均線糾結向上突破做多SOP]] 既有參數）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 均線糾結辨識、均線糾結向上突破做多SOP、均線多頭排列、成交量分類與倍數門檻定義
