# 低檔大量長下影線圖

- **Rule ID**: R-CLASSIC-26
- **名稱**: 低檔大量長下影線圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第11種（下冊 p.406-407，圖表12-4-63～65）
- **解讀**: 股價低檔出現帶爆大量的長下影線K棒（盤中大幅殺低後拉回收復），屬於一日反轉訊號，長下影線必須爆大量，量能不足的長下影線不算有效訊號。直接對應既有規則 [[槌子線與倒槌K線]]（下影線長、實體小的槌子線定義）。示意圖並標示後續走勢「回補」（缺口或跳空回補、長下影線回補二分之一）等量化判讀細節，範例可搭配突破前高、突破月線、[[低檔晨星]] 等多重確認。
- **可程式化**: 是
- **所需資料**: 當日OHLCV、成交量、MA20
- **計算公式**:
```
is_long_lower_shadow = [[槌子線與倒槌K線]].is_hammer(K) and volume(K) >= 大量門檻
is_at_bottom = [[頭頭低底底低空頭趨勢判定]](近期) == "低檔"

if is_long_lower_shadow and is_at_bottom:
    signal = "低檔大量長下影線，一日反轉買進候選"
    if close > MA20 or [[低檔晨星]].check(近期):
        signal += "，多重確認強化"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 槌子線與倒槌K線、低檔晨星、成交量分類與倍數門檻定義、均線上彎的支撐與助漲作用、空轉多打底量價與均線排列鎖股訊號
