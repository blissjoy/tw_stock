# 底部洗盤上攻大漲圖

- **Rule ID**: R-CLASSIC-21
- **名稱**: 底部洗盤上攻大漲圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第6種（下冊 p.400，圖表12-4-50、12-4-51）
- **解讀**: 股價打底完成後，先出現一次「破底洗盤」（假跌破前低，誘空震洗浮額，隨即爆量長黑收斂），之後才突破前高並突破買點向上噴出。破底洗盤與突破買點兩處皆須爆大量，破底洗盤是本型態辨識重點（區別於真跌破）。**規則庫覆蓋提醒**：現有規則庫沒有獨立命名的「洗盤」（假跌破誘空）通用規則，本規則以 [[量價因果關係與四維度綜合判讀原則]] 的量價異常判讀＋[[轉折低點支撐與跌破後轉壓力規則]] 的反例組合表達。
- **可程式化**: 部分（突破買點的偵測可程式化，但「假跌破洗盤」與「真跌破」的事前區分缺乏獨立底層規則，通常須靠事後反彈速度回溯判定）
- **所需資料**: 前低價位、洗盤K棒OHLCV、突破日OHLCV、成交量
- **計算公式**:
```
is_fake_breakdown = close < 前低 and volume >= 大量門檻 and 次日反彈迅速收回

if is_fake_breakdown:
    label = "假跌破洗盤（非真跌破）— 依 [[量價因果關係與四維度綜合判讀原則]] 判讀"
    if close > 近期前高 and volume >= 大量門檻:
        signal = "洗盤後突破前高，攻擊買進，[[多頭趨勢進場訊號]]"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）、次日反彈收回幅度門檻
- **可回測**: 是
- **信心**: 60/100 中（型態邏輯明確，但「洗盤」缺乏獨立底層規則庫定義，事前無法完全排除真跌破的可能）
- **底層規則依賴**: 量價因果關係與四維度綜合判讀原則、轉折低點支撐與跌破後轉壓力規則、成交量分類與倍數門檻定義、多頭趨勢進場訊號、空轉多打底量價與均線排列鎖股訊號
