# 雙弧底大量紅K突破圖

- **Rule ID**: R-CLASSIC-29
- **名稱**: 雙弧底大量紅K突破圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第14種（下冊 p.410，圖表12-4-70、12-4-71）
- **解讀**: 與型態28「雙盤底」類似但底部形狀為圓弧狀（U型底）而非平台狀——股價於低檔形成兩個相連的圓弧型打底，底部完成後以大量紅K突破買進壓力線。範例並標示末端可能出現長黑下跌，提醒漲勢仍會有獲利了結賣壓，不影響前段突破有效性。與型態18「大量雙腳反轉圖」、型態28「雙盤底」同屬底部打底家族的形狀變化型，規則庫彙整時可視為同一族規則、以「底部形狀」（V型雙腳／平台雙盤／圓弧雙弧）作為子分類參數，直接沿用 [[雙盤底大量紅K突破進場]] 的核心突破邏輯。
- **可程式化**: 是
- **所需資料**: 圓弧底區間OHLCV、突破日成交量
- **計算公式**:
```
arc_bottom = detect_rounded_bottom(price_series)   # 圓弧狀而非平台
resistance = [[密集盤整區區間支撐壓力規則]](arc_bottom.上緣)

if close > resistance and volume >= 大量門檻:
    signal = "雙弧底突破，套用 [[雙盤底大量紅K突破進場]] 核心邏輯（底部形狀參數＝圓弧）"
```
- **參數**: 底部形狀參數（圓弧 vs 平台）、大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 58/100 中（底部打底核心邏輯與型態28、18共通，但「圓弧底」形狀偵測本身未在既有規則庫找到獨立定義，屬形狀參數化推論）
- **底層規則依賴**: 雙盤底大量紅K突破進場、密集盤整區區間支撐壓力規則、成交量分類與倍數門檻定義
