# 大量雙頭反轉圖

- **Rule ID**: R-CLASSIC-03
- **名稱**: 大量雙頭反轉圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第3種（下冊 p.378-379，圖表12-4-7～9）
- **解讀**: 經典M頭／雙重頂型態：兩個相近高點中間夾一個回檔低點（頸線），第二個頭略低於或等於第一個頭，符合 [[頭頭低底底低空頭趨勢判定]] 的頭頭低條件。訊號確認為帶大量跌破兩頭之間的回檔低點（頸線），對應 [[盤整區上下頸線支撐壓力規則]]；頸線本身即為 [[頭部底部大量關卡支撐壓力轉換規則]] 定義的套牢壓力區。第二頭出現時常伴隨長上影線或 [[高檔長黑吞噬]]，跌破頸線後可用 [[型態頸線量測目標價支撐壓力規則]] 等距量測下跌目標價。
- **可程式化**: 是
- **所需資料**: 兩個高點與中間回檔低點（頸線）的價位、成交量序列
- **計算公式**:
```
head1, trough, head2 = detect_double_top(price_series)
is_lower_high = head2.high <= head1.high   # 頭頭低，依 [[頭頭低底底低空頭趨勢判定]]
neckline = [[盤整區上下頸線支撐壓力規則]](trough)

if is_lower_high and close < neckline and volume >= 大量門檻:
    target = [[型態頸線量測目標價支撐壓力規則]](head1, trough, neckline, 方向="向下")
    signal = "M頭頸線跌破，空頭確認，[[空頭趨勢進場訊號]]，目標價=" + target
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 盤整區上下頸線支撐壓力規則、型態頸線量測目標價支撐壓力規則、頭部底部大量關卡支撐壓力轉換規則、頭頭低底底低空頭趨勢判定、高檔長黑吞噬、空頭趨勢進場訊號、成交量分類與倍數門檻定義
