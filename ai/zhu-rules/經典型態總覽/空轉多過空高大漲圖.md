# 空轉多過空高大漲圖

- **Rule ID**: R-CLASSIC-22
- **名稱**: 空轉多過空高大漲圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第7種（下冊 p.401-402，圖表12-4-52～54）
- **解讀**: 股價原處於空頭趨勢，先出現空頭確認訊號的止跌反轉（跌破切線、跌破前低等空頭訊號止穩，[[空頭趨勢改變先知先覺訊號]]），股價止跌後反彈並多頭確認（[[頭頭高底底高多頭趨勢判定]] 首次出現higher low反彈），最終突破空頭反彈期間的「空頭高點」（依 [[轉折高點壓力與突破後轉支撐規則]]），此點突破即代表趨勢由空翻多，隨後展開大漲。空頭確認、突破買進兩處皆須搭配大量。這是「趨勢轉折確認」的總覽型態，整合空頭確認與多頭確認兩階段完整判斷邏輯。
- **可程式化**: 是
- **所需資料**: 空頭反彈期間高點、成交量、近期高低點序列
- **計算公式**:
```
is_bear_reversal = [[空頭趨勢改變先知先覺訊號]].check(近期) and volume >= 大量門檻
is_bull_confirm = [[頭頭高底底高多頭趨勢判定]].check_first_higher_low_bounce()

if is_bear_reversal and is_bull_confirm:
    bear_rebound_high = 空頭反彈期間高點
    if close > bear_rebound_high and volume >= 大量門檻:
        signal = "突破空頭反彈高點，趨勢空轉多確認，[[多頭趨勢進場訊號]]"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 空頭趨勢改變先知先覺訊號、頭頭高底底高多頭趨勢判定、轉折高點壓力與突破後轉支撐規則、多頭趨勢進場訊號、成交量分類與倍數門檻定義
