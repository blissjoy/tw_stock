# 突破高檔前高黑K出現大量要下跌圖

- **Rule ID**: R-CLASSIC-10
- **名稱**: 突破高檔前高黑K出現大量要下跌圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第10種（下冊 p.385-386，圖表12-4-21、12-4-22）
- **解讀**: 股價突破前波歷史高點（依 [[轉折高點壓力與突破後轉支撐規則]]，突破後理應轉為支撐），但突破後立刻爆大量收長黑（[[高檔長黑吞噬]]），收盤又跌破原本的前高價位，屬於「假突破／騙線」型態，代表前高支撐測試失敗、並未真正解除原壓力角色。**規則庫覆蓋提醒**：現有規則庫中沒有獨立命名的「假突破」通用規則（僅缺口分類有 [[缺口假封口判定規則]] 處理缺口封口的真假判定，適用範圍不同），本規則以「突破後轉支撐失效」的組合方式表達。
- **可程式化**: 部分（K線與量能條件可程式化，但「假突破」作為獨立型態缺乏底層規則庫的通用定義，需以支撐壓力角色互換規則反向推論）
- **所需資料**: 前波歷史高點、突破後1-2日OHLCV、成交量
- **計算公式**:
```
is_breakout = close > 前波歷史高點   # 依 [[轉折高點壓力與突破後轉支撐規則]] 理應轉為支撐
is_big_black_engulf = [[高檔長黑吞噬]].is_pattern and volume >= 大量門檻

if is_breakout and is_big_black_engulf(突破後1~2日內) and close < 前波歷史高點:
    signal = "假突破：前高支撐測試失敗、反轉為壓力，空頭確認"
    trigger = [[空頭確認爆大量與做空時機判斷]]
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 60/100 中（型態邏輯清楚，但「假突破」缺乏獨立底層規則支撐，屬跨規則組合推論）
- **底層規則依賴**: 轉折高點壓力與突破後轉支撐規則、高檔長黑吞噬、頭頭低底底低空頭趨勢判定、空頭確認爆大量與做空時機判斷
