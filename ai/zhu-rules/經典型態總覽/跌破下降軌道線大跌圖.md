# 跌破下降軌道線大跌圖

- **Rule ID**: R-CLASSIC-15
- **名稱**: 跌破下降軌道線大跌圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第15種（下冊 p.392-393，圖表12-4-35、12-4-36）
- **解讀**: 空頭排列中，股價沿著下降切線（上緣壓力）與下降軌道線（下緣支撐，與下降切線平行）構成的下降通道緩步下滑。一旦帶大量的長黑K棒跌破下降軌道線（下緣支撐），代表跌勢加速，後續會出現更陡、更猛烈的一波下跌。本型態直接逐字對應既有規則 [[下降軌道線畫法（切線平移取中間低點）]] 與 [[跌破下降軌道線支撐的大量長黑K]]（非推論，原書兩處敘述完全一致）。
- **可程式化**: 是
- **所需資料**: 下降軌道線（依既有規則畫出）、當日OHLCV、成交量
- **計算公式**:
```
channel = [[下降軌道線畫法（切線平移取中間低點）]](下降切線, A, B)
is_big_black = is_black(K) and volume >= 大量門檻

if close < channel(today) and is_big_black:
    signal = [[跌破下降軌道線支撐的大量長黑K]] 觸發
    note = "支撐轉壓力，跌勢由緩降轉為急跌"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 93/100 高（原文逐字對應，非推論）
- **底層規則依賴**: 下降軌道線畫法（切線平移取中間低點）、跌破下降軌道線支撐的大量長黑K、下降切線畫法與線不蓋線判準、頭頭低底底低空頭趨勢判定
