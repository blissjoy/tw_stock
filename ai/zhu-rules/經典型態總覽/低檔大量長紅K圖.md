# 低檔大量長紅K圖

- **Rule ID**: R-CLASSIC-16
- **名稱**: 低檔大量長紅K圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第1種（下冊 p.393-394，圖表12-4-37、12-4-38）
- **解讀**: 股價處於低檔（下跌一段時間後），出現一根帶爆大量的長紅K棒，且短時間內連續出現多根長紅K（範例①在低檔區間內反覆出現「長紅＋爆大量」共3次才確立反轉；範例②則一次爆量長紅後止穩，再一根長紅＋爆大量才確認轉強）。單一長紅但量能不足不算數，對應 [[低檔黑K紅K基本反轉型態]] 與 [[成交量分類與倍數門檻定義]]，整體是 [[空轉多打底量價與均線排列鎖股訊號]] 的起漲觸發訊號。
- **可程式化**: 是
- **所需資料**: 低檔區間OHLCV、成交量序列
- **計算公式**:
```
count = 0
for K in 近期低檔K棒序列:
    if is_red(K) and volume(K) >= 大量門檻 and [[低檔黑K紅K基本反轉型態]].check(K):
        count += 1

if count >= 2:   # 短時間內反覆出現大量長紅
    signal = "低檔大量長紅確認打底，[[空轉多打底量價與均線排列鎖股訊號]]觸發"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）、確認所需大量長紅根數（預設≥2）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 低檔黑K紅K基本反轉型態、低檔長紅吞噬、低檔長紅貫穿、成交量分類與倍數門檻定義、空轉多打底量價與均線排列鎖股訊號、頭頭低底底低空頭趨勢判定
