# 缺口之下續空圖

- **Rule ID**: R-CLASSIC-12
- **名稱**: 缺口之下續空圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第12種（下冊 p.387-389，圖表12-4-25～29）
- **解讀**: 股價向下跳空缺口後，在缺口下方反覆「盤整→跌破→再盤整→再跌破」，只要缺口不被回補，就持續視為空方主導，可持續在盤整區下緣尋找空點放空或加碼，對應 [[向下跳空缺口壓力規則]]／[[向下跳空缺口四層價位分級壓力規則]] 與 [[缺口真封口判定規則]]（反向判定：缺口未封口）。變化版強調缺口出現前股價先跌破MA20（[[均線下彎的壓力與助跌作用]]），確認多頭轉弱後才會加速形成缺口下續空格局；範例並顯示同一機制可能連環觸發，出現 [[高檔島型反轉規則]] 再次啟動循環。
- **可程式化**: 是
- **所需資料**: 缺口價位、缺口後盤整區間OHLCV、MA20
- **計算公式**:
```
gap = [[向下跳空缺口四層價位分級壓力規則]].detect(today, yesterday)
is_not_covered = not [[缺口真封口判定規則]].is_covered(gap)   # 缺口未回補

if gap and is_not_covered:
    zone = [[密集盤整區區間支撐壓力規則]](缺口下方走勢)
    if close < zone.low:
        signal = "缺口下再破底，續空/加碼放空點"
    if [[均線下彎的壓力與助跌作用]].is_broken(MA20) before gap:
        signal += "（月線先跌破，加速確認）"
    if [[高檔島型反轉規則]].detect(近期):
        signal += "，注意反轉循環再次啟動"
```
- **參數**: 缺口封口判定門檻（依 [[缺口真封口判定規則]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 向下跳空缺口四層價位分級壓力規則、向下跳空缺口壓力規則、缺口真封口判定規則、密集盤整區區間支撐壓力規則、均線下彎的壓力與助跌作用、高檔島型反轉規則、頭頭低底底低空頭趨勢判定
