# 大量長黑破切反轉圖

- **Rule ID**: R-CLASSIC-02
- **名稱**: 大量長黑破切反轉圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第2種（下冊 p.377-378，圖表12-4-5、12-4-6）
- **解讀**: 多頭沿 [[上升切線畫法與線不蓋線判準]] 所畫的上升切線上漲，頭部先出現 [[頭頭低底底低空頭趨勢判定]] 定義的「頭頭低」訊號（本次高點不過前高），隨後一根帶大量（甚至連續2次爆量）的長黑K直接跌破原本支撐股價的上升切線，對應 [[跌破上升切線分級與支撐壓力角色互換]]：切線角色由支撐轉為壓力。跌破切線後股價原地頭頭低小幅盤整，才正式確認空頭，可搭配 [[空頭確認爆大量與做空時機判斷]] 做最終確認。
- **可程式化**: 是
- **所需資料**: 上升切線（連接近期波段低點）、近期高點序列、當日OHLCV、成交量
- **計算公式**:
```
uptrend_line = [[上升切線畫法與線不蓋線判準]](近期低點序列)
has_lower_high = [[頭頭低底底低空頭趨勢判定]].check_first_lower_high(當前高點, 前高)
is_big_black = is_black(K) and volume >= 2 * avg_vol_5d

if has_lower_high and is_big_black and close < uptrend_line(today):
    result = [[跌破上升切線分級與支撐壓力角色互換]](uptrend_line, K)
    if result.破切等級 >= "顯著":
        signal = "空頭確認：上升切線轉壓力，[[空頭趨勢進場訊號]]"
        confirm = [[空頭確認爆大量與做空時機判斷]]
```
- **參數**: 大量倍數門檻（預設2倍5日均量）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 上升切線畫法與線不蓋線判準、跌破上升切線分級與支撐壓力角色互換、頭頭低底底低空頭趨勢判定、高檔長黑吞噬、空頭趨勢進場訊號、空頭確認爆大量與做空時機判斷
