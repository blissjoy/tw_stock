# 遇壓爆量黑K快跑圖

- **Rule ID**: R-CLASSIC-04
- **名稱**: 遇壓爆量黑K快跑圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第4種（下冊 p.380，圖表12-4-10）
- **解讀**: 股價上漲觸及前波套牢區／壓力位置，對應 [[轉折高點壓力與突破後轉支撐規則]] 或 [[頭部底部大量關卡支撐壓力轉換規則]] 所定義的壓力區。當天直接爆大量收長黑（甚至 [[高檔長黑吞噬]]），隨即出現頭頭低，構成空頭確認。與型態1、6等的差異在於本圖強調「見壓力位置＋爆量長黑」就應該獲利了結，不必等更多確認訊號，是 [[多頭高檔爆量停利訊號]] 中反應最快、最即時的版本。
- **可程式化**: 是
- **所需資料**: 近期壓力位置、當日OHLCV、成交量
- **計算公式**:
```
at_resistance = price_near([[轉折高點壓力與突破後轉支撐規則]](近期壓力位))
is_big_black = is_black(K) and volume >= 大量門檻

if at_resistance and is_big_black:
    signal = "遇壓爆量長黑，立即停利，不必等待更多確認"
    if [[頭頭低底底低空頭趨勢判定]].check_first_lower_high():
        signal += "，空頭確認強化"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 62/100 中（原書未另附獨立示意圖，僅以單一真實範例說明，型態本身邏輯與型態1、13高度共通）
- **底層規則依賴**: 轉折高點壓力與突破後轉支撐規則、頭部底部大量關卡支撐壓力轉換規則、高檔長黑吞噬、多頭高檔爆量停利訊號、頭頭低底底低空頭趨勢判定
