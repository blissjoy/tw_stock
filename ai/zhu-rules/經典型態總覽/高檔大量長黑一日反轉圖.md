# 高檔大量長黑一日反轉圖

- **Rule ID**: R-CLASSIC-01
- **名稱**: 高檔大量長黑一日反轉圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第1種（下冊 p.375-377，圖表12-4-1～12-4-4；原書12-4節「40年精華總結 熟背33種贏家圖像」15種多轉空祕笈圖之1）
- **解讀**: 全書33張精華圖之首，濃縮「多頭高檔＋爆量長黑吞噬＋一日反轉」的複合出場訊號。前提是股價處於多頭走勢高檔末端（符合 [[頭頭高底底高多頭趨勢判定]] 的多頭架構，短線急漲時獲利常已達15%以上），當日或隔日爆出天量，同時收出 [[高檔長黑吞噬]]（黑K實體完全覆蓋前一日紅K實體，帶跳空缺口者反轉力道更強）。停利SOP有兩種變化：①爆量長黑吞噬當天觀察，隔日股價跌破該黑K低點（依 [[前一日高低點支撐壓力規則]] 判定）才確認「一日反轉」全部出場；②當天已爆大量收長黑吞噬，可先賣出持股二分之一，隔日跌破黑K低點再賣出剩餘部位。此型態本質是 [[多頭高檔爆量停利訊號]] 與 [[高檔爆量三分類判讀]] 的具體實戰組合，若吞噬後股價再上漲卻不過前高，會進一步演變為 [[頭頭低底底低空頭趨勢判定]] 的頭部起點。
- **可程式化**: 是
- **所需資料**: 近期波段高低點序列（判定多頭高檔）、近期漲幅、當日/隔日OHLCV、前一日高低點
- **計算公式**:
```
is_top_zone = 頭頭高底底高多頭趨勢判定(近期波段) == "多頭末端" or 近期漲幅 >= 0.15
is_signal = [[高檔長黑吞噬]].is_pattern and 成交量分類與倍數門檻定義(當日量) in ["大量","天量"]

if is_top_zone and is_signal:
    action_A = "先賣出持股二分之一"
    if 隔日收盤價 < 黑K棒最低價:   # 依 [[前一日高低點支撐壓力規則]]
        action_B = "剩餘部位全數賣出，確認一日反轉"
    trigger = [[多頭高檔爆量停利訊號]]
    if 隔日反彈不過前高:
        note = "轉為 [[頭頭低底底低空頭趨勢判定]] 頭部起點"
```
- **參數**: 15%漲幅門檻（獲利判定基準）、天量倍數門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 頭頭高底底高多頭趨勢判定、高檔長黑吞噬、成交量分類與倍數門檻定義、前一日高低點支撐壓力規則、多頭高檔爆量停利訊號、高檔爆量三分類判讀、頭頭低底底低空頭趨勢判定
