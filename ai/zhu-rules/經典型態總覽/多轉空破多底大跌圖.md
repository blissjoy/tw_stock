# 多轉空破多底大跌圖

- **Rule ID**: R-CLASSIC-13
- **名稱**: 多轉空破多底大跌圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第13種（下冊 p.390，圖表12-4-30、12-4-31）
- **解讀**: 全書「頭底理論」的核心收尾規則：股價原本多頭排列（頭頭高、底底高，[[頭頭高底底高多頭趨勢判定]]），一旦跌破前一個重要的「多頭低點」（即上一波回檔支撐的底部），即代表多頭趨勢正式終結、轉為空頭，且往往伴隨大量出現快速大跌。此為 [[頭頭低底底低空頭趨勢判定]] 的實戰收官版本，前置流程常見於型態2「大量長黑破切反轉圖」（長黑破切→空頭確認→跌破做空）。
- **可程式化**: 是
- **所需資料**: 多頭趨勢下最後一個確認低點、當日OHLCV、成交量
- **計算公式**:
```
last_bull_low = [[頭頭高底底高多頭趨勢判定]].last_confirmed_low()

if close < last_bull_low and volume >= 大量門檻:
    signal = "跌破多頭最後低點，多頭趨勢終結，[[頭頭低底底低空頭趨勢判定]]確認，快速下跌警訊"
    trigger = [[空頭趨勢進場訊號]]
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 頭頭高底底高多頭趨勢判定、頭頭低底底低空頭趨勢判定、轉折波定義與取點演算法、跌破上升切線分級與支撐壓力角色互換、空頭趨勢進場訊號
