# 高檔大量長上影線反轉圖

- **Rule ID**: R-CLASSIC-06
- **名稱**: 高檔大量長上影線反轉圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第6種（下冊 p.381-382，圖表12-4-13、12-4-14）
- **解讀**: 高檔爆大量出現長上影線K棒（實體小、上影線長，代表當天上攻後被大量賣壓打回），屬於 [[高檔變盤線]] 家族的變化型。操作變化與型態1完全對稱：①爆大量長上影線出現後，等隔日跌破該K棒（或其大量區間）低點才賣出，構成一日反轉；②當天即可先賣出二分之一部位，隔日跌破大量低點再賣出剩餘。**規則庫覆蓋提醒**：現有 [[槌子線與倒槌K線]] 僅在形狀公式層次定義「上影線長」的倒槌線，未像 [[高檔長黑吞噬]] 一樣專門標注「高檔情境」下的反轉機率與量能門檻，本規則暫以 [[高檔變盤線]]＋量能條件組合表達。
- **可程式化**: 部分（K線形狀與量能條件可程式化，但「高檔情境下長上影線反轉機率」缺乏底層規則庫的量化基準，需依賴 [[高檔變盤線]] 的既有判準推估）
- **所需資料**: 當日OHLCV、成交量、近期波段高低點
- **計算公式**:
```
long_upper_shadow = upper_shadow(K) >= 2 * body(K) and volume(K) >= 大量門檻
is_at_top = [[頭頭高底底高多頭趨勢判定]](近期) == "高檔"

if long_upper_shadow and is_at_top:
    signal = [[高檔變盤線]] 觸發
    action_A = "先賣出二分之一部位"
    if 隔日收盤價 < K.最低價:
        action_B = "賣出剩餘部位，[[多頭高檔爆量停利訊號]]確認"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 60/100 中（K線形狀與停利SOP可回測，但「高檔長上影線」作為獨立命名型態未在既有K棒型態筆記中找到逐字對應段落，屬推論組合）
- **底層規則依賴**: 高檔變盤線、槌子線與倒槌K線、前一日高低點支撐壓力規則、多頭高檔爆量停利訊號、頭頭高底底高多頭趨勢判定、成交量分類與倍數門檻定義
