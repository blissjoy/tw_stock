# 突破大量黑K買進

- **Rule ID**: R-CLASSIC-24
- **名稱**: 突破大量黑K買進
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第9種（下冊 p.404，圖表12-4-58～60）
- **解讀**: 股價上漲過程中出現「大量黑K」（表面上是空方賣壓K棒），但只要後續股價突破該根大量黑K的高點（依 [[前一日高低點支撐壓力規則]]），黑K反而不是轉空訊號，而是續漲買進訊號。範例並標示「4線多排」（[[均線多頭排列]]）加強確認多頭格局，體現「黑K不代表看空，需視位置與後續突破而定」的觀念。
- **可程式化**: 是
- **所需資料**: 大量黑K的OHLCV、後續突破日OHLCV、均線排列狀態
- **計算公式**:
```
is_big_black_in_uptrend = is_black(K) and volume(K) >= 大量門檻 and [[均線多頭排列]].check(近期)

if is_big_black_in_uptrend:
    watch_high = K.最高價
    if close(後續K) > watch_high and volume(突破日) >= 大量門檻:
        signal = "突破大量黑K高點，非轉空、為續漲買進訊號，[[多頭趨勢進場訊號]]"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 黑K線、前一日高低點支撐壓力規則、均線多頭排列、多頭趨勢進場訊號
