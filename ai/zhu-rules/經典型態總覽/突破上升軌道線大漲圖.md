# 突破上升軌道線大漲圖

- **Rule ID**: R-CLASSIC-33
- **名稱**: 突破上升軌道線大漲圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第18種（下冊 p.415-416，圖表12-4-80～83；全書技術型態內文最後一則，p.417起為版權頁，全書終）
- **解讀**: 股價底部打底後，先以「上升切線」（[[上升切線畫法與線不蓋線判準]]）確立多頭雛形，隨後畫出與之平行的「上升軌道線」（[[上升軌道線畫法（切線平移取中間高點）]]，連接反彈高點的上緣線），當股價向上突破上升軌道線（代表漲勢從緩步盤堅轉為加速噴出），伴隨爆大量即為最強力的大漲訊號。本型態直接逐字對應既有規則 [[突破上升軌道線壓力的大量長紅K]]（非推論），為型態15「跌破下降軌道線大跌圖」的多空鏡像對稱版本，也是全書切線／軌道線家族的最終、最強勢多頭確認型態。
- **可程式化**: 是
- **所需資料**: 上升軌道線（依既有規則畫出）、當日OHLCV、成交量
- **計算公式**:
```
channel = [[上升軌道線畫法（切線平移取中間高點）]](上升切線, A, B)
is_big_red = is_red(K) and volume >= 大量門檻

if close > channel(today) and is_big_red:
    signal = [[突破上升軌道線壓力的大量長紅K]] 觸發
    note = "漲勢自緩步盤堅轉為加速噴出，全書最強力多頭訊號"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 93/100 高（原文逐字對應，非推論）
- **底層規則依賴**: 上升軌道線畫法（切線平移取中間高點）、突破上升軌道線壓力的大量長紅K、上升切線畫法與線不蓋線判準、多頭趨勢進場訊號
