# 高檔連2日大量被黑K跌破圖

- **Rule ID**: R-CLASSIC-05
- **名稱**: 高檔連2日大量被黑K跌破圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第5種（下冊 p.380-381，圖表12-4-11、12-4-12）
- **解讀**: 高檔連續2個交易日爆大量（一次「爆大量」＋一次「爆次大量」），代表主力籌碼在高檔劇烈換手，對應 [[高檔爆量三分類判讀]] 與 [[成交量分類與倍數門檻定義]]。之後只要出現一根黑K跌破這2日大量K棒的低點（依 [[大量K線支撐壓力回溯標記通用規則]] 回溯標記該區間低點），即視為「一日反轉」停利訊號，不需要等更多天數確認，與型態1「一日反轉」邏輯相同，屬同一套停利SOP的變化版本。
- **可程式化**: 是
- **所需資料**: 連續2個高檔大量交易日的OHLCV、成交量
- **計算公式**:
```
d1, d2 = 最近連續兩個高檔大量交易日(volume >= 大量門檻)
low_zone = min(d1.low, d2.low)   # 依 [[大量K線支撐壓力回溯標記通用規則]]

if any(close(後續K) < low_zone for 後續K in K_list) and is_black(該K):
    signal = "跌破高檔連2日大量低點，一日反轉停利，觸發 [[多頭高檔爆量停利訊號]]"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 高檔爆量三分類判讀、成交量分類與倍數門檻定義、大量K線支撐壓力回溯標記通用規則、前一日高低點支撐壓力規則、多頭高檔爆量停利訊號
