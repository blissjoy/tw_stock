# 反彈大量紅K跌破追空圖

- **Rule ID**: R-CLASSIC-09
- **名稱**: 反彈大量紅K跌破追空圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第9種（下冊 p.384-385，圖表12-4-19、12-4-20）
- **解讀**: 前提為股價已處於空頭下跌趨勢中（[[頭頭低底底低空頭趨勢判定]] 確認）。出現一根帶量的反彈紅K（即空頭中的逃命波／甩轎波），股價隨後跌破該反彈紅K的低點（依 [[前一日高低點支撐壓力規則]] 判定的「跌破空點」），確認空方力道未變，可加碼追空。直接對應 [[空頭反彈四大關鍵壓力放空訊號規則]] 與 [[空頭確認爆大量與做空時機判斷]] 的標準戰法。
- **可程式化**: 是
- **所需資料**: 近期空頭架構確認、反彈K棒OHLCV、成交量
- **計算公式**:
```
is_downtrend = [[頭頭低底底低空頭趨勢判定]](近期) == "空頭"
is_bounce_red = is_red(K) and volume(K) >= 平均量

if is_downtrend and is_bounce_red:
    watch_low = K.最低價
    if close(下一根K) < watch_low:
        signal = "跌破反彈紅K低點，追空點確認"
        trigger = [[空頭反彈四大關鍵壓力放空訊號規則]]
```
- **參數**: 反彈判定量能門檻
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 頭頭低底底低空頭趨勢判定、前一日高低點支撐壓力規則、空頭反彈四大關鍵壓力放空訊號規則、空頭確認爆大量與做空時機判斷、空頭短線選股與停損停利SOP
