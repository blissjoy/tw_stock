# 低檔連2日大量被突破圖

- **Rule ID**: R-CLASSIC-25
- **名稱**: 低檔連2日大量被突破圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第10種（下冊 p.405-406，圖表12-4-61、12-4-62）
- **解讀**: 低檔區間連續2天出現大量K棒（形成連2日大量高點），當股價突破這連續兩天大量K棒所形成的高點時，視為「一日反轉」式的多方轉強訊號。低檔連續大量本身即代表籌碼沉澱／換手，突破點須再度爆量確認。為型態5「高檔連2日大量被黑K跌破圖」的多空鏡像對稱版本，對應 [[大量K線支撐壓力回溯標記通用規則]] 與 [[多頭轉折關鍵大量K線操作對應]]。
- **可程式化**: 是
- **所需資料**: 連續2個低檔大量交易日的OHLCV
- **計算公式**:
```
d1, d2 = 最近連續兩個低檔大量交易日(volume >= 2*avg_vol)
high_zone = max(d1.最高價, d2.最高價)   # 依 [[大量K線支撐壓力回溯標記通用規則]]

if close(後續K) > high_zone and volume(突破日) >= 大量門檻:
    signal = "突破低檔連2日大量高點，一日反轉轉強，[[多頭轉折關鍵大量K線操作對應]]"
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 大量K線支撐壓力回溯標記通用規則、多頭轉折關鍵大量K線操作對應、成交量分類與倍數門檻定義、多頭趨勢進場訊號
