# 大量雙腳反轉圖

- **Rule ID**: R-CLASSIC-18
- **名稱**: 大量雙腳反轉圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4b-18種K線空轉多祕笈圖.md` 第3種（下冊 p.395-396，圖表12-4-41～43）
- **解讀**: 即「雙腳打底」型態——股價下跌後於低檔出現「第1支腳」（帶大量的探底K棒），反彈後再度回測出現「第2支腳」（同樣帶大量，但通常不破前低或僅小幅破低），第2支腳打底完成後突破頸線確認買點，並可用第1、2支腳的價差幅度D向上等距量測目標價。兩支腳低點都須出現大量，是判斷「探底有效」而非「持續破底」的關鍵，對應 [[密集盤整區區間支撐壓力規則]] 的W底概念，量測手法與 [[型態頸線量測目標價支撐壓力規則]] 一致。
- **可程式化**: 是
- **所需資料**: 兩支腳低點價位與成交量、頸線價位
- **計算公式**:
```
leg1, leg2 = detect_double_bottom(price_series)
both_large_vol = volume(leg1) >= 大量門檻 and volume(leg2) >= 大量門檻
neckline = [[密集盤整區區間支撐壓力規則]](between(leg1, leg2))

if both_large_vol and close > neckline:
    D = abs(leg2.低點 - neckline)
    target = [[型態頸線量測目標價支撐壓力規則]](neckline, D, 方向="向上")
    signal = "雙腳打底突破頸線，多頭確認，目標價=" + target
```
- **參數**: 大量門檻（依 [[成交量分類與倍數門檻定義]]）
- **可回測**: 是
- **信心**: 87/100 高
- **底層規則依賴**: 密集盤整區區間支撐壓力規則、型態頸線量測目標價支撐壓力規則、成交量分類與倍數門檻定義、空轉多打底量價與均線排列鎖股訊號、多頭趨勢進場訊號
