# 黑紅黑續跌圖

- **Rule ID**: R-CLASSIC-11
- **名稱**: 黑紅黑續跌圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第11種（下冊 p.386-387，圖表12-4-23、12-4-24）
- **解讀**: 三根K棒組合：黑K → 紅K（通常實體較小、未能收復前一根黑K的失土）→ 黑K，即「黑紅黑空方夾擊」。中間的紅K只是短暫止跌，未能扭轉空方力道，第三根黑K確認續跌。**規則庫覆蓋提醒**：現有K棒型態規則庫的三根K線組合集中在反轉型（[[一星二陽]]、[[一星二陰]]、[[上升三法]]、[[下降三法]]），沒有涵蓋這種「中間一根反向K被前後同向K夾擊、屬續勢而非反轉」的三日續勢組合，本規則以 [[黑K線]]／[[紅K線]] 基本定義＋空頭架構組合表達。
- **可程式化**: 部分（三日組合形狀可程式化，但「黑紅黑續勢」缺乏獨立底層K線組合規則，屬形狀層次的自訂偵測）
- **所需資料**: 連續3根K棒的開盤價、收盤價
- **計算公式**:
```
d1, d2, d3 = 最近三根K
is_black_red_black = is_black(d1) and is_red(d2) and is_black(d3)
red_fails_to_recover = d2.收盤價 < d1.開盤價   # 紅K未收復前黑失土

if is_black_red_black and red_fails_to_recover and [[頭頭低底底低空頭趨勢判定]](近期) == "空頭":
    signal = "黑紅黑空方夾擊，續跌確認"
```
- **參數**: 無額外參數（依黑紅黑三日開收盤價相對關係判定）
- **可回測**: 是
- **信心**: 35/100 低（本次比對未在既有K棒型態筆記中找到「黑紅黑」逐字命名段落，可能是財富篇新增的口訣總結，屬推論對應）
- **底層規則依賴**: 黑K線、紅K線、頭頭低底底低空頭趨勢判定、空頭確認爆大量與做空時機判斷
