# 跌破ABC下跌圖

- **Rule ID**: R-CLASSIC-14
- **名稱**: 跌破ABC下跌圖
- **分類**: 經典型態總覽
- **原文與頁碼**: `P12-C4a-15種K線多轉空祕笈圖.md` 第14種（下冊 p.391-392，圖表12-4-32～34）
- **解讀**: 股價創高後開始以ABC三波進行修正下跌（A為修正起點高點，B為拉回，C為次高不創新高，形成一段小幅下降切線）。訊號：股價跌破「上升切線」（[[上升切線畫法與線不蓋線判準]]）且跌破ABC修正的低點（空點），依 [[跌破上升切線分級與支撐壓力角色互換]] 確認空頭。停利／目標價估算以等幅測量法：用前一段漲幅D往下量測，推算下跌目標價。**規則庫覆蓋提醒**：現有 [[型態頸線量測目標價支撐壓力規則]] 是針對頭肩頂/W底頸線的量測，「用前一波段漲跌幅D等距外推」這種切線突破後的通用等幅測量法尚無獨立規則對應，本規則暫借用該規則的量測邏輯延伸表達。
- **可程式化**: 部分（ABC轉折點與跌破切線的偵測可程式化，但等幅測量目標價缺乏獨立底層規則，精確度中等）
- **所需資料**: ABC三波轉折點、前一段漲幅D、上升切線
- **計算公式**:
```
A, B, C = [[轉折波定義與取點演算法]].detect_ABC_correction(近期修正)
uptrend_line = [[上升切線畫法與線不蓋線判準]](前段低點序列)

if close < uptrend_line(today) and close < C.低點:
    D = 前一段漲幅(起漲點至A點)
    target = C.低點 - D   # 等幅測量法（借用 [[型態頸線量測目標價支撐壓力規則]] 量測邏輯）
    signal = "跌破ABC修正低點與上升切線，空頭確認，目標價=" + target
```
- **參數**: D值計算起訖點（前一段漲幅的起漲點與A點）
- **可回測**: 是
- **信心**: 58/100 中（ABC修正與切線跌破邏輯高，但等幅測量法本身未在切線章節找到逐字對應規則，屬推論延伸）
- **底層規則依賴**: 轉折波定義與取點演算法、上升切線畫法與線不蓋線判準、跌破上升切線分級與支撐壓力角色互換、型態頸線量測目標價支撐壓力規則、頭頭低底底低空頭趨勢判定
