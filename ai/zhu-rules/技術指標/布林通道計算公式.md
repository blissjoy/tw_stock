# 布林通道計算公式

- **Rule ID**: R-INDICATOR-20
- **名稱**: 布林通道計算公式
- **分類**: 技術指標
- **原文與頁碼**: P08-C5-布林通道.md，下冊 p.187-188
- **解讀**: 布林通道（Bollinger Bands，BBands，又稱布林格帶狀或保力加通道）由約翰·布林格（John Bollinger）提出，結合「移動平均線」與統計學上的「標準差」概念而成。原始參數設定：帶狀中軌＝20日移動平均線（MA20）；帶狀上軌＝中軌＋2個標準差（作為股價的壓力線）；帶狀下軌＝中軌−2個標準差（作為股價的支撐線）。換言之，布林通道就是「以MA20為中心、上下各取2倍標準差為範圍」的軌道操作方式。統計學依據：依常態分布曲線，距平均值±1個標準差之內涵蓋約68.2%的數值範圍，±2個標準差合計涵蓋約95.4%的股價出現機率，因此股價落在布林通道2倍標準差軌道之外的機率大約只有4.6%（即100%−95.4%），可作為極端乖離的統計參考。
- **可程式化**: 是
- **所需資料**: 收盤價（Close）之時間序列
- **計算公式**:
```
中軌(t) = MA(Close, 20)(t)
STD20(t) = 收盤價過去20日之母體或樣本標準差(Close[t-19..t])
上軌(t) = 中軌(t) + 2 * STD20(t)
下軌(t) = 中軌(t) - 2 * STD20(t)

# 統計參考：常態分布下，股價落於[下軌, 上軌]區間內機率約95.4%，
# 落在區間外（超出上軌或跌破下軌）機率約4.6%，屬低機率極端事件
```
- **參數**: 中軌天期=20日（MA20）、標準差計算天期=20日（與中軌同期）、標準差倍數=2倍（書中固定參數，未提及可調整版本）
- **可回測**: 是
- **信心**: 94/100 高（三線定義、參數天期、標準差倍數與統計依據書中皆明確標註）
