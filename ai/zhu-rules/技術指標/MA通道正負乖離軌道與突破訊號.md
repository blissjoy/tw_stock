# MA通道正負乖離軌道與突破訊號

- **Rule ID**: R-INDICATOR-18
- **名稱**: MA通道正負乖離軌道與突破訊號
- **分類**: 技術指標
- **原文與頁碼**: P08-C4-乖離與MA通道的運用.md，下冊 p.181-185
- **解讀**: MA通道又稱「MA軌道線」，作法是在固定一條移動平均線的上方與下方，各自設定一個乖離百分比，畫出上、下兩條「乖離線」（外包絡線），兩線之間形成一條區域軌道，用來判斷股價是接近軌道上緣的正乖離線（偏多、可能過熱），還是接近軌道下緣的負乖離線（偏空、可能過冷）。正常情況下，股價會在MA軌道內部游走；一旦股價帶量突破軌道線（上緣或下緣），代表趨勢轉強（向上突破上緣為偏多趨勢轉強，向下跌破下緣為偏空趨勢轉強）。參數設定：中心線（均線）通常以20日均線（MA20）作為研判多空方向的基準；乖離率參數設定為12%～15%，即上軌＝MA20×(1+12%~15%)，下軌＝MA20×(1−12%~15%)。書中範例圖表中實際標示的參數為UB_15.00%／LB_15.00%（即以MA20為中心，上下各平移15%），屬區間上限的常用設定。書中6檔個股範例圖顯示：軌道上下緣可作為壓力／支撐參考點（空頭下跌觸及上軌即受阻回落、觸及下軌有大量出現止跌；連續急漲伴隨連續大量、正乖離過大突破上軌時，常伴隨長上影線變盤訊號，提示過熱後可能反轉）。
- **可程式化**: 是
- **所需資料**: 收盤價（Close）、20日均線（MA20）、成交量（判斷「帶量」突破）
- **計算公式**:
```
中心線(t) = MA20(t)
上軌(t) = MA20(t) * (1 + r)    # r 建議區間 12%~15%，範例常用 15%
下軌(t) = MA20(t) * (1 - r)

if 下軌(t) <= Close(t) <= 上軌(t):
    狀態 = "軌道內游走（常態）"
elif Close(t) > 上軌(t) and 帶量(t):
    狀態 = "帶量突破上軌，偏多趨勢轉強"
    參考 = "亦可能為正乖離過大訊號，注意長上影線等變盤線出現"
elif Close(t) < 下軌(t) and 帶量(t):
    狀態 = "帶量跌破下軌，偏空趨勢轉強"

# 壓力/支撐參考（空頭格局中）
if Close(t) 觸及上軌(t) and 當下趨勢 == "空頭":
    參考 = "正乖離壓力，反彈易受阻回落"
if Close(t) 觸及下軌(t):
    參考 = "負乖離支撐，易有大量止跌"
```
- **參數**: 中心均線天期=20日（MA20）；乖離率r建議區間12%~15%（範例圖表常用15%）
- **可回測**: 是
- **信心**: 93/100 高（MA20中心線與12%~15%乖離率為書中正文明確給出的具體參數，範例圖表亦標示UB/LB 15%的實際設定值，並附6檔個股實例驗證軌道壓力/支撐與突破轉強的判讀邏輯）
