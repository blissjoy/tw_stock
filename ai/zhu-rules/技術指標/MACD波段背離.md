# MACD波段背離

- **Rule ID**: R-INDICATOR-06
- **名稱**: MACD波段背離
- **分類**: 技術指標
- **原文與頁碼**: P08-C1-MACD指標中期走勢的研判.md，下冊 p.158-159
- **解讀**: 這是「單一波段內」的背離判斷，尺度小於「[[MACD趨勢級背離]]」。①股價上漲、紅柱縮短→高檔背離，後續股價容易回檔。②股價下跌、綠柱縮短→低檔背離，後續股價容易反彈。與「[[MACD紅柱多頭動能7要點]]」第⑦點、「[[MACD綠柱空頭動能7要點]]」第⑦點的差異在於：本規則比較的是同一波段內股價續創高/創低過程中，OSC柱狀高度是否同步縮短，屬於單次波段內的動能觀察，而非跨波段的頭部/底部群組比較。
- **可程式化**: 是
- **所需資料**: 收盤價序列、OSC柱狀序列
- **計算公式**:
```
# 波段高檔背離（單一上漲波段內）
if 股價於本波段內續創高(Close(t) > max(Close[波段起點..t-1]))
   and OSC(t)紅柱高度 < OSC於本波段前次高點時的紅柱高度:
    signal = "波段高檔背離，提示回檔"

# 波段低檔背離（單一下跌波段內）
if 股價於本波段內續創低(Close(t) < min(Close[波段起點..t-1]))
   and abs(OSC(t)綠柱高度) < abs(OSC於本波段前次低點時的綠柱高度):
    signal = "波段低檔背離，提示反彈"
```
- **參數**: 「波段」的起訖點需依轉折波定義取點（可參照趨勢判定分類的轉折波規則），書中未提供獨立的波段切分演算法
- **可回測**: 部分（背離方向判定邏輯明確；波段起訖點的自動切分需額外引入轉折波演算法才能完整程式化）
- **信心**: 65/100 中（背離定義與方向明確；波段切分依賴外部轉折波規則，屬工程整合細節）
