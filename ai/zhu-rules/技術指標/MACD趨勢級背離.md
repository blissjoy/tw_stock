# MACD趨勢級背離

- **Rule ID**: R-INDICATOR-07
- **名稱**: MACD趨勢級背離
- **分類**: 技術指標
- **原文與頁碼**: P08-C1-MACD指標中期走勢的研判.md，下冊 p.160-161
- **解讀**: 這是比較「兩個以上波段的高點群／低點群」的跨波段背離，尺度大於「[[MACD波段背離]]」。①多頭「頭頭高」對應紅柱群「頭頭低」：多頭趨勢股價創新高（符合「[[頭頭高底底高多頭趨勢判定]]」的頭頭高條件），但兩波段對應的OSC紅柱群峰值卻一波比一波低（頭頭低）→屬趨勢高檔背離，後續趨勢容易由多轉空。②空頭「底底低」對應綠柱群「底底高」：空頭趨勢股價創新低（符合「[[頭頭低底底低空頭趨勢判定]]」的底底低條件），但兩波段對應的綠柱群谷值卻一波比一波淺（底底高）→屬趨勢低檔背離，後續趨勢容易由空轉多。書中特別強調兩種背離尺度（波段內 vs. 趨勢級）可先後出現於同一走勢中。
- **可程式化**: 是
- **所需資料**: 轉折波頭/底序列（同R-TREND-01輸出）、對應各波段的OSC紅柱峰值／綠柱谷值序列
- **計算公式**:
```
# 趨勢高檔背離（多頭，比較最近2個波段頭部）
# Head[-1], Head[-2] 為最近與次近的轉折頭部價格；OscPeak[-1], OscPeak[-2] 為對應波段的OSC紅柱峰值
if Head[-1] > Head[-2] and OscPeak[-1] < OscPeak[-2]:
    signal = "趨勢級高檔背離，提示多轉空"

# 趨勢低檔背離（空頭，比較最近2個波段底部）
# Bottom[-1], Bottom[-2] 為最近與次近的轉折底部價格；OscTrough[-1], OscTrough[-2] 為對應波段的OSC綠柱谷值（絕對值）
if Bottom[-1] < Bottom[-2] and abs(OscTrough[-1]) < abs(OscTrough[-2]):
    signal = "趨勢級低檔背離，提示空轉多"
```
- **參數**: 無額外可調參數；依賴上游轉折波取點演算法（均線天期5/10/20日，見均線分類規則）決定頭/底位置
- **可回測**: 是
- **信心**: 91/100 高（書中明確以「兩波段頭頭高/底底低對比OSC柱峰谷頭頭低/底底高」定義趨勢級背離，並附綜合應用範例圖表說明可與波段背離先後出現）
