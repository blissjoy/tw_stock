# KD不同週期依交易期程選用

- **Rule ID**: R-INDICATOR-10
- **名稱**: KD不同週期依交易期程選用
- **分類**: 技術指標
- **原文與頁碼**: P08-C2-反映股價變化的KD指標.md，下冊 p.166
- **解讀**: 依交易期程選用對應週期的KD：做短線進出，看日線KD操作；中期走勢進出，看週線KD操作；長期走勢進出，看月線KD操作。書中以台積電(2330)週線圖為例，展示週線KD的多次黃金交叉／死亡交叉可作為波段買賣參考點，呼應「中期走勢看週線KD」的原則。此規則本身不改變KD計算公式（見「[[KD計算公式（RSV、K、D）]]」），僅是將同一套計算公式套用在不同時間週期（日/週/月）的K線資料上。
- **可程式化**: 是
- **所需資料**: 日線、週線、月線之最高價/最低價/收盤價序列
- **計算公式**:
```
if 交易期程 == "短線":
    使用資料週期 = "日線"
elif 交易期程 == "中期":
    使用資料週期 = "週線"
elif 交易期程 == "長期":
    使用資料週期 = "月線"

# KD計算公式不變，僅替換輸入的K線週期資料（見 R-INDICATOR-08）
```
- **參數**: 無獨立參數，依賴KD計算公式本身的天期參數（常用9,3,3）套用於不同週期K線
- **可回測**: 是
- **信心**: 91/100 高（書中原文明確條列日/週/月線對應短/中/長期的選用原則，並附週線KD實例圖表）
