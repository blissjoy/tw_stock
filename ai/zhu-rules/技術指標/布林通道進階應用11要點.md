# 布林通道進階應用11要點

- **Rule ID**: R-INDICATOR-24
- **名稱**: 布林通道進階應用11要點
- **分類**: 技術指標
- **原文與頁碼**: P08-C5-布林通道.md，下冊 p.192-196
- **解讀**: 本節標題明確結合「趨勢型態＋布林通道＋MA5＋K線」四項工具交叉判讀，是與「[[布林通道4大買進訊號]]」「[[布林通道4大做空訊號]]」（8個使用原則）不同的另一組獨立編號11條要點，聚焦於「通道形狀變化」（收斂/開口/三軌方向）的中期判讀。①多頭股票上漲，布林上軌由走平轉為揚升張口、K線沿上軌上漲時，是短波段獲利段，可持股續抱，直到出現一根跌破MA5的黑K才停利出場。②多頭趨勢不變時，布林上軌、中軌保持向上，股價維持在上軌之上，代表多頭持續上漲。③上軌上揚一段後開始走平或下彎，代表多頭進入盤整或急速回檔階段；若回檔跌破中軌，則容易進一步觸及下軌。④布林通道上漲或下跌一波之後開口會收斂，進入橫盤整理，K線在上下軌之間來回震盪；當出現大量紅K突破盤整，布林通道會重新打開成「喇叭型」開口，股價沿上軌加速上漲，此時應把握進場做多。⑤與④對稱：通道收斂橫盤後，若出現大量黑K跌破盤整，布林通道同樣會打開成喇叭型開口，股價沿下軌加速下跌，此時應把握進場做空。⑥當上、中、下三軌同時向上運行時，代表價格強勢特徵非常明顯，短期內將延續上漲。⑦當上、中、下三軌同時向下運行時，代表價格弱勢特徵非常明顯，短期內將延續下跌。⑧上軌向上運行、而中軌下軌卻同時向下運行的組合，出現機率非常小（書中僅點出此組合罕見，作補充說明）。⑨當上軌向下運行、但中軌與下軌仍在向上運行時（通道整體縮小），代表價格開始回檔或漸趨整理，須依所處長期趨勢分別判讀：長期上升趨勢中屬回檔途中的強勢整理，可持股觀望或逢回後短線買進；長期下跌趨勢中屬下跌途中的弱勢整理，宜持空單續抱或反彈時做空。⑩當三軌幾乎同時呈水平橫向運行時，方向不明，須視當下走勢情境判斷，此時最好的作法是退出觀望、等待通道重新開口突破後再進場。⑪實際操作時，仍應以波浪型態（趨勢方向）與個人交易紀律為主要判斷主軸，再搭配布林通道輔助確認，布林通道應作為輔助工具而非唯一依據。
- **可程式化**: 部分
- **所需資料**: 收盤價/開盤價/最高/最低價（K線）、MA5、布林通道上/中/下軌序列（見「[[布林通道計算公式]]」）、成交量、長期趨勢判定
- **計算公式**:
```
上軌(t), 中軌(t), 下軌(t) = 見 R-INDICATOR-20
上軌斜率(t) = 上軌(t) - 上軌(t-1)   # 同理可算中軌斜率、下軌斜率

# ①持股續抱直到跌破MA5停利
if 上軌斜率(t) > 0（由走平轉揚升） and Close(t) 沿上軌上漲:
    持股條件 = True
if Close(t) < MA5(t) and K線收黑:
    signal = "跌破MA5黑K，停利出場"

# ③上軌走平/下彎，回檔跌破中軌
if 上軌斜率(t) <= 0（走平或下彎）:
    警示 = "多頭進入盤整或急速回檔階段"
    if Close(t) < 中軌(t):
        警示 += "；跌破中軌，容易進一步觸及下軌"

# ④⑤通道收斂後帶量突破/跌破
if 通道寬度(t) = 上軌(t)-下軌(t) 持續收斂 and 出現帶大量紅K突破盤整區間:
    signal = "通道重新開口向上（喇叭型），做多進場訊號"
if 通道寬度(t) 持續收斂 and 出現帶大量黑K跌破盤整區間:
    signal = "通道重新開口向下（喇叭型），做空進場訊號"

# ⑥⑦三軌同向
if 上軌斜率(t) > 0 and 中軌斜率(t) > 0 and 下軌斜率(t) > 0:
    signal = "三軌同時向上，強勢多頭延續"
if 上軌斜率(t) < 0 and 中軌斜率(t) < 0 and 下軌斜率(t) < 0:
    signal = "三軌同時向下，強勢空頭延續"

# ⑧罕見組合（補充說明，非交易訊號）
if 上軌斜率(t) > 0 and 中軌斜率(t) < 0 and 下軌斜率(t) < 0:
    備註 = "此組合出現機率非常小"

# ⑨上軌轉下但中下軌仍向上（通道縮小）
if 上軌斜率(t) < 0 and 中軌斜率(t) > 0 and 下軌斜率(t) > 0:
    if 長期趨勢 == "多頭":
        signal = "強勢整理，可持股觀望或逢低短多"
    elif 長期趨勢 == "空頭":
        signal = "弱勢整理，宜持空單續抱或反彈續空"

# ⑩三軌走平
if abs(上軌斜率(t)) ≈ 0 and abs(中軌斜率(t)) ≈ 0 and abs(下軌斜率(t)) ≈ 0:
    signal = "方向不明，建議觀望等待通道開口突破"
```
