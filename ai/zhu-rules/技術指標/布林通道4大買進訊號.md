# 布林通道4大買進訊號

- **Rule ID**: R-INDICATOR-22
- **名稱**: 布林通道4大買進訊號
- **分類**: 技術指標
- **原文與頁碼**: P08-C5-布林通道.md，下冊 p.189
- **解讀**: 本規則對應書中明確標題「布林通道的8個使用原則」中「一、4個買進訊號」的完整4條，與「[[布林通道4大做空訊號]]」合稱書中的「8個使用原則」，全部8條均已分別完整轉為規則、不省略。①空頭下跌至低檔，價格由下往上穿越下軌時，是買進訊號（搶空頭反彈）。②多頭上漲，價格由下往上穿越中軌時，是買進訊號（多頭回檔跌破中軌後，再站上中軌）。③價格在中軌與上軌之間向上運行，為多頭市場，持續做多或長線抱股。④布林3軌呈現平行（盤整）時，股價突破盤整區，做多買進。此4條為「穿越軌道/中軌/盤整區」的進場邏輯，與「[[布林通道觸及上下軌出場回補訊號]]」的「觸及軌道即出場/回補」持倉了結邏輯互為對照，構成完整策略框架。
- **可程式化**: 是
- **所需資料**: 收盤價（Close）、布林通道上/中/下軌序列（見「[[布林通道計算公式]]」）、當下趨勢判定（多頭/空頭/盤整）
- **計算公式**:
```
上軌(t), 中軌(t), 下軌(t) = 見 R-INDICATOR-20

# ①空頭低檔，價格由下向上穿越下軌 → 搶反彈買進
if 當下趨勢 == "空頭" and Close(t-1) <= 下軌(t-1) and Close(t) > 下軌(t):
    signal = "搶空頭反彈買進訊號"

# ②多頭回檔跌破中軌後，再站上中軌 → 買進
if 當下趨勢 == "多頭" and EXISTS 近期日d: Close(d) < 中軌(d) and Close(t-1) <= 中軌(t-1) and Close(t) > 中軌(t):
    signal = "多頭回檔後重新站上中軌，買進訊號"

# ③價格於中軌與上軌之間向上運行 → 多頭市場，持有/續抱
if 中軌(t) <= Close(t) <= 上軌(t) and Close(t) > Close(t-1):
    signal = "多頭市場，持續做多或長線抱股"

# ④三軌走平（盤整）+ 價格向上突破盤整區 → 買進
if 三軌斜率皆接近0（走平） and Close(t) > 盤整區間上緣:
    signal = "盤整突破，做多買進訊號"
```
- **參數**: 依賴布林通道計算公式參數（MA20、2倍標準差）；「三軌走平」之斜率容許誤差書中未量化，需自訂
- **可回測**: 部分（①②③條件明確可回測；④「三軌平行/走平」與「盤整區」邊界需自訂量化斜率門檻，可參照趨勢判定分類的盤整規則）
- **信心**: 89/100 高（4條買進訊號書中逐條明確列出，僅④的「走平」與「盤整區突破」邊界需搭配盤整判定規則量化）
