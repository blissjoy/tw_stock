# 布林通道4大做空訊號

- **Rule ID**: R-INDICATOR-23
- **名稱**: 布林通道4大做空訊號
- **分類**: 技術指標
- **原文與頁碼**: P08-C5-布林通道.md，下冊 p.189
- **解讀**: 本規則對應書中明確標題「布林通道的8個使用原則」中「二、4個做空訊號」的完整4條，與「[[布林通道4大買進訊號]]」合稱書中的「8個使用原則」，全部8條均已分別完整轉為規則、不省略。①價格由上往下穿越上軌時，是賣出訊號（搶多頭回檔）。②價格在中軌與上軌之間，由上往下跌破中軌，為賣出訊號（空頭反彈突破中軌後，再跌回中軌之下）。③價格在中軌與下軌之間運行，為空頭市場，持續做空。④布林3軌呈現平行（盤整）時，股價跌破盤整區，做空賣出。此4條與「[[布林通道4大買進訊號]]」互為多空對稱結構。
- **可程式化**: 是
- **所需資料**: 收盤價（Close）、布林通道上/中/下軌序列（見「[[布林通道計算公式]]」）、當下趨勢判定（多頭/空頭/盤整）
- **計算公式**:
```
上軌(t), 中軌(t), 下軌(t) = 見 R-INDICATOR-20

# ①多頭高檔，價格由上向下穿越上軌 → 搶回檔賣出/放空
if 當下趨勢 == "多頭" and Close(t-1) >= 上軌(t-1) and Close(t) < 上軌(t):
    signal = "搶多頭回檔賣出/放空訊號"

# ②空頭反彈突破中軌後，再跌回中軌之下 → 賣出/放空
if 當下趨勢 == "空頭" and EXISTS 近期日d: Close(d) > 中軌(d) and Close(t-1) >= 中軌(t-1) and Close(t) < 中軌(t):
    signal = "空頭反彈後跌回中軌下方，賣出/放空訊號"

# ③價格於中軌與下軌之間向下運行 → 空頭市場，持有空單
if 下軌(t) <= Close(t) <= 中軌(t) and Close(t) < Close(t-1):
    signal = "空頭市場，持續做空"

# ④三軌走平（盤整）+ 價格向下跌破盤整區 → 放空
if 三軌斜率皆接近0（走平） and Close(t) < 盤整區間下緣:
    signal = "盤整跌破，放空賣出訊號"
```
- **參數**: 依賴布林通道計算公式參數（MA20、2倍標準差）；「三軌走平」之斜率容許誤差書中未量化，需自訂
- **可回測**: 部分（①②③條件明確可回測；④「三軌平行/走平」與「盤整區」邊界需自訂量化斜率門檻，可參照趨勢判定分類的盤整規則）
- **信心**: 89/100 高（4條做空訊號書中逐條明確列出，僅④的「走平」與「盤整區跌破」邊界需搭配盤整判定規則量化）
