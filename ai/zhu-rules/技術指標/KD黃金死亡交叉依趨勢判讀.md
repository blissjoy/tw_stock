# KD黃金死亡交叉依趨勢判讀

- **Rule ID**: R-INDICATOR-09
- **名稱**: KD黃金死亡交叉依趨勢判讀
- **分類**: 技術指標
- **原文與頁碼**: P08-C2-反映股價變化的KD指標.md，下冊 p.164-165
- **解讀**: 這是KD指標應用的核心規則。K、D兩線上下波動時會產生交叉訊號：K線由下往上穿越D線＝黃金交叉；K線由上往下穿越D線＝死亡交叉。利用KD交叉判斷行情走勢時最容易誤判，原因是股價當下處於多頭、空頭或盤整這三種不同趨勢時，同一個交叉訊號的判讀方式並不一樣；因此使用前要先分辨當下股價處於哪一種趨勢（可用「[[頭頭高底底高多頭趨勢判定]]」「[[頭頭低底底低空頭趨勢判定]]」「[[盤整趨勢判定與型態]]」判定），才不會誤解讀。①股價走多頭：KD黃金交叉是參考買點，死亡交叉是多單參考的賣出點。②股價走空頭：KD死亡交叉是參考空點，黃金交叉是空單參考的回補點。③股價在盤整：KD死亡交叉與黃金交叉都不能進場（訊號無效，不宜依交叉訊號進出）。
- **可程式化**: 是
- **所需資料**: K線序列、D線序列、當下趨勢判定結果（多頭/空頭/盤整，來自趨勢判定分類規則）
- **計算公式**:
```
golden_cross(t) = K(t-1) <= D(t-1) and K(t) > D(t)
dead_cross(t)   = K(t-1) >= D(t-1) and K(t) < D(t)

if trend == "多頭":
    if golden_cross(t): signal = "參考買點"
    if dead_cross(t):   signal = "多單參考賣出點"
elif trend == "空頭":
    if dead_cross(t):   signal = "參考空點"
    if golden_cross(t): signal = "空單參考回補點"
elif trend == "盤整":
    signal = "訊號無效，不宜依KD交叉進出"
```
- **參數**: 依賴KD計算天期（見「[[KD計算公式（RSV、K、D）]]」，常用9,3,3）；趨勢判定依賴外部趨勢分類規則
- **可回測**: 是
- **信心**: 92/100 高（書中明確列出三種趨勢下交叉訊號的對應意義，並以2張範例圖表分別驗證多頭黃金交叉買進、多頭死亡交叉賣出的實例）
