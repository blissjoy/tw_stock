# MACD綠柱空頭動能7要點

- **Rule ID**: R-INDICATOR-05
- **名稱**: MACD綠柱空頭動能7要點
- **分類**: 技術指標
- **原文與頁碼**: P08-C1-MACD指標中期走勢的研判.md，下冊 p.157
- **解讀**: 與「[[MACD紅柱多頭動能7要點]]」對稱，書中針對空頭趨勢下OSC綠柱（負值柱狀體）列出7項判讀規則：①綠柱縮短→空頭反彈；②綠柱漸長→跌勢持續，空單續抱；③綠柱漸長但股價不跌或上漲→動能背離，空單準備回補；④綠柱由縮短再轉增長→跌勢再起，反彈後可再放空；⑤紅柱縮短並接近0軸、之後轉為綠柱→股價容易再跌（偏空訊號）；⑥綠柱縮短並接近0軸、之後轉為紅柱→股價容易續反彈（偏多訊號）；⑦低檔背離：空頭趨勢處於低檔，股價創新低（1～3次），但綠柱未同步創新低→稱為低檔背離，股價打底機會很大。
- **可程式化**: 是
- **所需資料**: OSC柱狀序列、收盤價序列、空頭格局判定（可用「[[頭頭低底底低空頭趨勢判定]]」或DIF/MACD皆在0軸下作為前提）
- **計算公式**:
```
# 前提：處於空頭格局
# ①綠柱縮短（絕對值變小）→ 反彈
if OSC(t) < 0 and abs(OSC(t)) < abs(OSC(t-1)):
    signal = "空頭反彈訊號"

# ②綠柱漸長（絕對值變大）→ 續抱空單
if OSC(t) < 0 and abs(OSC(t)) > abs(OSC(t-1)):
    signal = "跌勢持續，續抱空單"

# ③綠柱漸長但股價未創新低或上漲 → 動能背離
if OSC(t) < 0 and abs(OSC(t)) > abs(OSC(t-1)) and Close(t) >= min(Close[t-N..t-1]):
    signal = "動能背離，提示空單回補"

# ④綠柱縮短後轉為遞增（絕對值再擴大）→ 放空（彈後續跌）
if 連續數日abs(OSC)遞減後 abs(OSC(t)) > abs(OSC(t-1)) and OSC(t) < 0:
    signal = "放空訊號（彈後續跌）"

# ⑤紅柱縮短貼近0軸後翻綠 → 偏空
if OSC(t-1) > 0 and OSC(t-1) 持續縮小 and OSC(t) < 0:
    signal = "偏空訊號，股價易再跌"

# ⑥綠柱縮短貼近0軸後翻紅 → 偏多
if OSC(t-1) < 0 and abs(OSC(t-1)) 持續縮小 and OSC(t) > 0:
    signal = "偏多訊號，股價易續反彈"

# ⑦低檔背離
if 股價於近期創新低(1~3次) and 對應OSC綠柱谷值(絕對值)未同步創新低:
    signal = "低檔背離，打底機會升高"
```
- **參數**: 「創新低1~3次」為書中經驗描述，非固定數字門檻，程式化時需自訂比較窗口N
- **參考來源查證（2026-07-23，工程補充）**：與R-INDICATOR-04同批查證，MACD標準背離定義僅為單次新低比較（來源：[QuantPass MACD教學](https://quantpass.org/macd-2/)），查無「比較窗口應涵蓋幾次新低」的公認統一數字，N需自訂為工程估計值。
- **可回測**: 部分（①②④⑤⑥條件明確可回測；③⑦涉及經驗性描述，需自訂量化門檻後方可完整回測）
- **信心**: 62/100 中（規則邏輯與方向書中明確列出7條，惟部分規則缺乏精確數字定義）
