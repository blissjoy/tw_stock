# KD計算公式（RSV、K、D）

- **Rule ID**: R-INDICATOR-08
- **名稱**: KD計算公式（RSV、K、D）
- **分類**: 技術指標
- **原文與頁碼**: P08-C2-反映股價變化的KD指標.md，下冊 p.163
- **解讀**: KD指標是結合RSI、均線、量能等優點改良而成的指標，研究「高低價位」與「收盤價」的關係，反映價格走勢的強弱及超買、超賣現象。KD可選擇任何日數作為計算基礎，計算出來的數字一律介於0～100之間；K線為快速線，D線為慢速線。書中以5日為例示範公式：K值＝100×（C−L5）／（H5−L5）；D值＝100×（H3／L3），其中H3為最後3個「（C−Ln）」數值的總和，L3為最後3個「（Hn−Ln）」數值的總和。換言之K值即RSV（Raw Stock Value）的概念：當日收盤價在最近N日價格區間中所處的相對位置；D值則是把最近3天的「C−Ln」總和除以最近3天的「Hn−Ln」總和再乘以100，用3天加總的方式取代單日RSV，達到平滑效果。通常以最近9日的最高價、最低價及本日收盤價為計算基礎（對應常見的KD(9,3,3)參數）；若做短線交易，可採用5日作為計算基礎（5日恰為1週的交易時間）。書中5日範例的D值公式為「3天加總後相除」的簡化式，與業界常見的「RSV先算出後再取移動平均」（如RSV(t)=100×(C-Ln)/(Hn-Ln)，K(t)=2/3*K(t-1)+1/3*RSV(t)，D(t)=2/3*D(t-1)+1/3*K(t)的平滑移動平均法）並不完全相同，屬書中簡化例題的計算方式，程式化時可依需求選擇「書中3日加總簡化式」或「業界標準平滑移動平均式」，兩者數值會有差異，此點書中未特別區辨，屬工程實作補充。
- **可程式化**: 是
- **所需資料**: 每日最高價（High）、最低價（Low）、收盤價（Close）之時間序列
- **計算公式**:
```
# 書中5日範例公式（3日加總簡化式）
RSV(t) = (Close(t) - Ln(t)) / (Hn(t) - Ln(t)) * 100    # Ln, Hn 為最近N日內最低價/最高價
K(t) = 100 * (Close(t) - L5(t)) / (H5(t) - L5(t))        # 即當日RSV，N=5

H3 = SUM_{i=t-2..t}( Close(i) - L5(i) )
L3 = SUM_{i=t-2..t}( H5(i) - L5(i) )
D(t) = 100 * H3 / L3

# 常用參數：KD(9,3,3)，將上式中的 N=5 換成 N=9 即可
# 業界標準平滑移動平均式（補充，非書中原文）：
# RSV(t) = (Close(t) - Ln(t)) / (Hn(t) - Ln(t)) * 100
# K(t) = (2/3) * K(t-1) + (1/3) * RSV(t)
# D(t) = (2/3) * D(t-1) + (1/3) * K(t)
```
- **參數**: N（計算基礎天數）常用9日（KD(9,3,3)）；短線交易可用5日；D值平滑天數固定3日
- **可回測**: 是
- **信心**: 60/100 中（K值/RSV定義與常用參數9日、5日書中明確；但D值公式書中僅給出5日範例的3日加總簡化式，與業界標準平滑移動平均算法不同，程式化時須明確選定其一，此差異書中未討論，故信心評為中）
