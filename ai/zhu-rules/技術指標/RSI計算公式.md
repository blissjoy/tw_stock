# RSI計算公式

- **Rule ID**: R-INDICATOR-13
- **名稱**: RSI計算公式
- **分類**: 技術指標
- **原文與頁碼**: P08-C3-相對強弱變化的RSI指標.md，下冊 p.173
- **解讀**: RSI強弱指標由威爾德（Welles Wilder）創建，威爾德也是「亞當理論」的創立者。RSI用於衡量資產價格的相對強弱，判斷市場的超買或超賣狀態，幫助識別潛在的反轉點。RSI數值範圍固定介於0～100之間。當「平均漲幅」與「平均跌幅」剛好相等時，RSI＝50；因此RSI大於50向上移動代表股價往上轉強，RSI小於50向下移動代表股價往下轉弱。書中例題明確示範：「週期日上漲值平均數」＝設定天期N天內每日上漲幅度加總後除以N（不是只除以上漲天數）；「週期日下跌值平均數」則是N天內每日下跌幅度（取絕對值）加總後除以N（不是只除以下跌天數）。這是Wilder原始簡單平均算法，也是計算RSI平均漲跌幅時最容易寫錯的細節，程式化時務必以「統一除以週期天數N」為準，而非分別除以各自的漲跌天數次數。
- **可程式化**: 是
- **所需資料**: 每日收盤價（Close）之時間序列
- **計算公式**:
```
# N為週期天數（例題示範N=9）
up_i   = max(Close(i) - Close(i-1), 0)     # 第i日上漲幅度，若下跌則為0
down_i = max(Close(i-1) - Close(i), 0)     # 第i日下跌幅度（取絕對值），若上漲則為0

avg_up(t)   = SUM(up_i   for i in [t-N+1 .. t]) / N     # 除以N，非除以上漲天數
avg_down(t) = SUM(down_i for i in [t-N+1 .. t]) / N     # 除以N，非除以下跌天數

RSI(t) = avg_up(t) / (avg_up(t) + avg_down(t)) * 100

# 例題驗證（N=9，收盤價 30,32,35,32,29,30,32,35,34）：
# 上漲值加總=2+3+1+2+3=11，avg_up=11/9=1.22
# 下跌值加總=3+3+1=7，avg_down=7/9=0.77
# RSI = 1.22/(1.22+0.77)*100 = 61
```
- **參數**: 週期天數N，例題示範N=9；書中「判讀」「應用」段落實際使用N=6與N=12（短週期/長週期交叉判斷），背離範例圖同時採用RSI 6與RSI 12兩條不同天期的線
- **可回測**: 是
- **信心**: 93/100 高（書中以完整例題逐步演算，明確驗證「除以週期天數N而非漲跌天數」此一關鍵細節）
