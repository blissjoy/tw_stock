# RSI背離判斷（頭部底部背離）

- **Rule ID**: R-INDICATOR-16
- **名稱**: RSI背離判斷（頭部底部背離）
- **分類**: 技術指標
- **原文與頁碼**: P08-C3-相對強弱變化的RSI指標.md，下冊 p.175-177
- **解讀**: ①頭部（空頭）背離：股價走勢創新高、一底比一底高（頭部持續墊高，對應「[[頭頭高底底高多頭趨勢判定]]」的頭頭高型態），但RSI指標卻同期間走低（對應KD指標背離定義的「頭頭低」型態）。②底部（多頭）背離：股價走勢創新低、一底比一底低（對應「[[頭頭低底底低空頭趨勢判定]]」的底底低型態），但RSI指標卻同期間走高（對應「底底高」型態）。書中特別指出背離可重複出現（第1次背離、第2次背離），且訊號會隨背離次數增加而增強；範例圖表更展示連續兩次底背離後才展開反轉的實例。此定義與8-2節「[[KD背離判斷（峰背離、底背離）]]」定義結構一致，可視為同一套「量價背離」框架套用在不同指標上。書中另指出：RSI背離訊號常需搭配價格結構的「多頭確認」／「空頭確認」（即價格是否實際突破前高、跌破前低或關鍵頸線）作為進場觸發條件，單純背離本身僅是預警訊號，不宜單獨作為進出場依據。
- **可程式化**: 是
- **所需資料**: 轉折波頭/底序列（價格）、對應各波段的RSI峰值/谷值序列（通常RSI6、RSI12皆可觀察）
- **計算公式**:
```
# 頭部（空頭）背離：Head[-1], Head[-2] 為最近與次近的價格轉折頭部；RsiPeak[-1], RsiPeak[-2] 為對應的RSI峰值
if Head[-1] > Head[-2] and RsiPeak[-1] < RsiPeak[-2]:
    背離次數 += 1
    signal = f"RSI頭部背離（第{背離次數}次），預警訊號，需搭配價格是否跌破前低/頸線形成空頭確認才可進場"

# 底部（多頭）背離：Bottom[-1], Bottom[-2] 為最近與次近的價格轉折底部；RsiTrough[-1], RsiTrough[-2] 為對應的RSI谷值
if Bottom[-1] < Bottom[-2] and RsiTrough[-1] > RsiTrough[-2]:
    背離次數 += 1
    signal = f"RSI底部背離（第{背離次數}次），預警訊號，需搭配價格是否突破前高/頸線形成多頭確認才可進場"

# 訊號強化：背離次數越多，訊號強度越高（書中僅定性描述，未給量化加權公式）
```
- **參數**: 無固定數字參數；背離次數為累加計數，書中未提供「次數達幾次即應進場」的量化門檻
- **參考來源查證（2026-07-23，工程補充）**：查證外部資料，RSI/MACD背離文獻皆強調背離訊號本身「並非100%準確、可能連續背離而不反轉（背了又背）」（來源：[HyperCapital指標背離教學](https://hypercapital.blog/indicator-divergence-guide/)），佐證書中「訊號隨次數增強」屬合理方向，但**查無「累計幾次背離即應進場」的公認統一量化門檻**，此參數維持定性描述，程式化時需自訂或改以「背離+價格結構確認」複合條件替代單純計數。
- **可回測**: 部分（背離方向與次數計數可程式化；「訊號隨次數增強」及「需搭配多頭/空頭確認才進場」屬定性描述，須自訂量化規則（如結合突破前高/跌破前低的價格確認條件）才能完整回測）
- **信心**: 63/100 中（背離定義本身明確且與KD背離框架一致＝高；但「訊號增強程度」與「確認條件」缺乏量化標準，故整體信心評為中）
