# MACD紅柱多頭動能7要點

- **Rule ID**: R-INDICATOR-04
- **名稱**: MACD紅柱多頭動能7要點
- **分類**: 技術指標
- **原文與頁碼**: P08-C1-MACD指標中期走勢的研判.md，下冊 p.155-156
- **解讀**: 書中針對多頭趨勢下OSC紅柱（正值柱狀體）長度變化與股價關係，列出7項判讀規則：①紅柱縮短→多頭回檔；②紅柱漸長→漲勢持續，多單續抱；③紅柱漸長但股價不漲或下跌→動能與價格背離，多單準備賣出；④紅柱由縮短再轉增長→漲勢再起，可回檔後買上漲；⑤綠柱縮短並接近0軸、之後轉為紅柱→股價容易再漲（偏多訊號）；⑥紅柱縮短並接近0軸、之後轉為綠柱→股價容易續回檔（偏空訊號）；⑦高檔背離：多頭趨勢處於高檔，股價創新高（1～3次），但紅柱未同步創新高→稱為高檔背離，股價做頭機會很大。
- **可程式化**: 是
- **所需資料**: OSC柱狀序列、收盤價序列、多頭格局判定（可用「[[頭頭高底底高多頭趨勢判定]]」或DIF/MACD皆在0軸上作為前提）
- **計算公式**:
```
# 前提：處於多頭格局
# ①紅柱縮短 → 回檔
if OSC(t) > 0 and OSC(t) < OSC(t-1):
    signal = "多頭回檔訊號"

# ②紅柱漸長 → 續抱
if OSC(t) > 0 and OSC(t) > OSC(t-1):
    signal = "漲勢持續，續抱多單"

# ③紅柱漸長但股價未創新高或下跌 → 動能背離
if OSC(t) > 0 and OSC(t) > OSC(t-1) and Close(t) <= max(Close[t-N..t-1]):
    signal = "動能背離，提示多單賣出"

# ④紅柱縮短後轉為遞增 → 買進（回後上漲）
if 連續數日OSC遞減後 OSC(t) > OSC(t-1) and OSC(t) > 0:
    signal = "買進訊號（回後上漲）"

# ⑤綠柱縮短貼近0軸後翻紅 → 偏多
if OSC(t-1) < 0 and abs(OSC(t-1)) 持續縮小 and OSC(t) > 0:
    signal = "偏多訊號，股價易再漲"

# ⑥紅柱縮短貼近0軸後翻綠 → 偏空
if OSC(t-1) > 0 and OSC(t-1) 持續縮小 and OSC(t) < 0:
    signal = "偏空訊號，股價易續跌/回檔"

# ⑦高檔背離
if 股價於近期創新高(1~3次) and 對應OSC紅柱峰值未同步創新高:
    signal = "高檔背離，做頭風險升高"
```
- **參數**: 「創新高1~3次」為書中經驗描述，非固定數字門檻，程式化時可設定為「近N個波段高點」的比較窗口（N需自訂，書中未量化）
- **參考來源查證（2026-07-23，工程補充）**：查證外部技術分析資料，MACD高檔背離的標準定義僅為「股價創新高但OSC柱狀體未同步創新高」的單次比較（來源：[QuantPass MACD教學](https://quantpass.org/macd-2/)），**查無「比較窗口應涵蓋幾次新高」的公認統一數字**，書中「1~3次」屬原書經驗描述，程式化時N需自訂為工程估計值，非外部佐證數字。
- **可回測**: 部分（①②④⑤⑥條件明確可回測；③⑦涉及「創新高次數」「峰值比較窗口」等經驗性描述，需自訂量化門檻後方可完整回測）
- **信心**: 62/100 中（規則邏輯與方向書中明確列出7條，但部分規則如「創新高1~3次」缺乏精確數字定義，回測時需自行補充參數）
