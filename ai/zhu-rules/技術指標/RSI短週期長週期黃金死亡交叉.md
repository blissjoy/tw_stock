# RSI短週期長週期黃金死亡交叉

- **Rule ID**: R-INDICATOR-15
- **名稱**: RSI短週期長週期黃金死亡交叉
- **分類**: 技術指標
- **原文與頁碼**: P08-C3-相對強弱變化的RSI指標.md，下冊 p.174-175
- **解讀**: 通常短週期的RSI指標變動比較大（比較靈敏），長週期的RSI指標變動比較小（比較遲鈍），因此2條線會出現交叉、排列、背離等現象。①「短週期的RSI」向上突破「長週期的RSI」是黃金交叉，代表向上力道轉強，是參考買進的訊號。②「短週期的RSI」向下跌破「長週期的RSI」是死亡交叉，代表向下力道轉強，是參考賣出的訊號。書中在「應用」段落將此規則具體化為：RSI（6日、12日）黃金交叉，是多頭上漲買進參考訊號；RSI（6日、12日）死亡交叉，是空頭下跌做空參考訊號。此規則與均線分類中的「[[黃金交叉]]」「[[死亡交叉]]」概念相通，差異在於這裡交叉的兩條線是短、長週期的RSI，而非兩條均線。
- **可程式化**: 是
- **所需資料**: 短週期RSI序列（RSI6）、長週期RSI序列（RSI12）
- **計算公式**:
```
RSI_short(t) = RSI(6, t)     # 短週期RSI，見 R-INDICATOR-13
RSI_long(t)  = RSI(12, t)    # 長週期RSI

golden_cross(t) = RSI_short(t-1) <= RSI_long(t-1) and RSI_short(t) > RSI_long(t)
dead_cross(t)   = RSI_short(t-1) >= RSI_long(t-1) and RSI_short(t) < RSI_long(t)

if golden_cross(t):
    signal = "多頭上漲買進參考訊號"
if dead_cross(t):
    signal = "空頭下跌做空參考訊號"
```
- **參數**: 短週期=6日、長週期=12日（書中固定參數）
- **可回測**: 是
- **信心**: 91/100 高（書中「判讀」與「應用」兩段落一致明確給出6日、12日參數與黃金/死亡交叉之買賣意義）
