# 乖離率應用原則

- **Rule ID**: R-INDICATOR-19
- **名稱**: 乖離率應用原則
- **分類**: 技術指標
- **原文與頁碼**: P08-C4-乖離與MA通道的運用.md，下冊 p.180
- **解讀**: 書中就乖離率的實務運用列出3項原則：①乖離率不能單獨使用，必須搭配趨勢研判。②股價遠離均線越遠，拉回修正的機率越高；因此當乖離過大時，應避免追高買進或殺低賣出，宜等股價回檔／修正乖離後，再檢視是否符合原本的進場條件操作；乖離過大時也可留意是否出現可搶短的交易機會（急漲後的短空單，或急跌後的短多單）。③對於長期處於緩漲、緩跌、盤整的個股，多數時間乖離率會失效、難以發揮判斷作用（乖離率適合波動較大的走勢，不適合牛皮盤整股）。這些原則屬於乖離率使用時機與風險管理的定性指引，並非獨立的買賣訊號公式；具體的乖離過大買賣點請參照均線分類中的「[[葛蘭碧買點四大法則]]」（第4買點：連續下跌+負乖離過大→潛在反彈買點）與「[[葛蘭碧賣點四大法則]]」（第8賣點：連續上漲+正乖離過大→潛在回跌賣點），或本分類的「[[MA通道正負乖離軌道與突破訊號]]」（12%~15%固定通道參數）。
- **可程式化**: 部分
- **所需資料**: 乖離率序列（見「[[乖離率計算公式與正負乖離定義]]」）、當下趨勢判定、個股波動度（用以判斷是否為「緩漲緩跌盤整股」）
- **計算公式**:
```
# ①不能單獨使用，須搭配趨勢
if 乖離率(t) 極端 and 未搭配趨勢判定:
    警示 = "單獨依乖離率判斷風險高，須搭配趨勢研判"

# ②乖離過大時的操作原則
if abs(乖離率(t)) 明顯偏大（可參照MA通道12%~15%門檻，見R-INDICATOR-18):
    建議 = "避免追高買進或殺低賣出，等回檔修正乖離後再依原進場條件操作；亦可考慮搶短單（急漲後短空/急跌後短多）"

# ③適用個股類型判斷（書中定性描述，無量化公式）
if 個股長期屬於緩漲/緩跌/盤整型態:
    警示 = "乖離率多數時間會失效，不適合牛皮盤整股，僅適合波動較大的走勢"
```
- **參數**: 「乖離過大」之量化門檻依賴MA通道之12%~15%（書中未在此節單獨給出乖離率本身的固定門檻）；「波動較大/牛皮盤整」之區分書中未給量化標準（如年化波動度門檻），需自訂
- **可回測**: 部分（①②的操作邏輯可在有「乖離過大」量化門檻（借用MA通道12%~15%）後程式化回測；③「個股是否適合用乖離率」屬選股前置篩選，需自訂波動度量化標準才能程式化）
- **信心**: 60/100 中（原則方向書中三點明確列出；但均為定性描述，缺乏獨立於MA通道之外的量化數字，程式化須依賴其他規則的參數或自行補充門檻）
