# KD背離判斷（峰背離、底背離）

- **Rule ID**: R-INDICATOR-12
- **名稱**: KD背離判斷（峰背離、底背離）
- **分類**: 技術指標
- **原文與頁碼**: P08-C2-反映股價變化的KD指標.md，下冊 p.168-171
- **解讀**: ①高檔背離（峰背離）：當多頭股價走勢維持「頭頭高」（可用「[[頭頭高底底高多頭趨勢判定]]」判定），但KD指標的K線卻出現「頭頭低」，稱為高檔背離，是趨勢反轉的先期訊號；當股價走勢後續也呈現「頭頭低」時，代表趨勢即將反轉成空頭。②低檔背離（底背離）：當空頭股價走勢維持「底底低」（可用「[[頭頭低底底低空頭趨勢判定]]」判定），但KD指標的K線卻出現「底底高」，稱為低檔背離，同樣是趨勢反轉的先期訊號；當股價走勢後續也呈現「底底高」時，代表趨勢即將反轉成多頭。書中特別強調兩條重點：KD指標出現背離，表示行情有反轉可能，尤其出現底背離時要注意股價隨時會反彈或落底；且指標背離要在20～80之間才正常有效（即K值/D值落在20～80的「非鈍化區」時，背離訊號才可信；若KD已進入0～20低檔鈍化或80～100高檔鈍化區間，背離判讀容易失真，須回歸股價走勢與價量配合判斷，呼應「[[KD高低檔鈍化定義與因應]]」）。
- **可程式化**: 是
- **所需資料**: 轉折波頭/底序列（價格）、對應各波段的K值峰值/谷值序列
- **計算公式**:
```
# 峰背離（高檔背離）：Head[-1], Head[-2] 為最近與次近的價格轉折頭部；KPeak[-1], KPeak[-2] 為對應的K值峰值
if Head[-1] > Head[-2] and KPeak[-1] < KPeak[-2]:
    if 20 <= KPeak[-1] <= 80 and 20 <= KPeak[-2] <= 80:   # 須在20~80非鈍化區才有效
        signal = "KD峰背離，趨勢反轉風險升高（尤其後續價格也出現頭頭低時，確認轉空）"
    else:
        signal = "KD雖呈背離型態，但落在鈍化區，訊號可信度低，需回歸股價與價量判斷"

# 底背離（低檔背離）：Bottom[-1], Bottom[-2] 為最近與次近的價格轉折底部；KTrough[-1], KTrough[-2] 為對應的K值谷值
if Bottom[-1] < Bottom[-2] and KTrough[-1] > KTrough[-2]:
    if 20 <= KTrough[-1] <= 80 and 20 <= KTrough[-2] <= 80:
        signal = "KD底背離，股價隨時會反彈或落底（尤其後續價格也出現底底高時，確認轉多）"
    else:
        signal = "KD雖呈背離型態，但落在鈍化區，訊號可信度低，需回歸股價與價量判斷"
```
- **參數**: 背離有效區間＝K值/D值介於20～80之間（書中明確數字門檻）
- **可回測**: 是
- **信心**: 93/100 高（書中以「KD指標背離重點」獨立條列4條核心規則，且明確給出「20～80之間才正常有效」的量化門檻，並附多組範例圖表佐證）
