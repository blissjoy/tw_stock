# MACD零軸上黃金死亡交叉多頭格局判讀

- **Rule ID**: R-INDICATOR-02
- **名稱**: MACD零軸上黃金死亡交叉多頭格局判讀
- **分類**: 技術指標
- **原文與頁碼**: P08-C1-MACD指標中期走勢的研判.md，下冊 p.153-154
- **解讀**: 適用情境為DIF與MACD兩線皆位於0軸之上（多頭格局）。①黃金交叉買訊：兩線在0軸之上時，若DIF（快線）由下往上穿越MACD（慢線），視為多方買進訊號，代表行情再次上漲。②死亡交叉之區分：兩線在0軸之上時，若DIF由上往下穿越MACD（死亡交叉），視為多頭高檔訊號，可獲利賣出；但只要兩線拉回時未跌破0軸，仍應視為多頭格局中的回檔，而非趨勢反轉。此規則與均線分類中的「[[黃金交叉]]」「[[死亡交叉]]」概念類似，差異在於MACD交叉訊號的意義必須額外搭配「是否位於0軸之上/之下」這個條件才能正確判讀，單看交叉方向容易誤判。
- **可程式化**: 是
- **所需資料**: DIF序列、MACD（訊號線）序列
- **計算公式**:
```
zero_axis_bull = DIF(t) > 0 and MACD(t) > 0    # 兩線皆在0軸之上

golden_cross(t) = DIF(t-1) <= MACD(t-1) and DIF(t) > MACD(t)
dead_cross(t)   = DIF(t-1) >= MACD(t-1) and DIF(t) < MACD(t)

# 多頭格局黃金交叉：買進訊號
if zero_axis_bull and golden_cross(t):
    signal = "多方買進訊號"

# 多頭格局死亡交叉：短線獲利了結，但非趨勢反轉（除非後續跌破0軸）
if zero_axis_bull and dead_cross(t):
    signal = "短線獲利了結訊號"
    if DIF(t) > 0 and MACD(t) > 0:   # 拉回後兩線仍守0軸之上
        trend_status = "僅屬回檔，多頭格局不變"
    else:
        trend_status = "跌破0軸，格局轉為空頭，非單純回檔"
```
- **參數**: 無額外可調參數（依賴R-INDICATOR-01計算出的DIF與MACD序列）
- **可回測**: 是
- **信心**: 91/100 高（書中原文明確區分「0軸上黃金交叉＝買進」與「0軸上死亡交叉但未破0軸＝回檔非反轉」，並附範例圖表佐證）
