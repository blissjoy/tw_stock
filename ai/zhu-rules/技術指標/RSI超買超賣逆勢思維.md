# RSI超買超賣逆勢思維

- **Rule ID**: R-INDICATOR-14
- **名稱**: RSI超買超賣逆勢思維
- **分類**: 技術指標
- **原文與頁碼**: P08-C3-相對強弱變化的RSI指標.md，下冊 p.174-175
- **解讀**: RSI指標又被稱為「逆勢指標」。①RSI＞80為「超買」訊號，表示大家一窩蜂買進、市場過熱，短期內股價漲幅過高、脫離正常價格軌道，這時要逆勢思考，準備賣出或做空，價格容易回跌到正常軌道。②RSI＜20為「超賣」訊號，表示大家一窩蜂賣出、市場太悲觀，短期內股價跌幅過多，這時要逆勢思考，準備回補或做多，價格容易反彈到正常軌道。書中在「應用」段落將超買／超賣門檻明訂為80／20（而非坊間常見的70/30），與「判讀」段落敘述一致，屬本書作者朱家泓的既定參數，程式化時應直接採用80／20。此外書中特別提醒兩項限制：盤整區不宜使用RSI指標（訊號容易失真、反覆鈍化）；強勢連續上漲或連續急跌時，RSI指標都容易失真。
- **可程式化**: 是
- **所需資料**: RSI序列（見「[[RSI計算公式]]」）、趨勢/盤整判定（來自趨勢判定分類規則）
- **計算公式**:
```
if RSI(t) > 80:
    signal = "超買，市場過熱，逆勢思考準備賣出或做空"
if RSI(t) < 20:
    signal = "超賣，市場過冷，逆勢思考準備回補或做多"

# 使用限制（書中明文提醒，非量化門檻但屬明確使用條件）
if 當前趨勢 == "盤整":
    警示 = "盤整區不宜使用RSI指標，訊號容易失真"
if 連續強勢上漲 or 連續急跌:
    警示 = "強勢連續上漲或連續急跌時，RSI指標容易失真"
```
- **參數**: 超買門檻=80、超賣門檻=20（書中明確數字，非慣用之70/30）
- **可回測**: 部分（80/20門檻本身可完整回測；「連續強勢上漲/急跌」「盤整區」的判定需依賴趨勢判定分類規則或另訂量化定義）
- **信心**: 93/100 高（80/20門檻書中在判讀與應用兩段落重複明確標註，且特別注意本書採用80/20而非坊間常見70/30）
