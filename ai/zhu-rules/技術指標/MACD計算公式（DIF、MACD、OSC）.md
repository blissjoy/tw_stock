# MACD計算公式（DIF、MACD、OSC）

- **Rule ID**: R-INDICATOR-01
- **名稱**: MACD計算公式（DIF、MACD、OSC）
- **分類**: 技術指標
- **原文與頁碼**: P08-C1-MACD指標中期走勢的研判.md，下冊 p.152-153
- **解讀**: MACD是利用均線趨勢變動方向的特性設計出來的指標，用兩條不同週期的均線計算兩者的差離狀況，藉此研判買賣時機與趨勢變化。應用上以「12日均線」為快速均線、「26日均線」為慢速均線。指標圖由三個數值構成：①DIF（快線）＝兩均線的差離值，一般以12日均線減去26日均線；②MACD（慢線／訊號線）＝DIF的9日平均值；③OSC（柱狀體）＝DIF減MACD，OSC為正時於0軸之上顯示紅柱（正值越大紅柱越長，代表多方動能增強），OSC為負時於0軸之下顯示綠柱（負值越大綠柱越長，代表空方動能增強）。書中原文僅稱「12日均線」「26日均線」「9日平均值」，未明確標註是簡單移動平均（SMA）或指數移動平均（EMA）；但MACD為國際通用指標，業界慣例（含大多數看盤軟體）均以指數移動平均（EMA）計算DIF與訊號線，程式化時建議採用EMA以符合市場上看盤軟體顯示的數值，若改用SMA會與一般看盤軟體之MACD數值不一致。此點書中未明講，屬工程慣例補充，非原書明文規則。
- **可程式化**: 是
- **所需資料**: 每日收盤價（Close）之時間序列
- **計算公式**:
```
# 指數移動平均（EMA），採業界慣例，書中未明確標註EMA或SMA
EMA(N, t) = Close[t] * (2/(N+1)) + EMA(N, t-1) * (1 - 2/(N+1))

DIF(t)  = EMA(Close, 12)(t) - EMA(Close, 26)(t)          # 快線
MACD(t) = EMA(DIF, 9)(t)                                   # 慢線／訊號線
OSC(t)  = DIF(t) - MACD(t)                                 # 柱狀體

if OSC(t) > 0:
    柱狀顏色 = "紅柱"（多方動能，0軸之上）
elif OSC(t) < 0:
    柱狀顏色 = "綠柱"（空方動能，0軸之下）
```
- **參數**: 快線天期=12日、慢線天期=26日、訊號線平滑天期=9日（書中固定參數，未提及可調整版本）
- **可回測**: 是
- **信心**: 88/100 高（三個數值定義與參數天期書中明確標註；EMA/SMA計算方式為工程慣例補充，非書中明文，故此細節信心中）
