# KD高低檔鈍化定義與因應

- **Rule ID**: R-INDICATOR-11
- **名稱**: KD高低檔鈍化定義與因應
- **分類**: 技術指標
- **原文與頁碼**: P08-C2-反映股價變化的KD指標.md，下冊 p.166-167
- **解讀**: 當K值與D值上升到80以上，或下降到20以下，表示股價過熱或超跌。但股價有時仍會繼續上漲或繼續下跌，這時KD值會進入80～100區間盤整，稱為「高檔鈍化」；或進入0～20區間盤整，稱為「低檔鈍化」。此時KD已經無法反映股價繼續上漲或繼續下跌的狀況，故稱為「鈍化」。書中僅以「KD值進入80～100（或0～20）區間盤整」描述鈍化現象，並未給出連續天數的量化門檻（例如「連續N天K值≥80」），判斷仍需搭配圖表上KD值持續停留在該區間的走勢型態來認定；程式化時若要設定「連續N日」門檻須自行另訂，非原書明文規則。KD進入鈍化區時就失去參考價值，這時要回歸看股價走勢與價量的配合，不宜再單純依KD高低位置或交叉訊號進出。
- **可程式化**: 部分
- **所需資料**: K線序列、D線序列
- **計算公式**:
```
overbought(t) = K(t) >= 80 and D(t) >= 80    # 過熱
oversold(t)   = K(t) <= 20 and D(t) <= 20    # 超跌

# 鈍化：KD持續停留在80~100或0~20區間（書中未給連續天數門檻，此處N需自訂，例如N=3~5日）
high_dull(t) = ALL( K(i) >= 80 and D(i) >= 80  for i in [t-N+1 .. t] )   # 高檔鈍化，N自訂
low_dull(t)  = ALL( K(i) <= 20 and D(i) <= 20  for i in [t-N+1 .. t] )   # 低檔鈍化，N自訂

if high_dull(t) or low_dull(t):
    建議: "KD已失去參考價值，回歸股價走勢與價量配合判斷，不宜單純依KD高低位置或交叉訊號進出"
```
- **參數**: 高檔門檻=80、低檔門檻=20（書中明確數字）；鈍化認定所需的連續天數N書中未給定，需自訂（範例圖表顯示鈍化區間可達數日至數週不等）
- **參考來源查證（2026-07-22，工程補充）**：查證外部技術分析資料，較常見的鈍化起始定義為「KD連續3根K棒維持在80以上（或20以下）」即可視為鈍化開始（來源：[nStock名師專欄](https://www.nstock.tw/author/article?id=1013)）；另有回測研究指出以連續5~6天作為判定門檻時效果較佳（來源：[FinLab](https://www.finlab.tw/kd1/)）。程式化採N=3為預設起始判定值（貼近多數文章共識的最小門檻），並支援呼叫端依需求調整為5~6日的加強版門檻。
- **可回測**: 部分（80/20門檻可直接回測；鈍化「連續天數」門檻需自訂後才能完整程式化與回測）
- **信心**: 65/100 中（80/20門檻與鈍化現象定義明確＝高；但鈍化判定缺乏量化連續天數標準，程式化實作需自行補充，故整體信心評為中）
