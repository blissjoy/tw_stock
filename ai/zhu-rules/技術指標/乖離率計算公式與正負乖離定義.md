# 乖離率計算公式與正負乖離定義

- **Rule ID**: R-INDICATOR-17
- **名稱**: 乖離率計算公式與正負乖離定義
- **分類**: 技術指標
- **原文與頁碼**: P08-C4-乖離與MA通道的運用.md，下冊 p.179-180
- **解讀**: 乖離率是「移動平均線」的技術分析輔助工具，字義為「歪斜、偏離」，用於衡量股價與均線之間的距離。乖離率用來衡量股價與均線的偏離程度，可藉此判斷股價波動是否過大（漲太多或跌太深）。基本邏輯：股價通常跟隨均線同向移動；但一旦股價遠離均線過多，之後價格「回歸均線」修正的機率會提高。正乖離：收盤價位於移動平均線之上，稱為正乖離；正乖離過大表示「買超」，後續有下跌壓力。負乖離：收盤價位於移動平均線之下，稱為負乖離；負乖離過大表示「超賣」，後續有反彈動能。書中明確指出：乖離率達到何種程度屬於買點或賣點，目前沒有統一的原則，需搭配行情強弱經驗判斷，或以「[[MA通道正負乖離軌道與突破訊號]]」的固定百分比（12%~15%）輔助觀察。理論淵源為葛蘭碧（Granville Joseph）於1960年發表的移動平均線理論，其中做多第4買點與做空第8賣點即為乖離過大的逆勢操作點，此二法則已在均線分類的「[[葛蘭碧買點四大法則]]」「[[葛蘭碧賣點四大法則]]」中完整定義，本規則不重複展開。
- **可程式化**: 是
- **所需資料**: 收盤價（Close）、N日均線（MA_N）
- **計算公式**:
```
乖離率(t) = (Close(t) - MA_N(t)) / MA_N(t) * 100%    # N常用20（MA20）

if Close(t) > MA_N(t):
    乖離狀態 = "正乖離"
elif Close(t) < MA_N(t):
    乖離狀態 = "負乖離"
else:
    乖離狀態 = "無乖離（貼近均線）"

# 書中未給出「正/負乖離過大」的固定數字門檻，僅指出無統一原則，
# 需搭配趨勢研判或參考 MA通道的12%~15%固定百分比（見 R-INDICATOR-18）
```
- **參數**: N（均線天期），常用20日；正負乖離過大之數字門檻書中未定義（無統一原則）
- **可回測**: 部分（乖離率數值本身可完整計算與回測；但「乖離過大」的買賣判定門檻書中未給固定數字，需另訂或搭配MA通道規則）
- **信心**: 62/100 中（計算公式與正負乖離定義明確＝高；但乖離過大之門檻書中明言「沒有統一的原則」，此部分無法直接程式化為精確買賣訊號，故整體信心評為中）
