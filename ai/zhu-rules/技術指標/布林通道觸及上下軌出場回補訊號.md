# 布林通道觸及上下軌出場回補訊號

- **Rule ID**: R-INDICATOR-21
- **名稱**: 布林通道觸及上下軌出場回補訊號
- **分類**: 技術指標
- **原文與頁碼**: P08-C5-布林通道.md，下冊 p.188
- **解讀**: 基本操作邏輯：股價觸及下軌時偏多操作，觸及上軌時偏空操作，形成「軌道內來回」的高低點交易思路。這是「持倉了結」的出場邏輯，與「[[布林通道4大買進訊號]]」「[[布林通道4大做空訊號]]」的「穿越軌道進場」邏輯互為對照，共同構成一組完整的進場/出場策略框架。①做多出場：股價碰到上軌道，視為出場訊號出現，準備賣出——買進或做多後股價碰上軌，再往上漲的機會已經很小，可選擇短線出場。②做空回補：股價碰到下軌道，視為反彈訊號出現，準備回補——股價碰下軌後再續跌的機會不大，可回補短線空單。
- **可程式化**: 是
- **所需資料**: 收盤價（Close）、布林通道上軌/下軌序列（見「[[布林通道計算公式]]」）、持倉部位方向
- **計算公式**:
```
上軌(t), 下軌(t) = 見 R-INDICATOR-20

if 持有多單 and Close(t) >= 上軌(t):
    signal = "短線出場訊號（做多出場）"

if 持有空單 and Close(t) <= 下軌(t):
    signal = "短線回補訊號（做空回補）"
```
- **參數**: 依賴布林通道計算公式參數（MA20、2倍標準差）
- **可回測**: 是
- **信心**: 90/100 高（書中原文以清楚的做多出場、做空回補兩條規則明確條列）
