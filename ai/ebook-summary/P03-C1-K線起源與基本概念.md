## 0. 章節背景與起源（p.125）

- K線理論起源於17世紀日本大阪堂島稻米交易所；1750年江戶時代，本間家族么子本間宗久在阪田交易所交易時，每天記錄氣候狀況與交易價格漲跌，繪製成圖表分析出稻米價格變化的相關性，因而致富。後來歐美將此方法應用到股票市場，成為技術分析的重要理論（日文「技術分析」譯音為K，因此稱K線圖）。（上冊 p.125）

## 1. K線的畫法與4大元素（p.125-128）

### 1.1 K線的4個組成元素
- K線記錄股票市場每天買賣交易過程，組成元素共4個：**開盤價、收盤價、盤中最高價、盤中最低價**。（上冊 p.125）

### 1.2 紅K線（陽線）畫法
- [可能規則] 定義：**開盤價 < 收盤價 → K線為紅色（陽線）**。（上冊 p.125）
- 範例（圖表3-1-1，華上6289，2013/5/24）：開盤4.53、收盤4.67、最高4.78、最低4.44，走勢圖上4個關鍵時間點的價格分別對應K線圖上最高價／收盤價／開盤價／最低價的位置。（上冊 p.125-126）
- 圖表3-1-2「紅K線的形成」：以9:00～13:30的走勢折線圖示範，最高價、收盤價、開盤價、最低價四個價位分別用箭頭指到同一根K線棒上對應的四個高度位置，直觀說明K線是由走勢圖中這4個價位「濃縮」而成。（上冊 p.126）

### 1.3 黑K線（陰線）畫法
- [可能規則] 定義：**開盤價 > 收盤價 → K線為黑色（陰線）**。（上冊 p.126）
- 範例（圖表3-1-3，晶豪科3006，2013/5/24）：開盤46.9、收盤45.1、最高47.6、最低44，同樣示範走勢圖四個價位對應到黑K線棒上的位置。（上冊 p.127）
- 圖表3-1-4「黑K線的形成」：同上，但收盤價低於開盤價，形成黑色實體。（上冊 p.128）

## 2. K線基本概念（p.128-145）

### 2.1 紅/黑K線的判定原則與K線各部位名稱
- 收盤價高於開盤價 → 紅K線；收盤價低於開盤價 → 黑K線（重申4元素→紅黑判定的基本規則）。（上冊 p.128）
- 一、K線各部位名稱（圖表3-1-5「陰陽K線圖」）：K線棒由上到下依序為 **最高價 → 上影線 → 實體（收盤價/開盤價其中一邊）→ 下影線 → 最低價**；紅K實體上緣=收盤價、下緣=開盤價；黑K實體上緣=開盤價、下緣=收盤價。（上冊 p.128）
- [可能規則] 特別註記：**開盤價與收盤價「有價差」時才畫出矩形實體**（意即開盤價=收盤價時不畫矩形實體，只畫一橫線，即十字線/一字線的畫法邏輯）。（上冊 p.128）

### 2.2 紅K線或黑K線不代表當天股價上漲或下跌（與「前一日收盤價」比較才是漲跌）
- 重要觀念：紅/黑K線只反映「當天開盤與收盤的相對關係」，不代表「與前一交易日相比」的真實漲跌，判斷漲跌要看當日收盤價 vs. 前一日收盤價。（上冊 p.129）
- 1. 收盤高於開盤為紅K線，不一定代表當天股價上漲：範例，前日收盤50元，當日開盤47元、收盤49元 → 當日為紅K線，但實際比前一日下跌1元。（上冊 p.129）
- 2. 收盤低於開盤為黑K線，不一定代表當天股價下跌：範例，前日收盤50元，當日開盤53元、收盤51元 → 當日為黑K線，但實際比前一日上漲1元。（上冊 p.129）
- 3. 收盤高於開盤為陽線（紅K線），不一定代表上漲：範例，前日收盤51元，當日開盤46元、收盤50元 → 當日為紅K線，但實際比前一日下跌1元。（上冊 p.129）
- 圖表3-1-9「走勢圖中紅K、黑K的檢視」：實例標示「紅K線，股價下跌」「黑K線，股價上漲」「黑K線，股價收平盤」「紅K線，股價收平盤」四種情境，強化「顏色≠漲跌方向」。（上冊 p.130）
- 4. 不同情境的K線代表意義不同：即使當天收盤股價同樣上漲1元，K線的表現形態（實體大小、有無影線）所隱含的意義不同：
  - [可能規則] 紅K上漲1元、實體飽滿（開低走高類型）→ **多方上漲力道強，次日容易上漲**。
  - [可能規則] 黑K上漲1元、開高賣壓重（開高走低但收盤仍小漲）→ **次日容易下跌**。
  - [可能規則] 紅K上漲1元但留長上影線 → **向上有壓力，多空震盪，次日看開盤方向決定強弱**。
  （圖表3-1-10「K線組合的不同解讀」，上冊 p.130）

### 2.3 趨勢由K線組成（多時間週期的基礎框架）
- 圖表3-1-11「由K線形成趨勢的技術分析」：技術分析的推演邏輯鏈為 **價格 → K線 → 2根K線組合 → 3根K線組合 → 均線 → 波浪型態 → 多空市場 → 趨勢**；同時可延伸出RSI、KD、MACD等指標與波浪理論。書中註明「決定方向的力量：小到大」、「轉折訊號的速度：快到慢」，即單根K線的轉折訊號最快但力道最小，趨勢（波浪/多空市場）訊號最慢但力道最大。（上冊 p.131）
- 三、不同週期K線的基本認知：
  - [可能規則] 日線圖1根K線 = 當日交易紀錄（當日開盤～當日收盤）。
  - [可能規則] 週線圖1根K線 = 1週交易紀錄（**該週第1個交易日的開盤價**做為開盤、**該週最後一個交易日的收盤價**做為收盤，期間最高/最低取全週最高/最低）。
  - [可能規則] 月線圖1根K線 = 1個月交易紀錄（**月初第1個交易日開盤**、**月底最後一個交易日收盤**）。
  （上冊 p.131）
- 四、不同週期K線的趨勢不同：同一支股票同一天，日線、週線、月線三種週期解讀出來的趨勢可能不同（需要分別確認）。（上冊 p.131）
- 圖表3-1-12/3-1-13：以同欣電(6271)為例，日線呈現「多頭回後買上漲」，但同時期週線卻是「空頭反彈高檔接近壓力」，證明不同週期趨勢確實可能不一致，須以交易週期對應的K線圖判斷。（上冊 p.132）
- 圖表3-1-14：同欣電(6271)月線圖同時期呈現「多頭回後買上漲」，再補一個月線週期視角範例。（上冊 p.133）

### 2.4 K線「4+1」元素的意義與應用（核心：二分之一價）
- [可能規則] 除了開盤價、收盤價、最高價、最低價這4個電腦K線圖上會標明的元素外，朱老師額外定義**第5元素＝「二分之一成本價」**（即當日最高價與最低價的中點），合稱「4+1」元素，在短線操作判斷上非常重要。（上冊 p.133）
- 圖表3-1-15：以紅K、黑K圖示標出5個標記點：①最高價、②收盤價(紅K)/開盤價(黑K)、③最低價、④開盤價(紅K)/收盤價(黑K)、⑤二分之一價（位於實體中，但與開盤/收盤點分開標示，代表(最高+最低)/2，非開收盤中點）。（上冊 p.133）

#### （1）開盤價的意義
- [可能規則] (1) 反映利多或利空的消息面。
- [可能規則] (2) 反映主力大戶的企圖。
- [可能規則] (3) 用來判斷「今日是否變盤」：**開高可化解疑慮；開低則多頭要小心是否真的變盤**。
- [可能規則] (4) 是關鍵轉折位置，**次日開盤定強弱**（即隔日開盤價相對前一日關鍵價位的位置，用來判斷當下多空強弱是否延續）。
（上冊 p.134）
- 圖表3-1-16/17/18（分別以個股1904、宏達電2493、加權指數TSE為例）示範「由開盤價看主力企圖」的判讀邏輯，重複出現的規律：
  - [可能規則] 多頭上漲到高檔出現大量長紅K → 隔日**開低盤**代表主力要回檔整理。
  - [可能規則] 下跌到低檔出現（大量）長黑K線（常伴隨長下影線）→ 隔日**開高盤收紅K**代表主力（或有支撐）要反彈。
  - [可能規則] 接近前高壓力位置、開低盤收黑K → 代表主力要繼續盤整。
  - [可能規則] 接近前高壓力位置、開盤跳空大量 → 代表主力要突破盤整（突破後常見連續3天開高盤急漲）。
  - [可能規則] 下跌到低檔出現十字變盤線，次日開高盤收紅K → 代表主力要反彈。
  （上冊 p.134-135）

#### （2）收盤價的意義
- [可能規則] (1) 收盤價是當日多空雙方一整天決戰後的最終結果價格。
- [可能規則] (2) 可看出當日多空強弱，並可預期明日開高或開低的意圖。
- [可能規則] (3) **收盤價是否突破昨日高點（轉強）、或跌破昨日低點（轉弱）**，是重要觀察指標。
（上冊 p.136）
- 圖表3-1-19「從K線看多空力道」：以收盤價在K線實體內（相對於最高/最低價中點）的相對位置判斷多空力道——
  - [可能規則] 收盤價偏向K線區間上半部（接近最高價）→ **多方強**（紅K為典型）。
  - [可能規則] 收盤價偏向K線區間下半部（接近最低價）→ **空方強**（黑K為典型）。
  - [可能規則] 收盤價落在區間中段（十字線/小紅K附近）→ **多空平分**。
  （上冊 p.137）
- 圖表3-1-20：實例標示「收盤空方強」「收盤多方強」的判讀。（上冊 p.137）
- [可能規則] (4) **前一日最高點是（次日的）壓力，前一日最低點是（次日的）支撐**。（上冊 p.138）
- 圖表3-1-21「壓力與支撐的位置」：以前一日走勢圖的最高點(107)為壓力線、最低點(98)為支撐線；當日走勢圖同理以當日最高點(106.5)、最低點(99)構成新的壓力/支撐線，說明每日盤中高低點都會成為隔日的壓力/支撐參考線。（上冊 p.138）
- [可能規則] (5) **收盤突破前一日最高點 → 買方力量轉強；收盤跌破前一日最低點 → 賣方力量轉強**。（上冊 p.138）
- 圖表3-1-22：實例標示「收盤價跌破昨日K線最低點」「收盤價突破昨日K線最高點」。（上冊 p.138）

#### （3）最高價的意義
- [可能規則] (1) 當日多方力量所能攻到的最高價位。
- [可能規則] (2) 是空頭抵擋防守的價位，為空方防守線（即**次日的壓力線**）。
- [可能規則] (3) 是當天盤中多方測試壓力的位置。
- [可能規則] (4) **最高價來到前面壓力位置，若沒有過壓、或過壓又拉回，並留下長上影線，即表示該位置出現壓力訊號，次日要特別注意是否轉為遇壓回檔或拉回下跌**（上影線長度是判斷壓力強弱的具體訊號）。
- [可能規則] (5) 最高價次日被突破 → 多方力量轉強。
（上冊 p.139）
- 圖表3-1-23/24/25：多組個股實例標示「盤中最高點測壓力，留長上影線代表有壓」「盤中最低點測支撐，留長下影線代表有撐」。（上冊 p.139-141）

#### （4）最低價的意義
- [可能規則] (1) 當日空方力量所能到達的最低價位。
- [可能規則] (2) 是多頭抵擋防守的價位，為多方防守線（即**次日支撐線**）。
- [可能規則] (3) 是當天盤中空方測試支撐的位置。
- [可能規則] (4) **最低價來到前面支撐位置，若沒有跌破、或跌破又拉上來，並留下長下影線，即表示該位置出現支撐訊號，次日要特別注意是否轉為遇撐反彈或上漲**（下影線長度是判斷支撐強弱的具體訊號）。
- [可能規則] (5) 最低價次日被跌破 → 空方力量轉強；若跌破後又被突破（收復），則低檔止跌回升機率大增，可注意反彈。
（上冊 p.139-141）

#### （5）二分之一價（第5元素，核心規則）
- [可能規則] 定義（公式）：**二分之一價 = (當日最高價 + 當日最低價) ÷ 2**，代表當日多空交易的「平均成本」。（上冊 p.141）
- [可能規則] 走勢中只要出現**大量**的長紅K線或長黑K線，都要特別注意其二分之一價的位置，**尤其該K線位於相對高檔或低檔時**更重要。（上冊 p.141）
- [可能規則] 大量長紅K線出現後，若日後回檔**跌破**該K線的二分之一價 → 表示多方氣勢轉弱。（上冊 p.141）
- [可能規則] 大量長黑K線出現後，若日後反彈**突破**該K線的二分之一價 → 表示空方力道轉弱。（上冊 p.141）
- [可能規則] 股價漲到高檔出現大量長紅K線，該K線的二分之一價若被跌破 → **高檔做頭機率大增，要小心反轉**。（上冊 p.141）
- [可能規則] 股價跌到低檔出現大量長黑K線，該K線的二分之一價若被突破（延續同一邏輯，隱含股價止跌反轉訊號）。（上冊 p.141-142）
- 「假突破/假跌破」判讀（圖表3-1-26~33，多組個股實例）：
  - [可能規則] 大量長紅K線後接**大量長黑K線且收盤跌破前面長紅K的二分之一價** → **次日容易回檔**（延續轉弱）；但若隔日（次日）股價反而**上漲**（收復），則視為**假跌破**，不視為真正轉弱。（上冊 p.141-144）
  - [可能規則] 對稱地，大量長黑K線後接**大量長紅K線且收盤突破前面長黑K的二分之一價** → **次日容易反彈**（延續轉強）；但若隔日股價反而**下跌**，則視為**假突破**。（上冊 p.144-145）

## 3. K線的支撐壓力與強弱的改變（p.146-153）

### 3.1 1日K線盤中的支撐與壓力
- [可能規則] 股票當天開盤到收盤4.5小時交易期間，股價漲跌波動，**盤中最高點＝當天的壓力，盤中最低點＝當天的支撐**，此支撐壓力對「次日」走勢同樣具有參考意義。（上冊 p.146）
- [可能規則] **次日收盤突破今日最高點（壓力）→ 表示多方強；次日收盤跌破今日最低點（支撐）→ 表示空方強**。（上冊 p.146）
- 此支撐壓力邏輯不限日K，60分K同樣適用：前一根60分K的高/低點構成下一根60分K的壓力/支撐參考（圖表3-1-34、3-1-35）。（上冊 p.146-147）

### 3.2 大量長紅K線日後的支撐與壓力
- [可能規則] 多頭上漲的長紅K線，若在攻擊的關鍵位置通常伴隨**大量**，此量可視為進貨量、攻擊量或換手量。在多頭走勢中「價量配合攻擊」是重要現象；若出現股價攻不上甚至下跌，都是**重大警訊，必須設好停損、做好風險控制**。（上冊 p.147）
- [可能規則] 出現大量長紅K線、股價上漲之後的回測，該長紅K線構成重要支撐，支撐分為**3個重要觀察價位（由強至弱）**：
  1. **最高價**：股價在其上方時是**最強支撐點**。
  2. **二分之一價**：股價在其上方時是**平均成本支撐點**。
  3. **最低價**：股價在其上方時是**最弱支撐點**；若股價**跌破最低價，則多空易位**（原本的支撐轉為壓力）。
  （圖表3-1-36，上冊 p.147-148）
- [可能規則] 重點說明（黑K線測試長紅K線支撐的三層反應，圖表3-1-36延伸說明）：
  1. 黑K線跌破長紅K線的**最高點**：代表向上攻擊力道減弱，**必須在3～5個交易日之內站回該長紅K線高點之上**，否則要注意是否轉折向下。
  2. 黑K線跌破長紅K線的**二分之一價**：代表跌破該長紅K線當天交易的平均成本，容易產生大量賣壓，多方氣勢已被空方破壞。
  3. 黑K線跌破長紅K線的**最低點**：代表多空易位，轉為空方主導，同時該長紅K線反而變成**日後股價上漲的壓力**。
  （上冊 p.148）
- 圖表3-1-37/38（賓茂9907、宏泰2610等實例）：反覆驗證「多頭爆大量長紅K上漲，該K線最低點可作為日後支撐參考，只要沒有被跌破，仍維持多方力道」的規則。（上冊 p.149）

### 3.3 大量長黑K線日後的支撐與壓力（與長紅K完全對稱的鏡射邏輯）
- [可能規則] 空頭下跌行進中出現長黑K，股價繼續下跌且沒有止跌訊號出現時，**無論是否有大量，都不能低接**。（上冊 p.150）
- [可能規則] 連續出現下跌的長黑K，若出現**爆量或窒息量**，容易產生反彈；當出現上漲紅K**收盤價站過前一日高點**時，可短線搶反彈。（上冊 p.150）
- [可能規則] 出現大量長黑K線、股價下跌之後，該長黑K線構成重要壓力，壓力分為**3個重要觀察價位（由強至弱）**：
  1. **最低價**：股價在其下方時是**最強壓力點**。
  2. **二分之一價**：股價在其下方時是**平均成本壓力點**。
  3. **最高價**：股價在其下方時是**最弱壓力點**；若股價**突破最高價，則多空易位**（原本的壓力轉為支撐）。
  （圖表3-1-39，上冊 p.150-151）
- [可能規則] 重點說明（紅K線測試長黑K線壓力的三層反應）：
  1. 紅K收盤突破長黑K線的**最低點**：代表向下力道減弱，向上力道轉強，要注意是否轉折向上反彈。
  2. 紅K收盤突破長黑K線的**二分之一價**：代表該長黑K線當放空交易的平均成本已被多方突破，容易產生大量回補買單，空方氣勢可能被多方扭轉。
  3. 紅K收盤突破長黑K線的**最高點**：代表多空易位，轉為多方主導，同時該長黑K線反而變成**日後股價上漲的重要支撐觀察點**。
  （上冊 p.151）
- 圖表3-1-40/41（群聯2351等實例）：驗證上述規則。（上冊 p.152-153）

> 小結（可程式化的核心規則）：任何一根「帶大量」的長紅K或長黑K，都會在其**最高價、二分之一價（(高+低)/2）、最低價**這3個價位上留下日後的支撐/壓力刻度；多空雙方每突破一層價位，力道強弱就轉變一級，全部突破（或跌破）則視為「多空易位」，原支撐/壓力角色互換。

## 4. K線橫盤原則與進出場的確認（p.154-156）

- [可能規則] **橫盤/盤整的定義**：走勢圖中K線橫向走勢，股價一直沒有突破前一根K線的最高點，也沒有跌破前一根K線的最低點，**這樣的橫向K線連續超過3根以上**，可視為K線的橫盤或K線盤整。（上冊 p.154）
- [可能規則] **確認向上突破盤整的條件**：必須出現**1根中長紅K線，且收盤價突破橫盤區間的「上頸線」**（即橫盤期間的最高點），才能確認往上突破；若此時處於多頭趨勢，即為進場做多的買進位置。（上冊 p.154）
- [可能規則] **確認向下突破盤整的條件**：必須出現**1根中長黑K線，且收盤價跌破橫盤區間的「下頸線」**（即橫盤期間的最低點），才能確認往下跌破；若此時處於空頭趨勢，即為進場放空的位置。（上冊 p.154）
- 圖表3-1-42「K線橫盤的趨勢變化關鍵」：定義上頸線＝壓力，以「收盤確認突破第1根K線的最高點」判定；下頸線＝支撐，以「收盤確認跌破第1根K線的最低點」判定。（上冊 p.154）
- 圖表3-1-43/44/45：多組個股實例（中光電5371等）標示「K線橫盤突破」與「K線橫盤跌破」的實戰位置，驗證上述規則。（上冊 p.155）

## 5. K線轉折與趨勢關系（p.156-159）

### 5.1 趨勢與轉折的基本關係（核心口訣）
- 趨勢定義：股價一段時間內行進的方向。當趨勢沒有改變，**多頭趨勢會維持「漲多跌少」，股價逐漸墊高**，直到趨勢反轉成空頭；**空頭趨勢會維持「跌多漲少」，股價逐漸破低**，直到趨勢反轉成多頭。（上冊 p.156）
- 股價由上漲轉下跌、或由下跌轉上漲時，會出現「轉折」的變化，這些轉折變化是由K線來顯示的，因此K線可以提供股價轉折的訊號。
- 朱老師口訣（原文短句）：**「在看轉折時不要忘記看趨勢，在趨勢行進時不要忘記看轉折」**；圖表3-1-46 另有標註「**在趨勢中看轉折，看轉折不要忘了趨勢**」。（上冊 p.156-157）
- 換句話說：判斷股價漲跌時，**趨勢與K線轉折要一起看**，才不會判斷錯誤——即轉折訊號必須放在當前趨勢背景下解讀（例如多頭趨勢中的短暫轉折向下K線訊號可能只是回檔，而非真正反轉；需搭配後續K線與均線確認）。（上冊 p.156）

### 5.2 研判K線之前必須先確認的3個環境條件
- [可能規則] 1. 先確認行情趨勢目前是**多頭、空頭、還是盤整**。（上冊 p.157）
- [可能規則] 2. 要看K線在走勢中所處的相對位置：是在**低檔、高檔、行進中（半山腰）、接近壓力或支撐、還是處於關鍵突破或跌破的位置**。（上冊 p.157）
- [可能規則] 3. 要看**成交量的變化**，可幫助確認K線的能量大小以及籌碼變化（配合前面「大量長紅/長黑K」的支撐壓力規則使用）。（上冊 p.157）

### 5.3 多頭趨勢上漲的轉折K線訊號（實戰範例，圖表3-1-47，個股大西1453）
- [可能規則] 大量長紅K突破底部盤整，且均線4條呈多頭排列向上 → **多頭確認，開始做多**訊號。（上冊 p.158）
- [可能規則] 多頭趨勢上漲到高檔，出現**長下影線的變盤線** → 轉折下跌的訊號。（上冊 p.158）
- [可能規則] 紅K突破前述長下影線變盤線 → 多頭續漲的K線訊號，可把握機會做多。（上冊 p.158）
- [可能規則] 多頭趨勢上漲到高檔，出現**大量長黑K下跌** → 轉折下跌的訊號。（上冊 p.158）
- [可能規則] 紅K收盤突破盤整區間 → 多頭續漲的K線訊號，可把握機會做多。（上冊 p.158）
- [可能規則] 多頭趨勢上漲到高檔，出現**大量紡錘（變盤）K線** → 轉折下跌的訊號。（上冊 p.158）
- [可能規則] 多頭回檔修正後，紅K收盤突破5日均線 → 多頭續漲的K線訊號，可把握機會做多。（上冊 p.158）

### 5.4 空頭趨勢下跌的轉折K線訊號（實戰範例，圖表3-1-48，個股揚明光3504；此範例含高檔反轉到空頭續跌的完整循環）
- [可能規則] 多頭趨勢上漲到高檔（或上漲遇壓力），出現**大量長黑K下跌** → 轉折下跌的訊號。（上冊 p.159）
- [可能規則] 多頭趨勢上漲到高檔，出現**大量長黑K帶長上影線** → 轉折下跌的訊號。（上冊 p.159）
- [可能規則] 大量長黑K下跌**跌破前面低點** → 空頭確認，若同時月線下彎，可順勢做空。（上冊 p.159）
- [可能規則] 空頭趨勢下跌到低檔，出現**長紅K線** → 止跌訊號，也是轉折反彈的訊號。（上冊 p.159）
- [可能規則] 空頭趨勢反彈到高檔，再出現**大量長上影線變盤K線** → 轉折向下訊號，準備做空。（上冊 p.159）
- [可能規則] 空頭黑K收盤**跌破盤整下頸線** → 空頭續跌的K線訊號，可把握機會做空。（上冊 p.159）
- [可能規則] 空頭趨勢下跌到低檔，**連續3天出現變盤K線** → 止跌反彈的訊號。（上冊 p.159）
- [可能規則] 空頭反彈後再出現下跌黑K → 空頭續跌的K線訊號，可把握機會做空。（上冊 p.159）

> 對照多頭範例（5.3）與空頭範例（5.4）可看出朱老師的轉折判讀是「鏡射」邏輯：多頭續漲訊號（紅K突破變盤線／突破盤整／突破5日均線）對應空頭續跌訊號（黑K跌破盤整下頸線／空頭反彈後再破底）；轉折反轉訊號則是「長黑K/長上影線出現在高檔＝轉弱」對應「長紅K/連續3天變盤線出現在低檔＝轉強」。
