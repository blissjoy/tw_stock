## 1. 章節導言（下冊 p.152）

MACD 是利用均線趨勢變動方向的特性設計出來的指標：用兩條不同週期的均線，計算兩者的差離狀況，藉此研判買賣時機與趨勢變化。

---

## 2. MACD指標的內涵及計算（下冊 p.153）

- 應用上以 **12日均線為快速均線、26日均線為慢速均線**。
- MACD 指標圖由三個數值構成：
  1. **DIF（快線）**：兩均線的差離值，一般以 12日均線減去 26日均線，得出當日 DIF 數值。
     `[可能規則] DIF = MA(收盤價, 12) − MA(收盤價, 26)`
  2. **MACD（慢線）**：DIF 的 9日平均值。
     `[可能規則] MACD訊號線 = 9日移動平均(DIF)`
  3. **OSC（紅、綠柱狀體）**：快線減慢線（DIF − MACD）。OSC 為正數時，於 0 軸之上顯示紅柱，正值越大紅柱越長；OSC 為負數時，於 0 軸之下顯示綠柱，負值越大綠柱越長。
     `[可能規則] OSC = DIF − MACD訊號線；OSC>0 → 紅柱（多方動能）；OSC<0 → 綠柱（空方動能）`
- MACD 理論除了可用於確認中長期趨勢，也能用來判斷短期反轉點。

---

## 3. MACD多頭趨勢應用要點（下冊 p.153-154）

適用情境：DIF 與 MACD 兩線皆位於 0 軸之上（多頭格局）。

1. **黃金交叉買訊**：兩線在 0 軸之上時，若 DIF（快線）由下往上穿越 MACD（慢線，黃金交叉），視為多方買進訊號，代表行情再次上漲。（下冊 p.153-154）
   `[可能規則] 條件：DIF>0 且 MACD>0 且 DIF 上穿 MACD → 多方買進訊號`
2. **死亡交叉之區分**：兩線在 0 軸之上時，若 DIF 由上往下穿越 MACD（死亡交叉），視為多頭高檔訊號，可獲利賣出；但只要兩線拉回時**未跌破 0 軸**，仍應視為多頭格局中的回檔，而非趨勢反轉。（下冊 p.154）
   `[可能規則] 條件：DIF>0 且 MACD>0 且 DIF 下穿 MACD → 短線獲利了結訊號；若拉回後兩線仍守 0 軸之上 → 僅屬回檔，多頭格局不變`

範例圖表8-1-1（下冊 p.154）：某股 2023/06/30 附近日線圖，2 線持續在 0 軸之上多次黃金/死亡交叉，整體維持多頭格局，死亡交叉僅造成回檔而非趨勢反轉。

## 4. MACD空頭趨勢應用要點（下冊 p.154）

適用情境：DIF 與 MACD 兩線皆位於 0 軸之下（空頭格局）。

1. **黃金交叉之區分**：兩線在 0 軸之下時，若 DIF 與 MACD 黃金交叉，視為空頭格局中的低檔訊號，可作空單回補；但**反彈若未突破 0 軸**，只能視為空頭趨勢中的反彈，不可視為轉多。（下冊 p.154）
   `[可能規則] 條件：DIF<0 且 MACD<0 且 DIF 上穿 MACD → 空單回補訊號；若反彈未站上 0 軸 → 僅屬空頭反彈，空頭格局不變`
2. **死亡交叉賣訊**：兩線在 0 軸之下時，若 DIF 與 MACD 死亡交叉，為空頭賣出訊號，代表行情再次下跌。（下冊 p.154）
   `[可能規則] 條件：DIF<0 且 MACD<0 且 DIF 下穿 MACD → 空方賣出（做空）訊號`

範例圖表8-1-2（下冊 p.154）：個股 2023/08/22 附近日線圖，2 線持續在 0 軸之下多次黃金/死亡交叉，維持空頭格局，黃金交叉僅為反彈而非轉多。

## 5. 多頭格局OSC柱狀趨勢變化（下冊 p.155-156）

七項針對紅柱（OSC 正值）長度變化與價格關係的判讀規則：

1. 多頭趨勢紅柱縮短 → 多頭回檔。（p.155）
   `[可能規則] 連續紅柱高度遞減 → 研判為漲勢中的回檔訊號`
2. 多頭趨勢紅柱漸長 → 漲勢持續，多單續抱。（p.155）
   `[可能規則] 連續紅柱高度遞增 → 漲勢持續，續抱多單`
3. 多頭趨勢紅柱漸長，但股價不漲或下跌 → 動能與價格背離，多單準備賣出。（p.155）
   `[可能規則] 紅柱遞增但股價未創新高或下跌 → 動能背離，提示多單賣出`
4. 多頭趨勢紅柱由縮短再轉增長 → 漲勢再起，可回檔後買上漲。（p.155）
   `[可能規則] 紅柱縮短後轉為遞增 → 買進訊號（回後上漲）`
5. 柱狀由綠柱縮短並接近 0 軸，之後轉為紅柱時 → 股價容易再漲。（p.155）
   `[可能規則] 綠柱縮短貼近 0 軸後翻紅 → 偏多訊號，股價易再漲`
6. 柱狀由紅柱縮短並接近 0 軸，之後轉為綠柱時 → 股價容易續回檔。（p.155）
   `[可能規則] 紅柱縮短貼近 0 軸後翻綠 → 偏空訊號，股價易續跌/回檔`
7. **高檔背離**：多頭趨勢處於高檔，股價創新高（1～3 次），但紅柱未同步創新高 → 稱為高檔背離，股價做頭機會很大。（p.155）
   `[可能規則] 股價創近期新高（1~3次），但對應 OSC 紅柱峰值未創新高 → 高檔背離，做頭風險升高`

範例圖表8-1-3、8-1-4（下冊 p.156）：兩檔個股 2023/01/13 附近日線圖，分別示範「股價創新高但紅柱未創新高（高檔背離）」與「股價上漲、紅柱縮短」的多頭回檔情境。

## 6. 空頭格局OSC柱狀趨勢變化（下冊 p.157）

與多頭格局對稱的七項規則，針對綠柱（OSC 負值）：

1. 空頭趨勢綠柱縮短 → 空頭反彈。
   `[可能規則] 連續綠柱高度（絕對值）遞減 → 研判為跌勢中的反彈訊號`
2. 空頭趨勢綠柱漸長 → 跌勢持續，空單續抱。
   `[可能規則] 連續綠柱高度（絕對值）遞增 → 跌勢持續，續抱空單`
3. 空頭趨勢綠柱漸長，但股價不跌或上漲 → 動能背離，空單準備回補。
   `[可能規則] 綠柱遞增但股價未創新低或上漲 → 動能背離，提示空單回補`
4. 空頭趨勢綠柱由縮短再轉增長 → 跌勢再起，反彈後可再放空。
   `[可能規則] 綠柱縮短後轉為遞增 → 放空訊號（彈後續跌）`
5. 柱狀由紅柱縮短並接近 0 軸，之後轉為綠柱時 → 股價容易再跌。
   `[可能規則] 紅柱縮短貼近 0 軸後翻綠 → 偏空訊號，股價易再跌`
6. 柱狀由綠柱縮短並接近 0 軸，之後轉為紅柱時 → 股價容易續反彈。
   `[可能規則] 綠柱縮短貼近 0 軸後翻紅 → 偏多訊號，股價易續反彈`
7. **低檔背離**：空頭趨勢處於低檔，股價創新低（1～3 次），但綠柱未同步創新低 → 稱為低檔背離，股價打底機會很大。
   `[可能規則] 股價創近期新低（1~3次），但對應 OSC 綠柱谷值（絕對值）未創新低 → 低檔背離，打底機會升高`

範例圖表8-1-5（下冊 p.157）：個股 2023/01/13 附近日線圖，股價創新低但綠柱未創新低，屬低檔背離範例。

## 7. MACD指標的背離應用（下冊 p.158-161）

書中將「背離」拆成兩種尺度：單一波段內的背離，以及跨越多個波段（頭部/底部相互比較）的趨勢背離。

### 7-1 波段上漲或下跌的背離（下冊 p.158-159）

- **股價上漲、紅柱縮短 → 高檔背離**，後續股價容易回檔。（p.158，圖表8-1-6概念圖）
  `[可能規則] 單一上漲波段內：股價續創高但同步之 OSC 紅柱高度縮短 → 波段高檔背離，提示回檔`
- **股價下跌、綠柱縮短 → 低檔背離**，後續股價容易反彈。（p.158，圖表8-1-7概念圖）
  `[可能規則] 單一下跌波段內：股價續創低但同步之 OSC 綠柱高度（絕對值）縮短 → 波段低檔背離，提示反彈`

範例圖表8-1-8「波段上漲的背離」、圖表8-1-9「波段下跌的背離」（下冊 p.159）為對應個股日線圖實例（分別對照股價創高/創低與 OSC 柱狀縮短的同步變化）。

### 7-2 趨勢上漲或下跌的背離（下冊 p.160-161）

此類背離比較的是**兩個以上波段的高點群／低點群**，而非單一波段內的柱狀縮長：

- **多頭「頭頭高」對應紅柱群「頭頭低」**：多頭趨勢股價創新高（頭頭高），但兩波段對應的紅柱群峰值卻一波比一波低（頭頭低）→ 屬趨勢高檔背離，後續趨勢容易由多轉空。（p.160，圖表8-1-10概念圖）
  `[可能規則] 連續兩個上漲波段：價格高點依序墊高，但對應 OSC 紅柱波峰依序遞減 → 趨勢級背離，提示多轉空`
- **空頭「底底低」對應綠柱群「底底高」**：空頭趨勢股價創新低（底底低），但兩波段對應的綠柱群谷值卻一波比一波淺（底底高）→ 屬趨勢低檔背離，後續趨勢容易由空轉多。（p.160，圖表8-1-11概念圖）
  `[可能規則] 連續兩個下跌波段：價格低點依序探低，但對應 OSC 綠柱波谷（絕對值）依序遞減 → 趨勢級背離，提示空轉多`

範例圖表8-1-12「趨勢上漲的背離」（下冊 p.161）、圖表8-1-13「趨勢上漲、下跌的綜合應用」（下冊 p.161）為個股日線圖實例，後者同時標示同一走勢圖中先出現「波段上漲的背離」、後續又出現「趨勢下跌的背離」的複合案例，顯示兩種背離尺度可先後出現於同一走勢中。

> 補充區辨：第5、6節的「高檔背離／低檔背離」（p.155、p.157）是以「股價創新高/新低 1~3 次」對比「單一項紅柱或綠柱峰谷是否同步創新高/新低」；而本節 7-2 的「趨勢背離」則明確以「兩波段的頭與頭比較、底與底比較」為定義依據，兩者概念相通但比較的對象顆粒度不同（單次極值 vs. 兩波段極值群組），實作規則庫時建議分開建模。

## 8. 規則彙整速查表（供後續程式化參考）

| 類別 | 條件 | 結論 |
|---|---|---|
| 計算公式 | DIF = MA12(close) − MA26(close) | 快線 |
| 計算公式 | MACD = MA9(DIF) | 慢線／訊號線 |
| 計算公式 | OSC = DIF − MACD | 柱狀圖，正紅負綠 |
| 多頭 | 0軸上黃金交叉 | 買進訊號 |
| 多頭 | 0軸上死亡交叉、未破0軸 | 回檔（非反轉） |
| 空頭 | 0軸下黃金交叉、未站上0軸 | 反彈（非反轉） |
| 空頭 | 0軸下死亡交叉 | 賣出/做空訊號 |
| OSC柱狀 | 紅柱漸增/漸減 | 多方動能增強/減弱 |
| OSC柱狀 | 綠柱漸增/漸減 | 空方動能增強/減弱 |
| 背離(波段) | 價創高/低，同波段OSC柱縮短 | 波段高檔/低檔背離 |
| 背離(趨勢) | 價頭頭高/底底低，OSC柱峰谷頭頭低/底底高 | 趨勢級高檔/低檔背離 |
| 背離(高低檔) | 價創新高/低1~3次，OSC峰谷未同步創新高/低 | 做頭/打底機會大 |

## 本章補充：外部查證備註（2026-07-23，工程實作用，非書中原文）

對應規則庫 **R-INDICATOR-04（MACD紅柱多頭動能7要點）**與**R-INDICATOR-05（MACD綠柱空頭動能7要點）**：第5、6節（p.155、p.157）第7項「高檔/低檔背離」以「股價創新高/新低1~3次」定義比較窗口，查證外部技術分析資料，MACD背離的標準定義僅為單次新高/新低與OSC峰谷的比較（來源：[QuantPass MACD教學](https://quantpass.org/macd-2/)），**查無「比較窗口應涵蓋幾次新高/新低」的公認統一數字**，書中「1~3次」屬原書經驗描述，程式化時比較窗口N需自訂為工程估計值，非外部佐證數字。
