## 一、盤整趨勢的定義與成立條件（上冊 p.097-098）

- 當趨勢不是多頭，也不是處在空頭狀態，就代表是正處於盤整的趨勢中（上冊 p.097）。
- [可能規則] 依據道氏觀察理論：波浪走勢橫向盤整時，股價上下不規則波動，即波浪的底部與頭部會**相互交叉**；因此，除了多頭上漲、空頭下跌之外的橫向走勢，均歸為盤整波（上冊 p.097）。
- [可能規則] **盤整趨勢的4種型態**（圖表2-4-1）：
  1. 三角收斂盤整（頭愈來愈低、底愈來愈高，向中間收斂）
  2. 矩形盤整（頭大致等高、底大致等低，區間橫盤）
  3. 上升直角三角盤整（底逐漸墊高、頭大致等高）
  4. 下降直角三角盤整（頭逐漸走低、底大致等低）

重點說明（上冊 p.097-098）：
1. 不是多頭的架構，不是空頭的架構，即為盤整。
2. [可能規則] 確認盤整的股票操作原則：①不要操作、②畫上頸線及下頸線、③等待盤整結束後的突破做多或跌破做空。

## 二、盤整趨勢的操作策略（上冊 p.098-099）

- [可能規則] 1. 靠近下頸線未跌破，向上漲時買；靠近上頸線未突破，向下跌時賣。
- 2. 前面是多頭時，盤整區間**先買後賣**；前面是空頭時，盤整區間**先賣後買**（順勢操作，即依原趨勢方向操作盤整區間）。
- [可能規則] 3. 盤整區間幅度太小，或橫向盤整時交錯雜亂、無法確認波浪型態時，**不要進場交易**（**盤整上下幅度小於15%，則不宜在盤整區中進出**）。
- 4. 當盤整架構沒有改變之前，不能自己猜測未來方向要如何發展。
- 5. 當盤整末端**向上突破或向下跌破頸線**時，才能確定盤整結束後的方向，此時再決定做多或是做空。

盤整波浪型態走勢圖及進出場位置重點說明（圖表2-4-2，上冊 p.098-099）：
1. 盤整下頸線。
2. 盤整上頸線。
3. 下頸線買點。
4. 上頸線賣點。
5. 跌破下頸線的賣點。
6. 突破上頸線的買點。

盤整趨勢進出場操作範例①②③（圖表2-4-3～2-4-5，上冊 p.099-100）：
- 範例①：盤整區上頸線黑K下跌做空→盤整區下頸線支撐紅K上漲回補→下頸線支撐紅K上漲做多→接近上頸線黑K下跌停利→下頸線支撐紅K上漲做多→遇上頸線壓力黑K下跌停利→下頸線支撐紅K上漲做多→大量長紅K突破盤整區上頸線，多單加碼（共8個標記步驟）。
- 範例②：①底部多頭確認 → ②多頭上漲未過前高、頭頭低進入盤整 → ③下跌未破前低盤整確認 → ④大量K線上漲、突破盤整多單買進。
- 範例③：①頭部空頭確認 → ②空頭下跌未破前低、底底高進入盤整 → ③上漲未過前高盤整確認 → ④大量黑K線下跌、跌破盤整空單做空 → ⑤長黑K下跌、空頭彈後空下跌，空單做空。

## 三、盤整趨勢結束後操作步驟與實戰（上冊 p.101-103）

- 盤整可以是**反轉型態**，也會是**中繼型態**。當多頭「頭頭高、底底高」2個條件中任何一個條件改變，多頭趨勢進入盤整（此時要畫頸線）；當確認另一條頸線被突破/跌破時，盤整結束確認（上冊 p.101）。

- 空頭突破盤整反轉確認（圖表2-4-6）：盤整區上頸線被突破，反轉確認為多頭。
- 多頭突破盤整續勢上漲（圖表2-4-7）：盤整區下頸線不破，續勢維持原多頭方向上漲。

盤整趨勢操作範例（圖表2-4-8，上冊 p.101-102）：以個股中軸（示例代號中軸2400）2016/10/21日線圖為例：
1. 反彈突破前高，出現頭頭高。
2. 下跌未破前高，大量長紅K上漲，出現底底高，多頭確認。
3. 上漲未過前高黑K線下跌，進入盤整（畫出上頸線、下頸線）。
4. 大量長紅K上漲突破上頸線，盤整突破，做多買進。
