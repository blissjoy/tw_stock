## 一、實體紅K線 (p.213-217)

### 1. 長／中／小紅K的量化門檻 [可能規則]
書中用「開盤到收盤的漲幅百分比」（即 (收盤-開盤)/開盤價）來分類紅K的實體大小，而不是用「實體占最高減最低全距」的比例（圖表3-4-1，p.213）：

| 分類 | 門檻（開盤到收盤漲幅%） | 多方力道評語 |
|---|---|---|
| 長紅K | > 6.5% | 多方力道最強 |
| 中紅K | 3.5% ～ 6.5% 之間 | 多方力道強 |
| 小紅K | < 3.5% | 多方力道小強 |

[可能規則] 判斷式草案：`body_pct = (close - open) / open`；`body_pct > 0.065` → 長紅K；`0.035 <= body_pct <= 0.065` → 中紅K；`body_pct < 0.035` → 小紅K（僅在收盤>開盤、即紅K時適用）。

### 2. 多頭上漲趨勢中紅K的意義 (p.213-214)
逐項列出紅K在多頭走勢不同位置的解讀（原文編號1-7）：
1. 出現在**底部打底期間**：代表主力在接籌碼，常伴隨量增，但因主力尚未布局完成，不會立刻出現續漲攻勢。
2. 出現在**底部打底完成、向上收盤突破底部盤整**：通常配合攻擊量，是多頭確認的長紅K。[可能規則：突破盤整區的長紅K+放量 = 多頭確認訊號]
3. **多頭上漲行進中**的紅K：代表多方氣勢不墜，仍會續漲，此時常見「惜售量縮」。
4. **上漲一段後回檔修正、修正完成**：低點沒有跌破前面低點，此時出現上漲紅K，表示修正結束，多頭要繼續上漲，應把握買進做多機會。
5. **多頭上漲一段後的盤整末端**，出現大量紅K突破盤整：是多頭上漲的攻擊訊號，應把握機會買進做多。
6. **多頭上漲到高檔或連續急漲**後的紅K：要特別注意「成交量爆增、價量背離」，若遇壓力股價不漲或下跌，通常是主力在高檔利用散戶追高出量趁機出貨。
7. **多頭高檔盤頭區間**出現大量紅K：是主力分批出貨的訊號，若之後跌破盤整、空頭確認，代表頭部大量為主力出貨。

搭配圖表3-4-2「多頭趨勢不同位置的大量長紅K線」（p.214）標示了打底→多頭起漲→多頭確認→盤整突破→多頭高檔→多頭行進→開始盤整→空頭確認 的完整位階序列示意圖，以及圖表3-4-3、3-4-4 兩個實際個股範例（p.215）。

### 3. 空頭下跌趨勢中紅K的意義 (p.216)
1. 出現在**空頭下跌行進中的反彈紅K**：代表主力誘多，常伴隨量增，此類反彈不可搶進，容易被套；若反彈無量，則更不容易反彈，此時應觀察反彈不過前高的做空機會。[可能規則：空頭趨勢中反彈紅K，若量縮=反彈力道弱，可留意逢高放空]
2. 空頭跌到**低檔或急跌一段後**，出現紅K的止跌，或紅K突破前一根紅K的最高點：視為反彈信號。

搭配圖表3-4-5、3-4-6「空頭趨勢的紅K線範例①②」（p.217）。

### 4. 紅K進場注意位置（實戰篩選規則）(p.216) [可能規則]
**什麼位置的大量紅K可以買？**
1. 空頭轉多頭第1次過前高，多頭確認的大量長紅K（**須完成均線3線多排向上**）。
2. 多頭回檔止跌回升的大量長紅K。
3. 走勢當中盤整完成向上突破的大量長紅K。
4. 型態確認突破的大量長紅K。

**什麼位置的大量紅K不能買？**
1. 日線上漲3天以上、高檔的大量紅K。
2. 上漲接近壓力位置前的大量紅K。
3. 空頭反彈的大量紅K。
4. 多頭回檔跌破月線，在月線下的上漲大量紅K——要等站上月線紅K再買。
5. 多頭回檔跌破前低後出現上漲的大量紅K——趨勢已改變成為盤整。

## 二、實體黑K線 (p.218-221)

原文特別提醒：出現黑K線，**並不表示日後一定會繼續下跌**（p.218），解讀仍須看趨勢位置。

### 1. 長／中／小黑K的量化門檻 [可能規則]
與紅K完全對稱，同樣用「開盤到收盤跌幅%」分類（圖表3-4-7，p.218）：

| 分類 | 門檻（開盤到收盤跌幅%） | 空方力道評語 |
|---|---|---|
| 長黑K | > 6.5% | 空方力道最強 |
| 中黑K | 3.5% ～ 6.5% 之間 | 空方力道強 |
| 小黑K | < 3.5% | 空方力道小強 |

[可能規則] `body_pct = (open - close) / open`（黑K時，close<open），門檻同紅K：>6.5%長黑K、3.5%-6.5%中黑K、<3.5%小黑K。

### 2. 打底到多頭上漲趨勢中黑K的意義 (p.218-219)
1. 出現在**空頭低檔打底盤整期間**：代表底部尚未確認，可能是主力再次洗盤，同時仍沒有止跌訊號，**不可以認為股價跌深而去做多**。[可能規則：低檔打底階段黑K不構成止跌訊號，須額外等待確認]
2. **多頭向上行進中出現黑K**：通常是漲多的賣壓出現，容易回檔；若多頭趨勢未變，黑K下跌力道會隨量縮而減弱，屬於多頭回檔的正常走勢。
3. **多頭上漲到高點出現黑K**：是止漲的訊號，若伴隨大量或爆天量，主力出貨可能性很大，須密切注意次日股價走勢，一旦出現下殺要立刻出場。[可能規則：高檔黑K+大量/爆量 = 出場警訊，觀察次日是否下殺]

### 3. 空頭下跌趨勢中黑K的意義 (p.219-221)
1. **高檔出現頭頭低**，第2個頭出現長黑K的下跌：一般容易轉成空頭，有量的下跌威力較大，**無量也會下跌**。
2. **空頭初跌段**出現黑K：通常都會出量，行情繼續下跌。
3. **空頭下跌行進中**出現黑K，股價繼續下跌，沒有止跌訊號出現：無論是否有量，**都不能低接**。
4. **連續出現下跌的長黑K**，之後若再出現爆量或窒息量，容易反彈；當有上漲的紅K收盤過前一日的高點，可做短線的搶反彈。[可能規則：連續長黑後爆量/窒息量+隔日紅K收盤過前高 = 短線反彈訊號]
5. **空頭下跌行進中出現盤整**，當出現黑K跌破盤整，股價將繼續下跌一段，是做空的機會。

## 三、實體紡錘K線 (p.221-222)

### 型態定義
- **外型特徵**：當天實體上方有壓力、下方有支撐；**上下影線都很短，影線長度沒有超過實體棒的二分之一**（p.221）。[可能規則] `upper_shadow <= body/2 且 lower_shadow <= body/2`。
- 注意：此處書中定義的「紡錘K」不同於一般西方蠟燭圖術語「Spinning Top」（後者通常強調上下影線都長）；本書的紡錘K是**短影線、小實體**的概念，需比照紅K/黑K實體棒的長中小分類方式套用。
- 分為「紅K紡錘線」與「黑K紡錘線」兩種（圖表3-4-12，p.221）：
  - 紅K紡錘線：多空拉鋸，**多方力道強**。
  - 黑K紡錘線：多空拉鋸，**空方力道強**。
- **位階意義**：書中未對紡錘K線另外區分高檔/低檔的不同解讀規則，僅提示「次日再看K線的變化」，即紡錘K本身不足以判斷轉折，需等待次日K線確認方向。
- 範例：圖表3-4-13「不同位置的紡錘K線範例」（p.222），在同一走勢圖上標出多個高低點的紅K紡錘/黑K紡錘。

## 四、變盤槌子與倒鎚K線 (p.222-225)

### 型態分類（圖表3-4-14，p.222）
書中把槌子/倒槌依「顏色 x 形狀」分成4種，但**判斷邏輯完全相同**：
- 變盤紅K槌子
- 變盤黑K槌子
- 變盤紅K倒槌
- 變盤黑K倒槌

四種的圖說一致寫著：**「高檔易下跌，低檔易反彈」**——即書中不因顏色或槌子/倒槌形狀不同而給予不同的多空傾向，真正決定意義的是「出現的價格位階」。

### 外型特徵與量化門檻（重點說明，p.223）[可能規則]
1. 小實體在上方或下方（即槌子=下影線長、實體在上方；倒槌=上影線長、實體在下方），**實體的紅黑不重要**。
2. **下影線或上影線很長，至少是實體的2倍以上**。[可能規則] `shadow_length >= 2 * body_length`（槌子看下影線，倒槌看上影線）。

### 位階意義與次日確認規則（重點說明3-5，p.223）[可能規則]
3. **強勢上漲到高點，出現槌子或倒槌**，次日走勢非常重要：
   (1) 次日**向下跳空大跌** → 多頭結構可能反轉。
   (2) 次日**收盤跌破槌子或倒槌線的最低點** → 轉折向下確認。
4. **急速下跌到低點，出現槌子或倒槌**，次日走勢非常重要：
   (1) 次日**向上跳空大漲** → 空頭結構可能反轉。
   (2) 次日**收盤突破槌子或倒槌線的最高點** → 轉折向上確認。
5. **若在槌子或倒槌線出現大成交量，轉折的機率更高**。[可能規則：槌子/倒槌 + 大量 = 提高轉折訊號可信度的加權因子]

### [可能規則] 完整判斷邏輯彙整
```
若 shadow_length(上或下) >= 2 * body_length：
    候選 = 槌子（下影線長）或倒槌（上影線長），顏色不影響判斷
    若候選出現在「高檔」（強勢上漲後）：
        次日向下跳空大跌 → 反轉向下（強確認）
        次日收盤跌破該K線最低點 → 反轉向下（確認）
        若當根K線為大量 → 提高轉折機率
    若候選出現在「低檔」（急速下跌後）：
        次日向上跳空大漲 → 反轉向上（強確認）
        次日收盤突破該K線最高點 → 反轉向上（確認）
        若當根K線為大量 → 提高轉折機率
```

## 本章補充：外部查證備註（2026-07-22，工程實作用，非書中原文）
