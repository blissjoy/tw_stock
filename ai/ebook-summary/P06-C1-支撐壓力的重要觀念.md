## 一、支撐與壓力的基本定義（p.021）

- 技術分析兩大探討核心：「波浪趨勢」與「支撐、壓力」。趨勢行進中，多頭上漲到某位置會出現停滯盤整或回檔下跌，空頭下跌到某位置會出現止跌不跌或止跌回升——這些止漲/止跌的位置，就是壓力/支撐。
- **支撐的成因**：某個位置出現大量的買進力量，把股價拉抬上漲；因此在該位置下方買進的投資人，已經處於獲利狀態，會形成一股「捍衛獲利」的支撐力量。
- **壓力的成因**：某個位置出現大量的賣出力量，把股價打壓下跌；因此在該位置上方買進的投資人，被套牢在裡面，會形成一股「等解套」而在該價位附近出脫持股的壓力。
- 實務意義：事先掌握支撐、壓力位置，就能大概推估股價會漲到哪裡、跌到哪裡，進而掌握股價漲跌的節奏。

## 二、支撐與壓力並非絕對（p.021）

- **核心觀念：支撐與壓力都不是絕對的**，走勢中隨時可能被突破（壓力）或被跌破（支撐）。
- 因此是否真的「有」支撐/壓力，**必須等技術面上股價的反應確認後**才能判斷，不能單憑「靠近某條均線／某個價位」就直接認定有效。
- 書中舉例說明風險：多頭回檔遇到上揚的月線支撐，如果沒有先確認真的有支撐力道就先行買進，萬一次日繼續破線下跌、跌破月線，就會被套牢。
- 圖表6-1-1（p.022，範例股票代號6142）示範同一條月線在不同時間點的「有支撐」與「沒有支撐」兩種結果並存於同一張走勢圖上——說明同一條均線本身不保證每次都有效，必須逐次用當下K線反應去確認。[可能規則：不可僅憑「股價接近月線/趨勢線」即判定有支撐，須待當根或次一根K線收盤反應確認後才成立]

## 三、面臨支撐與壓力的可能變化——用當日K線變化研判（p.022–026）

這是全章的方法論核心：**不是看「價格有沒有到達某個位置」，而是看「到達之後當天K線收出什麼型態、次日如何反應」，來確認支撐/壓力是否真的成立**。以下兩組各4條判斷法則，是本章唯一具體、可操作的訊號規則來源。

### （一）接近壓力時的變化（p.022–023，圖表6-1-2、6-1-3）

當股價上漲接近壓力位置時，看當日K線的變化來判斷是否有壓力：

1. **遇壓力當天出現大量中長紅K**：代表多方買盤仍強，此時要注意**次日**是否能過壓力——如果次日收下跌黑K，即所謂「**闖關前爆大量，股價不漲要回檔**」，確認遇壓。[可能規則：壓力位當天大量中長紅K → 次日收黑K → 判定遇壓回檔]
2. **遇壓力當天出現大量長上影線K或中長黑K**：代表盤中曾一度衝過壓力，但收盤沒有真正站上壓力之上，表示當天在壓力處出現賣壓，**次日容易下跌**。[可能規則：壓力位當天大量長上影線／中長黑K（盤中破壓但收盤未站穩）→ 次日容易下跌]
3. **遇壓力當天出現大量變盤線**（十字線一類，多空雙方仍在膠著）：需再觀察**次日**股價變化——如果次日收下跌黑K，股價就確認遇壓回檔。[可能規則：壓力位當天大量變盤線 → 次日收黑K → 確認遇壓回檔]
4. **遇壓力當天出現大量中長紅K，且成功突破壓力**：仍要注意次日是否出現大量長黑K下跌——書中提醒此種情況「**假突破的機率很高**」，要預先做好多單停損的準備。[可能規則：突破壓力當天大量長紅K，次日卻出現大量長黑K → 高機率為假突破，須執行多單停損]

（圖表6-1-2「接近壓力的股價變化①」以個股（代號6242）為例，標示①大量紅K次日不漲、股價回檔；②大量變盤線、次日不漲股價回檔；③大量長紅突破壓力；④大量長黑下跌、昨日假突破。圖表6-1-3「接近壓力的股價變化②」以個股（代號5351）為例標示①②大量長上影線有壓、③④大量長黑/長紅有壓、⑤大量長黑有壓，均呼應上述4條法則。）

### （二）接近支撐時的變化（p.024–026，圖表6-1-4、6-1-5、6-1-6）

當股價下跌接近支撐位置時，看當日K線的變化來判斷是否有支撐（與壓力判斷邏輯完全對稱、方向相反）：

1. **遇支撐當天出現大量中長黑K**：代表空方賣盤仍強，注意**次日**是否會跌破支撐——如果次日收上漲紅K，即所謂「**過撐爆大量，股價不跌要反彈**」，確認有撐。[可能規則：支撐位當天大量中長黑K → 次日收紅K → 判定遇撐反彈]
2. **遇支撐當天出現大量長下影線K或中長紅K**：代表盤中曾一度跌破支撐，但收盤沒有真正跌破，表示當天在支撐處出現買盤力道，**次日容易反彈**。[可能規則：支撐位當天大量長下影線／中長紅K（盤中破撐但收盤未失守）→ 次日容易反彈]
3. **遇支撐當天出現大量變盤線**（多空雙方還在膠著）：需再觀察**次日**股價變化——如果次日收上漲紅K，股價就確認遇支撐反彈。[可能規則：支撐位當天大量變盤線 → 次日收紅K → 確認遇撐反彈]
4. **遇支撐當天出現大量中長黑K，且真的跌破支撐**：仍要注意次日是否出現大量長紅K上漲——書中提醒「**假跌破的機率很高**」，要預先做好空單停損的準備。[可能規則：跌破支撐當天大量中長黑K，次日卻出現大量長紅K → 高機率為假跌破，須執行空單停損]

（圖表6-1-4「接近支撐的股價變化①」（代號6732）示範「長下影有支撐」及後續「跌破支撐」兩種結果並列；圖表6-1-5「接近支撐的股價變化②」（代號1524）標示「大量長上影線有壓」「上升切線支撐」「長下影線有支撐」「跌破支撐」；圖表6-1-6「接近支撐的股價變化③」（代號3105）標示「假跌破」與「真上漲」＋「晨星反彈」，直接對應第4條法則的假跌破後反彈情境。）

> 兩組法則本質上是同一套判讀邏輯的鏡像：**看該根K線「盤中有沒有測試到支撐/壓力」＋「收盤有沒有站穩/守住」＋「隔日確認方向」三步驟**，而非單看價格本身是否觸及該位置。「量」是關鍵前提——書中4條法則的前提幾乎都是「大量」，代表這是判斷多空力道是否真的在該價位交手的必要條件。

## 四、支撐與壓力的進出位置判斷（p.026–027）★重點小節★

這是本章使用者特別要求要寫清楚的核心方法：**在什麼樣的「趨勢位置」上，遇到什麼樣的支撐/壓力，才是真正可以進場的關鍵買賣點**。書中分成多頭關鍵支撐買點、空頭關鍵壓力空點兩組，各列4種具體支撐/壓力來源。

### （一）多頭趨勢中關鍵支撐買進訊號（p.026–027，圖表6-1-7）

在多頭趨勢中，股價回檔遇到以下4種支撐位置之一，且出現「止跌的紅K」或「長下影線」時，即為準備次日上漲的買點：

1. **月線、季線支撐**：多頭回檔遇到月線（月均線）或季線（季均線）支撐，出現止跌的紅K或長下影線，是準備次日上漲的買點。[可能規則：多頭回檔觸及月線/季線＋當日收止跌紅K或長下影線 → 次日買進訊號]
2. **上升切線支撐**：多頭回檔遇到上升切線（即由兩個轉折低點連線畫出的上升趨勢線）的支撐，出現止跌的紅K或長下影線，是準備次日上漲的買點。[可能規則：多頭回檔觸及上升切線＋止跌紅K或長下影線 → 次日買進訊號]
3. **前低支撐**：多頭回檔遇到前一波的轉折低點（前低）支撐，出現止跌的紅K或長下影線，是準備次日上漲的買點。[可能規則：多頭回檔觸及前低價位＋止跌紅K或長下影線 → 次日買進訊號]
4. **下方大量跳空缺口支撐**：多頭回檔遇到下方出現過大量的跳空缺口作為支撐，出現止跌的紅K或長下影線，是準備次日上漲的買點。[可能規則：多頭回檔觸及下方大量跳空缺口價位＋止跌紅K或長下影線 → 次日買進訊號]

（圖表6-1-7「多頭支撐的買進位置」以個股（代號2301）為例，同時標示「月線支撐」「前低支撐」「季線支撐」三種支撐同框出現的走勢圖。）

四種支撐來源的共同判斷條件都是：**支撐位置本身（月線/季線/上升切線/前低/跳空缺口）＋當天在該位置收出止跌訊號K棒（紅K或長下影線）**，兩者缺一不可，才成立「次日買進」訊號——單純價格碰到支撐線不足以構成買點，必須有當日K棒的止跌反應確認。

### （二）空頭趨勢中關鍵壓力做空訊號（p.027–028，圖表6-1-8）

在空頭趨勢中，股價反彈遇到以下4種壓力位置之一，且出現「止漲的黑K」或「長上影線」時，即為準備次日下跌的空點（與多頭支撐買點邏輯完全對稱、方向相反）：

1. **月線、季線壓力**：空頭反彈遇到月線或季線壓力，出現止漲的黑K或長上影線，是準備次日下跌的空點。[可能規則：空頭反彈觸及月線/季線＋止漲黑K或長上影線 → 次日放空訊號]
2. **下降切線壓力**：空頭反彈遇到下降切線（由兩個轉折高點連線畫出的下降趨勢線）的壓力，出現止漲的黑K或長上影線，是準備次日下跌的空點。[可能規則：空頭反彈觸及下降切線＋止漲黑K或長上影線 → 次日放空訊號]
3. **前高壓力**：空頭反彈遇到前一波的轉折高點（前高）壓力，出現止漲的黑K或長上影線，是準備次日下跌的空點。[可能規則：空頭反彈觸及前高價位＋止漲黑K或長上影線 → 次日放空訊號]
4. **上方大量跳空缺口壓力**：空頭反彈遇到上方出現過大量的跳空缺口作為壓力，出現止漲的黑K或長上影線，是準備次日下跌的空點。[可能規則：空頭反彈觸及上方大量跳空缺口價位＋止漲黑K或長上影線 → 次日放空訊號]

（圖表6-1-8「空頭壓力的做空位置」以個股（代號3481）為例，同時標示「下降切線壓力」「前高壓力」「月線壓力」三種壓力同框出現的走勢圖。）

## 五、不同趨勢支撐、壓力的慣性變化（p.028–029）

本節說明：**支撐/壓力被觸及後的後續反應，可以回過頭來反推目前趨勢的強弱與是否延續**，是進出位置判斷之外，另一層用支撐壓力反應「確認/推翻」原趨勢的方法。

### （一）多頭趨勢支撐、壓力的變化（p.028–029，共5條）

在多頭走勢中觀察支撐與壓力的變化，可以掌握趨勢的強弱與趨勢是否改變：

1. **強勢多頭判斷**：多頭沿著10日均線上漲，回檔不破10日均線，為強勢多頭。[可能規則：股價持續運行於10日均線之上、回檔不破10MA → 判定強勢多頭]
2. **多頭趨勢不變**：多頭回檔不破前低支撐，且維持在月線之上、月線上揚，表示多頭趨勢不變，可以繼續做多。[可能規則：回檔守住前低＋守在月線上方＋月線上揚 → 多頭趨勢延續，續抱/續做多]
3. **多頭沒有改變 vs. 進入盤整**：多頭上漲突破前高壓力，表示多頭沒有改變；若上漲沒有突破前高壓力反而下跌，則表示多頭進入盤整。[可能規則：測試前高——突破＝多頭延續；未突破轉跌＝轉入盤整]
4. **多單逃命波賣點（趨勢改變警訊）**：多頭回檔跌破前低支撐，表示多頭趨勢改變；此後若再反彈卻無法突破前高壓力，即是多單逃命波的賣點。[可能規則：跌破前低（趨勢轉變）→ 之後反彈不過前高 → 執行多單逃命賣出]
5. **月線轉弱的壓力效應（3～5日視窗）**：多頭回檔跌破月線支撐，表示多頭趨勢轉弱；若3～5日內無法重新站上（突破）月線，月線會容易下彎，進而形成對股價的壓力。[可能規則：跌破月線後設定3～5個交易日觀察窗，若逾期未能收復月線 → 月線由支撐轉為壓力，趨勢判定轉弱]

### （二）空頭趨勢支撐壓力的變化（p.029，共5條，與多頭鏡像對稱）

在空頭走勢中觀察支撐與壓力的變化，可以掌握趨勢的強弱與趨勢是否改變：

1. **弱勢空頭判斷**：空頭沿著10日均線下跌，反彈沒有突破10日均線，為弱勢空頭。[可能規則：股價持續運行於10日均線之下、反彈不破10MA → 判定弱勢空頭]
2. **空頭趨勢不變**：空頭反彈沒有突破前高壓力，且維持在月線之下、月線下彎，表示空頭趨勢不變，可以繼續做空。[可能規則：反彈受阻於前高＋守在月線下方＋月線下彎 → 空頭趨勢延續，續抱空單/續做空]
3. **空頭沒有改變 vs. 進入盤整**：空頭下跌跌破前低支撐，表示空頭沒有改變；若下跌沒有跌破前低支撐反而上漲，則表示空頭進入盤整。[可能規則：測試前低——跌破＝空頭延續；未跌破轉漲＝轉入盤整]
4. **空單逃命波回補點（趨勢改變警訊）**：空頭反彈突破前高壓力，表示空頭趨勢改變；此後若再下跌卻無法跌破前低支撐，即是空單逃命波的回補點。[可能規則：突破前高（趨勢轉變）→ 之後下跌不破前低 → 執行空單回補]
5. **月線轉強的支撐效應（3～5日視窗）**：空頭反彈突破月線壓力，表示空頭趨勢轉強；若3～5日內無法重新跌破月線，月線會容易上揚，進而形成對股價的支撐。[可能規則：突破月線後設定3～5個交易日觀察窗，若逾期未能失守月線 → 月線由壓力轉為支撐，趨勢判定轉強]

### 本節收尾要點（p.029）

- 支撐、壓力因「趨勢、位置、成交量」等因素，力量大小各不相同。
- 實務應用：上漲時可先評估上方壓力是否過大（過大則突破機率不高）；下跌時可先評估下方某支撐力道是否很強（很強則大概不易被跌破）——藉此預先做好操作計畫，避免行情到位時手忙腳亂。
- 書中明示：不同支撐與壓力的「強弱判斷」方法，留待下一章（6-2 走勢圖中的支撐、壓力位置）詳述，本章只建立基本觀念與進出訊號判斷法則。

## 全章結構小結

本章（6-1）在全書支阻篇中扮演「奠基」角色，建立三層觀念，依序遞進：

1. **支撐/壓力為何存在**（成因：大量買/賣力量造成的獲利/套牢心理）與**不是絕對**（p.021）。
2. **如何用當日K線反應確認支撐/壓力是否成立**——遇壓4法則、遇撐4法則，共用「量＋當日K棒型態＋次日確認」三步驟框架（p.022–026）。
3. **具體進出場位置與訊號**——多頭4種關鍵支撐買點來源（月線/季線、上升切線、前低、下方大量跳空缺口）與空頭4種關鍵壓力空點來源（月線/季線、下降切線、前高、上方大量跳空缺口），皆需搭配止跌/止漲K棒確認（p.026–028）；以及**用支撐壓力反應回推趨勢強弱**的多頭5條、空頭5條慣性變化法則，其中月線的3～5日觀察窗是可量化的具體規則（p.028–029）。

下一章「6-2 走勢圖中的支撐、壓力位置」（p.030起）將進一步詳述不同支撐與壓力的強弱判斷方法，屬於本章的延伸，不在本筆記處理範圍內。
