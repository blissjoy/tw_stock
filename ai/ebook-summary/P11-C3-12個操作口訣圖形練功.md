## 口訣1：多頭大量不漲，股價要回檔（當日或後數日）

**口訣原文**（下冊 p.329）
「多頭大量不漲，股價要回檔（當日或後數日）。」

**對應圖形判斷條件**
1. `[可能規則]` 前提：股價處於**多頭上升趨勢**中。
2. `[可能規則]` 訊號：某交易日**成交量明顯放大**（大量），但當日股價**漲幅有限或收黑**（不漲）。
3. `[可能規則]` 結果：訊號成立後，股價會在**當日或接下來數個交易日**出現回檔整理，非立即發生也算成立。
4. `[可能規則]` 此型態可在盤整區反覆測試前高時多次出現（提示上方賣壓未消化），也可能出現在波段最高點，作為頭部反轉的警訊——需搭配所在位置（盤整中段 vs 波段高點）判斷「大量不漲」是短暫回檔還是趨勢反轉。

**書中範例**（下冊 p.329–330，圖表11-3-1～11-3-4，共4例）
- 圖表11-3-1（代碼2603）：走勢中多次於前波高點附近爆量卻未續漲（標示「大量不漲」），股價隨後拉回；最後一次「大量不漲」出現在波段最高點，成為頭部訊號。
- 圖表11-3-2（代碼1309）：同一走勢中「大量不漲」重複出現於相近的壓力水平，顯示此水平的賣壓區。

- 圖表11-3-4（代碼1737）：連續3次「大量不漲」對應3個不同轉折高點，最末端標示「空頭確認」，示範口訣1可重複驗證於同一走勢的多個節點，並在最後一次演變為趨勢反轉。

## 口訣3：多頭利多不漲，主力出貨做頭

**口訣原文**（下冊 p.334）
「多頭利多不漲，主力出貨做頭。」

**對應圖形判斷條件**
1. `[可能規則]` 前提：個股處於多頭高檔區間，且傳出**利多消息**。
2. `[可能規則]` 訊號：利多消息公布後，股價**未能同步上漲**（利多不漲）。
3. `[可能規則]` 意涵：代表主力正藉利多出貨（利用市場對利多的樂觀情緒出脫持股），而非續抱做多。
4. `[可能規則]` 結果：隨後型態轉為「空頭確認」，趨勢由多轉空，形成頭部。

## 口訣5：多頭該回不回，過高要大漲

**口訣原文**（下冊 p.335）
「多頭該回不回，過高要大漲。」

**對應圖形判斷條件**
1. `[可能規則]` 前提：股價處於上升趨勢，依技術面（如漲多、觸及壓力等）理應出現回檔整理。
2. `[可能規則]` 訊號：股價**遲遲不回檔**（該回不回），顯示多方力道極強、賣壓輕微。
3. `[可能規則]` 結果：股價隨後直接創新高，並出現顯著漲勢（過高大漲）。
4. `[可能規則]` 型態常以階梯狀反覆出現：「該回不回→過高大漲→再度該回不回→再過高大漲」，可作為多頭趨勢強度的持續驗證工具，每一次「該回不回」都是加碼或續抱的訊號，而非停利訊號。

## 口訣9：晨星多方主控，夜星空方主控，晨星對夜星，強多對強空

**口訣原文**（下冊 p.343）
「晨星多方主控，夜星空方主控，晨星對夜星，強多對強空。」

**對應圖形判斷條件**
1. `[可能規則]` **晨星**（低檔出現的「長黑K＋跳空小實體或十字星＋長紅K」三根K棒組合）成立後，代表**多方取得主導權**，股價隨後出現反彈甚至大漲（標記「晨星反彈，多方主控」）。
2. `[可能規則]` **夜星**（高檔出現的「長紅K＋跳空小實體或十字星＋長黑K」三根K棒組合，為晨星的鏡射型態）成立後，代表**空方取得主導權**，股價隨後出現下跌（標記「夜星下跌，空方主控」）。
3. `[可能規則]` 當同一走勢中晨星與夜星交替出現，可用兩者的反彈/下跌力道對照，判斷該波段當下多空力量的相對強弱（「強多對強空」），力道明顯偏強的一方通常預示下一波段的主導方向。

## 口訣10：一星二陽，長紅跌破，近日易大跌；一星二陰，長黑突破，近日易大漲

**口訣原文**（下冊 p.345）
「一星二陽，長紅跌破，近日易大跌；一星二陰，長黑突破，近日易大漲。」

**對應圖形判斷條件**
1. `[可能規則]` **一星二陽**：K線序列為「1根星形線（十字星或小實體）＋隨後2根陽線」；若接下來出現的長紅K棒表面收紅，實際上卻**向下跌破**原有支撐（標記「跌破長紅」、「空頭確認」），代表這是假突破陷阱，近日容易出現大跌。
2. `[可能規則]` **一星二陰**：K線序列為「1根星形線＋隨後2根陰線」；若接下來出現的長黑K棒表面收黑，實際上卻**向上突破**原有壓力（標記「長黑突破」、「突破壓力」），代表這是假跌破陷阱，近日容易出現大漲。
3. `[可能規則]` 核心判讀邏輯：**K棒表面顏色（紅/黑）與其實際突破方向相反時，該K棒是騙線／洗盤動作**，後續真正走勢方向會與K棒表面顏色相反，須以「收盤價是否有效突破/跌破前波支撐壓力」為準，而非單看K棒紅黑。

**書中範例**（下冊 p.344–345，圖表11-3-32～11-3-34，共3例）

## 口訣11：關前放大量，股價不漲要回檔

**口訣原文**（下冊 p.346）
「關前放大量，股價不漲要回檔。」

**對應圖形判斷條件**
1. `[可能規則]` 前提：股價接近前面某個關鍵壓力價位（前高，「關前」）。
2. `[可能規則]` 訊號：股價於接近前高之處出現**爆大量**（「關前爆量」），但股價**未能有效上漲**（「大量不漲」），代表上方套牢籌碼／賣壓仍重、尚未消化完畢。
3. `[可能規則]` 結果：訊號成立後股價會回檔整理；此型態常在同一走勢中**重複出現於多次測試同一前高**的時點，須耐心等待爆量後股價真正站穩並突破，才視為壓力解除，否則每次爆量不漲都應視為回檔訊號。
4. `[可能規則]` 若反覆測試後始終無法放量突破，最終可能轉為「空頭確認」、「大跌」，即壓力關卡久攻不下轉為反轉向下的頭部型態。

## 口訣12：上漲高檔久盤必跌，下跌低檔久盤必漲

**口訣原文**（下冊 p.349）
「上漲高檔久盤必跌，下跌低檔久盤必漲。」

**對應圖形判斷條件**
1. `[可能規則]` **高檔久盤**：股價於相對高檔區間長時間橫盤整理（久盤）卻未能向上突破，整理時間越久，代表多方力道逐漸衰竭；最終容易向**下跌破**整理區間（「跌破久盤」），並出現顯著跌勢（「大跌」）。
2. `[可能規則]` **低檔久盤**：股價於相對低檔區間長時間橫盤整理，最終容易向**上突破**整理區間（「突破久盤」），並出現顯著漲勢（「大漲」）。
3. `[可能規則]` 核心判讀邏輯：盤整持續時間越長，代表多空籌碼交換、消化越徹底；一旦整理區間的上緣或下緣被有效突破/跌破，累積的能量容易加大釋放，形成較大幅度的後續走勢，而非小幅假突破。

## Part11 操作篇全章總結（第1～3章完整脈絡）

> 說明：`ai/ebook-summary/` 目錄下現已有完整的 `P11-C1-進場前的準備工作.md`、`P11-C2-進場實戰操作SOP.md` 檔案，可查閱完整細節；以下總結是撰寫當下直接讀取相關印刷頁面（下冊 p.310–311、p.326–327）取得的第一手內容整理而成，與正式筆記檔案內容一致。

操作篇（Part 11）共3章，脈絡是一條完整的「進場前準備 → 實戰SOP → 口訣總複習」操作鏈：

**① 11-1 進場前的準備工作（下冊 p.303左右起～p.319左右，依p.310–311內容推估）**
- 直接讀取到的內容（p.310–311）聚焦於**停損方法的建立**，是進場前必須先設定好的風險控制基礎建設，共列出5種停損法：
  1. K線停損法：以特定K棒的高低點作為停損依據。
  2. 趨勢停損：做多時，回後買上漲，停損設在轉折波低點；盤整突破買上漲，停損設在突破點。做空則對稱設於轉折波高點或跌破點。
  3. 均線停損：以進場依據的均線設停損，例如多頭上5日均線買進，跌破5日均價即停損。
  4. 固定比例停損：以買進價的2%～10%為停損點，依個人風險容忍度決定，一般可用固定5%。
  5. 支阻停損：做多時，關鍵支撐跌破停損；做空時，關鍵壓力突破停損。
- 這一章建立的是「進場之前，就先想好在哪裡認錯出場」的紀律基礎，是後續11-2 SOP與11-3口訣能夠被嚴格執行、不淪為紙上談兵的前提。

**② 11-2 進場實戰操作SOP（下冊 p.320左右起～p.327）**
- 直接讀取到的內容（p.326–327，「飆股操作範例②③④」）具體示範了一套完整的多頭飆股操作標準流程（SOP），可歸納為8～9個步驟：
  1. 底部型態確認（一字底突破／盤整突破／底部碎步上漲），多頭確認，進場做多。
  2. 均線4線多頭排列確認（或大量紅K突破上升軌道線，多方力道轉強）。
  3. 掌握飆股第1波漲勢。
  4. 飆股第1波達到停利點，出場。
  5. 股價回後再度上漲，重新進場做多。
  6. 掌握飆股第2波漲勢。
  7. 獲利累積超過30%～60%，若出現大量長黑K，飆股第2波停利出場。
  8. 長黑跌破5日均線後，股價一路下跌，確認離場時機正確。
- 這一章把11-1的風險控制原則，套用到一個具體、可重複操作的「多頭波段操作」完整週期上，示範「進場→抱波段→停利→回檔加碼→再抱波段→再停利→出場確認」的全流程。

**③ 11-3 12個操作口訣圖形練功（下冊 p.328–350，本檔案）**
- 本章是操作篇的總複習與精華濃縮：把11-1的風控紀律、11-2的SOP流程在**實際盤面每一個判斷節點**會遇到的12種常見量價／K線／趨勢反轉情境，濃縮成12句口訣、以44個實戰圖例逐一驗證。
- 這12個口訣可以對應嵌入11-2 SOP流程的不同節點使用，例如：
  - 執行SOP步驟①「進場做多」前，可用口訣3、4判斷當前高檔/低檔是否為主力出貨/進貨的假訊號。
  - 執行SOP步驟③⑥「掌握飆股波段」過程中，可用口訣1、2判斷途中的爆量不漲/不跌只是正常拉回或反彈，不必因此提早出場；可用口訣5、6判斷波段是否仍在「該回不回、該彈不彈」的強勢延續階段，決定是否續抱加碼。
  - 判斷SOP步驟④⑦「停利出場」時機時，可用口訣11判斷股價在關鍵壓力（前高）反覆爆量不漲，是停利了結的訊號；可用口訣12判斷高檔久盤是否即將轉跌，提前減碼。
  - 判斷趨勢是否真正反轉、而非僅是波段回檔時，可用口訣7、8（完成反轉後的大漲/大跌）與口訣9、10（晨星／夜星、一星二陽／二陰的K線騙線辨識）作最終確認。
- 換言之，操作篇三章的整體結構是：**11-1建立「怎麼認錯出場」的紀律底線 → 11-2建立「怎麼進場、抱單、停利」的標準作業流程 → 11-3提供「盤面每個當下該怎麼判讀」的12組快速辨識口訣**，三者合一構成一套完整、可執行、也具備程式化規則基礎的操作系統。

**整體規則庫建議串接順序**：
1. 用11-1的5種停損法，為任一進場部位預先計算並掛好停損價位。
2. 用11-2的SOP流程（底部確認→均線多排→波段判斷→停利→回後加碼→再停利→離場確認），建立操作的主幹狀態機。
3. 用11-3的12個口訣，作為SOP流程各節點上的訊號過濾器與時機判斷模組——尤其口訣1/2（量價背離型回檔/反彈過濾）、口訣5/6（波段延續強度判斷）、口訣7/8（真反轉確認）、口訣11/12（停利/停損時機的最後把關），皆具備明確的量價條件，適合優先轉換為程式化規則。
4. 口訣3/4（利多不漲/利空不跌）與口訣9/10（晨星夜星、一星二陽二陰）則需搭配基本面消息面資訊或多K棒組合型態辨識，程式化難度較高，可列為次優先開發項目。
