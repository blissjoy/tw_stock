**貫穿全章的核心操作型定義（「高檔」的判定方式）**：書中對「高檔」不是給絕對股價或漲幅百分比門檻，而是**相對位置**定義——「上漲高檔」= 多頭趨勢的上漲高點，**或**空頭趨勢中反彈的相對高點（p.161、p.164、p.183 三節開頭皆用幾乎相同的句式重申這個定義）。也就是說本章所有型態在「空頭反彈到高點」時同樣適用，只是解讀上偏向「反彈結束、準備做空」，而非「多頭做頭、轉折向下」。[可能規則] 判斷「高檔」可用：(a) 近期一段上升趨勢或反彈段的相對最高點／近期爆量的K棒位置，而非固定漲幅或乖離率門檻。

另一個貫穿全章的加分條件：**大量**與**KD死亡交叉／價量背離**。書中在每一節的「重點說明」中反覆出現「爆大量」「死亡交叉」使反轉機率提高的敘述，但均未給出具體量增倍數門檻（例如未寫「量增達幾倍」），只以「大量」帶過，程式化時需自行定義量能門檻（如近5日均量的N倍）。

## 一、高檔變盤線的轉折變化（p.161-163）

- **型態本質**：非2根以上K線組合，而是**單一根「變盤線」K棒**（或連續數天都出現變盤線），出現在高檔時視為止漲訊號。
- **圖表3-2-1「高檔變盤K線」列出的8種K棒外觀**（p.161）：墓碑線、長T線、十字線、紡錘線、反錘線（紅／黑皆可)、槌子線（紅／黑皆可)、跌停線、長黑線。書中文字寫「變盤訊號有6種變化觀察」，但圖示實際列出8種K棒外觀（紅/黑同形只算變化不算新增分類推測是分類方式差異，非資料缺漏，程式化時建議直接以圖示8種外觀為準）。
- **強化轉折機率的4個條件**（p.162 重點說明）：
  1. 高檔出現上述任一種K棒，統稱為「變盤線」，是止漲訊號，須留意是否反轉；多頭高檔＝轉折向下回檔訊號，空頭反彈高檔＝準備做空訊號。
  2. [可能規則] 高檔爆出**大量** + 出現**價量背離** + **KD指標死亡交叉** → 更容易反轉走跌（三個條件疊加）。
  3. [可能規則] 上升走勢遇到壓力位置 + 爆大量 + 股價收盤未突破該壓力 + 同時出現變盤K線 → 反轉機率更高，須密切注意次日股價。
  4. 出現變盤線後次日不一定馬上下跌，但**所有**轉折下跌的位置事後回顧都會先出現變盤線；因此高檔一出現變盤線，次日走勢就要特別留意（隱含：變盤線是必要不充分條件）。
- **實戰範例**（圖表3-2-2〜3-2-5，共4檔個股）：
  - 訊芯-KY(6451)，2022/08-09區間：先出現「多頭高檔變盤線」後股價下跌回檔，隨後反彈至「空頭反彈變盤線」再度轉跌，圖上同步標示KD死亡交叉與大量。
  - 新光鋼(2031)，2022/07-09區間：走勢與上例相同結構（多頭高檔變盤線→回檔→空頭反彈變盤線→續跌），伴隨死亡交叉與大量標記。
  - 力致(3443)，2022/06-12區間：多頭高檔變盤線出現在「遇壓力」位置（价格上攻壓力線附近），隨後下跌回檔，死亡交叉、大量同時出現。
  - 樺漢(6414)，2021/11-2022/02區間：連續多次出現「多頭高檔變盤線」與「空頭反彈變盤線」交替，圖上多處死亡交叉與大量標記，顯示變盤線可在同一走勢中重複出現多次確認轉折。

## 二、高檔紅K＋黑K線的轉折變化（p.164-182）

- **統整概念**（p.164）：上漲高檔（定義同前）出現「紅K次日接黑K」的2根K線組合，依兩者**開高低收相對位置**不同，分成6種型態，代表意義與反轉力道強弱不同。書中依力道由弱到強大致排列如下（第1種為基礎款，第6種力道最強）：
  1. 基本型態（長紅長黑）
  2. 長黑遭遇（一日封口）
  3. 長黑覆蓋（烏雲罩頂）
  4. 母子懷抱（不懷好意）
  5. 長黑吞噬（主力出貨）
  6. 長黑貫穿（一路向下）

> 注意：書中此6種型態各自用「一、二、三…」小標題呈現，與本章大節「一、二、三」同一視覺層級，但實質上都是「二、高檔紅K＋黑K線的轉折變化」底下的子型態，此筆記維持原書標題文字但用「型態1〜6」加註避免與大節編號混淆。

### 型態1（原書小標「一」）：高檔2根K線轉折向下的基本型態（p.165-168）

- **組成**：固定2根K線。左邊為上漲幅度達**3%以上**的中長紅K，右邊為下跌幅度達**3%以上**的中長黑K。[可能規則] Day1漲幅 ≥3%（紅K），Day2跌幅 ≥3%（黑K）。
- **實體/影線要求**：2根都須為**中長實體**，可以有小的上下影線（未給影線長度上限比例，只排除大影線的變盤線型態）。
- **位置要求**：須出現在上漲高檔（定義見前）。
- **重點說明**（p.165，共6點）：
  1. 左紅右黑皆需中長實體漲跌幅≥3%（如上）。
  2. 屬於2根K線構成的上漲高點/頭部反轉向下型態。
  3. 2根K線須為中長實體，可有小上下影線。
  4. 在上漲高檔位置，視為一組轉折向下訊號。
  5. **風險提示**：若出現在多頭剛起攻階段，要提高警覺，可能是主力「假攻、真下跌」的騙線，非型態失效而是位置判斷需謹慎。
  6. 在空頭走勢中，此型態代表反彈結束訊號；若出現在盤整區，代表主力尚無上攻企圖。
- **3種情境**（圖表3-2-8）：多頭上漲高檔止漲、盤整的假突破、空頭反彈結束。
- **書中範例**（圖表3-2-9〜3-2-14，共6檔）：曄輝(3061) 2013/06；代號3011個股 2021/07（跌幅約4.75%，紅K+黑K後接「空頭確認」）；另2檔個股2021/07附近（分別標「多頭確認」與「空頭確認」）；正文(4906) 2017/07（紅K+黑K後標「真下跌」與「空頭確認」）；TPK-KY(3673) 2017/07（「高檔單一變盤K」先出現，隨後「上漲紅K+黑K」再「反彈紅K+黑K」，最終「空頭確認」）；一檔個股2020-2021（下跌趨勢中兩度出現「反彈紅K+黑K」，皆為反彈結束訊號，呼應重點說明第6點）。

### 型態2（原書小標「二」）：高檔「長黑遭遇」（一日封口）（p.169-172前段）

- **組成**：固定2根K線。中長紅＋次日中長黑，可有小上下影線。
- **判斷條件**（p.169 重點說明，共7點）：
  1. 屬2根K線構成的上漲高點/頭部反轉向下組合。
  2. 2根K線皆為中長紅／中長黑實體，可有小上下影線。
  3. [可能規則] 上漲到明顯高檔出現中長紅K；次日**開高**（跳空或跳高開盤）later走低，收盤**接近**前一日紅K的收盤價（即黑K收盤價 ≈ Day1收盤價，未深入紅K實體太多）→ 判定為「遭遇」。
  4. 也可視為「一日缺口封閉」：當天向上跳空開高，但收盤又回到昨日紅K收盤價，等於把開盤缺口封閉，屬弱勢表現。
  5. **力道最弱**：遭遇組合是本節6組反轉訊號中最弱的一種，只能視為止漲，多頭仍可能續漲，建議再觀察1天確認是否真反轉。
  6. [可能規則] 多頭上漲中出現遭遇組合 + 爆大量 → 轉折機率大增，做多不可追多。
  7. 空頭反彈出現遭遇組合 → 視為反彈結束訊號，可把握做空機會。
- **書中範例**（圖表3-2-16〜3-2-18，共3檔）：崑鼎(1707) 2013/06，圖上先標「長黑遭遇」，後接「多頭高檔變盤線」「紅K+黑K」「反彈遇壓變盤線」，最終「空頭確認」；代號2637個股 2023/01，同樣「長黑遭遇」接「紅K+黑K」後續走勢；第3檔個股2017/07，「長黑遭遇」後續出現「紅K+黑K」交替，最終「空頭確認」。

### 型態3（原書小標「三」）：高檔「長黑覆蓋」（烏雲罩頂）（p.171下段-174前段）

- **組成**：固定2根K線，中長紅＋次日中長黑，可有小上下影線。
- **判斷條件**（p.172 重點說明，共9點，含明確二分之一比例門檻）：
  1. 屬2根K線構成的上漲高點/頭部反轉向下組合。
  2. 2根K線皆為中長實體，可有小上下影線。
  3. [可能規則] 上漲高點出現中長紅K；次日**開高盤**、盤中**創新高價**（Day2最高價 > Day1最高價），但**收盤跌入紅K實體內部**（Day2收盤價 < Day1最高價，且落入紅K實體範圍）。
  4. 覆蓋（黑K）越長，轉折力道越強。
  5. [可能規則] 黑K收盤若跌入紅K實體，但**未達紅K實體二分之一**（即 黑K收盤價 > 紅K實體中點）→ 反轉力道較弱（「長黑弱覆蓋」），建議再觀察1天確認。
  6. [可能規則] 黑K收盤若**跌破紅K實體二分之一**（黑K收盤價 < 紅K實體中點）→ 可視為反轉訊號（「長黑強覆蓋」）。
  7. 黑K收盤越向下深入紅K實體，反轉向下可能性越高；若黑K收盤跌破紅K實體的**最低點**，型態就轉變為下方型態5「長黑吞噬」（兩型態之間有明確的强度遞進關係）。
  8. [可能規則] 覆蓋2根K線的最高點(H)、最低點(L)是重要壓力/支撐觀察點：收盤突破H → 容易一波上漲；收盤跌破L → 容易一波下跌。
  9. 覆蓋後走勢常在L～H區間橫向盤整，多為做頭訊號；盤整後再度出現轉折向下訊號時，頭部形成機率大增，可考慮做空；若覆蓋2根K線本身又爆大量，更容易反轉向下。
- **3種型態**（圖表3-2-19）：長黑弱覆蓋（未破二分之一）、長黑強覆蓋（跌破二分之一）、長黑覆蓋的支撐壓力（以H、L與二分之一線為判定依據）。
- **書中範例**（圖表3-2-20、3-2-21，共2檔）：路徑(3515) 2012/11，圖上標「長黑強覆蓋」，同時標「長黑吞噬」（即黑K已跌破紅K最低點，強度已達型態5等級）、「二分之一」判定線、「反彈遇壓變盤線」、「空頭確認」、「盤整跌破」；代號2450個股2012/11，同樣標「長黑強覆蓋」「二分之一」「空頭確認」。

### 型態4（原書小標「四」）：高檔「母子懷抱」（不懷好意）（p.174後段-176前段）

- **組成**：固定2根K線。上漲高檔出現中長紅（稱**母線**），次日出現**不創新高也不破低**的黑K線（稱**子線**）。
- **判斷條件**（p.174 重點說明，共3點）：
  1. [可能規則] Day2（子線）最高價 ≤ Day1（母線）最高價，且 Day2最低價 ≥ Day1最低價，即子線完全被母線的高低點範圍包覆（母子懷抱）。子線可為任意顏色/大小的K棒，只要求「不過高不破低」。
  2. 上漲到明顯高檔出現母子懷抱組合，代表上漲走勢突然變不確定；以2天而言，多空力量突然拉鋸、多頭力道減弱，方向可能轉折向下，是一組止漲K線訊號，須注意次日是否轉折向下。
  3. [可能規則] 若子線本身是變盤線（十字線、紡錘線等），反轉力道**大於**一般母子懷抱，屬強力反轉訊號（型態上近似「十字孕線」的加強版）。
- **2種型態**（圖表3-2-22）：一般型態（母紅子黑）、子線為十字線的加強型態。
- **書中範例**（圖表3-2-23〜3-2-25，共3檔）：一檔個股(1005) 2013/06，標「母子懷抱」；一檔個股2021/07（跌幅約5.8%），依序標「高檔單一變盤K」「母子懷抱」「長黑貫穿」「長黑吞噬」「空頭確認」，顯示同一走勢中多個型態疊加出現；代號2637個股2021/01-05，依序標「長黑吞噬」「高檔單一變盤K」「高檔單一變盤K」「母子懷抱」「空頭確認」。

### 型態5（原書小標「五」）：高檔「長黑吞噬」（主力出貨）（p.175後段-179前段）

- **組成**：固定2根K線。中長紅（Day1）＋次日長黑（Day2）。
- **判斷條件**（p.176-177 重點說明，共6點）：
  1. [可能規則] 黑K（Day2）實體必須**完全吞噬**前一日紅K的實體：即 Day2開盤價 ≥ Day1收盤價 且 Day2收盤價 ≤ Day1開盤價（黑K實體範圍 ⊇ 紅K實體範圍）。
  2. [可能規則] 型態意義：當日**開高走低**（開盤創高或至少不低開），收盤**跌破前一日最低點**（Day2收盤價 < Day1最低價），賣壓出籠、多空主控權易位，是本節6型態中**最強**的反轉訊號之一（與型態6並列最強）。
  3. 吞噬當日或前一日出現大量 → 反轉訊號越強，轉折向下機率越高。
  4. [可能規則] 若短線連續上漲或急漲、獲利已達**15%以上**，此時出現長黑吞噬 → 單日反轉機率很高，建議持有多單出場或至少減碼一半。
  5. [可能規則] 若1根黑K一次吞噬並跌破前面**2〜3根K線**的低點 → 反轉力道更強，書中稱為「3線反黑」。
  6. 高檔爆量長黑吞噬後下跌，日後若再上漲碰到當初吞噬K線的價位，會形成重大壓力（多單逃命波，不可追多），容易形成「頭頭低」頭部型態。
- **3種型態**（圖表3-2-26）：長黑吞噬主力出貨（基本型）、長黑吞噬3線反黑（跌破前2-3根低點）、長黑吞噬後出現「頭頭低」（頭部型態延伸）。
- **書中範例**（圖表3-2-27〜3-2-31，共5檔）：一檔個股(1410) 2013/01，標「長黑吞噬」「空頭確認」「母子懷抱」；一檔個股2021/07（跌幅約9.13%），標「長黑吞噬」「高檔單一變盤K」；代號2888個股2021/07，依序標「長黑遭遇」「母子懷抱」「長黑吞噬」「空頭確認」；一檔個股2021/07，標「長黑覆蓋」「長黑吞噬」「空頭確認」；一檔個股2012/12，標「長黑覆蓋」「長黑吞噬」「空頭確認」。多檔範例顯示長黑吞噬常是長黑覆蓋或長黑遭遇進一步惡化後的結果，呼應型態3重點說明第7點的強度遞進關係。

### 型態6（原書小標「六」）：高檔「長黑貫穿」（一路向下）（p.179後段-182）

- **組成**：固定2根K線。中長紅（Day1）＋次日長黑（Day2）。
- **判斷條件**（p.180 重點說明，共5點）：
  1. [可能規則] 黑K（Day2）**開盤即下跌**（非開高，即Day2開盤價 ≤ Day1收盤價附近或直接低走），收盤**跌破前一日中長紅K實體的最低點**（Day2收盤價 < Day1最低價）。與型態5「長黑吞噬」的關鍵差異：吞噬是「開高走低」（Day2開盤≥Day1收盤），貫穿是「開盤即下跌」（不需要開高，直接下跌破底）；但兩者共同點都是**收盤跌破前一日最低點**，同屬全書描述的「反轉最強」訊號。
  2. 貫穿當日或前一日出現大量 → 反轉訊號越強，轉折向下機率越高。
  3. [可能規則] 短線連續上漲或急漲、獲利達**15%以上**時出現長黑貫穿 → 單日反轉機率很高，建議持有多單**立刻**出場（措辭比型態5「出場或減碼一半」更急迫）。
  4. [可能規則] 1根黑K一次貫穿並跌破前面**2〜3根K線**低點（3線反黑）→ 反轉力道更強。
  5. 高檔爆量長黑貫穿後下跌，日後再上漲到當初貫穿K線價位會形成重大壓力，屬多單逃命波不可追多，容易形成「頭頭低」頭部型態。
- **2種型態**（圖表3-2-32）：破底貫穿一路向下、破底貫穿後續「頭頭低」。
- **書中範例**（圖表3-2-33〜3-2-36，共4檔）：一檔個股(1809) 2013/06，標「高檔單一變盤K」，連續出現2次「長黑貫穿」；代號1305/8046個股2013/06，連續2次「長黑貫穿」；代號5434個股2021/07，依序標「長黑遭遇」「長黑吞噬」「長黑貫穿」「紅K+黑K」；代號355個股2021/07，標「長黑貫穿」「長黑吞噬」「高檔單一變盤K」「母子懷抱」「長黑覆蓋」，顯示同一走勢中多型態反覆確認頭部。

## 三、高檔夜星組合K線的轉折變化（p.183-186）

- **組成**：固定（基本款）**3根K線**：左邊中長紅K ＋ 中間1根（或多根）「變盤線」 ＋ 右邊中長黑K。書名為「夜星」，[可能規則] 右邊黑K完成即代表轉折確認（型態成立的觸發點是第3根黑K收盤）。
- **6種組合**（圖表3-2-37）：標準型態（含各種變化）、孤島夜星、母子變盤、雙星變盤、雙鴉變盤、群星變盤——差異主要在中間變盤線的根數與型態細節（例如「群星變盤」代表中間變盤線不只1根）。
- **判斷條件**（p.183-184 重點說明，共4點）：
  1. [可能規則] 型態組成＝上漲高檔左邊中長紅K ＋ 中間夾雜1根以上「變盤線」（十字線/紡錘線/墓碑線/長T線等止漲型K棒）＋ 右邊中長黑K，稱為「夜星」，是高檔轉折向下**確認**的K線組合（暗示：光有紅K+變盤線只是「疑似」，要等右邊黑K出現才算確認）。
  2. [可能規則] 夜星組合同時**爆大量** → 轉折向下機率大大提高，且下跌力道強。
  3. [可能規則] 中間夾雜的變盤線天數越多（對應「群星變盤」等多天變化型態）→ 中長黑K下跌時空方控盤時間也越長（型態強度與中段天數正相關）。
  4. [可能規則] **型態失效條件**：若下跌的中長黑K的高點，被後續上漲的紅K線突破，轉折向下的結構就被破壞，空方力道轉為由多方主控——此情形常發生在「多頭回檔」情境下的夜星（也就是說夜星在多頭回檔中的可靠度低於真正做頭時的夜星，需留意黑K高點是否被突破作為停損/型態失效判準）。
- **書中範例**（圖表3-2-38〜3-2-42，共5檔）：
  - 代號2637個股 2023/01：先「高檔單一變盤K」，接「長黑吞噬」，圖上標2處「夜星轉折」。
  - 大立光(3008) 2013/07：連續標2處「夜星轉折」，中間夾「長黑覆蓋」，最終「空頭確認」。
  - 華新(1605) 2023/01：連續2處「夜星轉折」。
  - 譁裕(5468) 2023/01：先「高檔單一變盤K」，接2處「夜星轉折」。
  - 敏隆(3563) 2023/01：1處「夜星轉折」後明顯回落。
