## 一、K線多空操作獲利戰法（p.243-244）

本節把整個K線判讀邏輯簡化成一套**可直接程式化的機械式交易規則**，是全書K線篇中最接近「規則庫」雛形的段落。

### 1. 多頭K線交易規則（p.243）[可能規則]

以「每日收盤前」為決策時點，用「前一日高/低點」作為唯一觸發基準：

1. **進場**：收盤前確認股價，**突破前一日高點**時買進
2. **續抱**：每天收盤前檢視股價，**沒有跌破前一日低點**時續抱（即繼續持有多單）
3. **出場**：收盤前確認股價，**跌破前一日低點**時出場
4. **停損**：以進場當日K線股價的**最低點**為停損價，且**不能超過7%**

> 規則設計重點：這是一套「逐日滾動比較前一根K線高低點」的順勢加碼/出場機制，不依賴特定K線型態辨識，而是單純的價格突破邏輯（類似海龜交易法的簡化版），停損同時設有「K棒最低點」與「7%上限」雙重界線，兩者取更嚴格（更早觸發）者。

- **書中範例**：圖表3-6-1「多頭K線交易範例」，個股代號3553（力積），2013/07-09走勢圖，圖上標註連續10個交易訊號點，交替出現「①多頭確認→②跌破昨低賣出→③突破昨高買進→④跌破昨低賣出→⑤突破昨高買進→⑥跌破昨低賣出→⑦突破昨高買進→⑧跌破昨低賣出→⑨突破昨高買進→⑩跌破昨低賣出」，demonstrated即使在同一段多頭走勢中，機械規則本身也會產生數次來回進出（whipsaw），提醒此規則單獨使用時交易頻率高、需搭配趨勢過濾條件。

### 2. 空頭K線交易規則（p.244）[可能規則]

與多頭規則完全鏡射對稱：

1. **進場**：收盤前確認股價，**跌破前一日最低點**時放空
2. **停損**：以進場當日K線股價的**最高點**為停損價，且**不要超過7%**
3. **續抱**：每天收盤前檢視股價，**沒有突破前一日的最高點**時續抱（繼續持有空單）
4. **出場（回補）**：收盤前確認股價，**突破前一日最高點**時出場（回補）

- **書中範例**：圖表3-6-2「空頭K線交易範例」，個股代號3049（空運嘉），2015/06/18附近走勢圖，標註8個訊號點：「①空頭確認→②突破昨高回補→③跌破昨低做空→④突破昨高回補→⑤跌破昨低做空→⑥突破昨高回補→⑦跌破昨低做空→⑧突破昨高回補」，同樣示範一段空頭趨勢中規則會產生多次進出。

**[可能規則] 共通結構**：多／空規則都是「以前一日高低點為唯一比較基準的逐日狀態機」，可直接抽象成同一套函式，輸入方向參數（多/空）與7%停損上限即可通用；兩規則互為鏡像（多頭比高低點突破/跌破買賣，空頭比高低點跌破/突破放空回補）。

## 二、多頭走勢中的K線練功實戰（p.245-246上半）

> 讀取說明：這兩張圖的型態標籤是分散疊加在K線圖不同位置上的文字框，屬於空間標註而非線性敘述，PDF文字擷取無法完全還原標註的精確先後順序，以下改採「列出圖中出現的型態標籤＋整體示範重點」方式記錄，避免虛構順序。

- **圖表3-6-3｜多頭走勢練功實戰①**：個股代號3441，圖上出現的標籤包含：長黑覆蓋、一星二陰、高檔連3紅、長黑吞噬、壓力線、多頭確認、多頭起漲、黑K回檔、母子懷抱。示範重點：在同一段多頭走勢的**起漲階段**（多頭確認、多頭起漲）與**回檔階段**（黑K回檔、母子懷抱）都可能穿插出現看似偏空的單根/組合K線（長黑覆蓋、一星二陰、長黑吞噬），但只要沒有跌破多頭確認的關鍵支撐（壓力線/前低），仍屬健康回檔而非趨勢反轉。

- **圖表3-6-4｜多頭走勢練功實戰②**：個股代號3624，2018/03-2020/07走勢圖，圖上標籤包含：壓力線、突破前高、防跌線、黑K覆蓋、十字線、反彈、多頭起漲、倒槌、丁字線、多頭確認、空頭確認、開始盤整。示範重點：走勢中同時出現空頭確認又轉多頭確認的**趨勢反覆**過程，搭配倒槌、丁字線、十字線等變盤訊號K線標示可能的轉折點，最終在「開始盤整」處收尾，提示此類混合型態圖的重點在於練習**分辨變盤K線出現的位置（趨勢初期 vs 高檔）**決定其意涵不同。

- **圖表3-6-5｜多頭走勢練功實戰③**：個股代號6443，2020/06-2023/01走勢圖，圖上標籤包含：椿子、紅K+長黑、倒槌、多頭確認、多頭起漲、盤整突破、黑K回檔、變盤K、高檔變盤K、開始盤整、空頭確認。示範重點：完整一輪「多頭確認→多頭起漲→（黑K回檔數次）→盤整突破→再次多頭起漲→高檔變盤K→空頭確認」的循環，是本節對「多頭走勢完整生命週期」最完整的示範圖，可作為之後設計「趨勢狀態機」規則時的參考藍本。

## 四、K線活用練功（p.247-253）

本節是本章最具體、文字定義最完整、最適合直接轉成規則的部分：把「K線」與「軌道線（趨勢線的平行延伸）」結合，定義兩個對稱的突破/跌破訊號。

### 1. 突破上升軌道線壓力的大量長紅線（p.247-250）

**定義（p.247）**：多頭上漲時，連接**低點**畫出的趨勢線稱為「上升切線」（支撐參考線）；在切線之上，於**高點**畫一條與上升切線平行的線，稱為「**上升軌道線**」，是一條壓力線。當股價**突破上升軌道壓力線**，視為走勢轉強訊號，往往會出現**連續上漲的一波攻勢**。[可能規則：收盤價突破上升軌道線（上升趨勢線的等距平行上緣） → 標記為波段轉強訊號]

- 圖表3-6-8｜示意圖，標示元素：上升趨勢線（連低點）、上升軌道線（連高點，與趨勢線平行）、大量突破、突破長紅

**重點說明（p.248，共3點）**：

1. **突破當天，出現跳空上漲、帶大量的長紅K實體棒，力道最強。** [可能規則：突破訊號強度可依「是否跳空」＋「是否帶大量」＋「實體棒長度」加權評分]
2. **沿軌道區間上漲到趨勢高檔（波段第3波）才出現的突破長紅K，容易發生急漲，屬於末升段現象**，此時要注意隨後出現的變盤訊號，準備**停利出場**（操作建議：可另外畫一條「急切線」，若股價**跌破急切線**即停利）。[可能規則：波段第N波（N≥3）出現的軌道突破 → 標記為末升段警示，需搭配急切線停損/停利]
3. **上升軌道線本身只是壓力，沒有實際突破確認前，不可提早進場做多**；相反地，若股價上漲接近軌道線但未突破就回落，**可短線逆勢做空**。[可能規則：接近未破軌道壓力線後反轉向下 → 短線放空訊號]

- **書中範例（p.249-250，共4檔個股，皆為2019年中走勢圖，皆標示「上升趨勢線／上升軌道線／突破長紅」核心三要素）**：
  1. 圖表3-6-9，代號6153，2019/06/26：另標「高檔變盤K」「長黑吞噬」「空頭確認」——示範突破後仍在高檔遇阻反轉向空，呼應重點說明2的「末升段停利」提醒
  2. 圖表3-6-10，代號6176，2019/06/26：另標「長黑遭遇」「高檔變盤K」「空頭確認」——同樣示範突破後高檔反轉
  3. 圖表3-6-11，代號2455，2018/09-2019：另標「高檔變盤K」——突破後走勢持續，變盤K示警
  4. 圖表3-6-12，代號6486，2019/06/26：另標「多頭起漲」——突破後直接延續多頭起漲，未出現高檔警示，作為與前3例對照的「順利延續」案例

### 2. 跌破下降軌道線支撐的大量長黑K線（p.251-253）

與上一段完全鏡射對稱：

**定義（p.251）**：空頭下跌時，連接**高點**畫出的趨勢線稱為「下降切線」（壓力參考線）；在切線之下，於**低點**畫一條與下降切線平行的線，稱為「**下降軌道線**」，是一條支撐線。當股價**跌破下降軌道支撐線**，視為走勢轉弱訊號，往往會出現**連續下跌的一波跌勢**。[可能規則：收盤價跌破下降軌道線（下降趨勢線的等距平行下緣） → 標記為波段轉弱訊號]

- 圖表3-6-13｜示意圖，標示元素：下降趨勢線（連高點）、下降軌道線（連低點，與趨勢線平行）、大量跌破、跌破長黑

**重點說明（p.252，共3點，與上升軌道段落逐條鏡射對稱）**：

1. **跌破當天，出現跳空下跌、帶大量的長黑K實體棒，力道最強。** [可能規則]
2. **沿軌道區間下跌到趨勢低檔（波段第3波）才出現的跌破長黑K，容易發生急跌，屬於末跌段現象**，注意隨後出現的變盤訊號，準備**空單回補停利出場**（可畫「下降急切線」參考，股價**突破急切線**即停利回補）。[可能規則：波段第N波（N≥3）出現的軌道跌破 → 標記為末跌段警示，需搭配急切線回補]
3. **下降軌道線本身只是支撐，沒有實際跌破確認前，不可提早進場做空**；相反地，若股價下跌接近軌道線但未跌破就反彈，**可短線逆勢做多**。[可能規則：接近未破軌道支撐線後反轉向上 → 短線做多訊號]

- **書中範例（p.251-253，共4檔個股，皆為2019/08附近走勢圖，皆標示「下降趨勢線／下降軌道線／跌破長黑」核心三要素）**：
  1. 圖表3-6-14，代號3481，2019/08/20：僅標核心三要素，屬單純示範案例
  2. 圖表3-6-15，代號6217，2019/08/20：另標「空頭確認」——跌破前已先有空頭確認訊號
  3. 圖表3-6-16，代號3535，2019/08/23：另標「長黑吞噬」「空頭確認」——跌破前疊加組合型態訊號
  4. 圖表3-6-17，代號2612，2019/08/20：另標「空頭確認」，且**後續出現「多頭確認」「長紅吞噬」**——是4個範例中唯一示範「跌破下降軌道後，走勢後續反轉向上、重新出現多頭確認」的案例，提醒軌道跌破訊號並非保證持續下跌，仍需搭配後續K線確認。

**[可能規則] 上升/下降軌道段落共通歸納**：
1. 軌道線＝趨勢線的等距平行線，一條在高點延伸（壓力＝上升軌道線）、一條在低點延伸（支撐＝下降軌道線），兩者互為鏡像，可用同一函式處理（輸入方向參數）。
2. 突破/跌破訊號強度的三個加權因子完全對稱：跳空、大量、長實體棒。
3. 「波段第3波（或更後段）才發生的軌道突破/跌破」被明文定義為末升/末跌段警示，需要額外的「急切線」工具來管理停利/停損，這是本章少數要求**額外畫線輔助工具**的規則，程式化時需要先有波段分段（第1波、第2波、第3波…）的邏輯作為前置條件。
4. 「未突破/未跌破前不可提早進場，反而可逆勢短做」是本節唯一一條**明確允許逆勢交易**的規則，與其餘章節「先確認再順勢操作」的主軸略有不同，值得在規則庫中特別標記為「逆勢短線」子規則、與主要的「順勢突破」規則分開管理。
