## 一、晉升千萬富翁的目標設定（下冊 p.363-364）

- 股市／期貨市場每年都有機會賺1〜2倍。以10萬元本金、每年獲利100%（賺1倍）計算複利成長：
  1. 第1年：10萬元 → 20萬元
  2. 第2年：20萬元 → 40萬元
  3. 第3年：40萬元 → 80萬元
  4. 第4年：80萬元 → 160萬元
  5. 第5年：160萬元 → 320萬元
  6. 第6年：320萬元 → 640萬元
  7. 第7年：640萬元 → 1,280萬元
  8. 第8年：1,280萬元 → 2,560萬元
  （下冊 p.363）
- 依此複利觀念，**第7年即可達成千萬目標**，關鍵在於「1年是否能賺到100%獲利」。（p.363）
- `[可能規則]` 目標可逐層拆解為可執行的短週期報酬率門檻：年獲利100% → 每月獲利約8.33%（12個月）→ 每週獲利約2.1%（每月4週）→ **每2週獲利4.2%** 即可視為完成「年獲利1倍」目標的基本操作單位。作者將此列為技術分析操作應努力達成的具體目標。（p.363）
- 設定目標的意義：有目標才有方向可全力執行，並能定期檢討改進；具體的獲利目標也能幫助克制「不願認賠」「貪婪不捨賣」等人性弱點，適時落實停損與停利。（p.363-364，段落跨頁）
- 目標管理下的操作原則：選股須符合設定條件才進場、高勝率機會出現才執行進場、進場後依自訂交易規則操作、不貪不懼、停損時立刻出場，逐步累積達到設定目標。若目標本身無法達成，**不輕易更動目標**，而是檢討並改變操作方法。（p.364）
- 核心觀點：目標管理除了精通技術分析外，更重要的是訓練自己操盤的「定法」與「定心」。（p.364）

## 二、達成目標4大重要關鍵（下冊 p.364-365，逐條列出）

1. **停利比停損重要（目標達成）**：停損須嚴格執行以免虧損擴大；但要達成獲利目標，必須確實執行停利，否則獲利可能回吐，白白浪費一次獲利機會及操作時間。
   `[可能規則]` 觸及停利條件即應出場了結，不可因期待更高點而延遲執行。（p.364）
2. **短線比長線重要（獲利效率）**：短線操作能快速獲利、目標易於掌握；除非出現大波段機會，否則應把握每一個能快速獲利的短線操作機會。買進後若走勢轉為飆股，改用飆股戰法操作；若為強勢波段股，可長線短做；無論如何都以快速獲利為第一目標。
   `[可能規則]` 個股符合飆股走勢特徵時，優先切換至飆股戰法以追求獲利速度。（p.364）
3. **強勢比漲勢重要（時間成本）**：目標管理設有時間限制，須選擇強勢股操作；緩慢牛步上漲、1週或半個月漲幅不到5%的多頭股票，應換成強勢股操作。
   `[可能規則]` 持股於觀察週期（1週或半個月）內漲幅低於5%，且屬多頭排列的牛步股，觸發換股篩選條件。（p.364）
4. **紀律操作比選股重要**：即使選到好股票，要真正實現獲利，仍必須嚴格執行整套操作紀律，包括：選股條件的紀律、進場位置的紀律、停損的紀律、持股續抱的紀律、停利的紀律；每一步都要落實紀律，才能完成「每一次大賺小賠」的交易。
   `[可能規則]` 交易全流程（選股條件／進場位置／停損／續抱／停利）皆須各自訂定明確規則並嚴格執行，任一環節破例即視為紀律失守。（p.364-365，段落跨頁）

## 三、3階段設定達成長期贏家目標（下冊 p.365-366，逐條列出）

### 階段一：初階獲利目標設定（p.365）
目的：累積操作經驗、驗證個人操作方法。
1. `[可能規則]` 固定資金50萬元，練習紀律，不逆勢操作，不用槓桿操作。
2. `[可能規則]` 以1〜2檔股票作為操作標的。
3. 練習時間6個月。
4. `[可能規則]` 6個月獲利目標12%，每個月獲利2%（全年獲利24%）。
5. 每1個月獲利金額約1萬元。
6. 6個月初階練習完成後，需定出個人交易方法，包含以下項目：
   1. **選股條件**：基本面（公司類別、股本、財報、本益比、發展性等）、技術面（日／週／月走勢分析及指標分析）、籌碼面（融資融券、三大法人買賣超、主力進出成交量變化）。
   2. **進場條件**：多頭符合做多的基本進場條件，空頭符合做空的基本進場條件。
   3. **交易策略**：練習操作短線交易。

### 階段二：進階獲利目標設定（p.365-366）
目的：熟練個人操作方法、提高獲利目標、累積各種應變經驗。
1. `[可能規則]` 固定資金80萬元，練習紀律，不逆勢操作，不用槓桿操作。（p.365）
2. `[可能規則]` 以1〜3檔股票作為操作標的。（p.366）
3. 練習時間6個月。
4. `[可能規則]` 6個月獲利目標30%，每個月獲利5%（全年獲利60%）。
5. 每1個月獲利金額約4萬元。
6. 6個月練習階段完成後，需修正個人交易方法，包含以下項目：
   1. 熟練選股策略，以主流、強勢股操作。
   2. 熟練多空雙向操作。
   3. 熟練各種交易策略，分配資金，建立波段操作能力。
   4. 完成情緒穩定訓練，能依紀律進出。

### 階段三：終極獲利1倍目標設定（p.366）
目的：建立高勝率的操作模式，長波段搭配短波段操作。
1. `[可能規則]` 固定資金100萬元，開立融資戶，加大槓桿獲利操作。
2. `[可能規則]` 以2〜5檔股票作為操作標的。
3. `[可能規則]` 6個月獲利目標50%，每個月獲利8.33%（全年獲利100%）。
4. 每1個月要獲淨利約8.3萬元。
5. 6個月階段練習完成後，需固定個人交易方法，包含以下項目：
   1. 以主流、強勢股操作。
   2. 達到「無我境界」，用紀律機械式操作。
   3. 分配資金，多元化金融商品操作（期指、選擇權等），創造財富。
   4. 研究基本面，可用波段操作，提高大波段獲利能力。
