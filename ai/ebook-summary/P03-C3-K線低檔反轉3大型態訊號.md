## 一、低檔變盤線的轉折變化 (p.189-191)

### 1. 變盤線的型態清單（圖表3-3-1，p.189）
書中列出8種低檔變盤K線圖形：墓碑線、長T線（長下影線）、十字線、紡錘線、反鎚線、吊人線、漲停線、長紅線。
原文敘述「低檔單一變盤線或連續數天變盤線的變盤訊號有6種變化觀察」，但圖示實際並列8種K線圖形——**用詞「6種」與圖示數量「8種」不一致，程式化時建議直接以圖表3-3-1列出的8種具體K線形狀作為判斷清單，不糾結原文「6種」字面數字**。

### 2. 判斷條件 (p.189-190)
1. [可能規則] 出現條件：股價下跌到低檔，出現上述任一變盤K線 → 視為止跌訊號，須持續觀察是否反轉。
   - 空頭趨勢的低檔出現 → 視為轉折向上反彈的變盤線
   - 多頭趨勢的回檔低檔出現 → 視為準備做多的訊號
2. [可能規則] 加分條件：低檔爆出大量或窒息量，且出現價量背離現象、KD指標黃金交叉 → 更容易反轉上漲。
3. [可能規則] 位置條件：下跌走勢遇到「支撐位置」時出現變盤線 → 反轉走升機率高，須密切注意次日股價是否上漲。
4. 變盤線本身**不保證**次日就上漲，但凡轉折上漲的位置幾乎都會先出現變盤線——即變盤線是必要但非充分條件，次日走勢才是關鍵確認點。
5. [可能規則] 低檔變盤訊號**不一定**伴隨大量出現，須以「次日股價是否上漲」作為確認條件，量能不是必要確認變數。

### 3. 實戰範例 (p.190-191)
- 圖表3-3-2「低檔變盤K線範例①」（約2022/02區間）：空頭下跌低檔連續出現2次「空頭低檔變盤線」標註，其後出現「回檔遇撐變盤線」，隨後轉折上漲。
- 圖表3-3-3「低檔變盤K線範例②」（約2023/01區間）：標註「回檔遇撐變盤線」出現2次＋「空頭低檔變盤線」1次，屬多頭回檔遇撐再上漲案例。
- 圖表3-3-4「低檔變盤K線範例③」（約2021/12-2023/03區間）：連續3次「空頭低檔變盤線」標註，顯示同一走勢中多次出現變盤線但仍持續破底——說明變盤線不必然一次反轉成功，呼應第4點「須看次日走勢確認」。

## 二、下跌黑K＋紅K線的轉折變化 (p.192-204)

總說（p.192）：下跌低檔中，下跌黑K次日出現紅K（黑K＋紅K兩根K線組合變盤線），兩者「相互位置」有6種情況，代表意義不同（圖表3-3-5，編號①～⑥）：
① 基本型態：長黑＋長紅　② 長紅遭遇（一日封口）　③ 長紅覆蓋（旭日東升）　④ 母子懷抱（光明在望）　⑤ 長紅吞噬（主力進貨）　⑥ 破高貫穿（長紅貫穿，一路向上）

以上6種依「反轉力道」大致由弱到強排列（原文明確指出②長紅遭遇是「6組反轉訊號中最弱」），⑤長紅吞噬被明確稱為「雙K線向上轉折訊號中最強的組合」。

### (1) 低檔2根K線轉折向上的基本型態——長黑＋長紅 (p.192-193)
[可能規則] 組成與條件（圖表3-3-6）：
- 2根K線構成
- 左邊第1根：下跌幅度達 **3%以上** 的中長黑K線
- 右邊第2根：上漲幅度達 **3%以上** 的中長紅K線
- 兩根都須為中長實體，**可以有小的上下影線**（非要求光頭光腳）
- 出現位置：下跌低檔或底部
- 判讀差異（p.193 重點說明5-6）：
  - 若出現在「**空頭起跌**」的位置 → 要提高警覺，小心是主力假跌真拉的騙線（誘空），投資人不可不察
  - 若出現在「**多頭走勢**」中 → 是回檔結束訊號，若出現在盤整區，表示主力還沒有下殺的企圖

子分類（圖表3-3-7 低檔紅K、黑K的3種型態）：空頭下跌低檔止跌 ／ 盤整的假跌破 ／ 多頭回檔結束

實戰範例：
- 圖表3-3-8「低檔紅K黑K組合轉折範例①」（約2006/05-2013年區間）：標「黑K＋紅K」組合出現在下跌末端，後續「多頭確認」。
- 圖表3-3-9「範例②」（約2017/07區間）：K線橫盤突破後出現「黑K＋紅K」，標「多頭確認」，後續續漲。

### (2) 低檔「長紅遭遇」（一日封口） (p.194-196)
[可能規則] 組成與條件（圖表3-3-10）：
- 2根K線構成的下跌低點或底部反轉向上組合
- 2根K線皆為**中長黑、中長紅**實體，可以有小的上下影線
- 核心條件：下跌走勢到明顯低檔，出現中長黑K線；第2天**開低走高**，**收盤價收在接近前一天中長黑K線收盤價**附近（即紅K收盤價貼近黑K收盤價，形成「封口」）
- 強度評級：**6組反轉訊號中最弱**的一種，只能視為止跌，可能還會續跌，**最好再觀察1天確認是否反轉**

判讀規則（p.195 重點說明5-7）：
- 空頭下跌中出現遭遇型態：若**無量**、次日向上 → 弱勢反彈；若**急跌爆量**、次日向上 → 強勢反彈（都屬逆勢反彈，搶反彈要小心）
- [可能規則] 空頭下跌中出現遭遇組合，若**爆大量** → 轉折機率大增，做空要小心，不可追空
- [可能規則] 多頭回檔中出現遭遇組合 → 是回檔結束的訊號，若次日**出量上漲** → 是多頭續漲訊號，要把握做多機會

實戰範例：
- 圖表3-3-11「長紅遭遇轉折範例①」（約2011/02-2013區間）：標「母子懷抱」「長紅遭遇」。
- 圖表3-3-12「範例②」（約2012/08-2013/03區間）：標「長黑貫穿」「多頭確認」「長紅遭遇」「反彈確認」。
- 圖表3-3-13「範例③」（約2013/04-2017/07區間）：標「長黑吞噬」「反彈確認」「長紅遭遇」。

### (3) 低檔「長紅覆蓋」（旭日東升） (p.197-198)
[可能規則] 核心條件（圖表3-3-14／重點說明1-6）：
- 下跌低檔，出現下跌中長黑K線；次日**開低盤、盤中跌破新低價**，但**收盤突破平盤**——即紅K實體進入黑K實體範圍內
- **覆蓋比例門檻（50%分界線）**：
  - 紅K收盤價若進入黑K實體，但**未突破黑K實體的二分之一** → 反轉訊號力道較弱（「長紅弱覆蓋」），須再觀察1天確認是否反轉
  - 紅K收盤價若**突破黑K實體的二分之一以上**（稱為「強覆蓋」）→ 反轉向上可能性高
- [可能規則] 兩根K線的最高點H、最低點L是重要壓力與支撐：觀察點收盤**突破H** → 容易出現一波上漲；**跌破L** → 容易出現一波下跌
- [可能規則] 覆蓋的2根K線若同時**爆大量** → 更加容易反轉向上

子型態（圖表3-3-14）：長紅弱覆蓋／長紅強覆蓋／長紅覆蓋的支撐壓力（標H、L、二分之一線）

實戰範例：
- 圖表3-3-15「長紅覆蓋範例①」（約2002/08-2013/03區間）：標「反彈確認」「長紅覆蓋」「多頭確認」「長紅強覆蓋」「上漲確認」。
- 圖表3-3-16「範例②」（約2006/01-2013/03區間）：標「跳空長黑」「多頭確認」「反彈確認」「長紅覆蓋」「長黑覆蓋」。

### (4) 低檔「母子懷抱」（光明在望） (p.199-201)
[可能規則] 組成與條件（圖表3-3-17，2種型態：一般母子懷抱／母子＋十字線）：
- 2根K線構成，下跌低檔出現中長黑K線（**母**線），次日出現**不過高也不破低**的紅K線（**子**線，高低點皆被母線實體/區間包住）
- 子線可以是**十字線／變盤線**變體（母子懷抱＋十字）
- 意義：多空雙方力量突然呈現拉鋸，空頭下跌力道減弱，方向可能轉折向上 → 屬止跌訊號，須觀察次日是否轉折向上

關鍵規則（p.199-200 重點說明3-7）：
- 母線的**最高點被突破** → 反彈確認
- [可能規則] 成交量條件：**母量須大於子量**（量能遞減），若出現「價漲量縮」背離，須**次日放量上漲**才能化解背離，否則不易反轉成功，次日容易下跌
- 母子懷抱2根K線的最高點與最低點是重要觀察點：**向上突破最高點** → 多空易位，多方轉為主動掌控；**向下跌破最低點** → 空方繼續主導下跌
- 次日**開盤位置**是關鍵：開高容易向上反轉，開低容易下跌
- [可能規則] 若子線本身是「變盤線」型態 → 反轉機率**大於**一般母子懷抱型態，投資人要特別警覺，很容易發展成「低檔晨星轉折」的3根K線組合（即母子懷抱是晨星型態的雛形之一）

實戰範例：
- 圖表3-3-18「低檔母子懷抱範例①」（約2013/03區間）：連續多次「母子懷抱」「夜星轉折」「反彈確認」「高檔單一變盤K」「多頭確認」「長紅吞噬」「壓力線」「遇撐回檔」標註。
- 圖表3-3-19「範例②」（約2022/05-2023區間）：標「多頭確認」「母子懷抱」×2「反彈確認」×2「空頭確認」「長黑吞噬」「壓力線」。
- 圖表3-3-20「範例③」（約2013/06區間）：標「跳空長黑」「長紅吞噬」「母子懷抱」×2「反彈突破前高」「反彈確認」「上漲確認」「多頭確認」「長紅遭遇」。

### (5) 低檔「長紅吞噬」（主力進貨） (p.201-203)
[可能規則] 核心條件（圖表3-3-21／重點說明1-6）：
- 2根K線；下跌走勢到低檔，出現1根長紅K，**完全包覆（吞噬）前一日下跌黑K的實體**——紅K實體必須**完全**吞噬前一根黑K線的實體棒（比「長紅覆蓋」更嚴格：覆蓋只需超過二分之一即算強覆蓋，吞噬要求100%覆蓋前一根實體）
- 意義：當日**開低走高，收盤突破前一日高點**，多方買盤出籠，多空主控權產生易位。**長紅吞噬是雙K線向上轉折訊號中最強的組合**——空單要立刻回補
- [可能規則] 加強條件：吞噬當日或前一日出現**大量或窒息量** → 反轉訊號越強，轉折向上機率越高（須分辨是多頭回檔再上漲、還是空頭跌深的反彈）
- [可能規則] 擴展型態「**3線反紅**」：1根紅K一次吞噬突破前面2～3根黑K線的高點，反轉力道更強
- 低檔出現爆量長紅吞噬K線後上漲反彈一段：**日後再下跌到長紅吞噬K線區間會形成重要支撐**；若再跌破此支撐，是空單的逃命波，不可追空，容易形成「**底底高**」的底部型態

子型態（圖表3-3-21）：長紅吞噬／長紅吞噬前3根K線／長紅吞噬後出現「底底高」

實戰範例：
- 圖表3-3-22「長紅吞噬範例」（約2006/11-2013/03區間）：標「母子懷抱」「反彈確認」「多頭確認」「長紅吞噬」「盤整突破」。

### (6) 低檔「長紅貫穿」（一路向上） (p.203-205)
[可能規則] 核心條件（圖表3-3-23／重點說明1-4）：
- 2根K線；下跌走勢到低檔，出現轉折向上的「貫穿」紅K線
- 當日**開高走高，收盤突破前一日黑K的最高點**（比「長紅吞噬」更強——吞噬只要求覆蓋前一日實體，貫穿要求突破前一日整根K線含影線的最高點）
- 意義：多方當天向上貫穿，多空主控權易位
- [可能規則] 加強條件：貫穿當日或前一日出現**大量** → 反轉訊號越強，轉折向上機率越高
- [可能規則] 位置條件：短線連續下跌或急跌，**獲利（跌幅）達15%以上**時出現長紅貫穿K線 → **1日反轉機率很高**，持有空單應該立刻回補
- [可能規則] 擴展型態「3線反紅」：1根紅K一次貫穿突破前面2～3根K線的高點，反轉力道更強
- 低檔爆量長紅貫穿K線後反彈：日後再下跌到長紅貫穿K線位置會形成重大支撐，再下跌是空單逃命波，不可追空，容易形成「底底高」的底部型態

子型態（圖表3-3-23）：長紅貫穿／長紅貫穿前3根K線／長紅貫穿反彈後續「底底高」

實戰範例：
- 圖表3-3-24「長紅貫穿範例①」（約2005/09-2013/03區間）：標「長紅貫穿」「多頭確認」「高檔單一變盤K」×2。
- 圖表3-3-25「範例②」（約2011/11-2013/06區間）：標「多頭確認」「晨星轉折」「長紅貫穿」。

## 三、低檔晨星組合K線的轉折變化 (p.205-210)

總說（p.205）：下跌低檔中，「黑K＋變盤線＋紅K」的晨星組合，不同情況代表意義不同。

### 1. 標準型態與6種組合變化 (圖表3-3-26，p.206)
[可能規則] 組成：**3根K線**
1. 第1根（左）：中長**黑K線**
2. 第2根（中）：**變盤線**（可以是十字線或其他變盤K線，且中間可延伸為多根小K線）
3. 第3根（右）：中長**紅K線**

標準型態即「黑K → 變盤線 → 紅K」。書中列出的6種變化組合（依中間變盤部分的差異）：孤島晨星、母子變盤、雙星變盤、雙肩變盤、群星變盤，另有標準型態本身，共同特徵是「左黑右紅、中間夾一段轉折/變盤的K線」。

判讀規則（p.206 重點說明1-4）：
1. 下跌低檔出現「左中長黑K、右中長紅K、中間夾雜變盤線」的K線組合 → 是低檔轉折向上確認的K線組合
2. [可能規則] 加強條件：晨星組合同時**爆大量** → 轉折向上機率大大提高，且上漲力道強
3. [可能規則] 中間夾雜的變盤線**數量越多** → 之後中長紅K線上漲時，多頭控盤的時間也越長（中間變盤天數與後續多頭持續時間呈正相關）
4. [可能規則][**關鍵失效條件**] 晨星轉折組合，若上漲那根中長紅K線的**低點被後續下跌的黑K線跌破** → 轉折向上的結構就被破壞，向上多方力道轉為向下由空方主控。原文特別提醒此現象**經常發生在空頭反彈的「晨星」中**——即晨星在空頭趨勢中可能只是一次假反彈，須以「第3根紅K最低點不被後續K線跌破」作為型態有效性的判定標準，不能出現晨星就直接視為底部確認。
